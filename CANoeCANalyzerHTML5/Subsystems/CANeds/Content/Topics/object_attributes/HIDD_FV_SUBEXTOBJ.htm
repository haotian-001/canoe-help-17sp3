Sub extension object attributes

 

On this page the attributes of a sub extension object are shown. Sub extension objects that instantiate new subobjects.

There are two additional fields according to the subobject attributes:

Created per Module

Number of subobjects with this attributes that are created per module or number of bits that are created per module to build a new subobject.

Maximum Sub Index

If an array is filled up to the end, the next object may be used to create the remaining subobjects. This value defines the maximum subindex after which the next object is used. If the value is 0 or the entry is missing, the next object can not be used.
