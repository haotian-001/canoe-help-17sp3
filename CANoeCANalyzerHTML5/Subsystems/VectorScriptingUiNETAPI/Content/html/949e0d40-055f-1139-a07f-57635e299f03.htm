Vector Scripting UI API

DataEntryDialogShow Method (String, String, String, String, RangeCollectionInt32)
Opens a dialog to enter a value. A RangeCollection restricts the valid lengths of the string. The dialog can only be closed with "OK" if the entered value is valid. The provided "value" is used as default value.
