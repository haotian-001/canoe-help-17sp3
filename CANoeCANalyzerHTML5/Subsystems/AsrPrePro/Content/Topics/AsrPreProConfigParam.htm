Configuration Parameters

 

You can run the AUTOSAR Preprocessor from the command line with different parameters. The following table lists all available parameters. Parameters required for execution are marked in the Required column.

Note

You can specify the parameters either directly in the command line (CMD) or within a configuration file (Config) but not miced across both places. Parameters available on the command line (CMD) and in the Config file are mutually exclusive. This means, for example, it is not possible to set the output-file parameter via the CMD and additional use a Config file, which contains the remaining parameters. Once a configuration file is specified, all desired and available parameters must be specified in it.

verbosity-level <level>

Toggles how much output is output to stdout and stderr.

  • Level 0 (error notifications only)
  • Level 1 (additional warnings)
  • Level 2 (additional information)
  • Level 3 (additional debug outputs).

The default is level 1.

config-file <file>

Replaces the command line parameters marked in the Config column with settings stored in a configuration file. The configuration file has the YAML format. These parameters must not be specified at the same time as --config-file on the command line.
