Patches

 

You can use the patches in AUTOSAR Preprocessor to modify specific elements in the ARXML files. There are so-called Built-In patches, which are predefined. The following table contains an overview of the available patches:

Name

Identifier

Parameter

Description

Rename-Cluster-Patch

rename-cluster

current-name new-name

The cluster with the name current-name is renamed to the specified name new-name.

You can specify the patches only in the configuration file. You can specify as many patches as you like, which will be applied one after the other from top to bottom.

See the Use Case 2 page for an example of how to specify the patch list as well as a concrete patch in the configuration file in YAML syntax.
