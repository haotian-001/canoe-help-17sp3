AUTOSAR Preprocessor

 

The AUTOSAR Preprocessor is a command line tool for preprocessing AUTOSAR XML (ARXML) files, which allows you to merge individual ARXML extracts (SYS-E) into a system description (SYS-D). In addition, you can use the AUTOSAR Preprocessor with the help of filters and patches, specific elements from the ARXML files to filter out or modify. With the AUTOSAR Preprocessor you thus reduce ARXML files to elements that you need and thus accelerate subsequent work steps.

Usage

In many cases, no system descriptions are available but only individual extracts that describe an ECU with its inputs and outputs. For these extracts to be processed further in CANoe4SW Server Edition , they must be merged. When merging, OEM-specific modeling and splitting rules, in particular also non-AUTOSAR-compliant rules, must be taken into account. The larger the individual extracts, the longer the merging process takes. Therefore, it is a great advantage to perform this process only once and then use the system description generated from it for further work steps.

The merging of the individual extracts can be done with the AUTOSAR Preprocessor e.g. by database experts at the supplier and OEM. Content errors cannot be corrected automatically with the AUTOSAR Preprocessor.
