Error Codes

 

Number

Short Description

-3

No (matching) PDU definitions were found.

-2

The reservation of slots failed, e.g. because the cycle repetition value is not a divider of the cycle length (64).

-1

No such connection, i.e. creation of a connection failed or the given handle is unknown.

0

OK, no error.

1

Some mandatory settings are missing, e.g. the addresses to use.

2

The value given contradicts another setting made earlier, e.g. payload length is larger than maximum.

3

The request to send some data was rejected since another transmission is in progress.

4

A reception was active when a first frame or single frame was received.

5

A negative ACK was received for acknowledged connections.

6

The bus did not confirm transmission of a frame in time.

7

The transmitter did not receive a flow control frame in time.

8

The receiver did not receive the next consecutive frame in time.

9

A setting cannot be changed because data is transferred at the moment and an inconsistent state would result.

10

Transmission was aborted because application did not provide data in time.

11

The peer rejected the data since it is too long.

12

Peer sent an undefined Ack.

13

Peer sent a First Frame where a Single Frame would have sufficed. The First Frame is ignored, an error is indicated, and a warning is printed in the Write Window.

14

Receiver has sent too many wait frames and must therefore abort the transfer.

15

Receiver has requested the resending of an illegal frame, e.g. the Start Frame.

16

Receiver requested the transfer to be aborted.

17

The ML parameter of the LF does not match the received data or the given ML in the STF.
