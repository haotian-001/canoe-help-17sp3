Add-on DLLs

 

Simulation Setup

In order to use this node layer DLL in the CANoe Simulation Setup, the DLL must be loaded into the simulation nodes. This can be done in the configuration menu of the node in the Simulation Setup; please refer to the following pages for details:

  • Function and Modeling Libraries (Add-on DLLs)
  • Assigning of Modeling Libraries (Add-on DLLs)

Additional CAPL functions

After the DLL has been loaded into the node, its CAPL program can access the TP functionality via additional CAPL functions, all marked with the prefix FrTP_.

Please note that the CAPL Browser offers a shortcut menu (press right mouse button in an edit area) where you can select a CAPL function. All functions provided by this DLL are collected in the category FlexRay TP Functions.

Version information

In order to determine the version and location of the DLL loaded in CANoe, you can get additional information in the Write Window when you open the menu path Help|About|More Info.
