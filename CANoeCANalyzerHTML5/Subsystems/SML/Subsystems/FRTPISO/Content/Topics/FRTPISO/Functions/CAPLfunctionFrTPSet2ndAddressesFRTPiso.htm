FrTP_Set2ndAddresses

 

Note

Changing the addresses will affect future data transfers!

Function Syntax

long FrTP_Set2ndAddresses( long connHandle, dword flags, dword a2ndLocalAddress, dword a2ndRemoteAddress);

Description

Sets optionally 2nd ECU addresses that are used for incoming or outgoing connection data.

This allows e.g. in the context of the Tester to receive responses from the ECU from 2 different (remote) TP addresses. In the context of the ECU (simulation) this allows to accept/receive requests that are directed to 2 different (local) TP addresses. In normal case either the local address or the remote address should be valid according to the current application context (Tester or ECU simulation).

The flags control whether the new local or the new remote or both addresses should be accepted.

Mask for flags

Meaning

0x01

If set it enables the use of the 2nd remote address and enables the adaptation of the TP addresses in flow control frames to be sent.

0x02

If set it enables the use of the 2nd local address and enables the adaptation of the TP addresses in flow control frames to be sent.

0x04

If set additionally it disables the adaptation of the TP addresses in flow control frames.

Default: If the function is not called, then both 2nd addresses are disabled.

Parameters

connHandle

The handle of the connection.

flags

Controls which addresses should be used, and whether responding flow control frames should use the second address.

a2ndLocalAddress

The 2nd local address ([0x0000 – 0xFFFF] 16 Bit).

a2ndRemoteAddress

The 2nd remote address ([0x0000 – 0xFFFF] 16 Bit).

Return Values

see Error Codes

Example

Set additionally 2nd ECU address from CANoe/CDD parameter settings.

const int cIsTester = 0; // default: we are an ECU simulation
long localAddress;
long localAddress2;
long useAddress2;
long remoteAddress;
long remoteAddress2;
long addressType;
long gDiagConnHandle = -1;
...
localAddress = DiagGetCommParameter("FlexRay.LocalAddress");
remoteAddress = DiagGetCommParameter("FlexRay.RemoteAddress");
addressType = 2;
localAddress &= 0xFFFF;
remoteAddress &= 0xFFFF;
useAddress2 = DiagGetCommParameter("FlexRay.Has2ndECUAddress");
if (useAddress2 == 1)
{
  if (cIsTester)
  {
    remoteAddress2 =
      DiagGetCommParameter("FlexRay.RemoteAddress2");

    writeDbgLevel( 1, "%s: Setting %d byte addresses (0x%04X, 0x%04X)", gECU,
      addressType, localAddress, remoteAddress);
    FrTP_SetAddresses(gDiagConnHandle, localAddress, remoteAddress);

    writeDbgLevel( 1, "%s: Setting 2nd ECU address for reception 0x%04X", gECU,
      localAddress2);
    FrTP_Set2ndAddresses(gDiagConnHandle, 0x01, 0, remoteAddress2);
  }
  else
  {
    localAddress2 =
       DiagGetCommParameter("FlexRay.LocalAddress2");

    writeDbgLevel( 1, "%s: Setting %d byte addresses (0x%04X, 0x%04X)", gECU,
      addressType, localAddress2, remoteAddress);
    FrTP_SetAddresses(gDiagConnHandle, localAddress2, remoteAddress);

    writeDbgLevel( 1, "%s: Setting 2nd ECU address for reception 0x%04X", gECU,
      localAddress);
    FrTP_Set2ndAddresses(gDiagConnHandle, 0x02, localAddress, 0);
   }
}
else
{
  writeDbgLevel( 1, "%s: Setting %d byte addresses (0x%04X, 0x%04X)", gECU,
    addressType, localAddress, remoteAddress);
  FrTP_SetAddresses(gDiagConnHandle, localAddress, remoteAddress);
}
