FrNm_PNRequestInd

 

Note

This indication will be only called if parameter NM_PN_EIRA_CALC_ENABLED is set to 1.

Function Syntax

void FrNm_PNRequestInd(long partialNetworkId)

Description

Indication that a certain partial network is now requested. The indication will be only called if the partial network is included in the filter mask of the node.

Parameters

networkId

Specifies which partial network was requested.
