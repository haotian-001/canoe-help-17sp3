FrTP_GetStMinOrBC

 

Note

These functions are allowed to be called only during execution of the callback function FrTP_FrameRxInd!

Function Syntax

long FrTP_GetStMinOrBC();

Description

Returns the bandwidth control (StMin) parameter of a FC-CTS C-PDU.

Only valid for FC-CTS frames.

Parameters

—

Return Values

≥0

This is the appropriate requested value.

<0

An error occurred. (Cf. Error Codes).
