AUTOSAR FlexRay Transport Protocol

 

With the specification FlexRay Transport Layer (version 3.2.1 of 2011-04-19), AUTOSAR (https://www.autosar.org/) has defined a way to transfer large blocks of data between nodes connected to a FlexRay network.

This document describes the node layer DLL for Vector.CANoe.FlexRay that implement the transport protocol to allow nodes simulated by CANoe to interact with hardware nodes attached to a FlexRay network.

Name of the VMODULE/DLL File

This module description file (*.vmodule) and the corresponding DLLs are delivered together with the CANoe installation:

  • AutosarFlexRayTP3.vmodule (AUTOSAR FlexRay Transport Protocol)

Sample Configurations

In order to demonstrate the usage of the Transport Protocol, a respective sample configuration with a current DLL is installed along with CANoe:
