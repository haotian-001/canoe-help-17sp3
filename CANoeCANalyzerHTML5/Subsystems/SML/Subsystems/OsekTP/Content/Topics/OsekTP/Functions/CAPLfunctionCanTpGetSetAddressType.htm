CanTpGetSetAddressType

 

Note

The J1939 addressing modes normal fixed and mixed can be used with physical (one-to-one) or functional (one to many) addressing. This is indicated in the value of the protocol field of the J1939 CAN ID.

Function Syntax

long CanTpGetAddressType(long connHandle);

long CanTpSetAddressType(long connHandle; dword type);

Description

Get or set the addressing type, where 1 indicates functional, 0 physical.

Parameters

—

Return Values

—

Example

If the database connection is in mixed or normal fixed mode, activate functional addressing.

long dbHandle;
dbHandle = CanTpGetDBConnection();
if( dbHandle <= 0) return; // No connection configured in the database
if( CanTpGetAddressingMode( dbHandle) == 2
  || CanTpGetAddressingMode( dbHandle) == 3)
{
  CanTpSetAddressType( dbHandle, 1); // functional
}
