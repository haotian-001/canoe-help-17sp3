AUTOSAR UDP Network Management

 

With the specification UDP Network Management , AUTOSAR (www.autosar.org) has defined a way to coordinate the transition between normal operation and bus-sleep mode of an Ethernet network.

This document describes the functions for AUTOSAR Network Management (NM) protocol, that are implemented via module description file Enables support for different RT kernel architectures (64 bit/32 bit/Linux) without having to change the configuration. (*.vmodule) in Vector CANoe version starting with 15.2. These functions allow nodes simulated by CANoe to interact with hardware nodes attached to an Ethernet network.

Name of the VMODULE File 

This module description file Enables support for different RT kernel architectures (64 bit/32 bit/Linux) without having to change the configuration. (*.vmodule) is delivered together with the CANoe installation:

  • AUTOSAR UDP Network Management functions (AsrUdpNm.vmodule)

Sample Configurations

In order to demonstrate the usage of the network management, a respective sample configuration with a current VMODULE file is installed with the UDPNM Add-on.
