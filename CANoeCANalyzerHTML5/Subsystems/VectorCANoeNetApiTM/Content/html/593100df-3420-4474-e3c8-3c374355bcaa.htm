Vector CANoe .NET API for test modules

ReportTestCaseSkipped Method

This function can be used to mark a test case as not executed. This will t hen appear as a "skipped" Test Case in the Test Report. The verdict of the Test Module is not affected. This function may only be called in the context of the MainTest function.
