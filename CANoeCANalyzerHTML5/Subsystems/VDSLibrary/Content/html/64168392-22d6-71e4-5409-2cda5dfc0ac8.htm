Ecu Methods

The Ecu type exposes the following members.

Methods
Name Description
ActivateTesterPresent
Activates (or deactivates) the periodic sending of the tester present request message by the application.
Authenticate
Execution of the default authentication mechanism.
Authenticate(String)
Execute the authentication using the passed key value list as string. The format is "Param1=Value1;Param2=Value2".
AuthenticateByJob
Execute the job with the passed name.
ClearDtc
Clears the given Dtc in the Ecu.
ClearDtcs
Clears all Dtcs in the ECU.
ClearDtcs(UInt32)
Clears the groupOfDtcs.
Close
Invalidate ECU and its hierarchy; free resources.
CreateRequest(Byte)
Create a request using raw bytes as PDU.
CreateRequest(String)
Create a request using the service qualifier.
CreateRequestsBySemantic
Creates requests using the supplied semantic.
Deauthenticate
Execution of the deauthentication mechanism.
Dispose
Free resources allocated for this message. Call this method when the message is no longer used.
Finalize
Finalizes an instance of the Ecu class.
(Overrides ObjectFinalize.)
IdentifyAndSelectVariant
Identify and select ECU variant and give the variant qualifier back.
IdentifyVariant
IdentifyVariant returns the qualifier of the identified ECU variant.
IsTesterPresentActive
Get the current tester present sending state.
ReadDtcs
Reads all Dtcs for the Ecu.
ReadDtcs(Byte)
Reads all Dtcs which have the given DTC status
ReadExtendedDataRecords(Dtc)
Reads the extended data records for the given Dtc.
ReadExtendedDataRecords(Dtc, Byte)
Reads the extended data records with the given recordNumber for Dtc.
ReadSnapshotDataRecords(Dtc)
Reads the snapshot data records for the given Dtc.
ReadSnapshotDataRecords(Dtc, Byte)
Reads the snapshot data records with the given recordNumber for Dtc.
SelectVariant
Selects the ECU variant by its qualifier. In case an empty string is passed to SelectVariant it shall return VariantSelectionResult.VariantDoesNotExist.
ToString
Get the ECU identification name.
(Overrides ObjectToString.)
Unlock(Byte)
Unlocks the ECU in the specified level.
Unlock(Byte, String)
Unlocks the ECU in the specified level using an additional option string.
