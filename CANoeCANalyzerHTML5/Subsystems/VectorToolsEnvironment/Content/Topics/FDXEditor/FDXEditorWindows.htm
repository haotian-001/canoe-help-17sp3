Description of the Windows

 

The FDX Editor provides the following windows:

Screenshot

Short Description

Table Editor

FDX groups and items are displayed and manipulated in the Tabular Editor. The display is split. The focus of the Group Table (on the left) determines the currently displayed content of the Item Table (on the right).

Each newly opened FDX description is displayed on its own page so that multiple descriptions can be edited side-by-side.

Symbol Explorer

All available symbols are shown in the Symbol Explorer. These can be linked to FDX items using drag and drop or copy and paste.

Identifier List

The Identifier List serves as the source for unique identifiers of the FDX groups and items.

The content of the list is imported from text or CSV files. The identifiers can be linked to FDX groups or items using drag and drop or copy and paste.

Symbol Properties

The Symbol Properties Tool displays details of an FDX item and the symbol linked to it.

The properties can be edited by the user and do not have to match the CANoe database at the time of editing. It is sufficient if a CANoe symbol is clearly identifiable at runtime.

Graphic Layout

The Graphic Layout graphically displays the physical extent of the selected groups.

Different symbol types are labeled with corresponding icons and unique colors. The tool allows scaling of the resolution in the horizontal direction.
