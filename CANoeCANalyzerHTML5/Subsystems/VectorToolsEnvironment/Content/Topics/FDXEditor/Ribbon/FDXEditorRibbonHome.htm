Symbol

Description

Edit Group

Paste

With a click on the symbol you add the contents of the clipboard to the file.

Cut

With a click on the symbol you cut the selection and to transfer it to the clipboard. Click on Paste to add the selection again.

Copy

With a click on the symbol you copy the selection to the clipboard. This does not change the file. Click on Paste to add the selection again.

Delete

With a click on the symbol you delete the selection.

Identifier Group

Import Identifier

With a click on the symbol you open a dialog where you can import identifiers from a text or CSV file to the identifier list.

The dialog provides the option to replace currently loaded identifiers with the imported list and the option to trim the imported identifiers.

The data of a text file are interpreted line-by-line as identifiers during importing while identifiers from a CSV file are separated with the "," and ";" characters.

This menu item is only available if the focus is on the Identifier List (Structure).

Add Items Group

New Group

With a click on the symbol you create a new empty group in the active FDX description.

The new group is placed at the end of the description.

New Item

With a click on the symbol you create a new item at the end of the group the currently has the focus.

A dialog prompts you to select the symbol type to which the item is linked.

New Function

With a click on the symbol you create a new function.

Data Layout Group

Automatic Group Size

With a click on the symbol you activate and deactivate automatic calculation of the group size for the whole application.

The associated icon indicates whether the feature is currently activated.

Automatic Layout

With a click on the symbol you revise the physical arrangement of items in all selected groups so that they are seamlessly lined up in order.

The sizes of groups are adjusted to the revised layout.

Data Group

Move Up

With a click on the symbol you move the selected groups, items, or identifiers up one position.

This movement relates to the arrangement of elements in the FDX description or Identifier List and has no affect on the physical layout of groups.

Move Down

With a click on the symbol you move the selected groups, items, or identifiers down one position.

This movement relates to the arrangement of elements in the FDX description or Identifier List and has no affect on the physical layout of groups.

Reduce Offset

With a click on the symbol you reduce the physical offset of the selected items by one byte.

Increase Offset

With a click on the symbol you increase the physical offset of the selected items by one byte.

Layout Group

Here you can open/close the available windows and reset the window position

  • Symbol Explorer
  • Properties
  • Identifier List (Outline)
  • Graphical Layout
  • Project Documents
  • Output Window
  • Search
  • Reset Window Positions
    Resets the window position to the default settings.
