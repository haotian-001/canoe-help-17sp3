Identifier List (Outline)

 

The Identifier List serves as a source for group or item identifiers from the system perspective. The identifiers can be linked to items or groups using drag and drop or copy and paste.

Note

Identifiers are always optional and purely descriptive in nature. They have no affect on communication at runtime.

The list of identifiers is only a textual list of character strings to which no other semantics are assigned from the perspective of the FDX Editor. New identifiers can be imported and loaded identifiers can be removed. These actions do not modify the source file. When the editor is closed, the current list is saved for the user and is reloaded the next time the editor is started. The elements in the list can be edited and new entries can be created, e.g. using the Paste command or the Insert row. The order of the identifiers can be modified, but this has no further semantic relevance.

Context Menu of the Identifier List

Move Up
Moves selected identifiers up one position in the list.

Move Down
Moves selected identifiers down one position in the list.

Cut
Moves selected identifiers from the list to the clipboard.

Copy
Copies selected identifiers to the clipboard.

Paste
Inserts data from the clipboard as new identifiers.

Delete
Deletes selected identifiers from the list.

Search Option

You can search for a desired text in the Identifier List. You start the search with <Enter> or .

The search function does not support the wildcard character ("*").
