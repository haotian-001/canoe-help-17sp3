Vector Support

Use the Vector Support Website to find supportive answers for your work with Vector products from multiple resources.

If you don't find what you are looking for, you can open a support case to get individual help.

  • Vector Support Website (support.vector.com)

  • Vector KnowledgeBase (support.vector.com/kb)

  • Support E-Mail: support@vector.com

Help Feedback

This is how you can give us feedback about the help:

  1. Create a support case on the Vector support website; the Subject is prefilled with Help Feedback.
  2. Copy the URL of the help page about which you want to send your feedback.
  3. Paste the help page URL and add your comments in the Description field.
  4. Select the Affected Product.
  5. Click [Proceed].
