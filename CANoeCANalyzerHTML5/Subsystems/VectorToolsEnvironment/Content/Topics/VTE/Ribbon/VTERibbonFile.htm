Ribbon: Tab File

File

Home

 

Symbol

Description

New

With a click on the symbol you open a new file.

Open

With a click on the symbol you open the Windows Explorer to select a previously saved file.

With a click on you open the recent file list.

Note

If you select a panel in CNP format, the panel is converted to the current Panel Designer format (*.xvp) and opened in the Panel Designer. The target file automatically obtains the same name as the source file.

Save

With a click on the symbol you save the file.
The Save As dialog automatically opens when the file is saved for the first time.

Save As...

With a click on the symbol you open the Save As dialog.

Save All

With a click on the symbol you save all opened and changed files.

Options

In the Options dialog you can configure global settings.

With a click on the symbol you open the Options dialog.

Close

With a click on the symbol you close the active file.

If you have not yet saved all changes to the open file, you will be prompted to do so automatically. Select the appropriate option in the window which appears.

Close All

With a click on the symbol you close all open files.

If you have not yet saved all changes to the open files, you will be prompted to do so automatically. Select the appropriate option in the window which appears.

Help

With a click on the symbol you open the help.

About

With a click on the symbol you open a dialog that displays the version numbers of the Vector Tools Environment and the plug-ins, and the name of the licensee.

Exits the Vector Tools Environment.

If you have not yet saved all changes to the open files, you will be prompted to do so automatically. Select the appropriate option in the window which appears.
