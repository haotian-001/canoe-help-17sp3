Document View

 

In this view you can edit the vCDL files.

You can make settings for the view in the Options dialog.

Error Handling

Parsing errors and warnings are displayed in the Output Window. Double-clicking on an error or warning in the Output Window selects the corresponding line in the document view here.

Autocomplete Input Assistance

The Autocomplete Input Assistance makes the input easier. By entering a word or number sequence, the marker of the input assistance jumps to the next matching entry. If you have found the desired entry within the input assistance you can accept it with <Return>. If no matching entry can be found, the marker jumps to the first entry of the input assistance.

The Autocomplete Input Assistance is automatically displayed if you have activated it in the Options dialog. You can also activate it with the key combination <Ctrl>+<Space>.

Search Panel

Description

Key Combination

Opens the search panel in find mode and focuses the find text box.

<Ctrl>+<F>

Repeats the last search operation in a forward direction.

<F3>

Performs a forward find on the current selection, or the current word when there is no selection. Also opens the search panel in find mode.

<Ctrl>+<F3>

Repeats the last search operation in a backward direction.

<Shift>+<F3>

Performs a backward find on the current selection, or the current word when there is no selection. Also opens the search panel in find mode.

<Ctrl>+<Shift>+<F3>

Opens the search panel in replace mode and focuses the find what text box.

<Ctrl>+<H>

These key combinations are active when focus is within the search panel, providing quick access to common features:

Closes the search panel.

<Esc>

Performs a find operation.

<Enter> (focus in find text box)

Performs a replace operation.

<Enter> (focus in replace text box)

When in replace mode, replaces the next matched search result.

<Alt>+<R>

When in replace mode, replaces all matched search results.

<Alt>+<A>

Toggles the match case option.

<Alt>+<C>

Toggles the whole word option.

<Alt>+<W>

Toggles the use regular expressions option.

<Alt>+<E>

Toggles the use search up option.

<Alt>+<U>
