Start Stop Control

 

Use Case

With the Start Stop Control, you can start and stop a measurement in both online and offline modes.

Configuration

You can configure the control via the ribbon or the Properties Window.

If you start or stop the measurement from the CANoe toolbar, the control indicates the current status.

If you start the measurement via CANoe step-wise () or if you interrupt the measurement (), you can only stop the measurement using the control. If you want to restart the measurement after the step-wise measurement start or after the measurement interruption, you can only do this via the toolbar.

Keyboard Operations

The function that is focused (Start or Stop) can be initiated with the space bar.
