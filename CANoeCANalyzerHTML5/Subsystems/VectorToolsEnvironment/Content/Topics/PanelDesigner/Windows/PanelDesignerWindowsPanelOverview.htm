Outline

Ribbon|Home Tab|Layout|Outline

The tree view of the Panel Designer lists controls which are integrated into the currently selected panel.

In the tabular overview, you can

  • quickly identify which symbols are assigned to which controls
  • assign symbols to the controls
  • find controls which are hidden or located outside the panel

Toolbar

Icon

Description

All controls which you have integrated into the panel are displayed.

Only controls which you have integrated into the panel and to which a symbol can be assigned are displayed.

Integrated Group Boxes are always displayed, since they may include controls which can be linked to symbols.

This option allows you to display the fully qualified names of the assigned symbols.

Assigning Symbols

You can assign a symbol to the control in the Symbol Name column

  • via the button (Symbol Selection dialog)
  • via the selection list
  • via the context menu
  • from a different table cell using drag and drop
    This allows you to use the already assigned symbols for other controls.
  • from the Symbol Explorer using drag and drop

Sorting

You can sort the list by clicking the table header:

  • alphabetically in ascending/descending order by control name
  • in ascending/descending order by symbol type
  • alphabetically in ascending/descending order by symbol name

In all cases, the classification of the elements in the Group Box remains the same, the elements are sorted within the Group Box.

Multi-Selection

You can mark several controls (lines) and assign them the same symbol with a unique action (see above Assigning Symbols). The selected symbol is not assigned to controls that do not support the selected symbol type.

Further Information

Below the list of the symbols, further information is displayed for the symbol selected in the list, for example the name of the namespace.
