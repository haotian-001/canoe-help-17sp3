Properties

Ribbon|Home Tab|Layout|Properties

You can use the Properties Window to configure selected elements (panel and controls) on the working area.

The setting options for each selected element are displayed in the Properties Window in the form of a table. With a multi-selection of elements, you can change all the shared settings at once.

Properties Window Toolbar

Description

Settings are categorized (default setting)

Settings

Categories

The display of the categories depends on the selected control or panel. The description of the different categories can be found on the individual help pages of the controls.

Note

  • With a multi-selection of elements, only the settings that are available for all the selected elements will be displayed in the Properties Window.

Links

Below the categories different blue links are displayed.

Only the links actually available for the relevant control will be displayed.

  • With the Fit Image Size link you reset the selected image to its original size.
  • With the Reset values link you reset the initialization values of all selected controls to the values from the database.
  • You can remove symbol assignments with the Detach Symbol, Detach all Symbols, Detach selected Symbols links.
  • You can open the Symbol Selection dialog with the Attach Signal, Attach Variable, Attach Diagnostics Parameter, Attach Service Signal, Attach Application Layer Object links.

Description

A short description of the selected setting is displayed at the bottom of the Properties Window.

Color Selection

In the case of the color selection list, a window with the following tabs will appear: User-defined, Web and System.

If you want to set a user-defined color, you can either select one of the colors available via the User-defined tab or you can enter the RGB value for the color directly behind the color display. The values must be separated by a ";" character. An error message will be displayed if the entry is incorrect.

To open the Color Selection dialog from within the User-defined tab, right-click on any of the empty color fields.
You can then select the required color by either entering the RGB values or by selecting it directly from a color scale and clicking the [Add color] button to add it to the palette of user-defined colors.
You can also use this dialog to specify the chroma, saturation, and intensity of the color.
