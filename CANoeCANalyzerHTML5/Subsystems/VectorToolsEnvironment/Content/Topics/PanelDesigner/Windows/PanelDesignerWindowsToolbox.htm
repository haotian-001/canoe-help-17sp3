Toolbox

Ribbon|Home Tab|Layout|Toolbox

The Panel Designer toolbox displays the available controls.

The controls can be assigned to an open panel directly from the toolbox:

  • By drag and drop
  • By double-clicking
  • By left-clicking the required control to select it and then inserting it by left-clicking the panel

Once a control has been assigned, the marker in the toolbox will automatically turn into a pointer. You can then configure the controls on the panel exactly as you wish.

On the Data Types of Assigned Symbols page you can find an overview of all data types of symbols that can be assigned to the controls.

Control Types

  • Display Element
    With display elements value changes of assigned symbols can be displayed.
  • Control Element
    With control elements values of assigned symbols can be changed.
  • Static Element
    Static elements are used for graphic representation or explanatory purposes.No symbols can be assigned to them.
  • Special Element
    With special elements predefined actions can be performed. No symbols are assigned to them.

Control Groups

For better clarity, the controls are divided into groups and sorted in alphabetical order:

  • All: all controls
  • Common: controls that cover most use cases
  • Graphical: controls that display data in graphical form
  • Complex Data: controls that display complex data
  • Special: controls for special use cases
