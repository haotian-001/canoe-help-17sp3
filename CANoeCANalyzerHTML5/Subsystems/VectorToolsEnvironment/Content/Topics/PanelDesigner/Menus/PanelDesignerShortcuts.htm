Keyboard Operations

 

The following key combinations may be used when working in the Panel Designer:

Calling Help

Function

Key Combination

Opens help.

In dialogs/window, help is displayed for the particular dialog/window.

<F1>

Controlling the Cursor in the Panel’s Working Area

Function

Key Combination

Skip to the next control

<Tab>

Skip to the previous control

<Shift>+<Tab>

Skip to the next panel

<Ctrl>+<Tab>

If the focus is in the panel's working area, you can press <Esc> to select the parent element of the currently selected control.

Example

  • A Radio button located in a group box is selected. When <Escape> is pressed, the parent group box is selected.
  • Several elements are placed very close to one another on a panel. It is not possible to use the mouse to select the panel itself. The solution is to continue pressing <Escape> until the panel is selected.

<Escape>

Activating Menu Commands

Function

Key Combination

Selects the ribbon and activates the selection of the tabs.

After that the following functions are available:

<Alt>

Opens the tab for the selected character.

<Character>

Opens the menu/window for the selected character on th eselected tab.

<Character>

Opens the tab to the left or right of the active tab.

<Arrow key right>/
<Arrow key left>

Moves the selection of menu commands within the opened menu.

<Arrow key up>/
<Arrow key down>

Executes the command of the selected menu command.

<Enter>
