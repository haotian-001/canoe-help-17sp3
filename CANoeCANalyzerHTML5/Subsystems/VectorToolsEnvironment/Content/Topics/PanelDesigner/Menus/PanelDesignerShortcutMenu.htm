Context Menu

 

Pressing the right mouse button opens the context menu.

Panel Page

Close
Closes the active file. If you have changed the displayed file and not yet saved the changes, you are asked whether the changes are to be saved.

Close All But This
Closes all files except the active one. If you have changed files and not yet saved the changes, you are asked whether the changes are to be saved for each individual program.

Panel

Paste
Adds the contents of the clipboard to the panel, provided that they are valid controls.

Select All
Selects all controls on the panel.

Properties
Opens the Properties Window.

Controls

Depending on the control the following commands are available:

Cut
Cuts the selected control and transfers it to the clipboard.

Copy
Copies the selected control to the clipboard.

Paste
Adds the contents of the clipboard to the panel, provided that they are valid controls.

Delete
Deletes the selected control.

Properties
Opens the Properties Window.

  • Bottom

  • Vertically

Order

  • Bring to Front
  • Send to Back

Attach Signal
Opens the Symbol Selection dialog, to assign a signal to the selected control.

Attach Variable
Opens the Symbol Selection dialog, to assign a variable to the selected control.

Attach Diagnostic Parameter
Opens the Symbol Selection dialog, to assign a diagnostic parameter to the selected control.

Attach Service Signal
Opens the Symbol Selection dialog, to assign a service signal to the selected control.

Attach Application Layer Object
Opens the Symbol Selection dialog, to assign a application layer object to the selected control.

Detach Symbol
Removes a symbol assignment.

Detach all Symbols
Removes all symbol assignments.

Detach selected Symbols
Removes selected symbol assignments.

Reset Values
Resets the initialization values of all selected controls to the values from the database.

Fit Image Sizes
Resets the selected image to its original size.
