New Features

 

Cross Reference: Release Notes Since Version 17

Starting with CANoe version 17, you can find the release notes at the following link:

  • CANoe Release Notes

New features of previous versions:

Version 16

  • In the Symbol Explorer, the new Min, Max and Unit columns are available in the Network Symbols, Variables and Participants & Application Layer Objects views.

Version 15 SP3

  • The Event Control is replaced by the Complex Data Control. Complex data structures (data members of distributed objects) can now also be received and sent.
  • The Method Call Control and the Complex Data Control can now be labeled for a better overview.
  • The Method Call Control and the Complex Data Control now support the display and selection of symbolic values if a value table is stored.

  • You can also select Application Panels for the Panel Control Button.

Version 15

  • For better clarity in the Toolbox, the controls have been divided into groups.
  • With the Event Control you can now also receive events.
  • In the Hex/Text-Editor, the byte display in hex format can now also be copied and pasted.

Version 14

  • With the Event Control you can send events.
  • The length of a dynamic array (CO/DO) can be dragged onto a panel and connected to an Input/Output Box.

  • The LED Control can now display more than two states in color.

Version 13.0

  • Integration of the Panel Designer as Panel plug-in into the Vector Tools Environment.
  • The Path Dialog can now also be displayed as a link.

Version 12.0 SP3

  • Graphical User Interface (GUI)
    The user interface of the Panel Designer is now High-DPI capable. On high-resolution monitors, symbols and texts are adapted to selected resolution and scaling.
  • Input/Output Box
    In addition to the description, you can also display defined min/max values and a defined unit.
    The display can be set as default.
  • Method Call Control
    Complex structured data types can be displayed.
  • Switch/Indicator and Picture Box
    For both controls an error message will be displayed if an image that is to large is selected. A maximum image size (height/width) of 32767 pixels is supported. For larger images, the image size must be reduced in an image processing program.
  • Improved Loading Time for Configurations with Many Panels
  • When copying a control, the new control gets a unique name (control type with unique index).

Version 12.0

  • Improved performance when copying and pasting within the Panel Designers.
  • Improved loading times for configurations with many panels.
  • For the LED Control the symbol value can be interpreted as RGB and the LED control can therefore be colored accordingly.

Version 11.0 SP3

  • Input/Output Box
    If the text is too long, you can select in the Properties Window whether the start of the text or the end of the text should be displayed.
  • The Text and Control function on the Home ribbon tab aligns controls (Input/Output Box, Combo Box, Check Box, Radio Button) and adjusts the width of the label and element.
  • Working with Value Tables
    You can assign switch values from value tables for the Button, Check Box, LED Control, Media Player, Radio Button, and Switch/Indicator controls.

Version 11.0

  • Picture Box
    To shorten the loading times of an image, you can now select in the Properties Window whether an image should be kept in cache memory or not.
  • Switch/Indicator
    During a state change, besides concrete values, you can also enter value ranges to map, for example a digital fuel indicator.
  • Panel Overview
    The new window lists controls which are integrated into the currently selected panel. In the tabular overview, you can
  • quickly identify which symbols are assigned to which controls
  • assign symbols to the controls
  • find controls which are hidden or located outside the panel
