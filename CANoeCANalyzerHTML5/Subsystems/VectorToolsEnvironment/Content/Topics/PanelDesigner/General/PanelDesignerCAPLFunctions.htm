Access Panels using CAPL Functions

 

You can access panels or panel elements using following CAPL functions.

CAPL Function

Description

Panel Designer Controls

openPanel

Opens a panel or all panels.

—

closePanel

Closes a panel or all panels.

—

clockControlReset

Resets the Clock Control designed as stop watch.

Clock Control

clockControlStart

Starts the Clock Control designed as stop watch.

Clock Control

clockControlStop

Stops the Clock Control designed as stop watch.

Clock Control

deleteControlContent

Deletes the contents of the CAPL Output View.

CAPL Output View

enableControl

Activates or deactivates specific elements.

Button, Check Box, Combo Box, Hex/Text Editor, Input/Output Box, Numeric Up/Down, Panel Control Button, Path Dialog, Radio Button, Switch/Indicator, Track Bar

putValueToControl

Transfers defined values to the control.

CAPL Output View

setClockControlTime

Sets the time of the Clock Control.

Clock Control

setControlBackColor

Sets the background color of panel elements. The color value is determined using the MakeRGB function.

Analog Gauge, Check Box, Clinometer, Clock Control, Combo Box, Compass, Group Box, Hex/Text Editor, Input/Output Box, LCD Control, Meter, Numeric Up/Down, Panel Control Button, Radio Button, Static Text, Track Bar

setControlColors

Sets the background and text color of panel elements.

Analog Gauge, Button, CAPL Output View, Check Box, Clock Control, Combo Box, Hex/Text Editor, Input/Output Box, LCD Control, Numeric Up/Down, Panel Control Button, Radio Button, Static Text

setControlForeColor

Sets the foreground color of panel elements. The color value is determined using the MakeRGB function.

Analog Gauge, Button, CAPL Output View, Check Box, Clock Control, Combo Box, Compass, Hex/Text Editor, Input/Output Box, LCD Control, LED Band, LED Control, Numeric Up/Down, Panel Control Button, Radio Button, Static Text

SetControlProperty

Sets the specified property of the control.

Note

Only a limited and predefined selection of control properties is supported. For an overview, see the description of the CAPL function.

Button, CAPL Output View, Check Box, Combo Box, Group Box, Input/Output Box, LCD Control, Numeric Up/Down, Progress Bar, Radio Button, Static Text, Switch/Indicator, Track Bar

setControlVisibility

Sets the visibility at runtime.

All controls.

setDefaultControlColors

Sets back the background and text color of panel elements as defined in the Panel Designer.

Analog Gauge, Button, CAPL Output View, Check Box, Clock Control, Combo Box, Group Box, Hex/Text Editor, Input/Output Box, LCD Control, Meter, Numeric Up/Down, Panel Control Button, Radio Button, Static Text, Track Bar

setMinMax

Sets the minimum and maximum value of controls.

Note

When setting new values, check that the following properties match:

  • Numeric Up/Down: Increment
  • ProgressBar: OriginValue
  • TrackBar: LargeChange, SmallChange, TickFrequency

Numeric Up/Down, Progress Bar, Track Bar

setMediaFile

Replaces the media file of the Media Player.

Media Player

setPictureBoxImage

Replaces the image of the Picture Box.

Picture Box
