Compatibility

 

Loading panel files in CANoe

  • Created panel files can be loaded and displayed in CANoe.

Loading panel files from newer versions

  • Created panel files from newer CANoe versions can be loaded and displayed in this version.

Note

Only functions of this version are supported. Functions of the newer version will automatically be deactivated.

If you save a loaded panel file with this version, all functions of the new version will get lost.
  • Created panel files since CANoe version 7.0 SP4 can not be loaded and displayed in older versions.

Solution:

Install the current Service Pack for version 7.0. You can find this on www.vector.com

Convert panels into Panel Designer panels

Panel Editor panels (*.cnp) are automatically converted by opening them in the Panel Designer.

Panel Editor panel will be automatically converted when loading a CANoe configuration and saved in XVP format when saving the configuration.
