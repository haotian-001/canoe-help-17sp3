Edit Group

Paste

With a click on the symbol you add the contents of the clipboard to the panel, provided that they are valid controls. You can choose the position of the control using the cursor.

Cut

With a click on the symbol you cut the selected control and paste it to the clipboard. Click on Paste to add the control to the panel again.

Copy

With a click on the symbol you copy the selected control to the clipboard. This does not change the panel. Click on Paste to add the control to the panel again.

Delete

With a click on the symbol you delete the selected control. The control is not transferred to the clipboard in this action.

Arrange Controls Group

Here you can align the selected controls against each other:

  • Left
    All selected controls are left-aligned. Their vertical alignment remains unchanged.
    The left-hand edge of the first control selected is used as the reference point.
  • Center
    Centers the selected controls horizontally; their vertical alignment remains unchanged.
  • Right
    All selected controls are right-aligned. Their vertical alignment remains unchanged.
    The right-hand edge of the first control selected is used as the reference point.
  • Top
    All selected controls are top-aligned. Their horizontal alignment remains unchanged.
    The top edge of the first control selected is used as the reference point.
  • Middle
    Centers the selected controls vertically; their horizontal alignment remains unchanged.
  • Bottom
    All selected controls are bottom-aligned. Their horizontal alignment remains unchanged.
    The top edge of the first control selected is used as the reference point.
  • Text and Control (valid for Input/Output Box, Combo Box, Check Box, Radio Button)
    All marked controls are arranged as follows:
  • The control with the white marking points specifies the width for the description and the elements. Therefore, it is recommended to select the control with the widest description and element. The first selected control automatically receives the white marking points.
    If descriptions are truncated or elements are too short, you should check that you have selected the correct control to specify the widths. An arrangement can be undone, e.g. via <Ctrl>+<Z>.
  • All control descriptions are displayed on the left of the element.
  • All descriptions are given the same width (if the text is too long it is abbreviated with "...").
  • All descriptions are left-aligned.
  • All Input/Output Box and Combo Box elements are given the same width, Check Boxes and Radio Buttons maintain their size.
  • All elements are left-aligned with each other.

Bring to Front

With a click on the symbol you bring the selected control to the front.

When configuring panels, elements can be overlaid. This ordering of elements is known as the overlap sequence.
To change the overlap sequence, select the corresponding element and send it to the front or to the back.
When a new element is added to a panel, it is always sent to the front until the overlap sequence changes or another element is added.

Send to Back

With a click on the symbol you send the selected control to the back.

When configuring panels, elements can be overlaid. This ordering of elements is known as the overlap sequence.
To change the overlap sequence, select the corresponding element and send it to the front or to the back.
When a new element is added to a panel, it is always sent to the front until the overlap sequence changes or another element is added.

Here you can set the horizontal spacing between the selected elements:

  • Make Equal
    All selected controls are positioned with equal horizontal spacing. Their vertical alignment remains unchanged.
    The distance between the controls is calculated on the basis of an equal distance between the two outermost controls.
  • Remove
    All selected controls are lined up horizontally. Their vertical alignment remains unchanged.
    The first control selected is used as the reference point.

Here you can set the vertical spacing between the selected elements:

  • Make Equal
    All selected controls are positioned with equal vertical spacing. Their vertical alignment remains unchanged.
    The distance between the controls is calculated on the basis of an equal distance between the two outermost controls.
  • Remove
    All selected controls are lined up vertically. Their horizontal alignment remains unchanged.
    The first control selected is used as the reference point.

Size Group

Width

With a click on the symbol all selected controls are set to the same width.

The first control selected is used as the reference point.

Height

With a click on the symbol all selected controls are set to the same height.

The first control selected is used as the reference point.

Both

With a click on the symbol all selected controls are set to the same height and width.

The first control selected is used as the reference point.

Center in Panel Group

Horizontally

With a click on the symbol all selected controls are centered horizontally in relation to the panel. Their vertical alignment remains unchanged.

Vertically

With a click on the symbol all selected controls are centered vertically in relation to the panel. Their horizontal alignment remains unchanged.

Both

With a click on the symbol all selected controls are centered horizontally and vertically in relation to the panel.

More Group

Check Symbols

With a click on the symbol you can check on the currently active panel if the controls are linked with symbols (if possible) and if these symbols are saved in the current database.

Information on inconsistencies is displayed to you automatically in the Output Window.
