Panel

Properties

 

Symbol

Description

General Group

Name

Here you can give the panel a name. The name of the panel is required for access from CAPL.

Background Group

Color

Here you change the background color of the panel.

Image File

Here you change the background image for the panel.

After selection of the image the panel is automatically adjusted to the image size.

Frame Group


Here you change the frame of the panel:

  • Fixed 3D
    3D frame
  • Fixed Single
    Single frame
  • None
    No frame

Size Group

Width

Here you change the width of the panel [px].

Height

Here you change the height of the panel [px].

Resize Proportionally

With a click on the symbol you activate/deactivate the proportional increase/decrease (height = width).

Fit Image Size

With a click on the symbol you reset the selected image to its original size.

More Group

Show Properties

With a click on the symbol you open the Properties Window.

You can also configure the panel via the Properties Window.

Properties Window

Note

As default only the relevant settings for the selected control are displayed. For displaying all properties, please select in the toolbar of the Properties Window.

Appearance

Background Color

Defines the background color of the panel.

Background Image

Opens the dialog for image selection.

After selection of the image the panel is automatically adjusted to the image size.

Border Style

Specifies whether a frame is to be drawn around the panel and if so in which style.

Is Proportional

Specifies whether you want to change the panel size proportionally.

If you select False, you can then resize the panel as desired. The background image will be distorted.

You can reset the panel size back to the original image size using the Fit Size context menu command.

Layout

In the Layout category, you can enter settings relating to the size of the panel.

Note

You can also resize the panel by dragging its markers.
