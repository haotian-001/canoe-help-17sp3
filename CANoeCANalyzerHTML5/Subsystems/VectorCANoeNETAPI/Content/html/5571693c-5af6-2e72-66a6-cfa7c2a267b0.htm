Vector CANoe .NET API

Vector.CANoe.Runtime.Abstract Namespace
Contains the API for Abstract Binding specific Service Discovery and Event, PDU and Field subscription.
Classes
Class Description
OnEventSubscribedAttribute
Mark a method with this attribute, and it will be called when a consumer has subscribed for an event with a specific provider.
OnEventUnsubscribedAttribute
Mark a method with this attribute, and it will be called when a consumer has unsubscribed for an event from a specific provider.
OnFieldSubscribedAttribute
Mark a method with this attribute, and it will be called when a consumer has subscribed for a field with a specific provider.
OnFieldUnsubscribedAttribute
Mark a method with this attribute, and it will be called when a consumer has unsubscribed for a field from a specific provider.
OnPDUSubscribedAttribute
Mark a method with this attribute, and it will be called when a consumer has subscribed for a PDU with a specific provider.
OnPDUUnsubscribedAttribute
Mark a method with this attribute, and it will be called when a consumer has unsubscribed for a PDU from a specific provider.
ServiceDiscovery
Low level Service Discovery API for Abstract Binding.
Interfaces
Interface Description
IAbstractConsumedEvent
Interface for consumer side service event ports with abstract binding, providing high level event subscription methods.
IAbstractConsumedField
Interface for consumer side service field ports with abstract binding, providing high level field subscription methods.
IAbstractConsumedPDU
Interface for consumer side service PDU ports with abstract binding, providing high level PDU subscription methods.
