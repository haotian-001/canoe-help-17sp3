Remarks
For the J1939 frames defined in databases, specific classes are generated in type libraries. These specific classes are subclasses of J1939Frame and provide easy access to signal values. You can also create instances of J1939Frame directly. Then you should set the ID and the DLCEx manually before sending the frame. The data can only be set on a per-byte basis in the J1939Frame class. You also receive objects of this class in the OnJ1939Frame handler.
