EthernetAdapterStackParameterAccessorGetStackParameter Method

Gets the value of the given TCP/IP stack parameter of the EthernetAdapter.

To get the value of the node's stack parameter, use the NetworkStackParameterAccessor obtained via Parameters.

Remarks
The function depends on the selected stack. This functionality cannot be used in connection with the operating system TCP/IP stack.

To get the value of the node's stack parameter, use the NetworkStackParameterAccessor obtained via Parameters.
