Vector CANoe .NET API

Execution Fields

The Execution type exposes the following members.

Fields
Name Description
WAIT_ABORTED
Result of a wait showing that the wait has been aborted (e.g. measurement was stopped)
WAIT_EMPTY_CALL_LIST
Result of a WaitForAllAnswers or WaitForAnyAnswer in case the list of network function calls passed in is empty - so there is nothing to wait for.
WAIT_EXCEPTION
Result of a WaitForTask showing the test was interrupted due to an exception.
WAIT_ILLEGAL_RESULTVALUE
Result of a WaitForTask showing the test task returned an invalid value (allowed values are > 0) If you need an arbitrary value range, use the parameter object to pass information in and out.
WAIT_NOT_ALLOWED
Result of a wait called from non-Test, -Snippet or -Waiting Handler code (e.g. from a simulation node).
WAIT_TIMEOUT
Result of a wait showing that the wait has timed out.
WAIT_VALUE_STATE_RESET
Result of a wait showing that the value status has been reset.
