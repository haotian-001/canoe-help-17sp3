Vector.CANoe.Sockets Namespace
Contains the API for sockets communication services in CANoe.
Classes
Class Description
NetworkStream
Provides the underlying stream of data for network access.
OnIpReceivePrepareAttribute
Mark a method with this attribute, and it will be called right before a received packet will be dispatched to the TCP/IP stack.
OnIpSendPrepareAttribute
Mark a method with this attribute, and it will be called right before a packet will be sent by the TCP/IP stack.
