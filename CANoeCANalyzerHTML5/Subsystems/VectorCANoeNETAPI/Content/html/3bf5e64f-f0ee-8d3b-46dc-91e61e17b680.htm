Remarks
Use instances of this class to temporarily buffer values for a signal / environment variable / ..., which you wish to store in the CANoe runtime at a later point of time. In effect, this class just holds a buffered value (of type double) and a connection to the CANoe runtime. If you wish to use an instance of this class in a generic class or method with a type parameter T, the method should be declared with the constraints that T is a class implementing IRuntimeValue.
