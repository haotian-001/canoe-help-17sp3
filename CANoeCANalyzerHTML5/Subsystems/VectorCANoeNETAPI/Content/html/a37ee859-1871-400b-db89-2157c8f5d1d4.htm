Vector CANoe .NET API

Output Methods

The Output type exposes the following members.

Methods
Name Description
Clear
Clears all content of the Write window of the tools.
Clear(String)
Clears the specified tab of the Write window of the tools.
LogLine
Writes the given text as a comment line into ASCII log files. I.e., before the text, comment characters ("//") and a timestamp are written.
LogLineRaw
Writes the given text into ASCII log files. In contrast to LogLine, no comment characters and not timestamp is written.
WriteLine(String)
Writes the given text to the Write window of the tools.
WriteLine(String, Object)
Writes the given text to the Write window of the tools.
WriteLine(String, String)
Writes the given text to the specified tab of the Write window of the tools. If the tab doesn't exist yet, it is created.
WriteLine(String, String, Object)
Writes the given text to the specified tab of the Write window of the tools. If the tab doesn't exist yet, it is created.
