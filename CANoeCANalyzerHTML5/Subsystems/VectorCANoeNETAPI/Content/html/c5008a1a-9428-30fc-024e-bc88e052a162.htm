NetworkStackParameterAccessorGetStackParameter Method

Gets the value of the given TCP/IP stack parameter of the node.

To get the value of a parameter related to a specific EthernetAdapter, use the EthernetAdapterStackParameterAccessor obtained via StackParameters.

Remarks
The function depends on the selected stack. This functionality cannot be used in connection with the operating system TCP/IP stack.

To get the value of a parameter related to a specific EthernetAdapter, use the EthernetAdapterStackParameterAccessor obtained via StackParameters.
