Vector CANoe .NET API

Vector.CANoe.Threading Namespace
Contains the API for threading services in CANoe.
Classes
Class Description
Criterion
Represents a criterion which is composed of several simple value criteria or custom criteria.
CriterionAttribute
You can mark a method with this attribute to define your own criteria.
CriterionBase
Base class for a criterion. You should normally not use the base class itself directly.
DelegateCriterion
Helper class wrapping a criterion delegate for values. You do not need to use this class directly.
DelegateCriterionBase
Helper class wrapping a CriterionDelegate. You do not need to use this class directly.
DynamicDelegateCriterion
Helper class to define a criterion with a delegate which evaluates dynamically defined signals / variables.
DynamicValueCriterion
Represents a simple criterion which can be used to check the current value of a dynamically defined signal, environment variable or system variable.
Execution
Provides access to execution control in CANoe.
FrameDelegateCriterionT
Helper class wrapping a criterion delegate for frames. You do not need to use this class directly.
TaskCancelToken
Contains the information if a task is to be canceled. If a task processes a loop, check for IsCancellationRequested and exit if .
UserEventCriterion
Criterion which can be used to signal the main simulation / test thread that something happened on a background thread. See remarks.
ValueCriterionT
Represents a simple criterion which can be used to check the current value of a signal, environment variable or system variable.
ValueCriterionBase
Helper base class for criteria which check the current value of a signal, environment variable or system variable.
ValueEntityCriterion
Represents a simple criterion which can be used to check the current value of a value entity.
ValueEntityCriterionBase
Base class for criteria checking value entity state.
ValueEntityStatusCriterion
Represents a simple criterion which can be used to check the current status of a value entity.
Delegates
Delegate Description
CriterionDelegate
Delegate for the evaluation of custom criteria.
FrameCriterionDelegateT
Delegate for the evaluation of custom criteria based on frame reception.
TaskAction
Task delegate function. Implement to pass to. See: Execution.WaitForTask
