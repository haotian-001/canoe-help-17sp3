PayloadProtocolSetRawData Method (Int32, Byte, Int32)
Sets the payload data of the protocol within an EthernetPacket. If data is greater than the current length of the IPayloadProtocol's payload, the EthernetPacket is enlarged; if data is smaller, the length is not changed.
