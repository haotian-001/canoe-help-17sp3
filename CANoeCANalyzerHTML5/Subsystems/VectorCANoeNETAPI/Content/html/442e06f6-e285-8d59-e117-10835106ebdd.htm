Return Value

Type: Int32
return any value > 0 for own results. Values 0 and below are reserved for following return values:
Execution.WAIT_TIMEOUT: if an maxTime was defined in Execution.WaitForTask and the task did not return within maxTime milliseconds

Execution.WAIT_ABORTED: if the measurement was stopped during task execution

Execution.WAIT_EXCEPTION: if an exception occurred in the taskAction delegate

Execution.WAIT_ILLEGAL_RESULTVALUE: the result provided by the task is <= 0
