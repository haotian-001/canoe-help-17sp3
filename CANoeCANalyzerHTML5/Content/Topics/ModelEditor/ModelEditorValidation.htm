Model Editor

 

You can open the views Validation and References using the splitter at the bottom of the editor.

Validation

This view displays errors that occurred while creating/editing type definitions, communication/distributed objects and endpoints. The validation refers to all data in the editor.

With a double-click on an entry you navigate directly to the displayed object.

References

This view displays all references to an object (for example, the service interface of a serve or vice versa). To do this, choose the context menu commandFind All References of an object.

With a double-click on an entry you navigate directly to the displayed reference.
