Service Interfaces

If you have selected the Service Interfaces node in the tree view, all service interfaces with the following properties will be displayed:

  • Name
    Name of the service interface
  • Namespace
    Namespace of the service interface
  • Comment

You can edit all properties directly in the table and also create and delete service interfaces.

You can create a new service of the service interface

  • with in the toolbar
  • under Communication Objects|Services

Events

If you have selected an existing service interface in the tree view, the eventsSignals within a service are called events that this service interface describes will be listed here. The following properties will be displayed:

  • Name
    Name of the event
  • Data Type
    Data type of the event

    • (can be defined under Type Definitions|Data Types)

  • Relative Bit Offset
    Start bit of the event
  • Optional
  • Comment

You can edit all properties directly in the table and also create and delete events.

Use the splitter on the right-hand side to open the Data Type Properties of the selected object that you can edit directly there.

Methods

If you have selected an existing service interface, the methods that this service interface describes will be listed here. The following properties will be displayed:

  • Name
    Name of the method
  • Function Type
    Function type of the method

    • (must be defined under Type Definitions|Function Types)

  • Comment

You can edit all properties directly in the table and also create and delete methods.

PDUs

If you have selected an existing service interface, the PDUs that this service interface describes will be listed here. The following properties will be displayed:

  • Name
    Name of the PDU
  • PDU Type
    Template of the PDU

    • (must be defined under Type Definitions|PDU Types)

  • Layout
    Length of the PDU type (fixed or dynamic)
  • Bit Count
    Length of the PDU type in bytes if Fixed Length was selected as the layout.
  • Comment

You can edit all properties of the PDU directly in the table and also create and delete PDUs. The PDU template defines the layout and number of bits and describes the signals.

Fields

If you have selected an existing service interface, the fields that this service interface describes will be listed here. The following properties will be displayed:

  • Name
    Name of the field
  • Data Type
    Data type of the field

    • (can be defined under Type Definitions|Data Types)

  • Get Function
    Here you specify whether the Get function is created automatically for the field or not
  • Set Function
    Here you specify whether the Set function is created automatically for the field or not
  • Notification Event
    Here you specify whether an event is created automatically or not

You can edit all properties directly in the table and also create and delete fields.

Use the splitter on the right-hand side to open the Data Type Properties of the selected object that you can edit directly there.
