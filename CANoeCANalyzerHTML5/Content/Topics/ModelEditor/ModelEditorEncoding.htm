Encodings

If you have selected the Encodings node in the tree view, all encodings with the following properties will be displayed:

  • Name
    Name of the encodings
  • Type
  • Linear: Offset, factor and unit
  • Symbolic: Value tables
  • Data Type
    Data type of the encodings
  • Namespace
    Namespace of the encodings
  • Comment

You can edit all properties (except the type) directly in the table and also delete encodings.

Symbolic/Linear Encodings

If you have selected an existing encoding, the properties that this encoding describes will be listed here:

Symbolic

  • Display Value
    The symbolic value
  • Symbolic Value
    he symbolic value
  • Lower Bound
    The lower bound of the raw or implementation value
  • Upper Bound
    The upper bound of the raw or implementation value

Linear

  • Unit
    Unit of the physical value
  • Factor/Offset
    Factor and offset for calculating the physical value
    [Physical value] = ( [Raw/implementation value] * [Factor] ) + [Offset]
  • Lower Bound
    The lower bound of the raw or implementation value
  • Upper Bound
    The upper bound of the raw or implementation value

You can edit all properties directly in the table and also create and delete encodings.

Raw Value, Implementation Value, Physical Value

The raw value of a signal is the value as it is transmitted in the network.

The physical value of a signal is the value of the physical quantity (e.g. speed, rpm, temperature) that represents the signal.

The implementation value is the value used for calculation in the model.

The encoding is used to transform the raw value to a physical value or in the reverse direction.
