Model Editor

 

In the Model Editor, you can define your own application layer objects and edit the existing ones.

You can open the Model Editor via the CANoe AUTOSAR Converter or the Data Sources.

Concept

Under Type Definitions, you create templates that you need to create new objects.

  • Create a PDU type as template for new PDUs. In the PDU type you can describe signals.
    Create a new PDU under Communication Objects|PDUs.
  • Create a service interface as template for new services. In the service interface you can describe events, methods, PDUs and fields.
    Create a new service under Communication Objects|Services.
  • Function types, data types and encodings are required accordingly when creating events, methods, fields and signals.

In the left sub-window you select what you want to edit. Depending on your selection, the corresponding contents will be displayed on the right-hand side. Depending on your selection here, you can use additional splitters to display and edit data type properties.

Validation messages and references for the data are displayed in the splitter at the bottom of the editor.
