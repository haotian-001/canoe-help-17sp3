CAPL Browser: Concept

 

Overview

Using the CAPL Browser, you can write CAPL programs (Communication Access Programming Language) for stimulating, simulating, testing and diagnostics.

Thus the problem of remaining bus subscribers not yet being available for tests arises when developing control units, for example. To map the system environment, the data traffic of all the other network subscribers can be simulated, for example, using CAPL.

You can also use CAPL to write programs for problem-oriented analysis of the data traffic or program a gateway to exchange data between different buses.

The well-laid-out program interface with various windows offers you the option of composing and compiling complex program sequences and easily locating possible programming errors in the program code.

CAPL programs are added in CANoe as CAPL program nodes. Event handlers serve as an input in CAPL, with which you can respond to external events (e.g. the arrival of certain messages). These language elements and the symbolic access to the various symbols defined in the database enable easy creation of CAPL program nodes.

Environment Information and Data

The CAPL Browser monitors the symbols, i.e. database symbols, diagnostic parameters, system and environment variables are updated automatically if changes are made. Configuration data of the current CANoe configuration, such as networks, channels, nodes etc., is not updated automatically. They can be read in with the import CANoe/CANalyzer environment command from a running CANoe application.

In order for this environment information and data to be available to a CAPL program, CAPL programs should always be opened from CANoe, either by double-clicking a CAPL program node or via the Edit context menu command or button.

CAPL Programs

The display of the opened CAPL programs is divided into a Navigator and a Text Editor.

  • The entire source code of the program is displayed in the Text Editor.
  • Using the Navigator, you can easily navigate to the desired sections in the source code (e.g. an event handler). New functions and Include files can also be inserted, however.

When files opened in the CAPL Browser are changed in another application, a dialog is opened to take over this changes.

Functions and Symbols

All CAPL functions and event handlers available for the opened nodes are displayed in the CAPL Function Explorer. The display is adapted to the environment configured in CANoe.

Event handlers and predefined functions can be inserted into the Text Editor from the Function Explorer via drag and drop.

All symbols and messages defined in the databases/function catalogs assigned in CANoe are displayed in the Symbol Explorer or MOST Symbol Explorer.

Symbols and messages can be inserted into the Text Editor from the Symbol Explorer or MOST Symbol Explorer via drag and drop.

Filters

The filters in the CAPL Browser control the visibility of the predefined functions, event handlers and symbols and affect the display in the Function Explorer, the autocomplete input assistance and Symbol Explorer/MOST Symbol Explorer.

The filters are adjusted automatically when CAPL programs are opened. The functions, event handlers and symbols for the currently loaded CAPL program are displayed. Others can be displayed at any time as needed using the Filter ribbon tab.

Help

The help for the CAPL Browser can be opened via the File ribbon tab or <F1> on the activated windows.

In the Detail view at the bottom edge of the CAPL Function Explorer, a short description of the selected category or its functions or event handlers is displayed. With the help information on the respective function is opened.
The syntax options are also displayed for the functions. Dragging and dropping from the Detail View is not possible.

In the Technical Reference, you can find a detailed description on syntax, functions, parameters, return values and availability of the functions and event handlers (event procedures) for each function.
Pressing <F1> on a function opens the help information on the respective function in the Text Editor, CAPL Function Explorer and Navigator.
