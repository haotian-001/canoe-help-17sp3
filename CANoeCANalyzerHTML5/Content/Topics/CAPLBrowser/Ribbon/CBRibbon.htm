Quick Access Toolbar

As default in the Quick Access Toolbar the following functions are available:

  • Open
    With a click on the symbol you open a dialog to select a previously saved file. The file is opened in a new tab. By entering the name of a non-existent file in the file selection dialog, you can create a new document with the entered name. If the CAPL Browser cannot display the program, you are asked whether you want to view and edit the program in a text editor.
  • Save
    With a click on the symbol you save the active file. If you save a file for the first time, you will be prompted to enter a filename in a dialog.
  • Undo
    With a click on the symbol you undo the last action. This is possible until another procedure is selected.
  • Redo
    With a click on the symbol you repeat the previously undone action. This is possible until another procedure is selected.
  • Navigate backwards
    With a click on the symbol you navigate in the Navigator of the Document View backwards.
  • Navigate forwards
    With a click on the symbol you navigate in the Navigator of the Document View forward.
  • Options

Note

You can use the context menu of the symbols and group titles in the ribbon to add each symbol or group to the toolbar.

You can remove symbols again using the context menu of the Quick Access Toolbar.

Context Menu of the Ribbon

Minimize the Ribbon
