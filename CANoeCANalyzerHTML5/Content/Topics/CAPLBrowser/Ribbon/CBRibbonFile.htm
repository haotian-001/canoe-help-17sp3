Symbol

Description

Save

With a click on the symbol you save the active file.

If you save a file for the first time, you will be prompted to enter a filename in a dialog.

Save As...

With a click on the symbol you save the file with a new name and/or in a new directory.

If a file with the specified name already exists, you will be asked if you want to overwrite it.

Save As Encrypted…

With a click on the symbol you save the file encrypted with a new name and/or in a new directory.

Save All

With a click on the symbol you save all opened files.

If you save files for the first time, you will be prompted to enter a filename in a dialog.

Open

With a click on the symbol you open a dialog to select a previously saved file. The file is opened in a new tab. By entering the name of a non-existent file in the file selection dialog, you can create a new document with the entered name. If the CAPL Browser cannot display the program, you are asked whether you want to view and edit the program in a text editor.

Last Used Files

List of recently used programs.

New

Here you select if you want to create a new CAPL file or CAPL include file.

CAPL Browser opens a new tab in which the CAPL file can be created. When saving or closing a new file, you are asked to provide a name for the file.

Close

With a click on the symbol you close the active file.

If you have modified the active CAPL program and have not saved it yet, you are asked whether you want to have the changes saved

Close All

With a click on the symbol you close all files.

Print

With a click on the symbol you open the Print dialog.

Select your printer settings and print the active file.

Export

Here you select if you want to export a CAPL file as HTML or RTF file.

The exported file contains no CAPL Browser structure information. It can only be displayed by an import into a CAPL Browser window. The command asks you to enter a new filename by a File Save dialog.

Encrypt CAPL Files

Here you select the files you want to encrypt.

Help

With a click on the symbol you open the help for the CAPL Browser.

Info

With a click on the symbol you open a dialog with information about CAPL Browser.

Options

In the Options dialog you can configure global settings.

With a click on the symbol you open the Options dialog.

Exit

With a click on the symbol you close the CAPL Browser.

If changes of the opened files have not been saved yet, you are asked whether you want to have the changes saved to a file.
