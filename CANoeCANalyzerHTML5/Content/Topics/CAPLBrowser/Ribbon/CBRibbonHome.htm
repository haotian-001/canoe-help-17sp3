Group Clipboard

Paste

With a click on the symbol you insert the contents of the clipboard at the active cursor position in the text

Cut

With a click on the symbol you cut the selected text and paste it to the clipboard. Click on Paste to add the text to the file again.

Copy

With a click on the symbol you copy the selected text to the clipboard. This does not change the file. Click on Paste to add the text to the file again.

Delete

With a click on the symbol you delete the selected text. The text is not copied to the clipboard.

Group Compile

Compile

With a click on the symbol you create an executable CBF file (CAPL Binary Format, filename extension CBF) from the active CAPL program. The name of the CBF file is composed of the name of the CAPL program file and the CBF extension. The CBF file is always saved in the same directory as the CAPL program file. If the CAPL program was not read from a file and has not been saved in a file yet, CAPL Browser prompts for the name of the CBF file in a File Save dialog.

Compile All

With a click on the symbol you create executable CBF files (CAPL Binary Format, filename extension CBF) from all loaded CAPL programs. The name of the CBF files is composed of the name of the CAPL program file and the CBF extension. The CBF files are always saved in the same directory as the CAPL program files. If a CAPL program was not read from a file and has not been saved in a file yet, CAPL Browser prompts for the name of the CBF file in a File Save dialog.

Group CANoe/CANalyzer

Import CANoe/CANalyzer Environment

The CAPL Browser monitors the symbols, i.e. database symbols, diagnostic parameters, system and environment variables are updated automatically if changes are made.

Configuration data of the current CANoe configuration, such as networks, channels, nodes etc., is not updated automatically. With a click on the symbol you update the data from the CANoe environment.

Group Navigation

Navigate Forward

With a click on the symbol you navigate in the Navigator of the Document View forward.

Navigate Backwards

With a click on the symbol you navigate in the Navigator of the Document View backwards.

Go to Line

With a click on the symbol you open the Go to Line dialog. Enter here a line number for the active file you want to navigate to.

Group Outlining

Expand All

With a click on the symbol you expand all sections of the active file.

Collapse All

With a click on the symbol you collapse all sections of the active file.

Mark All

With a click on the symbol you select the whole text of the active file.

Comment Out

With a click on the symbol you comment selected lines out.

Therefore you can select single lines in the Text Editor or single sections from the Navigator.

Uncomment

With a click on the symbol you remove the comment.

Show Generated Code

With a click on the symbol you open/close generated code in the Text Editor

Remove Old Browser Information
