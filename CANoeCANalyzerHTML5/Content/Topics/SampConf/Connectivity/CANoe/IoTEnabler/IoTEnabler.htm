IoT Enabler

 

Click on the blue link to load the sample configuration in CANoe.

  • IoT Enabler – IoT Enabler.cfg
    Shows the use of the Vector IoT Enabler network interface VH4110.

Introduction

This sample configuration requires a Vector IoT Enabler network interface connected via USB. The sample configuration shows how to execute control commands on the network interface.

Desktops

This configuration contains the following desktops:

Analysis

On this desktop, a sinusoidal input signal and the corresponding output signal of the IoT Enabler network interface are displayed in the Graphics Window. The DeviceControl panel visualizes whether the device is currently connected to CANoe.

Press the t key on the device, the current operating temperature of the IoT Enabler network interface is queried, the result is displayed in the Data Window and Trace Window.

Press the l key on the device, the radio interfaces available on the IoT Enabler network interface are queried, the result is displayed in the Trace Window.

Modeling

The necessary configuration settings in the Communication Setup are visible on this desktop. The interfaces of the IoT Enabler network interface are described in the vCDL file included as data source.

A CAPL program is included as an application model. This shows how you can call the interfaces of the IoT Enabler network interface from CAPL.
