MOST System Demo - Analysis

 

The following desktops support the analysis of data:

Desktops

Setup

  • Filtering messages by using MOST Application Filters in the Measurement Setup.
  • Use of Logging Block for recording the data flow.
  • Rendering and analysis of recorded data in offline mode.

System View

  • Display of the bus registry in the MOST Central Registry Window.
  • Display of the current status of the individual MOST channels in the MOST State Window.

Parameters

  • Display of statistical data in the Bus Statistics Window and the Graphics Window.
  • Display of parameter values in the Graphics Window and Data Window by directly selecting from the MOST XML function catalog (regardless of the transfer method, e.g. parameter values from arrays of record).

Trace All

  • Flexible configuration of the Trace Window, e.g. displayed columns or filter settings.
  • Color highlighting of certain bus events (MOST Event Highlighting page in the Trace Window configuration).

Stress

  • Display of statistical data in the Bus Statistics Window and the Graphics Window.

Audio

  • Display of allocation table in the MOST Audio Routing Window.
