  • High Protocol – MOST_HP.cfg
    Demonstrates the use of the MOST High Protocol in CANoe. (Online).

This sample configurations demonstrate the use of the MOST High Protocol in CANoe.

The services of the high protocol are provided via a modeling library which is integrated by the two CAPL nodes A and B to be able to operate as sender or receiver of a high protocol transfer. The API of the DLL provides a series of functions and callbacks for this purpose with which the transfer can be established and controlled. A complete documentation (MHP_NodeLayer_Doc.PDF) of the API, including a brief introduction of its use, can be found in the folder: Open Folder

In addition to the display of the disassembled messages of the high protocol, the Trace Window also displays events for the combined transfer units Block and Packet that are generated by the MOST High Protocol Observer built into CANoe. When transferring small data volumes (< 64 kb), the payload data in the Packet event can also be viewed disassembled. Therefore a function catalog has to be assigned to the configuration, which defines the meaning of the function address and its parameter structure. In case of dynamic arrays, the description for the respective function Id may be missing.

The observer also checks for every transmission whether the protocol-specification has been observed and generates additional error events if deviations, such as time-out violations or repeating counters, are observed.

MOST High Protocol - Application

CANoe .MOST - High Protocol Multi-Connection

This sample configuration demonstrates the use of several high protocol connections running simultaneously in a single CAPL node. This is achieved using the same modeling library as in the standard high protocol sample configuration. The sample configuration is intended as a supplement, focusing entirely on the aspect of managing multiple connections and ignoring all the other features and configuration options that are already demonstrated in the standard high protocol sample configuration.
