System Configuration (A429)

 

Note

Click on the blue link to load the sample configuration in CANoe.

System Configuration (A429) - A429System.cfg

This configuration shows some of the basic functions of the option A429. ARINC words are generated via panels and IG and displayed in different configured Trace Windows.

Overview

This sample configuration shows some of the CANoe related features. It is suitable for online mode as well as offline mode. Please make sure that no devices are connected to the Vector VN0601 in online mode with Real Bus.

This sample configuration simulates a Fuel Control Management System (FCMS).

Desktops

This configuration contains several desktops.

FCMS Panels

In Simulated mode the FCMS can be operated by the panel Overhead Fuel Panel on this desktop. The Fuel Panel simulation is connected to channel 3. On channel 1 and channel 2 simulated tank sensors generate messages by means of an Interactive Generator and a Replay Block. The FCMS receives these sensor data and calculates the current system state. The results are sent via channel 4 and displayed in the Fuel Computer panel. The current aircraft gross weight and pump statuses are shown in this panel too.

Analysis

In this desktop several parameter values are shown in a Graphics Window and in a Data Window.

FCMS Trace

This desktop displays the current bus traffic in three Trace Windows with different window layouts. For separating input messages and output messages a Channel Filter as well as an Event Filter is used. The third Trace Window shows messages with Low Fuel signals only. These messages are sent from the FCMS and they are separated with a Trigger Block.

Configuration

This desktop contains the Bus Statistics Window, Simulation Setup and Measurement Setup. Minimum one Main Pump(1..4) and minimum one fuel Tank (Center Left and Right, Trim, RCT) need to be engaged to simulate fuel burn. In offline mode, a recorded simulation of the system is replayed.

How to start?

When you start the measurement with the key <F9> the buttons in the Overhead Fuel Panel are available. Just switch on the engines (Main 1..4) and activate the tanks. As soon as fuel is consumed the gross weight is decreasing. When a main tank is empty it is closed and you may select a standby tank.
