VH5110 Basic (CCS Listener)

 

Click on the blue link to load the sample configuration in CANoe.

  • VH5110 Basic (CCS Listener) – VH5110Basic.cfg
    Demonstrates how the VH5110-CCS-Listener can be used to listen in on the communication between an electric vehicle and a charging station.

Introduction

This configuration uses the CANoe Option SmartCharging to display the communication in the Trace Window in a readable way.

Hardware Requirements

A VH5110-CCS-Listener is required for the execution. In the Hardware ribbon tab, you may need to map the application channel to the hardware using Channel Mapping.

Desktops

This configuration contains the following desktops:

VH5110Basic

VH5110 Control Panel

The VH5110 Control panel displays the last read values for Frequency, Duty Cycle and Voltage on the CP line, as well as the current configuration of the VH5110. In particular, you can see whether the measurement method is operating in Poll mode or Push mode.

Poll mode

The three values are explicitly queried. Either individually, via the Get Status, or cyclically via the toggle switch with the interval specified to the right of it (Cyclic [ms]).

Push mode

The three values are automatically sent to the sample configuration according to the configuration stored in the VH5110 as soon as they change within the specified range.

You can change the configuration parameters in the input boxes and write them to the VH5110 using the Set Conf button.

Note

The change remains saved in the VH5110 even after the sample configuration is terminated! The same applies to the last measurement method set.

The Graphics Window displays the three values for Frequency, Duty Cycle and Voltage. Scaled values are used for the display.

In the State Window the PWM signal is displayed as an equivalent current quantity and the CP state is displayed symbolically.

Protocol Panel

The Protocol panel allows you to follow the communication sequence of the DIN70121/ISO15118 protocol. As messages are received, the current protocol status is highlighted in the sequence view. For simplicity, not every displayed status corresponds to an individual message.

Configuration

This desktop contains the Measurement Setup, the Simulation Setup and the Write Window.
