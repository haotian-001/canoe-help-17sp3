GNSS Models

 

Click on the blue link to load the sample configuration in CANoe.

  • GNSS Models – GNSSModels.cfg
    Transmitting of position data along different models.

With this sample configuration you can define a path which is replayed by a simulated GNSS receiver. You can use the three models Line, Rectangle and Meander to build this path.

Introduction

After measurement start the desktop Simulation is activated. Then you can set the values for the different models and add them to the path with [Add ...]. After pressing [Start] the simulated GNSS receiver rides along the defined path. The GNSS Monitor records the path by using the received position data.

You can change the speed of the GNSS receiver with the slider or the input field in the Speed area.

With [Remove Waypoints] you can delete the defined path.

Desktops

This configuration contains the following desktops:

Setup

Desktop Setup contains the Simulation Setup with a node simulating the GNSS receiver, and the Measurement Setup with the GNSS Monitor, a Trace Window and a Data Window. Furthermore a Write Window is contained.

Simulation

Desktop Simulation contains a panel for the GNSS receiver, the GNSS Monitor, and a Trace Window.
