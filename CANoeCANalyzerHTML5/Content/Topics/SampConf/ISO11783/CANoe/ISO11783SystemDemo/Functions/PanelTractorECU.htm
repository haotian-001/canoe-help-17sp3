Tractor ECU

 

Panel with a Class 2 Tractor ECU with front mounted implement support and navigational message support.

On measurement start the key switch is enabled by default and the engine is running. The LEDs PWR and ECU_PWR show the state of the power lines at the implement connector.

If the gear is in neutral mode, you can set the engine speed with the slider. The [D] button enables the drive mode. In this mode the slider sets the speed of the tractor. The [Brake] button decrease the speed and enables the stop lights.

The Up and Down buttons control the position of the hitch. The button below engages the PTO.

The buttons in the Lights section enables the turn lights, marker and rear work lights.

With the button [Valves] you can open a new panel, which controls the state of the auxiliary valves.

With the [Schematic] button you can open a new panel, in which a schematic presentation of the tractor is shown.

With the panel Tractor Diagnostics you can simulate error states of the Tractor Engine Speed by using the slider. In neutral position no error state is reported. If you move the slider to the left, the value will be to small. If you move it to the right, the value will be to big. Both states signal error states that are sent cyclically (1 s) as DM1.
