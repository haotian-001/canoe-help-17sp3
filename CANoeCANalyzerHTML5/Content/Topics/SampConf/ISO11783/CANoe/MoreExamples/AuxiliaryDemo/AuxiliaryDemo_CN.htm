Auxiliary

 

Click on the blue link to load the sample configuration in CANoe.

  • Auxiliary – ISO11783AuxiliaryDemo.cfg
    Use of different auxiliary inputs.

This sample configuration demonstrates the usage of different auxiliary inputs in CANoe. Via the Virtual Terminal the functions of a Working Set Master are assigned to the controls of the auxiliary inputs.

All nodes are using the ISO11783 Interaction Layer. The configuration of the nodes is done in the database and therefore no CAPL program is necessary. The controls in the panels are connected to the Object Pool via system variables. The system variables are created by the interaction layer. The start positions of the controls are defined by the start values of the corresponding system variables.

Introduction

After measurement start the desktop Simulation is activated and the Working Set Master sends its object pool to the Virtual Terminal. The assignments of the Working Set Master functions to the controls of the auxiliary inputs are sent from the Working Set Master to the Virtual Terminal (Preferred Assignment). The Virtual Terminal then publishes them.

If an assigned control of an auxiliary input is actuated the value change is displayed in the Virtual Terminal (Data Mask page).

To change the assignments switch to page Auxiliary Function of the Virtual Terminal. On the left you can see the functions of the Working Set Master, on the right you can choose an appropriate control with the left mouse button.
