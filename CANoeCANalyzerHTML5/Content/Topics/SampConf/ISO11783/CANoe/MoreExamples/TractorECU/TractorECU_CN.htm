Tractor ECU Class 2

 

Click on the blue link to load the sample configuration in CANoe.

  • Tractor ECU Class 2 – ISO11783TractorECU.cfg
    Simulation of a tractor ECU of class 2.

This sample configuration contains a Tractor ECU of Class 2 with front mounted implement support and navigational message support. (See also Tractor ECU).

Introduction

After measurement start the desktop Panels is activated and the tractor is started. If button [D] in the tractor panel is pressed then the tractor starts moving.

With the keys <d>, <l> and <t> you can send requests for parameter groups TD (PGN 65254), LIC (PGN 65089) and TFR (PGN 65033) by using the Interactive Generator.

Cross Reference

Further information on operating the Tractor ECU can be found here.

Desktops

This configuration contains the following desktops:

Setup

Desktop Setup contains the Simulation Setup with a node for the tractor ECU, a node for the implement, an Interactive Generator, and a GPS node that transmits the positions of the tractor. Furthermore, it contains the Measurement Setup and a Write Window.

Measurement

Desktop Measurement contains a Trace Window that displays all messages as well as a data and Graphics Window that show some important signals of the tractor ECU.

Panels

Desktop Panels contains a panel for the tractor ECU and a panel for the implement that displays the received signals of the tractor. In the GNSS Monitor the position data are shown and in the data and Graphics Window also some signals of the tractor are shown.
