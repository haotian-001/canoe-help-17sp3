How to Use

After the measurement is started, the test module can be started manually. CAN and CAN FD messages will be transmitted. They will be recorded and analyzed by the Scope.

A CAPL test library contains the definition of the bitmasks. The library is also controlling the test steps, and will generate a test report with the analysis results.

With the TestControl panel you can limit the data bytes to be analyzed.

The complete test contains the following steps:

  • Test group CAN: Transmit and analyze a CAN message
  • Test group CAN FD: Transmit and analyze a CAN FD message
  • Test group Provoke Errors: Transmit and analyze a CAN message. Bitmask violations will be notified due to a bitmask which is too large.
  • Test group Other Tests: Transmit a CAN FD message and read out the bit parameters (bit length, voltage levels, logical value). The result is written in a ASC logging file which is linked to the test report.
