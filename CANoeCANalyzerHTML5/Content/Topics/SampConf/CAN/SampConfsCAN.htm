Sample Configurations CAN

 

Note

Click on the blue link to load the sample configuration in CANoe.

Note

If there is no description for the sample configuration in the help, you find a description in a TXT file next to the sample configuration in the same folder.

After you have loaded the sample configuration in CANoe you can show this TXT file via the Environment ribbon tab|Documents.

Testing with CANoe - Test Feature Set

Central Locking System – CentralLockingSystem.cfg
Test a vehicle's central locking system and the power window in vTESTstudio. All available test design editors are used (Test Table Editor, CAPL Editor, C# Editor, Test Diagram Editor and State Diagram Editor).

View Description

Seat Test – SeatTest.cfg
Demonstrates how the functionality of a seat and the communication of a seat ECU can be tested. The test is realized with test tables and CAPL test functions in vTESTstudio.

Diagnostics

CAN Diagnostics

The following four sample configurations demonstrate the use of Diagnostics over CAN in CANoe for typical use cases.

 

UDS Diagnostic Basic – UDSBasic.cfg
UDS based simple ECU simulation and diagnostic test.

View Description

UDS Diagnostic System Configuration – UDSSystem.cfg
UDS based ECU simulation demonstrating the interactive diagnostic windows (Diagnostic Console, Fault Memory Window and Diagnostic Session Control) as well as analysis and test features for diagnostics.
