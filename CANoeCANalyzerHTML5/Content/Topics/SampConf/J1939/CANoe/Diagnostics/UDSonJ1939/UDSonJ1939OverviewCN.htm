UDS

 

Click on the blue link to load the sample configuration in CANoe.

  • UDS – UDS_on_J1939.cfg
    Simulation of a J1939 engine control unit with UDS support.

The sample configuration shows how CANoe supports diagnostics with UDS (Unified Diagnostic Services) using a simulated J1939 engine management system as an example.

Introduction

The engine is started automatically when the measurement starts. You can increase the speed and set fault conditions on the EMS panel. You can query diagnostic-specific data using the Diagnostic Console and the Fault Memory Window.

Desktops

This configuration contains the following desktops:

Setup

Measurement

Diagnostics

Setup [▲ back]

This desktop displays the Simulation Setup and Measurement Setup and a Write Window.

The Simulation Setup contains the simulated engine management system. The Measurement Setup contains a J1939 scanner, a Graphics Window, and two Trace Windows. The Trace Window is configured with the predefined filters of the Trace Explorer.

Measurement [▲ back]

This desktop displays various windows for the network analysis.

EMS Panel

You control the engine with this panel.

You use the slider to influence the position of the gas pedal. In Auto simulation mode, you use this to change the speed, torque, and temperature. If you click the [Auto] button, a switchover to Manual mode occurs in which you can change the speed, torque, and temperature separately.

To switch off the engine, click the Ignition Switch. To restart the engine, double-click with the right mouse button.

To generate faults in the fault memory, click one of the two [OK] buttons in the Faults area.

Trace Window

The Trace Window shows all messages of the two buses.

Write Window

Information of the engine management system is output in the Write Window.

J1939 Scanner

The J1939 scanner displays the devices active on the bus. In addition, you can request Parameter Group messages using a diagnostic request.

Graphics Window

The Graphics Window displays important values of the simulation.

Diagnostics [▲ back]

This desktop displays the windows for the diagnostics.

EMS panel

You control the engine with this panel. A detailed description of the panel can be found here.

Trace Window UDS

The predefined filters of the Trace Explorer are set in such a way that only diagnostic messages (ISO TP) and their interpretation are displayed here.

EMS Diagnostic Console

This window is based on the diagnostic description for the ECU defined on the J1939 property page in the Diagnostics/ISO TP Configuration dialog (Diagnostics & XCP ribbon tab|Diagnostics/ISO TP).

The services defined in the file UDS_On_J1939_Ecu.cdd are displayed in the Diagnostic Console. After execution of a diagnostic request, the result can be viewed in the Trace area.

EMS Fault Memory Window

This window is also based on the above-indicated diagnostic description. If a fault is stimulated on the EMS panel, it can be made visible in the Fault Memory Window after clicking the button [Update fault memory list].
