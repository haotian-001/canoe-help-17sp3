Template Configuration J2735 2016

 

Note

Click on the blue link to open the directory of the sample configuration for copy procedure.

Template Configuration J2735 2016 - Open Folder

Note: This configuration is intended to be used as template for new configurations. To do this, copy the complete folder V2xTemplateJ2735-2016 to the desired directory.

Template configuration with a Car2x database and extended data structures for typical SAE J2735 2016 application messages to provide comfortable handling with these application messages.

The following application messages are included:

  • BasicSafetyMessage
  • MAP
  • SPAT
  • CommonSafetyRequest
  • EmergencyVehicleAlert
  • IntersectionCollision
  • NMEA_Corrections
  • ProbeDataManagement
  • ProbeVehicleData
  • RoadSideAlert
  • RTCM_Corrections
  • SignalRequestMsg
  • SignalStatusMessage
  • TravelerInformation
  • PersonalSafetyMessage

Additionally, the database contains the WAVE Service Advertisement (WSA) from IEEE 1609.3.
