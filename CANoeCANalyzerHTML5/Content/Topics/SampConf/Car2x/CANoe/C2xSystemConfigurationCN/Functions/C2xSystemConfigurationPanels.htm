Panels

 

With panels you have simple, clear display and user interfaces available in order to influence the simulation model and analyze measurements.

The following panels are available:

  • Traffic Light: This panel displays the current state of the traffic light. The time shows when the traffic light switches to the next state.
  • Dashboard SimCar1: This panel displays internal information as well as data of the »RSU that have been send with WLAN messages. Vehicles that are located in the communication range of SimCar1 are listed in the panel as well as the calculated distance to SimCar1.
  • RWW_Configuration: This panel configures the »DENM of the roadworks warning to be sent. Values that are changed in this panel (e.g. the Relevance Area), are sent with the next DENM.
  • SimCar EEBL and SimCar EVA to activate/ deactivate Electronic Emergency Brake Light and Emergency Vehicle Approaching DENMs.
  • Configuration Device Under Test to configure the Device under Test

The panels use system variable that are set or read in the simulated nodes.
