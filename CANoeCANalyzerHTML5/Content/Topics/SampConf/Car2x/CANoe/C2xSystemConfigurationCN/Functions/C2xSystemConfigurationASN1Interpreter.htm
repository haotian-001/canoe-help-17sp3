ASN.1 Interpreter

 

The »ASN.1 Interpreter allows the access to the content of application messages.

The »ITS stations transmit their application signals in application messages, that are contained as payload in the WLAN packets. The application messages are »PER coded referring to the included ASN.1 descriptions.

The sample configuration shows the functionality of the ASN.1 Interpreter:

  • The content of the application messages are decoded and displayed in interpreted form in the Detail view of the Trace Window. They are provided as single signals too for displaying them in the Data and Graphics Window.
  • Simulated nodes in send direction: Signals of the application messages are set and get with functions of the Car2x Interaction Layer. With C2xCompletePacket the application messages are coded with PER.
  • Simulated nodes in receive direction: Application messages in the payload of the received WLAN packets are decoded and interpreted so that the signals can be read with the functions of the Car2x Interaction Layer

The ASN.1 Interpreter gets information about content and coding (in this case PER) of the application messages from the Car2x database.
