Template Configuration CSAE0053 (CV2x)

 

Note

Click on the blue link to open the directory of the sample configuration for copy procedure.

Template Configuration CSAE0053 (CV2x) - CV2xTemplate-CSAE0053.cfg

Note: CV2x stands for Cellular Vehicle to X. This configuration is intended to be used as template for new configurations. To do this, copy the complete folder CV2xTemplate-CSAE0053 with the configuration file CV2xTemplate-CSAE0053.cfg to the desired directory.

Template configuration with a Car2x database and extended data structures for typical C/SAE Car2x application messages to provide comfortable handling with these application messages.

The sample configuration allows comfortable working with different C/SAE Car2x application messages. A database is already assigned to the configuration for this purpose.

The following application messages are included:

  • Basic Safety Message (BSM)
  • Map Data (MAP)
  • Road Side Information (RSI)
  • Signal Phase and Time (SPAT)
  • Roadside Safety Message (RSM)
  • Road Traffic Signs (RTS)

ReadMe

Start desktop when loading the configuration. Contains a short description of the template.

Setup

Contains the simulation and the measurement setup. The database CSAE0053_MsgSet is included in the Simulation Setup. It contains the ASNV modules for the Chinese application messages. In addition, a network node BasicNode is included in the Simulation Setup. This already includes the generated data structures of the application messages for a more comfortable working with the messages, furthermore the callbacks for sending and receiving are already prepared.

Measurement

Contains a Trace Window to display incoming and outgoing messages.
