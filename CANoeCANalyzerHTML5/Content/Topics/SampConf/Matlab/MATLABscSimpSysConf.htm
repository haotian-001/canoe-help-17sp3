MATLAB Integration: Simple System

 

This sample configuration is only available after you have installed the MATLAB Integration Package: Open Setup Folder

See further: MATLAB Integration and MATLAB Integration Setup Introduction

Click on the blue link to load the sample configuration in CANoe.

Simple System – MATLAB SimpleSystemDemo.cfg
The generated code of two Simulink models has been added to the Simulation Setup. (MATLAB Integration Package installation required).

This configuration is derived from the CAN System Demo configuration that is supplied with CANoe. In addition to the original configuration the generated code of two Simulink models has been added to the Simulation Setup.

The first model – engine.MDL – implements a simple vehicle speed control system. Vehicle and engine speed are the input variables and the speed is controlled as a function of the pedal position and gear. You will find this model in: Open Folder
...\samples\SimpleSystemDemo\Models\engine\engine.MDL

Open the model in MATLAB. The model contains five CANoe specific blocks:

  • Simulation step
  • Actual speed
  • Engine speed
  • Pedal position
  • Gear

The first block Simulation step is only relevant to the simulation in offline mode and synchronous mode. The other blocks are respectively two input blocks System Variable Input and two output blocks System Variable Ouput. They read and write directly to the corresponding CANoe system variables. Double click on the Environment blocks to see the actual controlled variable names.

The second model – door_ri.MDL – lowers the window of the right door automatically to the lowest position when the user presses the window down button longer than a specified time. You will find the model in: Open Folder
...\SimpleSystemDemo\Models\door_ri\door_ri.MDL

Generate code for the models as described in the Signal Access example above. For your convenience you will find all prebuilt DLLs in: Open Folder
...\SimpleSystemDemo\Exec32

After you have generated code …

  • Start CANoe and open the configuration SimpleSystemDemo.cfg (see blue link on top).
  • Compile all nodes via Environment ribbon tab|Compile All Nodes
  • Open the Simulation Setup (Simulation ribbon tab) to verify that CANoe has detected the two DLLs.
  • Start the measurement to view the models in action.
  • Open the Engine desktop and try the [Brake] button. Changes are shown immediately in the CANoe Graphics Window.
  • Open the Door desktop and play with the window up and window down buttons of the doors and watch the effects.
