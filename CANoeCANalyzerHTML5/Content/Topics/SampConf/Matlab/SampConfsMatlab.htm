Sample Configurations MATLAB/Simulink

 

Setup of the MATLAB Integration Package

The Setup of the MATLAB Integration Package includes several sample configurations, that are available (working) in some cases only after installation of the Package: Open Setup Folder

Storage Locations of the Sample Configurations

The sample configurations delivered with CANoe can be found in the following folder: Open Folder

The additional sample configurations provided with the MATLAB Integration Package Setup are located in the following folder. The specific name (X.X) depends on the MATLAB for which the Package is installed:

SAMPLESDIR = %PUBLIC%\Documents\Vector\Matlab Integration\CANoe Matlab Integration Package for Matlab X.X\CANoe Sample Configurations\Matlab

You can also get an overview of the sample configurations on the File ribbon tab clicking the Sample Configurations entry. There are all the configurations listed in the group MATLAB/Simulink, with their absolute Paths.

Note

Click on the blue link to load the sample configuration in CANoe.
