MATLAB Integration: CAPL Handler

 

This sample configuration is only available after you have installed the MATLAB Integration Package: Open Setup Folder

See further: MATLAB Integration and MATLAB Integration Setup Introduction

Click on the blue link to load the sample configuration in CANoe.

CAPL Handler – MATLAB CaplHdlrWParams.cfg
Shows how a CAPL function can be called from a Simulink model with this block. (MATLAB Integration Package installation required).

The sample configuration CAPL Handler illustrates the use of the CAPL Handler block in Simulink. A CAPL function can be called from a Simulink model with this block.

  • Open the CANoe sample configuration CAPLHdlrWParams.cfg (see blue link on the top).
  • Start the measurement in CANoe.

The Graphics Window shows a ramp that is generated via a Signal Generator. This ramp is transferred, together with a threshold value that can be set via the slider, to the Simulink model. In the Simulink model, a counter is implemented that only increments its value when the current value of the ramp matches the threshold value. The current counter value is transferred back to CANoe as a parameter of a CAPL function.
