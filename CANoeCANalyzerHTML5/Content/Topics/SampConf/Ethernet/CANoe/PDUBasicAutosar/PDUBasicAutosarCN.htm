Basic AUTOSAR I-PDU

 

Click on the blue link to load the sample configuration in CANoe.

  • Basic AUTOSAR I-PDU – PDUBasicAutosar.cfg
    Shows how the AUTOSAR Ethernet interaction layer (CANoeILNL_AUTOSAR_Eth.vmodule) is used to send and receive an I-PDU.

In the ARXML, a PDU is configured that is sent cyclically from the VGW node to the IC node. The simulated nodes use two interaction layer VMODULE files:

  • AsrPDUIL.vmodule
    This is responsible for the sending behavior, e.g. cyclic sending.
  • CANoeILNL_AUTOSAR_Eth.vmodule
    This IL transmits the PDUs and performs the service discovery.

Both interaction layers are configured from the ARXML.

Introduction

At the start of measurement, the service offered by the IC node is subscribed to, and cyclic transmission of the PDU EMotorStatus starts.

Using the panels, you can change the signal value or activate Signal Generators.

Desktops

This configuration contains the following desktops:

Setup

Simulation

Trace

 

Setup [▲ back]

The Simulation Setup contains the two simulated nodes VGW and IC. Both use a simple CAPL program to make it possible to activate the service via the panels. The main work is handled by the interaction layers, which are assigned via the Node Configuration dialog on the Components page.

In the Start Values Window, the system variables for activating the service are set in such a way that the service is subscribed to immediately at the start of measurement.

The Measurement Setup contains a CAPL program that monitors the transmitted signal values. In addition, an on pdu EMotorStatus handler is implemented, which outputs a message on the PDUObserver panel (Desktop Simulation) if a limit value is exceeded.

Simulation [▲ back]

At the start of measurement, a changeover to this desktop occurs. The transmitted signals are displayed here in the Graphics Window and Data Window. If the Signal Generators are activate, a sine curve is output to the corresponding signal.

Trace [▲ back]

The Trace Window shows the cyclic PDU and the service discovery frames.
