Ethernet – EthernetChannelBased.cfg
Shows the channel-based access to the Ethernet network.

Open Folder

Global Event Handler – CANoeGlobalEventHandler.cfg
Shows the introduction function cclOnDllLoad, which must be implemented in every CANoe C library.

In this DLL file additional event routines are registered for unloading the DLL as well as start and stop of a CANoe measurement.

Open Folder

LIN – CANoeLIN.cfg
Shows the access to the LIN bus with help of a LIN Master node.

Various event routines in the DLL file demonstrate how to access LIN frames before they are sent, how to react to value changes of system variables and how to start and stop the scheduler, send individual LIN frames or put the LIN network to sleep (go-to sleep/wake-up).

Open Folder

Service Discovery with Communication Objects – CANoeServiceDiscovery.cfg
Shows the dynamic connection setup with service endpoints generated during measurement using abstract bindings.
