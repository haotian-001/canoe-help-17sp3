Note

Click on the blue link to load the sample configuration in CANoe.

Note

If there is no description for the sample configuration in the help, you find a description in a TXT file next to the sample configuration in the same folder.

After you have loaded the sample configuration in CANoe you can show this TXT file via the Environment ribbon tab|Documents.
