Control Plugin API

 

Control Plugin API – Open Folder
Implement your own controls in Panel Designer.

With help of the Control Plugin API you can use your own controls for creating panels of the Panel Designer. You can use these controls in the same way as built-in controls of the Panel Designer. You can display and modify values of numeric symbols in CANoe.

To do so, you must create a .NET assembly:

  • The Vector.PanelControlPlugin.dll component must be referenced in the assembly.

Note

Reference the newest version, which is located in the \Exec32\Components\Vector.PanelControlPlugin directory. Do not use the DLL that is found in the Programing sample configurations. Ensure that Copy Local is set to False in the project settings.
  • The IPanelControlLibrary and IPanelControlPlugin interfaces are defined in the Vector.PanelControlPlugin namespace. Create two classes that implement these interfaces.

Note

For details on the interfaces, refer to:
AN-IND-1-017_PanelControlPlugIn.pdf
  • Build the assembly.
  • Copy the created DLL to the directory Exec32\ControlLibraries.

After restart of CANoe the Panel Designer will contain a new Toolbox-Library containing the created control(s).
