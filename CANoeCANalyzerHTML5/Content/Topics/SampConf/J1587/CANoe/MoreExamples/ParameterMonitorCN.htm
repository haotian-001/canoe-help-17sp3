Parameter Monitor

 

Click on the blue link to load the sample configuration in CANoe.

  • Parameter Monitor – J1587ParameterMonitor.cfg
    Functional demonstration of the Parameter Monitor.

This example demonstrates the Parameter Monitor to display and symbolically interpret J1587 parameter and signal values.

Desktops

This configuration contains the following desktops:

Setup

The Simulation Setup contains three CAPL nodes, which - triggered by panels - send parameters while the measurement is running.

Parameter Monitor

This desktop (automatically activated at the start of measurement) shows three panels, each belonging to a CAPL node, on which the user can set and send parameters. The Parameter Monitor on the right side shows the interpretation of their contents appended with additional information (if assigned through the CAN database).
