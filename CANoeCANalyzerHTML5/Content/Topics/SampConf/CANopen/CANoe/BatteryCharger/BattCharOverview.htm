Battery Charger

 

Click on the blue link to load the sample configuration in CANoe.

This sample configuration is not part of the standard installation of the option CANopen! You can download the appropriate add-on from the download center of the Vector homepage that contains the sample configuration.

  • Battery Charger – battery_charger.cfg
    Simulation of a battery charger (DS-418/419).

The sample configuration simulates a much simplified battery charger according to CiA418/419. The simulation was created by the simulation generator and then extended by the application code.

This sample configuration should show that it is quite simple to extend the general CAPL code by special application code. It does not matter how the relevant data was transmitted (in which PDOs) in this case.

Introduction

The battery charge procedure is started immediately after the start of the measurement. The current charging status is displayed in the Graphics Window.
