RS232

 

Click on the blue link to load the sample configuration in CANoe.

  • RS232 – RS232.cfg
    Demonstrates the access to the RS232 serial port.

Note

If there are less RS232 ports as needed, you can connect USB-to-RS232 adapters. With these adapters you can use USB ports like serial RS232 ports.

The communication of the serial interface can be tested in three the different scenarios.

For these scenarios the configuration with two panels can be used. Every panel demonstrates a serial port on your computer. The input field Data Transmission and the display field Data Reception are used to insert and display data in hexadecimal or text format.

To send data use the Send Data button. When receiving data the gray point on the bottom right corner is lightning green.

To clean the input of both display fields please use the Clear Data button.

The computer has two serial ports

If the computer has two serial ports, please connect both using a null modem cable. Use the panels Port A and Port B to configure the ports. Default values are already inserted.

The data written to the input field of panel Port A is displayed in the display field of panel Port B after sending.

The computer has only one serial port

If the computer has only one serial port an additional computer is needed to test the communication with serial ports.

Start the same CANoe configuration on the additional computer or use a terminal program. Connect both computers with a null modem cable.

Working with two CANoe installations

Use on every computer one of the panels to configure the used serial port.

Working with CANoe and a terminal program

If you are using a terminal program only one panel has to be configured as port. The port configuration of the terminal program is done in the terminal program itself.

The written and send data on one computer are displayed in the display field on the other computer.

Note

Windows delivers a terminal program that can be found via the Start menu:

Start|Programs|Accessories|Communication|HyperTerminal

In Windows RS232 ports are listed as COM ports.

Using an external device

Please use an external device with a serial port for the second serial port, e.g. bar code scanner or any measurement device.

Use on the computer one panel to configure the serial port of CANoe. Send data to the external device and receive data respectively using the input and display fields of the panel.
