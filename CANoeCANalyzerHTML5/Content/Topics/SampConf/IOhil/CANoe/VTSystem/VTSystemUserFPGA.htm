VT System Basic User FPGA

 

Click on the blue link to load the sample configuration in CANoe.

  • VT System Basic User FPGA – VTSBasicUserFPGA.cfg
    Shows how the User FPGA for works for different VT System modules.

The following sample configuration shows how the User FPGA works for various VT System modules. Each module has its own desktop based on its specific hardware possibilities. Input channels and output channels are used, which are available on the individual modules.

Cross Reference

  • You will find further information about the User FPGA on this help page User FPGA (VT System) and in the following PDF:
    VT System FPGA Manager - User Programmable FPGA
  • You will find general information about the VT System on the corresponding help pages for the VT System or in the following PDF:
    VT System Manual
  • You will find an overview of all VT System modules (not just those with User FPGA extension) on this page Modules and Extensions (VT System).

Configuration

  • To set up the sample configuration, you need a supported module with User FPGA.
  • The sample configuration is automatically copied to the workbench when the manager is set up initially and can be restored using the menu path Tools|Option and the Restore Demo Projects button.
  • The code is precompiled and can be written directly to the FPGA or to the EEPROM of the module.
  • You must select connection to the VT System module in CANoe:
  • On the Hardware ribbon tab click on VT System and then on Configuration.
  • Click on the Hardware command.
  • In the Network adapter list select the network card that the VT System is connected to.
  • On the desktop of the corresponding module you can see how the module must be connected.

Introduction

  • Connect the appropriate hardware to the module as shown on the corresponding desktop.
  • Start the measurement. Modules that are not connected are set to offline mode.
  • The possibilities of the respective module are explained on the corresponding desktop.

Desktops

This configuration contains the following desktops:

  • VT1004
  • VT2004
  • VT2516
  • VT2816
  • VT2848

These desktops show the supported modules, as well as the corresponding panels and required wiring.

CAPL

The CAPL desktop shows the CAPL files used.

  • CalculateFrequency
    Calculates the frequency of the PWM signal.
  • SetThreshold
    Sets the threshold value for modules that support such a system variable.
  • StartSettings
    Sets the start values of the individual modules (e.g. output settings).
