VT System Online - Start

 

Once you have set up the hardware for the test setup, connected it to the computer and supplied it with power, you need to set up the sample configuration.

Configuring the VT System in CANoe

The first step is to configure the VT System in CANoe. Doing so checks whether CANoe recognizes the VT System correctly and whether the channels of the VT System are assigned correctly to the internal values of the sample configuration.

This is how you do it:

Open Options dialog (Extensions|VT System). Activate VT System active and make sure that the network card that is connected to the VT System is selected.

To configure the VT System, open VT System Configuration dialog. Then click on Adapt to connected HW… to search for VT modules automatically. After this you need to make sure that correct namespaces are assigned to the connected modules. The assignments are as follows:

  • VT1004 (if available) — Load
  • VT2004 (if available) — Stimulus
  • VT7001 (if available) — Power
  • VT2516 (if available) — Digital
  • VT6104 (if available) — Network

Deactivate Unneeded Nodes

In order to compile this sample configuration, you need to deactivate all unneeded network nodes from the Simulation Setup.

To do this, right-click the nodes of any VT modules that are not in the VT Rack. Now deactivate the node by choosing Block Active from its context menu. If, for instance, your VT Rack contains only a VT1004 and a VT2004 module, you need to deactivate the VT2516, VT6104 and VT7001 nodes.

Deactivate Unneeded Test Cases

As in the previous step, you now need to deactivate all test cases that use VT modules which are not connected to the computer. To do this, deactivate the corresponding test cases in the Test Setup.

If, for instance, you do not have a VT2516 module, you need to deactivate test cases CAPL: VT2004_Ch1 to VT2516_Ch1 and XML: 2516_Ch1 to VT1004_Ch1.

Compiling the Nodes and Test Cases

Now you should be able to compile all nodes and test cases in your configuration. In case there are still unfound variable names make sure you followed the steps above carefully.

Introduction

Once you have set up the hardware and the configuration as described above, you can start it by clicking Start on the CANoe Home ribbon tab.
