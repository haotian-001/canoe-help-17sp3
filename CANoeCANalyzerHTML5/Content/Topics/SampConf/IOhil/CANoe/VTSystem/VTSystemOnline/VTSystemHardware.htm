VT System Online - VT System Hardware

 

Introduction

You need at least one of the supported VT modules in order to use this sample configuration. These are the VT1004, VT2004, VT2516 and VT7001.

To exploit the full functionality of this sample configuration, you need one of each of the modules. However, only one card of each type is supported.

To use the VT modules with this configuration, you need to supply the VT Rack with power and connect it to the computer.

Connecting the Modules

To use and check the functions of the individual VT modules, you need to connect them with each other. When doing so, it makes sense to connect modules that generate signals with modules that measure signals. To obtain reliable results, it is important that you use the connectors on the rear of the modules.

A few examples of such connections and the tests you can run with them are provided in the Sample Test Setups page.

If you only have a single VT System module, you can still connect it to a multimeter or a power supply unit. Alternately you can connect individual module channels with each other.
