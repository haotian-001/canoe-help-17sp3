VT System - Online

 

Click on the blue link to load the sample configuration in CANoe.

  • VT System - Online – VTDemoOnline.cfg
    Set up and check VT modules.

The supported modules (VT1004, VT2004, VT2516, VT2816, VT2820, VT2848, VT6104 and VT7001) are displayed as CANoe panels, which you can use to interact with the actual VT System hardware.

Note

You have to activate the VT System hardware application. Select the VT System active options in the Options dialog (Extensions|VT System).

Screenshot

These panels support most of the functions offered by the real modules, e.g. generating and measuring constant voltages, PWM signals and short circuits.

VT1004 [▲ back]

This desktop contains the panel for the VT1004 module. The desktop also has a Graphics Window, which displays a history of the signal sequence for the generated resistance, the generated voltage and the generated PWM frequency at channel 1.

The VT1004 panel gives you access to three of the VT1004 module's essential functions:

  • Generating a short circuit between the input terminals
  • Measuring a constant voltage between the input terminals
  • Measuring the frequency of a PWM signal between the input terminals

As with the real VT1004 module, 4 channels are available here. The panel is structured as follows:

Meaning of the LEDs

The panel uses three of the VT1004's LEDs per channel. The Short circuit LED lights up when a short circuit occurs at the corresponding channel. The red and blue LEDs between the channel's input terminals show whether the voltage is positive (red) or negative (blue).

Using the Panel

To generate a short circuit between the input terminals, simply click on the Short circuit switch. The short circuit is then signaled both by the switch closing and the corresponding LED lighting up.

To measure a constant voltage, tick the Activate measurement check box in the Voltage measurement area. The measured voltage is then displayed both as a number and in the form of a status bar. The value range for the measurement is -27 and +27 Volt.

You can start a PWM frequency measurement in the same way. The permissible value range here is 0 to 10 kHz.

Note

  • Please take great care when generating short circuits. Before you connect other devices to the VT1004, make sure that they will not be damaged when a short circuit occurs.
  • Please take great care when generating short circuits. Before you connect other devices to the VT1004, make sure that they will not be damaged when a short circuit occurs.

VT2004 [▲ back]

This desktop contains the panel for the VT2004 module. The desktop also has a Graphics Window, which displays a history of the signal sequence for the generated resistance, the generated voltage and the generated PWM frequency at channel 1.

The VT2004 panel gives you access to four of the VT2004 module's essential functions:

  • Generating a short circuit between the input terminals
  • Generating a resistance between the input terminals
  • Generating a constant voltage between the input terminals
  • Generating a PWM signal between the input terminals

As with the real VT2004 module, 4 channels are available here. The panel is structured as follows:

Meaning of the LEDs

The panel uses three of the VT1004's LEDs per channel. The Short circuit LED lights up when a short circuit occurs at the corresponding channel. U int indicates that the internal voltage generator is active, while R int indicates that the internal decade resistor is active.

Using the Panel

To generate a short circuit between the input terminals, simply click on the Short circuit switch. The short circuit is then signaled both by the switch closing and the corresponding LED lighting up.

You can adjust the resistance, constant voltage and PWM signal using the corresponding sliders. When you adjust one of the sliders, the VT System module automatically switches to the corresponding output mode.

A Signal Generator for each output mode is also available at channel 1. It is deactivated by default.

Note

  • Please take great care when generating short circuits. Before you connect other devices to the VT2004, make sure that they will not be damaged when a short circuit occurs.
  • Click the Help button in the upper right corner of the VT2004 panel to access a summary of key points regarding how to use the panel.

VT2516 [▲ back]

This desktop contains the panel for the VT2516 module. The desktop also has a Graphics Window, which displays a history of the signal sequence of the output voltage and PWM frequency as well as the measured input voltage and PWM frequency at channel 1.

The VT2004 panel gives you access to four of the VT2516 module's essential functions:

  • Generating a digital output signal between the input terminals
  • Generating a PWM signal between the input terminals
  • Measuring a constant voltage between the input terminals
  • Measuring the frequency of a PWM signal between the input terminals

Unlike the real VT2516 module, only 4 channels are available here. The panel is structured as follows:

Meaning of the LEDs

Like the real VT2516 module, the panel uses one LED per channel. This shows whether there is a voltage (internal or external) at the corresponding channel.

Using the Panel

To generate a digital output signal between the input terminals, activate/deactivate the check box in the Digital voltage output area. A voltage of 0V is used when the value is 0, and a voltage of 5V when the value is 1. In addition, a Signal Generator is available at channel 1. You can use it to output complex bit sequences.

To output a PWM signal, use the slider in the PWM generator area. Frequencies here can range from 100 Hz to 10 Hz. The first channel also has a Signal Generator.

To measure a constant voltage, activate the Activate measurement check box in the Voltage measurement area. The measured voltage is then displayed both as a number and in the form of a status bar. The value range for the measurement is -27 and +27 Volt.

You can start a PWM frequency measurement in the same way. The permissible value range here is 0 to 10 kHz.

VT2816 [▲ back]

This desktop contains the panel of the VT2816 module. A Graphics Window is also available on this desktop. It displays the signal characteristic over time of the output voltage and PWM frequency as well as the measured input voltage and PWM frequency at output channel 1.

With the VT2816 panel, you have access to three essential functions of the VT2816:

  • Voltage measurement
  • Current measurement
  • Output of voltage characteristics

In contrast to the 12 input channels on the real VT2816, this panel offers only 2 input channels for sake of clarity. For the same reason, only 2 channels are used on the output side instead of the 4 real channels.

Meaning of the LEDs

As in the case of the real module, the red LED (LED_In_Plus) indicates that the input or output voltage is >+1 V or the input or output current is >+10 mA.

The blue LED (LED_In_Minus) lights up when the input or output voltage is <-1 V or the input or output current is <-10 mA. If both LEDs light up for a channel, the input or output voltage is in the range -1 V…1 V or the input or output current is in the range -10 mA…10 mA.

The third LED of the input channels (LED_In_Cur) indicates whether the current measurement is activated.

Operation of the Panel

You can set the measurement mode for the two input channels using the options. Voltage measurement or Current measurement is supported.

For the two output channels, you can output either a constant voltage or one of three predefined voltage curves in each case.

Predefined voltage characteristics are:

  • Square
  • Sawtooth
  • Sine

To output a constant voltage, you can set the desired value in the left panel area either numerically or via a slider. To use the predefined voltage curves, you select the type in the list. Then, click the Download and Start button to download the desired curve type to the module and start the output.

You can use this panel, for example, to measure an output voltage characteristic with a VT1004A. To do so, connect an output of the VT2816 to a channel of the VT1004A. The measurement channel of the VT1004A is then put into a voltage measurement mode.

Alternatively, you can measure the voltage output of a correspondingly configured VT2004A channel. To do so, connect an input channel to the corresponding output of a VT2004A.

VT2820 [▲ back]

This desktop contains the panel of the VT2820 module. You can use this panel to access four of twelve possible switch-on channels and four of eight possible switchover channels of the VT2820. Each of the switch-on channels can be placed independently on one or both bus bars of the module.

Meaning of the LEDs

For the four switch-on relays, the respective LED on the left indicates whether the relay is switched on, i.e., connected through. The respective center LED indicates whether the channel is also placed on bus bar A. The respective LED on the right indicates whether the channel is also placed on bus bar B.

For the switchover channels, the LED is active when the alternative position of the relay is switched.

Meaning of the Panels

With the check boxes for the four switch-on channels, you can independently close the individual channels (Relay), place them on bus bar A (Relay bus bar a), and place them on bus bar B (Relay bus bar b) in each case.

With the check boxes on the four switchover channels, you can activate the alternative position of the relay (Relay) in each case.

You can use the switch-on channels, for example, to switch external equipment in your Test Setup. With the switchover channels, it is possible, for example, to place two different simulation channels of a VT2004A alternatively on an ECU input or a VT1004A measurement channel.

VT2848 [▲ back]

This desktop contains the panel of the VT2848 module. A Graphics Window is also available on this desktop. It displays the signal characteristic over time of the output voltage and PWM frequency as well as the measured input voltage and PWM frequency at channel 1.

With this panel you can use essential functions of the VT2848 on four of 48 possible channels:

  • Output of digital voltage values
  • Measurement of digital input signals
  • Determination of the pulse ratio of PWM signals

Meaning of the LEDs

The LED indicates for each channel whether it is currently active or not (input and output). In the case of overvoltage (>60 V), the LED flashes and the measurement is stopped. After removal of the overvoltage, a measurement restart is possible.

Meaning of the Panel

You can use the corresponding check box to output a binary voltage for each channel. In activated state, the voltage is +5 V to ground. In deactivated state the voltage is 0 V.

The four channels can be connected to digitally operated input and output channels of an ECU. Alternatively, you can for example connect a PWM output of the VT2848 to an input channel of the VT2516A and thus measure the generated digital signal.

Note for operation

The high level of the digital signal must be supplied externally, for example from a VT7001A or a VT2004A with constant voltage output.

VT6104 [▲ back]

This desktop contains the panel for the VT6104 module.

The VT6104 panel gives you access to five of the VT6104 module's essential functions:

  • Generating a short circuit between two CAN or LIN lines
  • Simulating a line disturbance (high and/or low)
  • Short circuiting a line (high and/or low) to ground
  • Short circuiting a line (high and/or low) to Ubat
  • Removing the terminating resistor

Meaning of the LEDs

The real VT System module uses the Activity LED to indicate the status of the bus communication on the channel. This sample configuration does not simulate this behavior, which means that this LED is not activated in the panel.

The Disturbed LED indicates whether or not an error relay is active on the corresponding channel. When this is the case, the LED lights up red, which indicates a disturbance of the bus communication on this channel.

Using the Panel

The VT6104 panel offers four ways to disturb the bus communication for each channel. You can activate and deactivate these using the different switches on the panel.
The first switch (Short circuit) lets you generate a short circuit between the two signal lines.
The second set of switches lets you influence the two signal lines separately. You can therefore disturb the high and low lines separately.
The next two sets of switches let you short circuit the two signal lines separately to ground or Ubat.
The last switch lets you remove the terminating resistor between the high and low lines.

VT7001 [▲ back]

This desktop contains the panel for the VT7001 module. The desktop also has a Graphics Window. The Graphics Window displays a history of the output voltage at outputs 1 and 2 along with the corresponding current flows. It also shows the activation state of the VT7001 module.

The VT7001 panel gives you access to two of the VT7001 module's essential functions:

  • Generating a voltage between the output terminals at output 1
  • Measuring the current that is currently flowing at outputs 1 and 2

You can also use the panel to control the voltage at output 1 using the internal power supply. The voltage at output 2 is determined by the external power supply.

Meaning of the LEDs

As soon as you activate the VT7001 module, the Int. Supply and Ground LEDs at output 1 and/or the Supply 1 and Ground LEDs at output 2 light up. You can now use the module and its panel.

The red LED indicates a positive voltage at the corresponding VT7001 output.

Using the Panel

First of all you need to start the VT7001 module. Click the Start VT7001 button in the panel to do this. Alternately, you can run the Start VT7001 test case.

Once the module is activated, you can adjust the output voltage using the slider. The value range here is 3 to 18 Volt. You can connect an external power supply to output 2. Its voltage is also shown in the panel.

The currently flowing current is measured for each output. This is displayed both as a number and in the form of a status bar.

Testing [▲ back]

This desktop contains the Test Setup. This contains five sample test cases that illustrate ways to test the VT modules using the CANoe Test Feature Set.

Each of these test cases, which are implemented in XML , CAPL or .NET, serves to test the interaction between two supported VT modules.

The corresponding modules need to be available in order to start the configuration. If a module is not available, you need to deactivate any test cases that use this module before starting the configuration. The Setting up and starting the sample configuration section explains how to do this.

XML: VT2516_Ch1 to VT1004_Ch1

Language

This test module is defined in XML.

Prerequisites

A VT1004 and a VT2516 module are needed to execute these test cases. These need to be connected with each other at channel 1.

Sequence

In the test cases of the first test group, a constant voltage is output at channel 1 of the VT2516. The VT1004 module measures its value. If it is identical to the value output by the VT2516, the test case is passed. If it differs, the test case fails. This test case is then repeated for different voltages.

The second test group tests the generation and measurement of PWM signals. A PWM signal is output at channel 1 of the VT2516. The VT1004 measures its frequency. If it is identical to the value output by the VT2516, the test case is passed. If it differs, the test case fails. This test case is then repeated for different frequencies.

The third test group containts a test case for checking the short circuit relay on channel 1 of the VT1004. A constant voltage is applied to the output of the VT2516 and a short circuit is generated at the VT1004's input. The voltage at the VT1004 is now measured. This should be 0V because of the short circuit.

CAPL: VT2004_Ch1 to VT1004_Ch1

Language

This test module is implemented in CAPL.

Prerequisites

A VT1004 and a VT2004 module are needed to execute these test cases. These need to be connected with each other at channel 1.

Sequence

Test group 1 consists of several test cases, each of which generates a constant voltage at the VT2004's output. The voltage is measured at the VT1004's input and compared with the generated voltages. If the voltages are identical, the corresponding test case is passed.

Test group 2 also consists of several test cases, each of which generates a PWM signal at the VT2004's output. The VT1004 measures the frequency of this signal. If it is identical to the output frequency, the corresponding test case is passed.

CAPL: VT2004_Ch1 to VT2516_Ch1

Language

This test module is implemented in CAPL.

Prerequisites

A VT2004 and a VT2516 module are needed to execute this test group. These need to be connected with each other at channel 1.

Sequence

In the test cases, a PWM signal is output at channel 1 of the VT2004. The VT2516 measures its frequency. If it is identical to the value output by the VT2004, the test case is passed. If it differs, the test case fails. This test case is then repeated for different frequencies.

CAPL: VT7001_Ch1 to VT1004_Ch1

Language

This test module is implemented in CAPL.

Prerequisites

A VT1004 and a VT7001 module are needed to execute these test cases. These need to be connected with each other at channel 1.

Sequence

In the test cases belonging to test group 1, a constant voltage is output at channel 1 of the VT7001. The VT1004 module measures this voltage. If it is identical to the value output by the VT7001, the test case is passed. If it differs, the test case fails. This test case is then repeated with different voltage values.

The second test group contains a short circuit test. As in the previous case, a constant voltage is generated at the VT7001. A short circuit is now generated in the VT1004. The current flowing through the VT7001's output should now be greater than 1 mA. The test case checks whether this is the case.

Configuration [▲ back]

This desktop represents the setup of the configuration. It contains the Simulation Setup and the Test Setup.

The Simulation Setup contains one node per VT System module. The nodes are implemented in CAPL. The nodes are responsible for communication between each VT System module and its panel.

If you don't have all the supported VT modules available, you can deactivate the missing modules in the Simulation Setup and in the Test Setup. The Setting up and starting the sample configuration section explains how to do this.
