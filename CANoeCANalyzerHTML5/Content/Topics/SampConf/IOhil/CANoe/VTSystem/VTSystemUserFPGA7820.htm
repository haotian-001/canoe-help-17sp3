Notice!

Relay Switch and Output Type must both always be set to voltage or current, otherwise incorrect values may be output and the VT7820 and external hardware may be damaged.

Configuration

  • To set up this sample configuration, you need a VT7900 with VT7820 and User FPGA.
  • The used VHDL program is preinstalled on the VT7900 and does not have to be installed separately on the module. In addition, the VHDL program is included in CANoe and can be loaded onto the module with the VT System FPGA Manager.
  • The sample configuration is automatically copied to the Workbench when you first set up the manager and can be restored later using the menu path Tools|Option|Restore Demo Projects.
  • The code is precompiled and can be written directly to the FPGA or EEPROM of the module.
  • You must select the connection to the VT System module in CANoe:
  • On the Hardware Ribbon Tab click VT System and then Configuration.
  • Click the Hardware command.
  • In the Network adapter list select the network adapter to which the VT System is connected.
  • On the Test Run desktop you see how the module has to be connected.

Introduction

  • Connect an oscilloscope or other hardware to graphically display a PWM signal.
  • Start the measurement.
  • A simple driving behavior is automatically played back.
