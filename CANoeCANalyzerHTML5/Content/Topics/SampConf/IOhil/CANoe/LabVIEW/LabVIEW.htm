LabVIEW

 

Click on the blue link to load the sample configuration in CANoe.

This sample consists of two parts:

  • LabVIEW Projekt – Open Folder
  • LabVIEW – LabVIEW.cfg
    Demonstrates how to couple CANoe with LabVIEW.

This sample configuration demonstrates how to use LabVIEW with CANoe using network variables.

Note

You must have the LabWindows CVI Runtime installed on your computer in order to use the CANoe LabVIEW integration. This runtime can be downloaded from the National Instruments website free of charge.

To get started, open the LabVIEW project. Right-click the Network Variables library VariablesLibrary.LVLIB and choose Deploy all. This will register the all variables used in this sample. Then start the virtual instrument Demo VI.

Now open the CANoe sample configuration and start the measurement. In the Graphics Window you will see two signals:

  • CANoe_LabVIEW is a sine wave generated by CANoe. This signal is transferred to LabVIEW.
  • LabVIEW_CANoe is sent back from LabVIEW to CANoe. It contains the processed sine wave, which is transformed to a rect signal by the LabVIEW VI.
