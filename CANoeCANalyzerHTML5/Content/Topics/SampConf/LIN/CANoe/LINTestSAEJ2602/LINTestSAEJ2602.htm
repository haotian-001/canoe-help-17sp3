SAE J2602 Test (LIN)

 

Click on the blue link to load the sample configuration in CANoe.

  • SAE J2602 Test (LIN) – LINTestSAEJ2602.cfg
    Demonstrates a J2602 slave conformance test.

Introduction

Note

This sample configuration cannot be operated in the Simulated Bus mode of CANoe. The default setting is therefore Real Bus mode. Furthermore, a network interface that supports LIN must be connected.

This sample configuration shows how a Conformance Test for an SAE J2602 slave can be implemented. The following CANoe LIN functions are also shown:

  • LIN Slave Conformance Tester
  • LIN Test Feature Set

The LIN Slave Conformance Tester allows you to perform a Slave Conformance Test within an already existing configuration. In generates a test specification in XML on the basis of the LDF file assigned to the bus. Furthermore, you can assign an NCF to the module.

Start the CANoe measurement and then start the LIN Slave Conformance Tester 1 test module by clicking on .

Located on this desktop are two panels:

Also located here is a Trace Window, in which you can observe all bus traffic.

The LIN Slave Conformance Tester 1 test module is used to control the test. Here, you can start, pause and stop the test module as well as open the automatically generated test report.
