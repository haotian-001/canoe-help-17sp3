For EtherCAT Basic

 

Click on the blue link to load the sample configuration in CANoe.

    For EtherCAT Basic – For EtherCAT Basic.cfg
    The configuration demonstrates the reception of measured values from a CSM measuring module connected via EtherCAT®.

To be able to execute the configuration, it is necessary to have the license for the option .EtherCAT on the one hand and the CSM hardware ECAT ADMM4 HS. The regular Ethernet driver of the Intel® network card must be replaced by the EtherCAT driver from the installation directory, as described in the EtherCAT help.

Introduction

The configuration menu of the .EtherCAT option allows a scan for the connected EtherCAT devices. The ESI file for the CSM measurement module ADMM4 HS is enclosed to this configuration and must be added explicitly in the configurator for EtherCAT by the ESI Manager.

Desktops

This configuration contains the following desktops:

Trace For EtherCAT Basic

Configuration

Analysis For EtherCAT

 

Trace For EtherCAT Basic[▲ back]

Within this desktop the measured values of the EterCAT measuring module ADMM4 HS are visible as system variables. System messages are displayed in the Write window.

Configuration [▲ back]

This desktop allows you to further configure this example of your own purposes.

Analysis For EtherCAT[▲ back]

The system variables of the ADMM4 HS are displayed in the Graphics window and the Data window. A CAPL program generates additional average values from the measured values, which are displayed in a panel.
