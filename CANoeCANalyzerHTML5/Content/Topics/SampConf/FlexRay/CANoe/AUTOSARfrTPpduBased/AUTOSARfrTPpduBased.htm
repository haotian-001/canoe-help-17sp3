AUTOSAR TP PDU Based (FlexRay)

 

Click on the blue link to load the sample configuration in CANoe.

  • AUTOSAR TP PDU Based – AUTOSAR_FlexRay_TP_PDU_based.cfg
    This sample configuration illustrates the use of the AUTOSAR 3.0 FlexRay Transport Protocol Module for CANoe.

Introduction

See also the PDF manual for the AUTOSAR FrTP DLL.

Note

The TP settings are not read from the FIBEX database. They are hard coded in the CAPL programs.

Desktops

This configuration contains the following desktops:

TP Demo Trace

Use the panels ESP, ECM, Tester->ESP and Tester->ECM on this desktop to send large data packets with a transport protocol.

CAPL Key

Description

1 - 6

Start parallel transmissions for different scenarios.

p

Switching between diagnostics and flashing on all ECUs simultaneously.

x

Remove data in all TP panels.

<space>

Display all TP panels.

h

Hide all TP panels.

You can change some transport protocol settings by using the buttons Additional Settings (Diagnostics) or Additional Settings (Flashing) on the TP panels. For correct operation, you must ensure that the transport protocol settings for both nodes match.
