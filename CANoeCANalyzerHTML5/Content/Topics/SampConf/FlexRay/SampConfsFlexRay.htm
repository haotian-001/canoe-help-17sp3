Sample Configurations FlexRay

 

Note

Click on the blue link to load the sample configuration in CANoe.

Note

If there is no description for the sample configuration in the help, you find a description in a TXT file next to the sample configuration in the same folder.

After you have loaded the sample configuration in CANoe you can show this TXT file via the Environment ribbon tab|Documents.

General

System Configuration (FlexRay) – FlexRaySystemDemo.cfg
Simulation of a complete FlexRay network with FlexRay specific functions.

View Description

AUTOSAR Network Management (FlexRay) – AsrFrNM.cfg
Demonstrates the usage of the CANoe AUTOSAR NM Node Layer.

View Description

AUTOSAR TP PDU Based – AUTOSAR_FlexRay_TP_PDU_based.cfg
This sample configuration illustrates the use of the AUTOSAR 3.0 FlexRay Transport Protocol Module for CANoe.

View Description

In Cycle Mux (FlexRay) – FlexRayInCycleMuxDemo.cfg
Demonstrates In Cycle Repetition and In Cycle Responses of frames.

View Description

2 Cluster Gateway (FlexRay) – FlexRay_2Cluster_Gateway.cfg
Demonstrates the simulation of two FlexRay networks that are interconnected by a FlexRay-FlexRay gateway.

View Description

ISO TP PDU Based (FlexRay) – FlexRay_ISO_TP_PDU_based.cfg
Demonstrates how to use the ISO 10681-2 Transport Protocol in CANoe on FlexRay with a FIBEX database in version 3.0 using FlexRay PDUs.

View Description

Diagnostic Tester AUTOSAR TP PDU (FlexRay) – UDSsimExt_Fr_AsrTP_PDU_based.cfg
Shows the interactive Diagnostic Features in the environment of FlexRay Diagnostics (Diagnostic Console, fault memory and variant coding) as well as the analysis functions for diagnostics, based on the AUTOSAR transport protocol.

View Description

Diagnostic Tester ISO TP PDU (FlexRay) – UDSsimExt_Fr_ISOTP_PDU_based.cfg
Shows the interactive Diagnostic Features in the environment of FlexRay Diagnostics (Diagnostic Console, fault memory and variant coding) as well as the analysis functions for diagnostics, based on the ISO transport protocol.

View Description

CAPL-on-Board (FlexRay) – CAPLonBoard.cfg
Demonstrates the determinism (reliability) and performance of the CAPL execution on the network interface.

Bitmask Analysis (FlexRay - Scope) – FlexRayBitMaskAnalysis.cfg
Demonstrates how you can execute a bitmask analysis for FlexRay with the Scope option.
