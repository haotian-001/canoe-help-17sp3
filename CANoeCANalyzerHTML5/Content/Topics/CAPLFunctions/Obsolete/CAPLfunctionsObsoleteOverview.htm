Deprecated and Removed CAPL Functions

 

Note

Deprecated Function

If a function is deprecated you get the CAPL Compiler notification Function is deprecated. See help for more information.

In CANoe at the start of measurement different scenarios are possible:

  • The function has full functionality.
  • The functionality of the function is limited, please see notification in the Write Window.
  • The function no longer has any effect, please see notification in the Write Window.

Removed Function

Some functions were deprecated for a long time and were therefore removed (obsolete). These functions are neither known to the CAPL compiler nor to CANoe.

You get the CAPL Compiler notification unknown function and in CANoe at the start of measurement a notification in the Write Window.

For the display of the notifications see settings in the Options dialog.
