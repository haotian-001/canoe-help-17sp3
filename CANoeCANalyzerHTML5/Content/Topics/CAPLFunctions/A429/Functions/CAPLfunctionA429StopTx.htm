a429StopTx

 

Function Syntax

dword a429StopTx( a429word myA429word );

Description

This function is necessary to stop the cyclic transmission of an ARINC word in the hardware scheduler. The function modifies the selector TxCtrl.

Parameters

myA429word

ARINC word

Return Values

0

could not modify TxCtrl

1

TxCtrl successfully set to CYCLIC_OFF
