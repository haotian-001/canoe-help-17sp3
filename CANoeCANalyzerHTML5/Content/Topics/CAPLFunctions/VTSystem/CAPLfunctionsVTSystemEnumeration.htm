eVTSCurveType

Value

Enum

Short Description

0

eVTSCurveTypeConstant

Constant value, determined by the corresponding output system variable. Nevertheless, this value can also be used dynamically by changing the value of the variable in CANoe.

1

eVTSCurveTypePWM

PWM

2

eVTSCurveTypeAnalogWaveform

Analog wave form; loaded using LoadWFVoltage (for VT2004 and VT2816 only) or LoadWFResistance (for VT2004 only), then started using StartStimulation.

3

eVTSCurveTypeBitStream

Bitstream (= digital wave form); loaded using LoadWFBitStream, then started using StartStimulation (for VT2516 and VT2848 only).

4

eVTSCurveTypeUserFPGA

User FPGA. If the output of a module shall be controlled by a user-programmable FPGA this type has to be set. This type can only be used if the module contains a user programmable FPGA.
