ISO11783 Node Layer
(Only available with ISO11783!)

 

ISO11783 NL

To use the CAPL functions the node layer ISO11783DLL.dll must be included via the module description file ISO11783DLL.vmodule.

You can include the node layer by using the configuration dialog of the node on page Components or via the node attribute NodeLayerModules in the database.
