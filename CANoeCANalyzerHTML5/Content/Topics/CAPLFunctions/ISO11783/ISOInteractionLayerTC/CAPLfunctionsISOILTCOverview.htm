Functions

Short Description

Is called if an address conflict is detected.

Is called if an address violation is detected.

TCIL_OnCA

TCIL_OnChangedState

Is called if the IL has changed its state.

TCIL_OnError

Is called if an error has occurred.

TCIL_OnRequest

Is called if a request (0xEA00) is received.
