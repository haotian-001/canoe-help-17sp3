ISO11783 Interaction Layer
(Only available with ISO11783!)

 

ISO11783 IL

To use the CAPL functions the node layer Iso11783_IL.dll must be included via the module description file Iso11783_IL.vmodule.

You can include the node layer by using the configuration dialog of the node on page Components or via the node attribute NodeLayerModules in the database.

More information about setting up the ISO11783 Interaction Layer you can found here.

Tractor Implement Management (TIM) API[▲ back]

Note

The following functions are supported by both the TIM Client and the TIM Server. Many functions refer to a specific client/server connection.

The counterpart is defined by a node from the data basis. The ISO11783 device name of the node specified in the database must match the device name of the ECU.
