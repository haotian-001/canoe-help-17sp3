Configuration

 

Network Nodes

Node Name And MAC Address

The node name and MAC address can be changed in the overview of all network nodes (item Network Nodes selected) as well as in the view of the respective node (individual node selected).

Assign Tx Messages

Each node can be assigned Tx messages, thus application messages that are sent from this node. To do so, select the Tx messages entry below the respective network node and then select the relevant application messages in the list on the right side.

Application messages

Configuration of XML Modules

When creating a new application message in the toolbar, you may select in the following dialog the XML module of the application message to be inserted. In the check box Application Message ASNV file (.xml) (Dialog Add Message), you may select the XML module.

All XML modules in the folder of the database are offered for selection. You can create the XML modules using the ASN.1 Import Wizard and then copy them into this folder.

Configuration of the Transmission

When creating a new application message (Dialog Add Message) or in the information area of the overview of the messages (Select Message) you may specify how the message is to be transmitted. Dependent on the used standard there are the following typical settings.
