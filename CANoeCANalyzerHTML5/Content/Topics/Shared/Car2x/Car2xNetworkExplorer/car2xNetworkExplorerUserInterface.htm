The Attributes view is empty for databases created with CANoe Version 8.5 SP4 or earlier. To add attributes to the database, this button must be clicked once.

Navigation Area

Icon

Description

Shows information about the database in the information area on the right side. You can enter information about the database (e.g. project, version, author, change history, etc.).

The network nodes are listed in the navigation area under the Network Nodes entry. The send nodes of WLAN packets are identified based on the Address 2 field in the WLAN header (corresponds to the MAC address of the sender) in the information area.

You may assign Tx-Messages to each node.

The application messages are listed in the navigation area on the left side under the entry Messages.

Information Area

The information area is located on the right side. If you select a single message (e.g., CAM) in the navigation area, the elements of the hierarchical ASN.1 data structure are displayed in the information area as a tree view.

Icon

Elements

message elements that can contain sub-elements (SEQUENCE, SEQUENCE-OF)

optional message elements that can contain sub-elements (SEQUENCE, SEQUENCE-OF)

scalar message elements and arrays

optional scalar message elements and arrays

For each element additional information is given in the columns Type, Range, Format and Initial Value.

Note

If a value table is available for an element, you can find it by moving the mouse over the Range column.

Search Options

At the top of the information area, the filter function assists you when searching for particular elements. In the text field at the top right, you enter the text to be searched for. The tree is filtered when the corresponding text is found.

You can set the following in the search options:

Search Option

Description

Contains

Select whether the text contains the search word that the searched for item begins with or ends with.

Match case

Specify whether the search is case sensitive.

Hide subtrees

Specify whether structures located below found elements are to be hidden.

CAPL Code Area

The CAPL code area is located in the lower left corner of the information area. Here you may specify the necessary data for generating the CAPL code.

In the CAPL main instance field, you specify the instance name of the main element of the CAPL data structure.
