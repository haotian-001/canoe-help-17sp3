Symbolic Access to Message Attributes

 

You can access specific properties (attributes) of CAN and LIN frames, parameter groups (PG) as well as FlexRay frames symbolically.
These properties are:

  • ID
  • DLC
  • Transmitter
  • Attributes defined the database of these objects

Here the access can be statically or dynamically.

Static Access

Static access is an access to a known object. Thereby the access can be done directly or by using a local variable.

Example

Static Access

int myID;

myID = EngineData.ID



message msg EngineData;

myID = msg.ID

Direct static access on a message ID

void foo (int id)
{
if (id == EngineData.id)
{
   // do something
}
}

Direct static access on a database attribute

void foo ()
{
int cycleTime;
cycleTime = EngineData.GenMsgCycleTime;
}

See also: Access to Database Attributes

Static access using a variable

message EngineData msg;

void foo( int id)
{
if (id == msg.id)
{
   // do something
}
}

Dynamic Access

Dynamic access is accessing a property without knowing the name of the object. This is common for event procedures using a wildcard selector, e.g. on message *.

Example

Dynamic access

on message *

{

myID = this.ID;

}

Dynamic access to a DLC of a CAN message

on message *
{
  int currentDLC;
  currentDLC = this.DLC;
}

or

void foo( message * msg)
{
if (msg.id == 0x100)
{
   // do something
}
}

Particularities

Note

The Transmitter property as well as database attributes can only be read dynamically over DBLookup.

Example

on message *

{

int myAttributeValue;

myAttributeValue = DBLookup(this).MyAttribute;

write(this.Transmitter); // compiler error

write(DBLookup(this).Transmitter); // OK

}

Using DBLookup following method

on message EngineData

{

int myDLC;

myDLC = DBLookup(this).DLC;

}

may deliver not the same result as

myDLC = EngineData.DLC;

Reason: The first command returns the DLC stored in the database while the second command returns the current value of the specific message. They can differ because the DLC can be changed, e.g. within a CAPL program.
