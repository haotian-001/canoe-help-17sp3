Keywords: Test

 

Keyword

Description

testcase

Defines a test case. A » test case is the semantic test entity. It returns a unique test result in form of a » verdict and appears in the » test report.

Example:

testcase MyTestCase()
{
}

CAPL Test Module | .NET Test Module | XML File

testfunction

Defines a test function. A test function is a function, that can be (re-)used in » test cases and » test sequences. In difference to other functions a test function is reported as a block in the » test report automatically.

Example:

testfunction MyTestFunction()
{
}

.NET Test Module | XML File

Note

CAPL test functions do not provide a return value.
