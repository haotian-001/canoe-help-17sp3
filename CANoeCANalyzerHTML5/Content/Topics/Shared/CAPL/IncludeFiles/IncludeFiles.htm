Include Files: Overview

 

You can source out constants, variables and functions that are needed identically in multiple CAPL programs.

You can integrate Include files directly in the code of the CAPL program (see below or Insert Include Files).

For Include files in CAPL there are common and different behaviors compared to Include files in programming language C.

Common Behavior:

  • The Include file has to be programmed in the same programming language as the main CAPL program.
    (CAPL can only include files that are programmed in CAPL, C can only include files that are programmed in C)
  • The name of the Include file is freely selectable.

Different Behavior:

  • An Include file is not inserted as true text. It is treated similar to a function library.

Example

File Simparameter.CIN ::

variables

{

  int _SimPar_Granularity_ms   = 10;  // Simulation frequence in milliseconds

}

File Door_Left.CAN ::
includes
{
  #include "Simparameter.CIN"
}

variables
{
  msTimer cyclicTimer;
}

on start
{
  setTimer(cyclicTimer, _SimPar_Granularity_ms);
}

  on Timer cyclicTimer
{
  // Main simulation loop Door left
  setTimer(cyclicTimer, _SimPar_Granularity_ms);
}

File Radio.CAN ::
includes
{
  #include "Simparameter.CIN"
}

variables
{
  msTimer cyclicTimer;
}

on start
{
  setTimer(cyclicTimer, _SimPar_Granularity_ms);
}

on Timer cyclicTimer
{
  // Main simulation loop Radio
  setTimer(cyclicTimer, _SimPar_Granularity_ms);
}
