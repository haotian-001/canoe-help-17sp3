Include Files: Insert

 

In CAPL Browser the Include file is displayed on the top of the Browser tree with the icon .

New Include files can be added either by means of manual input or using a file selection dialog (context menu command Add).
If you use the file selection dialog, the Include command will always be added at the end of the Include section.

Include files are displayed in the Browser tree under Include files.
If the syntax is correct, the display is updated as soon as

  • a new Include file is added via the file selection dialog or
  • the CAPL program is compiled.

When choosing an Include file

includes
{
}

is automatically generated in the text editor.

An empty Include section is not saved by the Browser.

If a file is included that has been opened and changed in the Browser before, the current version of the file will be included.

Compiler messages in the Include file are displayed in the Message Window. Double-click the message to go directly to the error location in the Include file.
