Keyboard Operations

 

The following shortcuts are available when working with the CAPL Browser:

Call Help

Description

Key Combination

Opens Help.

If it is available, Help for the word appearing at the cursor position is displayed in the text editor.

If Windows Help has already been opened, the use of the Help system is explained.

<F1>

Moving the Cursor

Description

Key Combination

Moves the cursor within an editing field to the left, right, up or down.

<Left Arrow> or <Right Arrow>
<Up Arrow> or <Down Arrow>

Moves the cursor to the end of the line.

<End>

Moves the cursor alternately to the beginning of the text in the corresponding line or to the beginning of line.

<Home>

Moves the cursor to the end of the text.

<Ctrl>+<End>

Moves the cursor to the beginning of the text.

<Ctrl>+<Home>

Moves the cursor to the end of the next block of text.

<Ctrl>+<Right Arrow>

Moves the cursor to the beginning of the previous block of text.

<Ctrl>+<Left Arrow>

Moves the cursor laterally up or down.

<Page Up> or <Page Down>

Function Keys in Dialogs

Description

Key Combination

Moves the insertion point to the next field.

<Tab>

Moves the insertion point to the previous field.

<Shift>+<Tab>

Executes the current command, or accepts the changes (Same as [OK]).

<Enter>

Closes the dialog without executing an action (Same as [Escape]).

<Esc>

Activate Menu Keys

Description

Key Combination

Selects the ribbon and activates the selection of the tabs.

After that the following functions are available:

<F10>

Opens the tab for the selected character.

<Character>

Opens the menu/window for the selected character on th eselected tab.

<Character>

Opens the tab to the left or right of the active tab.

<Arrow key right>/
<Arrow key left>

Moves the selection of menu commands within the opened menu.

<Arrow key up>/
<Arrow key down>

Executes the command of the selected menu command.

<Enter>

Insert Objects and Symbols in Text Editor

Description

Key Combination

Inserts a signal.

<Ctrl>+<I>

Inserts a system variable or an environment variable.

<Ctrl>+<E>

Inserts a message or a frame.

<Ctrl>+<D>

Inserts a PDU.

<Ctrl>+<Alt>+<D>

Inserts a diagnostic object (service).

<Ctrl>+<G>

Inserts a diagnostic parameter.

<Ctrl>+<T>

Inserts a diagnostic target qualifier.

<Ctrl>+<Alt>+<T>

Inserts a diagnostic primitive.

<Ctrl>+<Alt>+<G>

Edit, Mark, Search and Replace Texts

Description

Key Combination

Selects the entire text of the currently active Text Editor.

<Ctrl>+<A>

Comments a selection or selected line out.

Sequentially <Ctrl>+<K>, <Ctrl>+<C>

Removes the comment for a selection or selected line.

Sequentially <Ctrl>+<K>, <Ctrl>+<U>

Inserts messages with MOST function catalog.

<Ctrl>+<M>

Autocomplete input assistance

<Ctrl>+<Space>

Deletes the currently selected text or the character to the left of the cursor.

<Backspace>

Deletes the selected text or the character to the right of the cursor.

<Del>

Deletes the selected text from the document and places it in the clipboard.

<Ctrl>+<X>

Copies the selected text to the clipboard.

<Ctrl>+<C>

Pastes the contents of the clipboard at the active position in the text.

<Ctrl>+<V>

Undoes the previously executed action.

<Ctrl>+<Z>
<Alt>+<Backspace>

Redoes the last undone action.

<Ctrl>+<Y>

Expands the text selection one character to the left or right.

<Shift>+<Left Arrow> or <Right Arrow>

Expands the text selection to the beginning of the previous block of text.

<Ctrl>+<Shift>+<Left Arrow>

Expands the text selection to the end of the next block of text.

<Ctrl>+<Shift>+<Right Arrow>

Expands the text selection one line up or down.

<Shift>+<Up Arrow> or <Down Arrow>

Expands the text selection to the end of the line.

<Shift>+<End>
Expands the text selection alternately to the beginning of the text of the corresponding line or to the beginning of line.

<Shift>+<Home>

Expands the text selection to the end of the text.

<Ctrl>+<Shift>+<End>

Expands the text selection to the beginning of the line.

<Ctrl>+<Shift>+<Home>

Opens the Find and Replace dialog, Quick Find

<Ctrl>+<F>

Opens the Find and Replace dialog, Find in Files

<Ctrl>+<Shift>+<F>

Opens the Find and Replace dialog, Quick Replace

<Ctrl>+<H>

Opens the Find and Replace dialog, Replace in Files

<Ctrl>+<Shift>+<H>

Repeats the last executed find or replace action.

<F3>

Sets or deletes a bookmark.

<Ctrl>+<K>, <Ctrl>+<K>

Goes to previous bookmark.

<Ctrl>+<K>, <Ctrl>+<P>

Goes to next bookmark.

<Ctrl>+<K>, <Ctrl>+<N>

Clears all bookmark.

<Ctrl>+<K>, <Ctrl>+<L>

Other Keys

Description

Key Combination

Displays a file selection dialog in which the user can select an existing document.

<Ctrl>+<O>

Creates a new CAPL program.

<Ctrl>+<N>

Creates a new include file.

<Ctrl>+<Shift>+<N>

Saves the active CAPL program.

<Ctrl>+<S>

Saves the program in a file with the new name or at another location on your hard disk.

<Ctrl>+<Shift>+<S>

Opens a dialog in which you can enter a line number for the currently active file which you would like to navigate to.

<Alt>+<G>

Prints the active CAPL program.

<Ctrl>+<P>

Closes the active CAPL Browser window.

<Ctrl>+<F4>

Compiles the program in the active window.

<F9>

Compiles all loaded CAPL programs.

<Shift>+<F9>

Exits the CAPL Browser.

<Alt>+<F4>

Scrolls up the window contents. The active cursor position still remains.

<Ctrl>+<Up Arrow>

Scrolls down the window contents. The active cursor position still remains.

<Ctrl>+<Down Arrow>

Opens the system menu.

<Alt>+<Space>

Switches between the active Windows programs.

<Alt>+<Tab>
Output Window:
The cursor is positioned at the previous error location in the code.		 <F4>
Output Window:
The cursor is positioned at the next error location in the code.		 <Shift>+<F4>
