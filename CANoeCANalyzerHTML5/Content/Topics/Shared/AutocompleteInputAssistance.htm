Autocomplete Input Assistance

 

Note

Starting with CANoe Version 9.0 SP3, the Autocomplete Input Assistance feature in the CAPL Browser supports diagnostic ECUs/targets and services:

After inserting diagRequest, diagResponse and diagRequestSent a list of configured ECU qualifiers is offered. After inserting an ECU qualifier, a list of services defined for this target is offered. For event handlers also diagnostic classes are offered for selection.

Using the autocomplete input assistance you can enter or complete names of CAPL functions, symbolic messages signals and key words (e.g. int, message,...) in the CAPL Browser.

The input assistance can be opened automatically by entering text in the source code or via the key combination <Ctrl>+<Space>.

Within the input assistance all predefined functions, networks, messages and symbols available in the configuration are displayed in alphabetical sequence.

By entering a word or number sequence, the marker of the input assistance jumps to the next matching entry. If you have found the desired entry within the input assistance you can accept it with <Return>.
If no matching entry can be found, the marker jumps to the first entry of the input assistance.

With <Back> you can delete within the editor window the character on the left of the cursor.
With <Esc> you can close the Input Assistance without completing the current input.
