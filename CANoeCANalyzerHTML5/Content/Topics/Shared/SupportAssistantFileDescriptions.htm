Support Assistant File Descriptions

 

In the Vector Support Assistant the following file descriptions are displayed:

Program

Files

Description

 

Product Information

CANoe

CANalyzer

CAPL Browser

vector.ini

Contains information about the language setting as well as the version numbers of the DLLs used.

CANoe

CANalyzer

CAPL Browser

vcustom.ini

Contains information about installation (e.g. installed options) as well as links to the setting and user directories.

CANoe

CANalyzer

can.ini

Contains various settings that, for example, can be made in the Options dialog.
The paths to the last used configurations are also stored here.

CANoeCANalyzer_MoreInfo.txt

This file is generated if you

  • open the Vector Support Assistant via the Help menu,
  • call the Info menu command via the Help menu and select [More Info…] there.

The file contains information about the channels used at the time, installed plugins and add-ons, as well as information about the computer (operating system, processor, memory size, etc.).

WriteLog.txt

This file contains the content of the Write Window. If the recording of messages is activated in the configuration dialog of the Write Window, the filename specified there is applied.

canconfReport.log

Contains hardware information.

CAPL Browser

CAPLBrowser.settings
CAPLBrowser.workspace.settings

Local application settings, e.g GUI layout, at last opened files, global options.

CAPLBrowser.log

Logging files with logged errors and warnings during program sequence for a later error analysis.
