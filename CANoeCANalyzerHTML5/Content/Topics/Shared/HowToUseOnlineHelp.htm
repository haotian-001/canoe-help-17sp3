Tips for Using the Help

 

Navigation Within the Help

Table of Contents

The table of contents on the left margin lists only the main topics with the overview pages. You navigate to further subpages via the hyperlinks on the help pages.

Not all helps have a table of contents.

Search

Troubleshooting

  • If you can't find any or very little content when searching in the help, deleting the browser cache may solve the problem.

  • Some characters (such as colon :, pointed clamps < >, percent %, or slash /) are not interpreted correctly or are passed on incorrectly by the search in the help, so the search text will not produce any results. Instead of these characters, use the wildcard *, which stands for any character.

Basic Rules

  • The search is not case-sensitive.
  • The search contains a substring search starting with three related characters.

Search for Multiple Words / Usage of Search Operators

Search for multiple words:

Entry

Search Result: All help pages are listed, that ...

... contain the word trace, trace-window or window.

... contain the word trace, window or both.

... contain the phrase trace window.

Search with the search operators AND, OR, NEAR, NOT:

Entry

Search Result: All help pages are listed, that ...

Trace AND Window

... contain the words trace AND window.

Trace OR Window

... contain the word trace OR window.

Trace NEAR Window

... contain the word trace AND within eight words also the word window.

Trace NOT Window

... contain the word trace but NOT the word window.

Combinations of the above two cases:

Entry

Search Result: All help pages are listed, that ...

Lookup AND "CAPL Function"

... contain the word Lookup and the phrase CAPL Function.
Note: Each CAPL function help page contains the function name as index entry and the phrase CAPL Function as bracket addition.

(Event AND CAPL) NOT message

... contain the words event AND CAPL but NOT the word message.

English US and Synonyms

The help texts are written in English US. Accordingly, you must enter the search texts with English US spelling.

For certain search words synonyms are stored, which are searched parallel to the search word and generate additional hits in the search result list. For example, if you enter the search word deinstall, the word uninstall is also searched for and corresponding search results are displayed and vice versa. To prevent such synonym coupling, enclose the search word in quotation marks, e.g. "uninstall".

Search Filters

To the right of the search input box you can select search filters, to reduce the search to the corresponding range.

The Search without Topic Filter command removes the filters for the search.

Not all helps have search filters.

Page Search

With the key combination <Ctrl>+<F> you open the search input box for the current page in the browser. This way you can search the content of a single page.

Collapsed texts are not taken into account in page search.

Text areas that can be expanded via a but are currently collapsed are not included in this page search.

Structure Path

The help pages are structured thematically and hierarchically. The lower the page is in the hierarchy, the longer the structure path becomes. You can use the hyperlinks of the structure path to navigate to the overview pages of the next higher levels.

Navigation Path

If elements of the graphical user interface of the tool are explained on the help page, the navigation path indicates how you can open the element in the tool. Page Navigation

On the help page you will find hyperlinks that lead to further subpages or to text parts on the page.

Browser / Links Between Tool and Help

Notes for Browsers

The help opens in your default browser.

If you have changed the default browser in the operating system, this change will take effect with respect to the context-sensitive help of the Vector tool only after a restart of the Vector tool.

The following browsers are supported:

  • Google Chrome (recommended, since no problems are known here).
  • Mozialla Firefox (when opening links to PDFs, CFGs and other files a security dialog may appear).
  • Microsoft Internet Explorer (blocked content must be allowed with every help call, because of ActiveX elements).
  • Microsoft Edge:
  • When opening links to PDFs, CFGs and other files, a security dialog may appear..
