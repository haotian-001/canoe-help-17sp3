Find and Replace

Ribbon: Home Tab|Find and Replace

Find

The options in this window specify whether the search covers only whole words and whether capitalization is to be taken into account for the search. Matching instances in the text are highlighted by the search function. The search can be carried out either in the currently selected document or in all opened documents. If a document is opened in the Document View, the respective page is activated.

During the search, a differentiation is made between

  • Quick Find

    With [Find Next], you navigate to the individual search results within the currently active file or all opened files.
  • Find in Files

    With [Find All], the search results are displayed in the Find Results Window. Double-clicking a line in the Find Results Window allows you to navigate to the individual search results.

Replace

The options in the window specify whether only whole words are to be replaced, whether capitalization is to be taken into account for the search and whether found instances in the text are to be replaced. The search can be carried out either in the currently selected document or in all opened documents. If a document is opened in the Document View, the respective page is activated.

When replacing, a differentiation is made between

  • Quick Replace
  • With [Find Next], you navigate to the individual search results within the currently active file or all opened files.
  • With [Replace], you can replace a specific instance in the source code.
  • With [Replace All], you can replace all found texts.
    A message indicates the number of instances of the text which have been replaced. The replaced lines are displayed in the Output Window.
  • Replace in Files
  • With [Replace All], all lines in which the searched text was replaced are displayed in the Find Results Window. Double-clicking a line in the Find Results Window allows you to navigate to the individual results.
