Options

Ribbon: File Tab|Options

This dialog consists of following pages:

General

Menu

Here you can set the number of previously opened files to be displayed in the File ribbon tab. Up to 24 files can be displayed.

Using [Clear List], you can delete all the listed files.

Compatibility

Automatically remove old CAPL Browser information from CAPL files removes all old CAPL Browser information when saving the file.

Compile

Here you can specify whether all changed files must be saved before compiling.

CAPL File Encryption

Here you can specify whether an encrypted CAPL file must be generated when the files are saved.

Editors

Text Editors

The editor settings all relate to the CAPL editor and C# editor.

Here you can activate the following options:

  • Enable bracket highlighting

    Highlights associated function brackets in color, e.g. when entering syntax. You can select the color under Fonts and Colors.

  • Highlight current line

    Highlights the current selected line in color. You can select the color under Fonts and Colors.

  • Show line numbers

    Shows the line number on the left-hand side in the editor.

  • Show whitespaces

    Shows the spaces in the source code.

  • Mark line modification

    Highlights modified lines in color.

    • Yellow: Line was modified, but the file has not yet been saved
    • Green: Line was modified and file was saved

  • Show indentation guides

    Shows the indentation guides in the source code using guide lines.

  • Enable error highlighting

    Highlights error code.

  • Show autocomplete when typing

    Activates the autocomplete input assistance.
    If autocomplete is not shown on typing, you can open the autocomplete box manually with <Ctrl>+<Space>.

  • Enable word wrap

    Automatically wrap lines that are too long.

  • Highlight occurrences of selected text

    Additional occurrences of the selected text will be highlighted.

System Environment

  • Show symbols of system environment in autocomplete

    If the symbols of the system environment (signals, system variables, environment variables) are not shown in autocomplete, memory is saved This improves the performance when working with a large amount of symbols.

Text Formatting

The editor settings all relate to the CAPL editor and C# editor.

General

Here you can activate the following options:

  • Enable smart indentation

    Inserts spaces into program sequence automatically for improved overview.

  • Automatically insert closing bracket

    If you type "(", for example, the bracket is automatically closed.

  • Automatically insert closing apostrophe

    If you type an apostrophe, a closing apostrophe is automatically inserted.

  • Codepage

    Here you can specify the code page that is used for newly created files.

Tabs

Here you can specify the tab width and the behavior when inserting new tabs.

Fonts and Colors

Here you can select the character set and colors used in the Editor for highlighting certain displayed objects.
The colors selected here are also used, for example, in the Detail View of the CAPL Function Explorer.

A text sample with the character set and the respective color is displayed under Sample.

With [Reset] you can reset the setting to their default.

Messages

Here you can select the hideable messages which are to be displayed or hidden.
