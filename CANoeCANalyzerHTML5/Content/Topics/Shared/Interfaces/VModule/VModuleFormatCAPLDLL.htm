CAPL-DLL

 

XML Element CAPL-DLL

The CAPL-DLL XML element contains additional data for modules of type CAPL-DLL. This element duplicates the information of the CAPL Export Table and is used by the CAPL compiler during the translation of a CAPL program.

  • For Windows-x86 and Windows-x64 platforms, this element is optional.
    If it is not specified, the CAPL compiler uses the CAPL Export Table contained in the Windows DLLs.
  • This element is mandatory for the Linux-x64 platform, since the CAPL compiler cannot load Linux libraries and therefore has no access to the contained CAPL Export Table.

It contains the following XML elements:

XML Element

Description

CDLL_VERSION

Version information identical to the version in the CAPL Export Table.

CallingConvention

Calling convention of the exported functions.

Possible values:

Value

Short Description

CDECL

Corresponds to the setting CAPL_DLL_CDECL in the CAPL Export Table.

PASCAL

Corresponds to the setting CAPL_DLL_PASCAL in the CAPL Export Table.

CAN_MSG_VERSION

Version of the format for parameter transfer of CAN frames.

Possible values: 0…7

Corresponds to the macros CAPL_DLL_CAN_MSG_VER_0 to CAPL_DLL_CAN_MSG_VER_7 in the CAPL Export Table.

LibraryVersion

Exported version number of the CAPL DLL.

Corresponds to the exported variable caplDllLibraryVersion; see Version Control.

VersionCheckMessage

Text to be output if the version check fails.

Corresponds to the exported variable caplDllVersionCheckMessage; see Version Control.

RequiredVersionCheck

Specifies whether a version check must be carried out.

Corresponds to the exported variable caplDllRequireVersionCheck; see Version Control.

ExportTable

List of exported functions.

Contains XML elements of type Function (see below).

CallbackTable

List of CAPL callback functions.

Contains XML elements of type Function (see below).

XML Element Function

The XML element Function describes a single function. Its XML attribute arch specifies the platform(s) for which this description is valid. The XML attributes minAppVersion/maxAppVersion contain an optional restriction regarding the CANoe version of this function.

It contains the following XML elements:

XML Element

Description

FunctionName

Name of the function.

CategoryName

Name of the function category.

HintText

Note describing the function.

UsageContext

Context in which the function may be used.

If this element is not specified, the function is allowed in any context.

Context

Short Description

RT

Realtime context

ANALYSIS

Analysis context

TEST

Test node

CANOE

CANoe

TESTUNIT

Test Unit

TESTUNITGEN

Generated Test Unit

SIMNODE

Node in Simulation Setup

FunctionFlags

Some additional features of the function.

Flag

Short Description

HIDDEN

Function is not displayed in the function list in the CAPL browser.

DEPRECATED

Function is deprecated.

THREADSAFE

Function can be used for parallel execution.

Parameter

List of function parameters.

Contains XML elements of type Parameter (see below).

Result

Value type of the return parameter.

Contains the XML element Type with a type name; see Table with Type Specifications.

ExportSymbol

Identifier of the function in the symbol table of the library.

This element is currently only used for the CaplOnBoard-ARM32 platform.

XML Element Parameter

The XML element Parameter describes a single passing parameter of a function.

It contains the following XML elements:

XML Element

Description

Name

Name of the parameter.

Type

The type of the function parameter; see Table with Type Specifications.

The optional attribute rank defines the dimension of the parameter.

Value

Short Description

0

Scalar

Default value if the attribute rank is not specified.

1

1-dimensional array

2

2-dimensional array

ParameterPassing

How the parameter is passed when the function is called.

Value

Short Description

CallByValue

The parameter serves as input for the function.

Default value if the attribute ParameterPassing is not specified.

CallByReference

The parameter can be used as input and output for the function.
