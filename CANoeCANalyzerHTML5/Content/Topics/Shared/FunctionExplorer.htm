CAPL Function Explorer

Ribbon: Layout Tab|CAPL Functions

All CAPL functions available for the opened nodes are displayed in the CAPL Function Explorer. The display is adapted to the environment configured in CANoe. If a MOST bus is not used in the CANoe configuration, no MOST functions are displayed.

You can add or remove these filters through the Filter ribbon tab.

If the CANoe environment is changed while the CAPL Browser is running, the data in the CAPL Browser can be updated via in the toolbar.

Separation into categories gives a better overview and enables easier access to the functions.

  • Event Handlers
  • <Bus System/Options>
    Contains event handlers for bus- or option-specific events, e.g. CAN, VT System.
  • System
    Contains event handlers for system events.
  • Value Objects
    Contains event handlers for events on symbols.
  • System Functions
  • All Functions
    Contains all functions, sorted alphabetically.
  • General
    Contains the most frequently required CAPL functions.
  • Additional Categories
    Contains CAPL functions, categorized in application areas such as system variables, replay, file access etc.
  • Test Support
    Contains test-specific CAPL functions.
  • Diagnostics
    Contains diagnostic-specific CAPL functions.
  • <Bus Systems/Options>
    Contains bus- and option-specific CAPL functions, e.g. CAN, VT System.
  • User Defined Groups
    Contains all available user-defined functions (local and from included files), ordered by function type. User-defined groups are also displayed.
  • <Opened Files>
    Contains all available user-defined functions (local and from included files), ordered by filename.
    Additionally they are also ordered by function types and user-defined groups are also displayed.
  • <External CAPL DLLs>
    Functions of the assigned CAPL and modeling libraries, e.g. CANOEILNLVECTOR.DLL.

Descriptions of Event Handlers and Functions

In the Detail View at the bottom edge of the Explorer, a short description of the selected category or its functions or event handlers is displayed.
The syntax options are also displayed for the functions. Dragging and dropping from the Detail View is not possible.
The Detail View can be opened and closed with a mouse click via the Splitter function.

In the help (Technical Reference), you will find a detailed description of the syntax, functions, parameters, return values and availability of the functions and event handlers (event procedures) for each function.
Selecting <F1> on a function or opens the help for the respective function.

Inserting Event Handlers and Functions in Text Editor

Event handlers and predefined functions can be inserted into the Text Editor from the Function Explorer via drag and drop.
If you insert a function, the name of the function will be displayed in the source code. When entering parameters, support is provided by the autocomplete input assistance.
If you insert an event handler, the entire body of the procedure is inserted.

Search Option

You can search for a desired text in the Function Explorer. Press [Enter] or to start the search. Only those functions which have been found are then displayed in the tree view (filter function). You can remove the filters with .

The search function does not support the wildcard character ("*"). Instead you can use multiple search terms separated by spaces. All functions will be displayed which contain all search terms in their name.
