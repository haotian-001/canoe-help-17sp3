System Variables for Statistics

 

Certain CAN/LIN/MOST/FlexRay and Ethernet statistics can be evaluated using automatically defined statistics system variables. These system variables are available for each configured network channel (Options dialog). A complete list of system variables for statistics is available in the Symbol Explorer under the namespace _Statistics.

Using these system variables it is possible to make use of bus statistics in different function blocks and windows of the Measurement Setup, for example in the Graphics Window or in program nodes.

System variables are available for the following bus systems:

  • CAN
  • LIN
  • MOST
  • FlexRay
  • Ethernet

Note

  • The system variables for statistics are updated independent of any Statistics Window. For example, even if there is no CAN Statistics Window in the Measurement Setup, all CAN-specific statistic system variables are updated.
  • The system variables for statistics can be added to the windows in the Measurement Setup by using drag and drop from Symbol Explorer or from the corresponding Statistics Window (CAN, LIN, FlexRay, MOST).
  • The values of the system variables for statistics are only available in the Measurement Setup.
