System Variables for Statistics

 

Note

In Vector Ethernet interfaces since 2016 (VN5610A, VN5640) an application channel can be assigned to a segment with several hardware channels. Therefore, additional statistics system variables are generated for each configured Ethernet application channel in a separate namespace and for each potential hardware channel. This allows adequate access to the statistics data of each hardware channel.

Example

The statistics system variable Eth1::LinkSpeed in the namespace of the application channel corresponds to the newly introduced variable for the first hardware channel Eth1::HwChannel1::LinkSpeed.
The statistics system variables in the namespace of the application channel were only intended to be used for downward compatibility and should not be used any longer.
