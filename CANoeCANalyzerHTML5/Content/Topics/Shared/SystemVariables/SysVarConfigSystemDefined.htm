System Variables - System-Defined

Ribbon: Environment Tab|System Variables|System Defined

User-Defined

System-Defined

Name Display

 

On this page automatically created system variables and struct definitions are displayed. You cannot edit, delete or move this type of variable or struct definition. All system variables are displayed in a tree view at the left side of the dialog, the struct definitions in a tree view at the right side of the dialog.

You can open the tree view of the struct definitions via the splitter ().

The following variables and struct definitions are displayed on this page:

  • Statistics variables and struct definitions
  • VT variables
  • XCP variables
  • Variables that are created using COM
  • Variables that are created using CAPL
  • Variables that are created using modeling libraries (DLL files)

Columns

You can modify the displayed columns by moving them via drag and drop to another position.

The following columns are available:

Column Name

Brief Description

System Variables

Variable

Shows the name of each system variable or the associated namespace.

Data type

Shows the data type of a system variable.

Min

Shows the minimum value, if defined.

Max

Shows the maximum value, if defined.

Initial Value

Shows the initial value, if defined.

Read Only

Indicates whether the variable is read only or can be modified.

Analysis Only

Indicates whether a variable is only available in the measuring setup.

Source

Shows the memory location of the variable. This can be a file or the configuration itself.

Sync ERT Update

This column is visible if the Extended Real Time feature is used and only applies for system variables with Source = ERT. If the selective synchronization of ERT variable updates option is activated, the user can manually select if an ERT variable should be synchronized.

Comment

Shows the comment for this namespace/variable, if defined.

Struct Definitions

Struct Definition

Shows the name of each struct definition, the associated members or namespace.

Datatype

Shows the data type of a struct member.

Min

Shows the minimum value of a struct member, if defined.

Max

Shows the maximum value of a struct member, if defined.

Initial Value

Shows the initial value of a struct member, if defined.

Source

Shows the memory location of the struct definition. This can be a file or the configuration itself.

Array Size

If the struct member is an Array-type member, the size of the array is shown here.

Comment

Shows the comment of the namespace/struct definition, if defined.

Value Table

Indicates whether or not a value table is defined for a struct member.
