System-Defined

Name Display

 

On this page, you can specify how much path information of user-definedsystem variables and struct definitions is to be displayed.

Cross Reference

For the Name Display feature itself, a separate help page already exists since this function is also used for other windows.

So that changes can be made on the Name Display page, select the Custom command from the Show Details list.

Columns

You can modify the displayed columns by moving them via drag and drop to another position.

The following columns are available:

Column Name

Brief Description

Namespace

Shows the namespace designation.

Show in Name

If this check box is activated, the namespace is shown in the variable or struct definition name.

Comment

Shows the comment for this namespace, if defined.
