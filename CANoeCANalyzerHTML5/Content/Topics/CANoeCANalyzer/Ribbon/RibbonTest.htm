  • The available symbols depend on the installed options.

Symbol

Description

Test Units group

Test Setup for Test Units

In this window all test configurations and Test Trace Windows within a CANoe configuration are managed.

With a click on the symbol you open the Test Setup for test configurations and test traces.

Test Configuration

In this window you control the sequential execution of one or more test units.

You can add and remove test configurations in the Test Setup for Test Units.

Here can you open the windows of existing test configurations and control their test execution directly during the measurement.

Test Trace Window

The Test Trace Window is to observe and analyze single or multiple tests during the test run.

You can add and remove Test Trace Windows in the Test Setup for Test Units.

Here you open the windows of existing test traces.

Debugger

The debugger enables you to analyze programs that run during CANoe measurement.

Only available if you already opened the debugger for a test unit.

Here you open the debugger for test units.

Test Modules group

Test Setup for Test Modules

A test environment in CANoe consists of a directory structure in which the individual test blocks are inserted.

With a click on the symbol you open the Test Setup for test environments and test modules.

Test Module: Execution

You can use this window to control the execution of test modules.

You can add and remove test environments and their test modules in the Test Setup for Test Modules.

Here can you open the windows of existing test modules and control their test execution directly.

Debugger

The debugger enables you to analyze programs that run during CANoe measurement.

Only available if you already opened the debugger for a test module.

Here you open the debugger for test modules.

LIN group

LIN Slave Conformance Tester

With the LIN Slave Conformance Tester you can check the conformance of a Slave node within a CANoe configuration.

You can add and remove LIN Slave Conformance Tester in the Test Environment for Test Modules window.

Here you open the windows of existing LIN Slave Conformance Testers.

LIN Disturbance Block

With the LIN Disturbance Block you can disturb frames, headers and dominant/recessive bus levels.

You can add and remove LIN Disturbance Blocks in the Test Setup for Test Modules.

Here you open the windows of existing LIN Disturbance Blocks.

LIN Stress Interactive Generator

With the LIN Stress IG block you can configure and send frames, headers, frame errors, header errors and valid/invalid response updates.

You can add and remove LIN Stress Interactive Generators in the Test Setup for Test Modules.

Here you open the windows of existing LIN Stress Interactive Generators.

Conformance Test group

Smart Charging EV

The CANoe Test Package EV generates conformance tests for smart charging.

Here you can open the CANoe Test Package EV if the CANoe Option .SmartCharging is available and the CANoe Test Package EV is installed.
