Symbol

Description

Network group

CANdb++ Editor

Using CANdb++ you can create, visualize and edit CAN databases.

You can assign databases in the Simulation Setup.

Here you open assigned databases in CANdb++.

FIBEX Explorer

Using FIBEX Explorer you can visualize and edit FIBEX files for FlexRay.

With a click on the symbol you open the FIBEX Explorer.

AUTOSAR Explorer

Using AUTOSAR Explorer you can visualize and edit the communication aspects of AUTOSAR system description files and ECU extracts of system descriptions.

With a click on the symbol you open the AUTOSAR Explorer.

AUTOSAR Preprocessor

With the AUTOSAR Preprocessor you can merge individual ARXML extracts (SYS-E) into a system description (SYS-D). In addition, you can use the AUTOSAR Preprocessor with the help of filters and patches, specific elements from the ARXML files to filter out or modify. With the AUTOSAR Preprocessor you thus reduce ARXML files to elements that you need and thus accelerate subsequent work steps.

With a click on the symbol you open the AUTOSAR Preprocessor.

CANoe AUTOSAR Converter

With the CANoe AUTOSAR Converter you can convert ARXML files to vCODM files.

With a click on the symbol you open the CANoe AUTOSAR Converter.

vCDL Editor

With a click on the symbol you open the vCDL Editor.

LDF Explorer

Using LDF Explorer you can create, visualize and edit LDF files and NCF files.

With a click on the symbol you open the LDF Explorer.

Car2x Network Explorer

Using Car2x Network Explorer you can visualize Car2x databases. Furthermore you can edit basic node and message properties.

With a click on the symbol you open the Car2x Network Explorer.

Scenario Editor

With the Scenario Editor you can configure traffic scenarios.

With a click on the symbol you open the Scenario Editor.

CANeds

Using CANeds you can create and edit CANopen Electronic Data Sheets (EDS) and Device Configuration Files (DCF).

With a click on the symbol you open CANeds.

AME Aerospace Message Editor

Using Aerospace Message Editor (AME) you can create and modify messages, signals and virtual links for aerospace bus systems (incl. ICD import, DBC export as well as a library with standardized ARINC-429 words).

With a click on the symbol you open the Aerospace Message Editor.

Security Manager

Using Vector Security Manager you can manage and configure security profiles (e. g. SecOC, diagnostics, authenticationâ¦) for usage in Vector tools.

With a click on the symbol you open the Vector Security Manager.

More group

CAPL Browser

Using CAPL Browser you can write individual CAPL programs for stimulating, simulating, testing and diagnostics.

With a click on the symbol you open CAPL Browser.

Debugger

Using Debugger you can analyze programs that run during CANoe measurement.

Only available if you already opened the debugger for a network node, test module, test unit or application model.

Here you open the debugger for network nodes, test modules, test units and application models.

Panel Designer

Using Panel Designer you can create graphic panels on which the values of symbols can be modified and displayed interactively by the user during simulation.

With a click on the symbol you open the Panel Designer.

FDX Editor

Using FDX Editor you can create and edit FDX descriptions defining a HIL communication via FDX protocol.

With a click on the symbol you open the FDX Editor.

GL Logger Suite

You can use some loggers as network interface devices in CANoe. The logger is connected to the CANoe computer via Ethernet connection.

In the Options dialog you can set the different logger modes.

With a click on the symbol you open the installed logger.

vFlash

Using vFlash you can flash single ECUs interactively via a graphical user interface. Using the automation interface vFlash can also be controlled from CANoe.

With a click on the symbol you open vFlash.

Logging File Conversion

In this dialog you can:

  • save a Logging File in a new format
  • save the contents of the Trace Window and Graphics Window to a file

With a click on the symbol you open the Logging File Configuration dialog.
