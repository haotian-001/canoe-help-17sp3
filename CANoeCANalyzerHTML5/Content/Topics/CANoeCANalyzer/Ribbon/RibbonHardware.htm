Symbol

Description

Channels group

Channel Usage

Here you can configure the number of channels you wish to work with in your active configuration. Besides affecting the measurement, the channel definition also affects the input options of the various configuration dialogs. Only defined channels are offered to you for selection.

With a click on the symbol you open the Options dialog.

Application Channel Mapping

On measurement start the assistant will detect incorrect configured application channels. It will provide a suggestion on how this conflict may be resolved using currently attached network hardware devices.

With a click on the symbol you open the Application Channel Mapping dialog.

Port Configuration

The Port Configuration dialog displays all ports contained in the configuration. You can add and delete ports.

This command is only available with a loaded network-based configuration or with the activated Access network via Ethernet ports option in Options dialog.

With a click on the symbol you open the Port Configuration dialog.

Ethernet Migration Wizard

The Migration Wizard helps you to migrate your channel-based CANoe and interface configuration to a network-based configuration.

This command is only available with a loaded channel-based configuration (legacy).

With a click on the symbol you open the Migration to Network-Based Access window. Here you can start the Migration Wizard.

Network Hardware Configuration

In the Network Hardware Configuration dialog you can modify the settings for the available channels defined in the current configuration.

You can activate/deactivate channels in the Options dialog.

With a click on the symbol you open the Network Hardware Configuration dialog.

VT System group

VT System Configuration

In this dialog you can create or modify a VT System configuration. You can add modules to or remove modules from your configuration. In addition, you can change the order of modules in the configuration via drag and drop.

You can activate/deactivate the VT System in the Options dialog.

With a click on the symbol you open the VT System Configuration dialog.

The VT System Configuration will only be opened if you have activated the VT System in the Options dialog. Otherwise the Options dialog will be opened.

VT System Control

In this dialog you can manually manipulate a previously configured VT System during an active measurement. Outside of the measurement, the channels of the VT System can also be configured and their states stored as start states. In addition, information about the hardware connected to the VT System can be entered in this dialog. This can be used to generate various types of documentation (such as a system overview).

You can add and remove VT System modules in the VT System Configuration dialog.

With a click on the symbol you open the VT System Control dialog.

VT System Tools

  • FPGA Manager
  • Calibration Manager
  • Application Board Designer
  • Firmware Updater

With a click on a symbol you open the corresponding tool.

VIO System group

Configuration

In this dialog, you can create or modify your VIO System configuration.

With a click on the symbol you open the VIO System Configuration dialog.

Tools

  • VIO System Configuration Tool

With a click on the symbol you open the corresponding tool.

Group For EtherCAT

Configuration

With a click on the symbol you open the Configurator for EtherCAT dialog.

Diagnostics

With a click on the symbol you open the Configurator for EtherCAT dialog during the measurement.

Network Adapter Selection

With a click on the symbol you open the Network Adapter Selection for EtherCAT dialog.

Sensors group

Sensor Configuration

In this dialog you can configure option Sensor. You can create different types of channels and configure the dedicated sensors and ECU's. Depending on the use case you can choose which devices are connected to the network interface and which devices shall be simulated by CANoe.

With a click on the symbol you open the Sensor Configuration dialog.

I/O Hardware group

Vector I/O Configuration

The Vector I/O Configuration dialog is used to configure Vector I/O modules, such as IOpiggy, VN1630/VN1640/VN0601 and other I/O modules. The menu bar contains two selection fields.

With a click on the symbol you open the Vector I/O Configuration dialog.

GNSS Configuration

This dialog allows you the configuration of GPS receivers for usage in CANoe. The received data will be available as system variables in CANoe. You can visualize this data directly in the Map Window.

With a click on the symbol you open the GPS Configuration dialog.

Video Window Configuration

In this dialog, you can manage and configure Video Windows.

With a click on the symbol you open the Video Window Configuration dialog.

Other

In this dialog, you can configure the connection to other measurement hardware, e.g. from National Instruments.

With a click on the symbol you open the PortLink Configuration dialog.
