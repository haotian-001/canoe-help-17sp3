Note

  • The available symbols depend on the installed options.
  • The group symbols, e.g. , are only displayed in the Quick Access Toolbar if the group was added there.

  • The ribbon shows a submenu for many created windows/panels when the mouse pointer is positioned over the entry.

Symbol

Description

Symbols group

Symbol Explorer

You can assign symbols to the windows via drag and drop from the Symbol Explorer.

With a click on the symbol you open the Symbol Explorer.

System Variable Configuration

In the System Variables Configuration dialog you can manage the system variables used in CANoe.

With a click on the symbol you open the System Variables Configuration dialog.

Symbol Mapping

In the Symbol Mapping dialog, you can map system variables and signals.

With a click on the symbol you open the Mapping Configuration dialog.

Start Value Window

In the Start Values Window, values can be specified for symbols (system variables, signals, distributed objects). The values are set either at the start of the measurement or when the value is entered.

With a click on the symbol you open the Start Value Window.

More group

Compile All Nodes

With a click on the symbol all CAPL nodes contained in the data flow plan are recompiled.

Documents Window

In the Documents Window, you can compile a list of documents you want to open within a configuration.

With a click on the symbol you open the Documents Window.

Licensing

In the Licensing dialog you get an overview of the found licenses and you can activate and deactivate bus system options.

With a click on the symbol you open the Licensing dialog.

Customer Plugin

With the customer plugin you can add own functions to the ribbon.

This entry is only available with a own .NET assembly.

You can find information about adding a customer plugin in the Readme.txt: Open folder

Tool Couplings

  • Activate/deactivate MATLAB Integration
  • FMI Configuration

Here you can open the Options dialog (MATLAB integration) and the FMI configuration.

Car2x group

Car2x Station Manager

In the Car2x Station Manager, you can configure and manage ITSIntelligent Transport System stations.

With a click on the symbol you open the Car2x Station Manager.

Scenario Manager

In the Scenario Manager, you can manage traffic scenarios in CANoe.

With a click on the symbol you open the Scenario Manager.

Simulink group

Simulink Model Viewer

This window is used to displayMATLAB/Simulink models.

This entry is only visible if any Model Viewer information is available.

With a click on the symbol you open the Simulink Model Viewer.

Simulink Analysis Model

In this dialog, you can configure Simulink models in such a way that you can perform evaluations during the measurement.

This entry is only visible if any Model Viewer information is available.

With a click on the symbol you open the Simulink Analysis Model dialog.

LabVIEW group

LabVIEW Integration

This entry is only visible if you have already activated the LabVIEW Integration under Tool Couplings.

With a click on the symbol you open the LabVIEW Integration dialog.
