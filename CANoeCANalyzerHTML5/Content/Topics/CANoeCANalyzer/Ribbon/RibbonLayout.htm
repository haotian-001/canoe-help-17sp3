  • The available symbols depend on the installed options.

Symbol

Description

Windows group

Cascade

Arranges the windows so that they overlap on the screen.

Tile Horizontally

Arranges the windows uniformly from left to right.

Tile Vertically

Arranges the windows uniformly from top to bottom.

Activate Window

Sets the selected Window to the foreground of the current desktop.

More Windows

Automatic Fit

Select this option in order to adjust the size of the contents of the child windows together with the main window. However, some windows, such as panels, have a fixed size. These windows are not adjusted.
You can also activate/deactivate this option in the Options dialog.

Magnetic Windows

This feature facilitates the window layout. Position and/or size of a window are aligned relatively to adjacent windows. If you move or resize a window there is a capture range depending on adjacent windows.
You can also activate/deactivate this option in the Options dialog.

Windows can also be arranged on other desktops by dragging them with the mouse or using the context menu command Move to of the window's title bar.
