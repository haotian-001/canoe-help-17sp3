Customize Ribbon

Ribbon: Context Menu

In the Ribbon Customization dialog you can adapt the ribbon.

You can create new user-defined tabs or groups and add commands for the groups. Thus you can configure the commands that are most important to you on a separate tab or expand an existing tab to include new groups and commands.

Choose Commands

In this area you can choose individual commands that you want to display on a ribbon tab.

You can have the commands displayed sorted by commands (All Commands) or by tabs (All Tabs).

Customize Ribbon

In this area, all configured tabs are displayed.

Here you can:

  • Activate/deactivate Ribbons and tabs via the selection field
  • Add or remove new tabs
  • Add or remove new groups
  • Add and remove commands for user-defined groups
  • Rename user-defined categories, tabs, or groups
  • Import/export file with customizations

Notes

To make the ribbon easier to read, commands that are not available in the current configuration are hidden. An example would be commands for a bus system that is not being used in the configuration. If, as a result of this, a group or tab does not contain any commands, this element is also hidden. All elements that are hidden in the current configuration are marked with (*).

Please note the following restrictions:

  • Commands cannot be renamed.
  • Predefined standard tabs and groups cannot be renamed.
  • Predefined standard tabs and groups cannot be removed. However, it is possible to hide them.
  • No commands can be removed from standard groups.
  • No commands can be added to standard groups.

Buttons

[New]

  • New Category
    Creates a new category that can be displayed either stand-alone or with multiple categories.
    The order of the categories cannot be changed.
  • New Tab
    Creates a new tab in which groups can be inserted.
  • New Group
    Creates a new group in which commands can be inserted.

[Rename]

  • Here, depending on selection, you can change the name of a user-created category, tab or group. You cannot rename commands.

[Import/Export]

  • Here you can export or import your settings as an XML file.
    After installation of a service pack, you can apply the settings of the previous program version.
    To do so, import the automatically generated ToolBarLayout CANoe_backup.xml file. You can find the file in the directory of your application data (Options dialog).

[Reset]

  • Resetting the selected ribbon tab
  • Reset all customizations

Context Menu

New Category
Creates a new category that can be displayed either stand-alone or with multiple categories.
The order of the categories cannot be changed.

New Tab
Creates a new tab in which groups can be inserted.

New Group
Creates a new group in which commands can be inserted.

Rename
Opens a dialog for renaming a selected category, tab or group. Only user-defined categories, tabs or groups can be renamed.

Remove
Deletes the selected user-defined category, tab or group.

Add
Copies a complete tab from the command selection on the left to the selected category on the right.
Copies a complete group from the command selection on the left to the selected tab on the right.
Copies an individually selected command on the left to the selected group on the right.

Move Down
Moves a selection down.

Move Down
Moves a selection up.

Add/Remove Commands

You can add or remove commands

  • with the arrow buttons for a group
  • add or delete commands via the context menu

Example

Creating a category with tab, group and command

  1. Use the [New] button to create a new category.
    In the overview the new category is automatically created with a tab and a group.
  2. Select the newly inserted group.
  3. In the command view on the left, select the display All Commands.
  4. Select a command and add it with the context menu command Add for the selected group.
