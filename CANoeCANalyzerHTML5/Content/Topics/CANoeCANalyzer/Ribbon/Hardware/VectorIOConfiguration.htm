VN I/O Modules

Ribbon: Hardware Tab|Vector I/O|VN I/O Modules

Dependent on the selected module the following settings are available:

Connector Overview

Note

The exact pin assignment for the individual VN-I/O modules can be found in the supplied hardware manual.

Pin Configuration (only in Custom mode available)

The custom configuration lets you customize the functionality of the individual pins. You can adjust the settings in the table. One row in the table corresponds to one pin in the connector.

The columns in the table contain the following information:

Column

Description

Group

Signal group of the pin.

Pin

Pin number on the connector.

Name

Name of the pin, which you can modify.
The pin name is used to generate the system variables.

Disabled

If you select this option, the corresponding pin is not used.

Function

Function of the pin.
The following functions are available for the various pins:

Function

System Variable

Note

<Pin name>

Analog: Voltage in volts

The pin acts as a digital or analog input.

Digital: Level (Low = 0, High = 1)

Output

<Pin name>

Output level (Low = 0, High = 1)

The pin acts as a digital or analog output.

Output (oc)

<Pin name>

 

The pin acts as a digital output with open collector behavior.

Output (pp)

<Pin name>

 

The pin acts as a digital output with push pull behavior.

Output (pp) / Activation Line

<Pin name>

 

The pin acts as a digital output with push pull behavior. It can be used for controlling an Ethernet activation line.

Output (hold)

<Pin name>

 

The pin acts as a digital output. The minimum lengths of output high and low phases are user configurable.

Input / Output

<Pin name>

 

The pin works bidirectional. When set to high, it will drive a high output. When set to low, it will act as an input and report measured values.

Relay

<Pin name>

 

Two adjacent pins in relay mode can be short circuited via a semiconductor switch.

PWM

<Pin name>

 

The pin outputs a PWM signal with user configurable frequency and duty cycle.

Capture

<Pin name>

 

The pin captures PWM signals and measures the frequency and duty cycle.

Acquisition

Cyclic Acquisition

Cycle time: Configures the interval in which measurement values will be captured by the I/O device.

Digital Trigger

Generate update events when input value changes: If checked, the I/O device will capture additional samples, whenever the values of the input pins change.
