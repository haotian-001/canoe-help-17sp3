VN8900: Operating System Update

Ribbon: Hardware Tab|Network Hardware|Operating System Update

In certain rare cases, when a software or driver update doesn't suffice, it can be necessary to update the operating system (including all pre-installed drivers) running on the VN8900. This is done by loading a memory image (provided by Vector) to the internal flash memory of the device.

To prepare for this operation, unplug the VN8900 from the power supply. Open the device (for details, see the device manual) and remove the network hardware insert.

Remove the CFast card from the drive of the device and connect the card to the user computer using the provided card reader.

Updating the Operating System

In CANoe, the memory image can only be loaded via the Network Hardware Configuration dialog (Hardware ribbon tab). If at least one of the application channels currently in use is assigned to a hardware channel on the VN8900 in the driver configuration, the [Device] button appears in the corresponding dialog. Click on this button and select Update Operating System... from the menu. This opens the Vector VN8900 Flash Program dialog. Proceed as follows:

  1. In the upper part of the window, enter the path to the image file provided by Vector support.

  2. Under Device storage medium, select the system drive of the connected VN8900; it is labeled VN8900Sys.

  3. Now start the update by clicking the [Update] button.

    You can observe the update progress in the lower part of the dialog. Please note that the update will take a few minutes.

    Notice!

    • If you select another USB drive connected to the computer, instead of selecting the VN8900 drive as the storage medium, all files on the selected drive will be overwritten.

    • Do not disconnect the device during the update.

    • When the device is connected to the computer via its internal USB port, the VN8900's internal Flash memory drives are also visible in Windows Explorer.

      If you delete or modify files via this connection, the device may no longer operate correctly.

Once the operating system is updated successfully, re-insert the CFast card in the device's drive and re-mount the module for the network hardware (for details, see the device manual). You can now use the device in the usual way with your computer using the external USB port. Depending on the exact contents of the image that you loaded and the installed CANoe version on the user computer, you may first need to execute a software or driver update on the device.

Note

First start-up may take up to two minutes while the D1 (or "staus") LED keeps blinking orange.
This is because some final installation steps are performed when the system is booted the first time.
