Channel specific

Tx Request

By activating TxRq a time stamp is preserved for each message that is written in the transmit register of the CAN controller. The message is reported to CANoe as a TXRQ message. This makes it possible to perform difference measurements between the transmit request and time of transmission.

Transmission Error Notification

When sending a message you can set the number of transmission attempts (single shot mode). If it wasn't possible to send the message and transmission error notification is active you will be notified in CAPL with a TxReq message and the TxFailed message selector.

CAN FD specific

Protocol Exception

With activated protocol exception support the CAN controller switches to bus integration state when receiving a recessive res bit; additionally a message will be displayed in the Write Window. It will not send an Error Frame.

Edge Filter

IN CAN FD mode the edge filter is activated by default. It prevents that short dominant phases (<2 BTL-Zyklen) during bus integration are being interpreted as synchronization edge.

SSP-Offset

The Secondary Sampling Point Offset (SSP) is used for the compensation of the Transmitting Node Delay (TD). To compensate the TD the transmitter must be able to compare its transmitted bit to the current bit on the CAN network.
The SSP is then measured from the beginning of the transmitters received bit.
As of baudrates = 1Mbit/s the TD compensation is activated automatically. Therefore for the SSP offset a default value is used. The default value can be overwritten here.

CAN Disturbance Interface

You have to activate the interface for the corresponding channel in order to use the disturbance functions of the CAN Disturbance Interface.

As an option, you can assign a Device ID. The standard device ID corresponds to the application number of the CAN channel.

Note

The device ID is required for various CAPL functions and unambiguously identifies the CAN Disturbance Interface. It is not possible to use the same device ID for another CAN channel.
