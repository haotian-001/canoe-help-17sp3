CAN Controller

Mode

You can operate the channel either in CAN or in CAN FD mode.

You can activate the CAN FD mode when using Vector Informatik interfaces with CAN FD support (CAN with flexible data rate).

Baudrate

When you change the baudrate, the CAN controller parameters are automatically re-calculated. However, you can set specific values at any time on the Setup page.

CAN mode: If you use a database that contains baudrate information you can select the Use database settings option to accept this values.
