File

Ribbon: File Tab

When you start CANoe, you can

  • load a configuration
  • create a new configuration
  • get information and help about the program
  • open sample configurations
  • open the Vector Support Assistant
  • close the program

If you activate the Load the last used configuration at application start option, the last loaded configuration will be loaded automatically the next time CANoe is started.

The other functions are only available after loading a configuration.

New

Available Templates

Here you will find an overview of templates, sorted by the various bus systems.

Create Configuration From Template

Select a template and click Create Configuration From Template. In the Create Configuration dialog, select a configuration name and a directory. If you activate the option Create directory for configuration, a subdirectory with the name of the configuration will be created automatically. The new configuration has the settings of the selected template.You can also open the Create Configuration dialog by double-clicking on a template.

Create Configuration From 'venvironment.yaml'

Click on Create Configuration From 'venvironment.yaml' to open the Create Configuration From 'venvironment.yaml' dialog. Select here the directory where the YAML file is located. If scenarios are defined in the YAML file, you can also select a scenario here whose settings are to be adopted for the configuration. If no scenarios are defined, the Default scenario is selected.

Click [OK] to open the Create Configuration dialog. Select a configuration name and a directory here. If you activate the option Create directory for configuration, a subdirectory with the name of the configuration will be created automatically.

Note

Please note that the full feature set of the current venvironment.yaml schema is not supported during import.

Delete Template

Under Available Templates select the template you want to delete and click Delete Template.

Save as Template

Here you can save an open configuration as a template. No directories are saved.

After saving, it is displayed under Available Templates|User Defined.

Template Details

Here you see the following details for the template selected under Available Templates:

  • Filename
    The name under which the template is saved is shown here.
  • Channels
    The configured channels of the template are shown here.
  • Display Name
    The name that is included in the template list is shown here. You can change the name here.
  • Category
    The category in which the template is listed is shown here. You can move the template to a new or existing category by entering a category here.
  • Description
    You can enter a description for your template here.

Sample Configurations

Here you will find an overview of templates, sorted by the various bus systems.

Open Sample Configuration: Opens the selected sample configuration.

Description of the Sample Configurations: Opens sample configuration help.

Details

Here you see the following details for the sample configuration selected under Sample Configurations:

  • Filename
    The memory location of the sample configuration is shown here.
  • Description
    A short description of the sample configuration is shown here. You can find a detailed description in the help.

Options

In the global Options dialog you can configure global settings for the current configuration and for the application.

With a click on the symbol you open the Options dialog.

Support

Using Vector Support Assistant you can assemble all necessary system files and configuration files into a report (VSUPPORTREP file). The necessary files are sent compressed and password-protected to the Vector Support.

See also Vector Support Assistant File Descriptions.

With a click on the symbol you open the Vector Support Assistant.
