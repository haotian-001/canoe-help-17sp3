Symbol Mapping: Context Menus

 

Context Menu within the Table

Add Mapping Relation...
Adds a new (empty) mapping relation to the mapping list.

Add Application Layer Object...
Opens the Symbol Selection dialog. Here you can select a value of a application layer object as source or destination for a mapping relation.

Add Signal...
Opens the Symbol Selection dialog. Here you can select a signal as source or destination for a mapping relation.

Add Variable...
Opens the Symbol Selection dialog. Here you can select a variable as source or destination for a mapping relation.

Add System Variable Namespace...
Opens the Symbol Selection dialog. Here you can select a namespace as source or destination for a mapping relation.

Activate
Activates the selected lines.

Deactivate
Deactivates the selected lines.

Deactivate All Invalid Mappings
Deactivates all invalid mappings (with error icon).
This function is especially useful because the dialog can only be closed (using the [OK] button) if all active mapping relations are valid.

Delete
The selected lines are removed from the list.

Delete All
All lines are removed from the list.

Cut
Cuts the selected lines and copies into the clipboard.

Copy
Copies the selected lines into the clipboard.

Paste
Pasts the content of the clipboard above the selected line.

Formula Assistant
Opens the Formula Assistant, that supports the input of Factor and Offset.

Context Menu of the Column Headers

Best Fit
The column width of the selected column is optimized to the content of all cells of the column.

Best Fit (All Columns)
The column width of all columns is optimized to the content of all cells of the respective column.
