Context Menu: Toolbar

The toolbar provides the following icons for the configuration.

Note on Export/Import of VMAP Files

Import mapping relations only from an export of the same CANoe version. The VMAP format is subject to continuous development and may therefore not be compatible with another version.

If you want to exchange mapping relations between different CANoe versions, you must first save the CANoe configuration in the desired version before exchanging mapping relations by export/import.
