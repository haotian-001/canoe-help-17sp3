Symbol Mapping: Formula Assistant

Context Menu: Formula Assistant

You can open the Formula Assistant dialog via the context menu of the mapping list or via in the toolbar of the Symbol Mapping dialog. The Formula Assistant provides various input options and automatic calculations for factor and offset. When you confirm the dialog with [OK], the calculated values are applied to the mapping list.

Input by Sampling Points

Here, you can enter two sampling points that will be used automatically as the basis for calculating factor and offset.

Example:
The output voltage of a power section can be set using a control voltage. It is known that the control voltage can be set between 0 V and 3 V and that this yields an output voltage between 0.2 V and 12.2 V. The two resulting sampling points for the linear conversion are thus:

Point 1: 0 and 0.2
Point 2: 3 and 12.2

If you enter these sampling points at the bottom of the formula assistant, the values for offset 0.2 and factor 4 appear on the right.
