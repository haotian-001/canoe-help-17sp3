Configuration: SIL Kit

Ribbon: File Tab|Options|Extensions|SIL Kit

Use the Enable SIL Kit option to specify whether you want to enable the SIL Kit for the current configuration.

In the Registry URI input field, you specify the URI under which the registry was started. CANoe joins the simulation of this registry as a participant when the measurement is started. With the registry URI, several different simulations can be run simultaneously on one computer. All simulation participants that are to work together on a common simulation must use the same registry URI. If you leave this field empty, CANoe joins the default registry that CANoe starts at measurement startup.

In the Participant name input field, enter the name that CANoe should use when it joins a simulation with SIL Kit. Within a simulation domain, all participants must have a unique name.

For Configuration file, select the name and path of a text file containing configuration parameters for the SIL Kit. The runtime component of the SIL Kit needs these parameters to participate in a distributed simulation. The file must be in UTF8 format with byte order mark or in ASCII format. The file is read by CANoe at the start of measurement and its content is passed to the runtime component of the SIL Kit. CANoe itself does not evaluate the parameters from the configuration file. If you leave this field empty, CANoe uses an empty participant configuration for the SIL Kit simulation.

In the input field Duration for a simulation step you can configure the time length of a single simulation step for the working mode Simulated Bus (Slave Mode). Leave this field empty to use the default value of one tenth of a millisecond. For other working modes this parameter has no effect.

The input field Timeout for lifecycle events is also only relevant for the working mode Simulated Bus (Slave Mode). In this mode, the start and stop of the measurement is synchronized with the other simulation participants of the SIL Kit. The value of this input field determines the maximum time CANoe waits for the other simulation participants. Leave this field empty to use the default value of 30 seconds.
