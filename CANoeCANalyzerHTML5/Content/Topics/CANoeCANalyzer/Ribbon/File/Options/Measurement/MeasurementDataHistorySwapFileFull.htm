Swap File Full

Ribbon: File Tab|Options|Measurement|Data History

CANoe stores data for the history of the analysis window in a global buffer on the hard disk.

Your data buffer is now full, and old data is deleted from it bit by bit. This slows down the system.

You can now deactivate swapping to increase the system speed. The previous history is deleted here, and the data is then no longer available in the windows. Deletion of the data takes some time, and data could be lost during this time, e.g. for Logging Blocks as well. The speed will increase when this is complete.

If you do not want to deactivate swapping, select [Continue Swapping].

The settings for the swap file can be changed in the Options dialog. Here you can also select a faster hard disk or a larger file, for example, so that the system is fast enough.
