Libraries in the Simulation/Test/Measurement Setup

You can write own libraries (Windows DLLs or Linux SOs) with functions for CAPL programs.

To be able to use the functions in CAPL programs, you must add the library or a module description file to the lists.

Here you can select, if you want to use the library for the Simulation/Test Setup and/or for the Measurement Setup.
If the library should be used in Simulation/Test Setup and Measurement Setup, you must select the library for both areas.

The listed libraries may be activated or deactivated. Deactivated libraries behave like libraries which have not been added to the list.
