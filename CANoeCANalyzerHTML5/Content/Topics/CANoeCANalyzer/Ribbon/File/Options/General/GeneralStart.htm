Configuration: Windows/Program Start

Ribbon: File Tab|Options|General|Windows/Program Start

Note

These settings are configuration-independent settings of CANoe.

Windows Start

Here you can configure the action that has to be performed when you start Windows.

  • Perform no action
  • Activate CANoe quick start feature
    If activated, parts of CANoe are copied to the main memory when you start Windows.
    Thereby all subsequent starts of CANoe are accelerated.
  • Start CANoe and measurement
    When you activated this option CANoe will be started automatically next time you start Windows. The last opened configuration will be loaded/used and the measurement will be started. When you exit Windows, the measurement will be stopped and CANoe will be shut down regularly. A possible use case is using a computer installed in a vehicle as a data logger that shall start logging every time the vehicle is started.
    This setting remains active until you manually deactivate it (here in this dialog) or you install a new version of CANoe.

    		•

    Note

    This setting is user-specific and not a system setting (Admin rights would be needed in this case). Since CANoe cannot be started like a system service in a non-interactive environment, this feature needs a user login on the next system start.

Save configuration when exiting Windows
When you activate this check box the configuration will automatically be saved on exiting windows after changing the configuration with activated Start CANoe and measurement option. This may be useful for automated measurement, e.g. when you configured a Logging block to increment the logging filename automatically, the configuration has to be saved to avoid that your logging files will be overwritten (same names) during the next measurement.

Program Start

If you activated the option Permit only a single instance of CANoe, the program can be opened only once.

Otherwise, you can open CANoe multiple times.

Deactivate the Check for newer version at program start option if you do not want to check for available newer versions at program start. The Vector Application Updater must be installed and there must be an Internet connection for the check. The option is not displayed if the Vector Application Updater is not installed.

When a new version is available, the icon is displayed in the status bar of the program.

Clicking on the icon starts the Vector Application Updater, where new versions can be downloaded and installed.
