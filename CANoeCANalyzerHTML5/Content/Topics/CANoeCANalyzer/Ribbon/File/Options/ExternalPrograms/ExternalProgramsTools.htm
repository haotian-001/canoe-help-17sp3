Configuration: Tools

Ribbon: File Tab|Options|External Programs|Tools

On this page you can select a tool to open the displayed file types.

With [Change...] you can open a dialog to select a new tool. With [Reset] you can reset this setting to its default.

The [Reset All] button can be used to reset all tools to their default values.

Note

These settings are configuration-independent settings of CANoe.

List of Tools

DBC file editor (CANdb++)
The specified editor is used to display and edit DBC files (CANdb++).
With the default "CANdb" the application from the CANoe Exec32 directory will be started.

LDF file editor
The specified editor is used to display and edit LDF files. With the default LDF Explorer will be started.

MOST function catalog editor
The specified editor is used to display and edit MOST function catalog.

FlexRay FIBEX file editor
The specified editor is used to display and edit FlexRay FIBEX files.

FlexRay AUTOSAR system description editor

CAN AUTOSAR system description editor

CDD file editor (CANdelaStudio)
The specified editor is used to display and edit CDD files.
With the default CANdelaStudio View application (provided with CANoe) will be started.
Alternatively you can enter the path to the full version of CANdelaStudio.

ODX/PDX file editor (ODX Studio)
The specified editor is used to display ODX/PDX files.

MDX file editor
The specified editor is used to open MDX files.

ASCII logging file viewer
The specified viewer is used to display ASCII logging files.

MDF logging file viewer
The specified viewer is used to display MDF logging files.
Signal-based MDF files can be opened in vSignalyzer; see CANSystemDemo — vSignalyzer.vbs (COM Sample).

BLF logging file viewer
The specified viewer is used to display BLF logging files.

CAPL file editor
The specified editor is used to display and edit CAPL files.
When you enter CAPLBrowser.EXE the CAPL Browser will be used which enables you to compile files and select CANdb symbols.
When specifying a VisualBasic Script (*.vbs) any editor may be configured and started based on the settings submitted by CANoe.
As an alternative you can open a CAPL file in a different editor by entering this editors name; in this case you have no compiler integration.

.NET file editor
The specified editor is used to display and edit .NET application files.

Text file editor
The specified editor is used to display and edit text files.

HTML file editor
The specified editor is used to display and edit HTML files.

XML test module editor
The specified editor is used to display and edit XML specification of XML test modules.

Editor for parameter files in test units
The specified editor is used to open and edit test unit parameter files. Select here Excel or a text editor.

A2L file editor

ZIP Program

Note on Names/Descriptors and on Zip/Unzip

Some Zip programs have problems with certain characters, e.g., umlauts (mutated vowels; "ä", "ö", "ü"), "ß", space.

  • These characters are altered during a zip/unzip operation without any feedback/confirmation.

  • This results in configurations that do not work (properly) any longer after unzipping.

  • Please use only names/descriptors that do not contain any of these characters.
