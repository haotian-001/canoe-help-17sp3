Options » Appearance » Program

Configuration: Program

Ribbon: File Tab|Options|Appearance|Program

Note

These settings are configuration-independent settings of CANoe.

Language

You can configure the language for all program functions.
CANoe must be reopened before the new language setting takes effect.

Date and Time Settings

Via the Format list you can set the format of the ASCII and Write Window time stamps independent of the selected GUI language.

There are three formats:

  • Die Dez 17 15:06:04.278 2013 (German 24h)
  • Tue Dec 17 03:06:04.278 pm 2013 (English 12h am/pm)
  • Tue Dec 17 15:06:04.278 2013 (English 24h)

Application Window

If you activate the Hide the program window, when it is minimized option the CANoe program window will be hidden, if it is minimized.

Instead of that a tray icon is displayed in the status bar.

Windows Settings

  • Fit windows automatically

    You can activate this menu command to have the window layout automatically adjusted to the new size of the main window if the size of the main window is changed. If this function is disabled, the sizes of subordinate windows remain unchanged, even if the size of the main window is changed.
  • Magnetic windows
    When you activate this function the individual windows of a configuration automatically attempt to dock to one another after being moved.
