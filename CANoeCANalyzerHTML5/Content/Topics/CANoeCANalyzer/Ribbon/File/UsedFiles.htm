Files Used

Ribbon: File Tab|Configuration Overview|Files Used in this Configuration

Her you can find an overview of the files used by the active configuration.

You have select from the following:

  • With [Show File List] a list of the used files is determined and displayed.
  • With [Save File List] a file list (*.LST) is created.
  • With [Zip Files] a ZIP archive is created using the file list.

Example

Create a ZIP archive of the configuration after measurement stop

  • Select Configuration Overview on File ribbon tab after measurement stop. Under Used Files in this Configuration a list of all used files is displayed.
  • Open the Pack Files into Archive dialog via [Zip Files] button to select an archive file.
  • Select a name for the ZIP file and the storage folder.

Additionally to the used files in the configuration a TXT file is stored in the ZIP file. It contains system information, driver information, installed components and the license. In the ZIP file the path structure remains the same.

Note on Names/Descriptors and on Zip/Unzip

Some Zip programs have problems with certain characters, e.g., umlauts (mutated vowels; "ä", "ö", "ü"), "ß", space.

  • These characters are altered during a zip/unzip operation without any feedback/confirmation.

  • This results in configurations that do not work (properly) any longer after unzipping.

  • Please use only names/descriptors that do not contain any of these characters.
