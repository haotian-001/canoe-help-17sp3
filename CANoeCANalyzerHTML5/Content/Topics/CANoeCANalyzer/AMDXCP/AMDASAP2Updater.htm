ASAP2 Updater

 

The Option AMD/XCP comes with an ASAP2 Updater tool. This tool enables you to update measurement object addresses in an existing A2L file based on the corresponding Linker Map file. An address update is needed e.g. to insert the new addresses for the measurement objects into the A2L file after performing a software update on the ECU.

Please note that only the ASAP2 Updater tool comes with the Option AMD/XCP.
Vector offers other tools for creating and merging A2L files, but these are not included with the option.

  • The ASAP2 Updater can be launched via command line, so you can embed it automatically into build and update processes: <CANoe Installation>\ASAP2Updater\ASAP2Updater.EXE
  • A GUI that lets you execute address updates interactively is also available: <CANoe Installation>\ASAP2Updater\ASAP2UpdaterGUI.EXE
  • User manual: <CANoe Installation>\ASAP2Updater\Manual.PDF
