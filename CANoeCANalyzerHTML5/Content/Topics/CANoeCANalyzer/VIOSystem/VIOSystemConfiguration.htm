VIO System Configuration

Ribbon: Hardware Tab|Configuration (VIO System)

In the VIO System Configuration dialog, you can create or modify your VIO System configuration.

Open Dialog

You can open the VIO System Configuration dialog via the menu path Hardware| Configuration (VIO System).

Menu Bar

The Configuration menu provides the following options:

  • Create New Configuration
    Replaces the current configuration with a new configuration.

  • Open Card Capability Description Folder...
    Opens the folder that contains the Card Capability Description (CCD) files of the I/O cards. Save new CCD files in this folder to make them available in CANoe.

Pages

Hardware

The Hardware page shows the configured VIO System devices with their cards and channels in a tree view.

Toolbar

The toolbar at the top of the page provides the following functionality:

  • Add VIO System
    Opens a menu in which you can choose from the following options:

    • New VIO System
      Initiates a search for a VIO System in a different IP subnet.

    • VIO System From the Network...
      Allows the preconfiguration of a VIO System.

    • Available VIO Systems
      Lists all available VIO System devices.

  • Add Card
    Opens a menu in which you can add I/O cards to a selected VIO System.

  • Adapt to Hardware...
    Opens a dialog in which you can adapt your configuration to the connected VIO System.

Configuration Functions

The VIO System tree elements can be configured as follows:

  • Rename Cards

    To rename a card, select the card in the tree view and enter the new name in the field Card name of the Card Configuration area.

  • Card and Channel Configuration

    Select a card or a channel in the tree view. Depending on the selected I/O card or channel, the available configuration options are displayed in the Card Configuration or Channel Configuration area.

  • Multi-Selection of Channels

    Multi-selection is supported so that multiple channels can be adjusted at the same time. The new setting entered in the Channel Configuration area is then made accordingly for all selected channels.

  • Reset Channel To Default Value

    In the Channel Configuration area, a small square to the right of the drop-down list indicates whether a value deviates from the default (square is filled in red) or not (square is not filled).

    To reset a value, click on the small square.

Visualization Options

The VIO System is visualized as follows:

  • Assembled VIO System

    In the VIO System Configuration area, a symbolic front view of the VIO System selected in the tree view is displayed. The view shows all inserted cards at their respective positions in the rack.

    Show Image

  • Pin Assignments of I/O Cards

    In the Card Configuration area, the Pin Assignment of the I/O card selected in the tree view is displayed.

    Show Image

Tasks

The Tasks page shows the measurement and stimulation/settings tasks of the configured VIO System devices.

Toolbar

The toolbar at the top of the page provides the following functionality:

  • Add Empty Measurement Task
    Adds an empty measurement task to the selected VIO System.

  • Add Empty Stimulation/Settings Task
    Adds an empty stimulation/settings task to the selected VIO System.

  • Delete All Tasks
    Deletes all measurement and stimulation/settings tasks. To only delete specific tasks, use the context menu command of the respective node or task.

Configuration Functions

The VIO System tasks can be configured as follows:

  • Add Default Tasks
    To add tasks with default values, select Measurement tasks or Stimulation/settings tasks in the tree view and click the context menu command Add Default Tasks For|All Cards In The VIO System or click the entry for a specific I/O card.

  • Rename Tasks
    To rename a task, select the task in the tree view and enter a new name in the field Task name of the Task Configuration area.

  • Task Configuration
    Select a task in the tree view. Depending on the selected task, the available configuration options are displayed in the Measurement Task Configuration or Stimulation/Settings Task Configuration area.

  • Add Value
    To add a value to a task, click  Add Values... in the Task Configuration area. In the  Add Values dialog that opens, select the physical value you want to add and click [OK]. The physical value mapping is added to the table with a default system variable name.

  • Change Value Mapping
    To change a value mapping, select the mapping in the table of the Task Configuration area and click the [...] button. In the  Change Value Mapping dialog that opens, select the physical value you want to assign and click [OK]. The physical value mapping is updated.
