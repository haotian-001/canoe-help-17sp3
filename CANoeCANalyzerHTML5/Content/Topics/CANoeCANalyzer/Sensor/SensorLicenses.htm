Required Licenses to Use Option .Sensor

 

In general, you can always configure and edit all channels. However, you need the corresponding licenses to start the measurement procedure.

You CANoe can display your licenses in the Options dialog.

CANoe Basic License

The SPI, UART and I2C protocols are included in the CANoe basic license. This allows you to

  • Configure channels
  • Edit channels
  • Start measurements

Required hardware: VT System module VT2710

License for the .Sensor Option

The PSI5 and SENT protocols are included in the license for the .Sensor option. This allows you to

  • Configure channels
  • Edit channels
  • Start measurements

Required hardware: VT System module VT2710

Standalone Basic License

The Standalone Basic license required for the standalone mode includes the license for the .Sensor option. This allows you to

  • Start measurements

Required hardware: VT6000 - Realtime modules and VT2710 module
