Configuration of the UART Sensor Channel

Ribbon|Hardware Tab|Protocol Configuration|UART

The elements and the settings of the UART, RS485 and LVDS sensor channel are described in this section. All system variables available for a channel are also described here.

Overview of the UART, RS485 and LVDS Network Elements

Element

Description/Configuration


Network

UART Settings

No configuration settings are required.


Channel

A channel describes the interconnection between a master and one or more slaves.

All the settings that apply to the elements within a channel are configured on this level.

UART Channel Settings

  • Assigned hardware channel
    Specifies which physical hardware channel is assigned to this sensor channel. In case no channels have been configured, the sensor channel will be automatically deactivated at the start of the measurement.

Protocol Settings

  • Baudrate
    Specifies the data rate of the communication on this channel
  • Number of data bits
    Specifies the number of data bits transmitted within a frame.
  • Number of stop bits
    Specifies the number of stop bits transmitted at the end of each frame.
  • Parity mode
    Specifies if a parity bit is used and how. The possible settings are:
  • Odd: The parity bit completes the bit chain in order to get an odd number of bits showing a one. This means the parity bit is 1 if the quantity of the other ones in the frame is even.
  • Even: The parity bit completes the bit chain in order to get an even number of bits showing a one. This means the parity bit is 1 if the quantity of the other ones in the frame is odd.
  • Mark: The parity bit is always 1.
  • Space: The parity bit is always 0.

Voltage Levels (only for UART channels)

Here you can specify the voltage levels which are used for the communication on this channel. Basically there are two options:

  • RS232: The logical one is implemented by a negative voltage, the logical zero by a positive voltage.
  • User-defined: In this mode the level for the logical zero is set on 0 V. The level for the logical one can be freely selected. Depending on the interface hardware used, the voltage can be set directly in the Sensor Configuration dialog or in the CANoeVT System Configuration dialog.
