Model Generation Wizard

Ribbon: Simulation Tab|Model Generation Wizard

General

The model generation provides a comfortable way to create a CANoe configuration for remaining bus simulation with help of a database. This CANoe configuration uses the original send model, network management and offers an specific user interface with panels.

The Model Generation Wizard generates a simulation model (CANoe configuration) with extensive node simulation and operation panels:

  • Node Simulation
  • Interaction Layer
  • Network Management (OSEK / AUTOSAR)
  • Panels
  • Disturbances for CRC/CHK, message counters and cyclically transmitted messages can be defined directly from the panels.
  • A NM Panel displays the IL and NM status of the simulated nodes and allows the control of the NM requests and the NM and IL activity of the simulated nodes.

CAN, Ethernet and FlexRay networks are supported by the Model Generation Wizard.

Various generation options can be optionally enabled or disabled (suitable defaults are applied on the selection of a database).

With the installation of an OEM add-ons the model generation wizard supports the OEM specific models too.

Note

  • CANoe is started automatically.
    The Simulation creation runs about 30 seconds up to several minutes, depending on your hardware configuration and the complexity of the model. You can track the status of the generation within the Write Window of CANoe.
  • The run of the automatic simulation creation requires the confirmation of the disclaimer of CANoe.
  • If old CAPL files are found during the generation process, they will be kept. But be aware: parts of the CAPL files may be deleted and will be newly generated. These sections are marked by special comments.
