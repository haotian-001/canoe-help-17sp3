Installation: OptoLyzer (MOST50)

 

Install the OptoLyzer G2 3050e or OptoLyzer G2 3050e production by following the directions in the OptoLyzer user manual.

Notes

Note that a license for the Vector OptoLyzer Integration Package (OIP) is required, either on Vector hardware or a USB dongle, or as a license key.

The OptoLyzer G2 is addressed on the following Ethernet ports. If you are using a firewall, it must not block these ports.

Spy: 27998

Node: 27999

The computer automatically assigns the ports. If you want to use specific ports, you need to adapt the CAN.INI file.

The following code is valid for the OptoLyzer G2 3050e (MOST50) on channel 1:

[OptoLyzerG2_50_1]
MyPortNode=
MyPortSpy=

Using TCP/IP improvement software (e.g. Vodafone Mobile Client) can lead to data losses during communication between the OptoLyzer G2 and the computer. If necessary please remove this software.

Sync line for spy channel time stamps

If you want to synchronize the OptoLyzer time stamps of the spy channel with the time stamps of any Vector network interfaces that are connected in parallel, you will need a sync line. It connects the Trigger Extension Box of the OptoLyzer with the sync input of the Vector hardware.

Schema:

                                            OptoLyzer OL3050e
Vector Sync                                 +-----+    +-----+
                                            |  9  7  5  3  1 |
   GND (brown) o----o GND (Pin7)            | [] [] [] [] [] |
Signal (green) o----o Trigger Input (Pin8)  | [] [] [] [] [] |
                                            | 10  8  6  4  2 |
                                            +----------------+
                                                  Trigger

Trigger input pin:

Pin8 is used for the trigger input by default. If you use different circuitry settings (pins 2, 4, 6 and 8 are available) you will need to adapt the CAN.INI file.

For OptoLyzer OL3050e on channel MOST 1 the following section is relevant:

[OptoLyzerG2_50_1]
; 0x80 -> pin8 (default); 0x40 -> pin6; 0x20 -> pin4; 0x10 -> pin2
HwSyncTriggerMask=0x80
