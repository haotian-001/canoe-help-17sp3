Installation: OptoLyzer (MOST150)

 

Install the OptoLyzer G2 3150o or OptoLyzer G2 3150o production corresponding to the OptoLyzer user manual.

Notes

Note that a license for the OptoLyzer Integration Package (OIP) of Vector on a Vector hardware, USB dongle, or as a license key is required.

The OptoLyzer G2 is addressed on the following Ethernet ports. A potentially installed firewall must not block these ports.

Spy: 27998

Node: 27999

The computer automatically assigns the ports. However, if specific ports are used the file CAN.INI has to be adapted.

The following section is valid for the OptoLyzer G2 3150o (MOST150) on channel 1:

[OptoLyzerG2_1]
MyPortNode=
MyPortSpy=

The usage of TCP/IP improvement software (e.g. Vodafone Mobile Client) can lead to data loss during communication between the OptoLyzer G2 and the computer. If necessary please remove this software.

Sync line for spy channel time stamps

If you want to synchronize the OptoLyzer time stamps of the spy channel with the time stamps of any Vector network interfaces that are connected in parallel, you will need a sync line. To do this, use the SyncCable to connect the OptoLyzer's Trigger Extension Box and the sync input of the Vector hardware.

Example

Trigger input pin:

Pin8 is used for the trigger input by default. If you use different circuitry settings (pins 2, 4, 6 and 8 are available) you will need to adapt the CAN.INI file.

For the OptoLyzer G2 3150o on channel MOST 1 the following section is valid:

[OptoLyzerG2_1]
; 0x80 -> pin8 (default); 0x40 -> pin6; 0x20 -> pin4; 0x10 -> pin2
HwSyncTriggerMask=0x80
