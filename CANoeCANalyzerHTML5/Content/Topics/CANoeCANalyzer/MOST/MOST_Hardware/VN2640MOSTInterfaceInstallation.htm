VN2640 MOST Interface Installation

 

Requirements

  • VN2640
  • External power supply unit (included in delivery)
  • Vector Driver Disk with driver for VN2640
  • Free USB port (USB 2.0 or - with limitations on performance/throughput – USB 1.1)
  • Windows 7 (32/64 bit)
  • Administrator rights

Note

In Windows 7 it is not possible to install the drivers from a network drive. If you got your update from the Vector product page in the Internet, please copy the files to your local hard drive.

Step by Step Procedure

Install your drives before connecting the hardware. If you have already connected your hardware, close the Hardware Assistant with [Cancel].

  1. Insert the delivered driver CD in your computer.
    The installation setup will be started automatically.

If the installation is not started automatically please start the program by your own. You can find the installation program here:

  • \Drivers\32_Bit\setup.EXE (for Windows 7 32 Bit)
  • \Drivers\64_Bit\setup.EXE (for Windows 7 64 Bit).
  1. Select on the driver page the hardware for the driver installation.
  2. Start the installation with [Install].
  3. After successful installation close the setup with [Close].
  4. Connect your hardware with the computer.
