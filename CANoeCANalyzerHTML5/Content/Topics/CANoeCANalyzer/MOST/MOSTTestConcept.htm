Test Concept

 

CANoe supports testing of MOST systems with a bundle of specialized XML test patterns, checks and CAPL functions:

MOST specific XML Patterns

Test Pattern

Description

FBlocksExistence

This pattern checks the existence of FBlocks in a device to be tested.

FunctionsInFBlock

This pattern checks the existence of functions in an FBlock.

IsProperty

This pattern detects whether a given function address is a property or not.

NetBlockComplete

This pattern checks the completeness of the NetBlock of a device.

Replay

In the Replay test pattern a Replay file is played back.

Request/Response

In the Request/Response test pattern, a message or message sequence is sent and a corresponding response or several logically interconnected responses are awaited.

MOST specific CAPL Functions

CAPL Function

Description

TestGetWaitEventMostAMSMsgData

Gets the message data of the AMS message that resolved the last wait condition.

TestGetWaitEventMostMsgData

Gets the message data of the CMS message that resolved the last wait condition.

TestGetWaitEventMostPkt

Provides access to packet data of a waiting condition.

TestGetWaitEventMostRawMsgData

Retrieves the CMS raw message data of a waiting condition.

TestJoinMostAMSMessageEvent

Adds a further wait condition for AMS messages to the current set of Joined Events.

TestJoinMostAMSReportEvent

Adds a further wait condition for AMS response messages (OpType > 8) to the current set of Joined Events.

TestJoinMostAMSSpyMessageEvent

Adds a further wait condition for AMS Spy messages to the current set of Joined Events.

TestJoinMostAMSSpyReportEvent

Adds a further wait condition for AMS Spy response messages (OpType > 8) to the current set of Joined Events.

TestJoinMostMessageEvent

Adds a further wait condition for messages to the current set of Joined Events.

TestJoinMostPktEvent

Adds a packet to the set of joined events.

TestJoinMostReportEvent

Adds a further wait condition for response messages (OpType > 8) to the current set of Joined Events.

TestJoinMostSpyMessageEvent

Adds a further wait condition for Spy messages to the current set of Joined Events.

TestJoinMostSpyPktEvent

Adds a spy packet to the set of joined events.

TestJoinMostSpyReportEvent

Adds a further wait condition for Spy response messages to the current set of Joined Events.

TestMostRegGetData

Gets the result of an immediately preceding, successful Write or Read operation on the MOST hardware chip.

TestMostReadReg

Reads one or more registry values from the MOST hardware chip.

TestMostWriteReg

Writes one or more registry values in the MOST hardware chip.

TestSendMostAMSMessage

Sends a specified AMS message and waits for a delivery receipt.

TestSendMostRawMessage

Send a CMS raw message and waits for Tx acknowledgment result.

TestWaitForMostAllBypass

Determines whether the bypass of the MOST hardware connected to the channel is set to the specified value within the given waiting time.

TestWaitForMostCriticalUnlock

Wait for the occurrence of a LightLock event indicating a critical unlock state.

TestWaitForMostAMSMessage

Waits for the specified AMS message to occur.

TestWaitForMostAMSReport

Waits for an AMS response message (OpType > 8) to occur that matches the specified definition.

TestWaitForMostAMSResponse

Sends a specified AMS message and waits for the corresponding response message.

TestWaitForMostAMSResult

Sends a specified AMS message with OpType StartResult or StartResultAck and waits for the reception of the appropriate Result or ResultAck message.

TestWaitForMostAMSSpyMessage

Waits for the specified AMS Spy message to occur.

TestWaitForMostAMSSpyReport

Waits for an AMS Spy response message (OpType > 8) to occur that matches the specified definition.

TestWaitForMostApInstID

Waits for the FBlock InstanceId on the given channel to change.

TestWaitForMostAsRegistry

TestWaitForMostGroupAdr

TestWaitForMostLightOff

TestWaitForMostMessage

TestWaitForMostMPR

TestWaitForMostNetState

TestWaitForMostNodeAdr

TestWaitForMostNPR

TestWaitForMostPkt

TestWaitForMostRawSpyMessage

TestWaitForMostReport

TestWaitForMostSBC

TestWaitForMostSpyMessage

TestWaitForMostSpyPkt
