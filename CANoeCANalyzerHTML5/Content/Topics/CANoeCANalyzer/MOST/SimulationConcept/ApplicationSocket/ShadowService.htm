Shadow Service

 

The shadow service can be used to locate an appropriate function block in the network. Applications with controller functionality, which send command messages to a function block, obtain the logical address of the device containing the function block with the help of this service.

If a function block has been registered with MostAsShdEnable(FBlockId, InstId), the service sends a NetworkMaster.CentralRegistry.Get(FBlockId, InstId) request if the following events occur:

  • The network status switches from ConfigNotOk to ConfigOk (see OnMostNetState()).
  • The MOST NetworkMaster signals that the registered function block is newly present in its central registry (NetworkMaster.Configuration.Status(New, FBlockIdInstId List) message).
  • The NetworkMaster.Configuration.Status(Ok) message arrives while the network status is ConfigOk. The MOST NetworkMaster thus signals that a major change has been made in its bus registry.
  • MostAsShdEnable is called when the network status is ConfigOk and the device address of the function block is not yet known in the simulated device.

The MOST NetworkMaster's response message (NetworkMaster.CentralRegistry.Status(RxTxLog, FBlockId, InstId)) adds the new function block instance and its device address to the copy of the bus registry in the simulated device. This causes a message to be sent to the CAPL program indicating that the bus registry has been changed (see OnMostAsRegistry()).

When commands are sent to the function block, its logical device address is now known.

Notes

  • Registration with MostAsShdEnable(fblockId, 0xFF) causes the service to request the device addresses of all function blocks with the appropriate FBlockId at the MOST NetworkMaster.
  • Use of the shadow service is demonstrated in the controller applications of the supplied sample configurations.
  • The service does not send any requests to the MOST NetworkMaster if the automatic inquiry of the central registry has been enabled for the entire channel (see Hardware Configuration: MOST - Automatic Inquiry of the Central Registry ).
