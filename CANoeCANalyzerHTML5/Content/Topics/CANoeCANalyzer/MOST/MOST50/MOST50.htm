MOST50

 

CANoe version 7.2 SP3 and newer supports MOST50.

Besides meeting the special requirements of the MOST50 system, Vector also took the needs of experienced MOST25 users into consideration, and training effort is kept very low thanks to the unchanged user interface. For the most part, the programming interface has been kept identical, too. The tool accepts existing simulations and test scripts. However some differences in the user interface and the programming interface have to be considered.

Network Interfaces

CANoe supports OptoLyzer MOCCA compact 50e, OptoLyzer G2 3050e and OptoLyzer G2 3050e Production from K2L to access the MOST50 network. OptoLyzer MOCCA compact 50e is connected over USB to the computer. The required drives are part of the delivery of K2L. OptoLyzer G2 3050e and OptoLyzer G2 3050e Production are connected over Ethernet. Beside a standard Ethernet network adapter, no additional drivers or software packages need to be installed.

To use the interfaces, a license for the OptoLyzer Integration Package (OIP) from Vector is needed. Please contact Vectorâs sales department for a quotation.

Features

This version of MOST50 support allows access to the control channel and packet channel as node or spy. Many practice-proven analysis features are offered, such as the Audio Routing, Trace Windows, Data Windows and Graphics Windows , and the MOST FBlock Monitor. The Interactive Generator allows a flexible way to stimulate the MOST50 ring. Services like AMS and the MOST High Protocol have already been adapted to the new specifications.

The System Demo MOST50 for CANoe is very similar to the MOST25 counterpart. In fact this sample configuration reuses almost 100 % of simulation and test code of the MOST25 version and thereby show that a migration of configurations may be as easy as to select the newer speed grade in the Network Hardware Configuration dialog.
