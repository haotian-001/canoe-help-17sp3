MOST150

 

CANoe version 7.1 SP4 and newer supports MOST150.

Besides meeting the special requirements of the MOST150 system, Vector also took the needs of experienced MOST25 users into consideration, and training effort is kept very low thanks to the unchanged user interface. For the most part, the programming interface has been kept identical, too. The tool accepts existing simulations and test scripts. However some differences in the user interface and the programming interface have to be considered.

Network Interfaces

Starting with version 7.5 SP2, CANoe supports the Vector interface VN2640. It is connected to the computer by means of USB 2.0 and requires an appropriate, high-performance driver developed by Vector. The MOST option supports MOST25 (VN2610), MOST50 and MOST150 (VN2640) with the same activation.

In MOST150 networks with optical physical layer the Vector interface VN2640 and K2L’s OptoLyzer G2 3150o and OptoLyzer G2 3150o Production are applicable. Coaxial networks using simplex (unidirectional) transmission can be accessed with the same interfaces if an additional converter module (e.g. MOST150 C-O Converter of K2L) is joined up in circuit. Starting with version 8.5, CANoe supports coaxial networks with duplex (bidirectional) transmission and the interface OptoLyzer MOCCA compact 150c of K2L.

OptoLyzer G2 3150o, OptoLyzer G2 3150o Production and OptoLyzer MOCCA compact 150c require a license for the OptoLyzer Integration Package (OIP) from Vector is needed. Please contact Vector's sales department for a quotation.

Features

CANoe allows access to the control channel and data packet and Ethernet packet channel of MOST150 as node or spy. All practice-proven analysis features are offered, such as the Audio Routing, Trace, Data and Graphics Windows , and the MOST FBlock Monitor. The various possibilities to filter for relevant data were also adapted to MOST150. The Interactive Generator allows a flexible way to stimulate the MOST150 ring. Services like AMS and the MOST High Protocol have already been adapted to the new specifications.

The System Demo MOST150 for CANoe is very similar to the MOST25 counterpart. In fact this sample configuration reuses almost 100 % of simulation and test code of the MOST25 version and thereby show that a migration of configurations may be as easy as to select the newer speed grade in the Network Hardware Configuration dialog.
