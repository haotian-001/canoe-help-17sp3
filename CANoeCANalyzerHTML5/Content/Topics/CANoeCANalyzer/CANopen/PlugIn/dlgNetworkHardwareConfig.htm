Bus Configuration (Baudrate)

Ribbon: Hardware Tab|Network Hardware|CAN|CANopen Settings

Setup

Acceptance Filter

Options

CANopen

J1939

 

The option CANopen offers the possibility to set the baudrate and the bus timing register comfortably. For this, you must select Hardware ribbon tab|Network Hardware|CAN. On the dialog that appears, you can select the desired CANopen specific baudrate with CANopen settings.

On the dialog, all baudrates defined according to /2/ are available for selection. The set baudrate and the resulting entries of the bus timing register are taken over after the dialog has been closed with [OK].
