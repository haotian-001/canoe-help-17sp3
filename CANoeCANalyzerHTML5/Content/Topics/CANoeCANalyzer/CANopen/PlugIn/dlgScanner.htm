CANopen Scanner

Ribbon|Analysis Tab|System Scanner|View / New

The CANopen scanner evaluates the CAN messages and displays the active nodes in a list. Additional node-specific information such as state, device name and master properties are also output.

Using the CANopen Scanner

The CANopen scanner must be inserted into the Measurement Setup. For this, you must select the menu command Insert CANopen Scanner from the context menu of the Measurement Setup.

In the Measurement Setup a new analysis block with the label CO Scanner is now displayed. A double-click on the block opens the scanner window.

Basic Principle

So that the scanner recognizes the nodes on the bus, the error control (CAN ID 1793..1919) and SDO messages (CAN ID 1409..1535) are evaluated.

If an error control message is received, the state of the node is displayed in the list. Furthermore, a colored icon is used. If only SDO messages are received or if the error control messages are missing for 10 seconds, then the icon is displayed in gray.

If one of the nodes transmits the message with CAN ID 113 (identification message of a master), then it is marked as active master in the Master column and its priority is indicated.

The scanner evaluates the SDOs on the bus and can thus display additional information about a node in a tooltip. For this, the mouse pointer must remain over the node in the list for a brief time. Since the scanner is passive and no SDOs are sent, these objects must be exchanged with SDOs by the node on the network.
