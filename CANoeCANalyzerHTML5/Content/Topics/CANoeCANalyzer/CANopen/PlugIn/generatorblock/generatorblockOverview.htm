CANopen Interactive Generator

 

The CANopen Interactive Generator enables the creation of message sequences that can be sent once or cyclically. Messages can be selected from a list of CANopen specific message templates and combined into a sequence. The messages are configured via dialogs.

If the CANoe configuration has already been assigned a database, the messages described in the database are available for selection.

Insert the CANopen Interactive Generator

The CANopen Interactive Generator can be inserted into the Simulation Setup. Right-click on the line that symbolizes the bus and select the Insert CANopen Interactive Generator entry. The CANopen Interactive Generator can be opened in the Simulation Setup with a double-click on the icon.

The messages are sent on the CAN channel that is assigned to the network in the System View.

Settings

A click of the [Configure...] button opens the CANopen IG settings dialog.

In the Pause between services (dec.) input field, it is possible to specify what minimum time must elapse between the transmission of two messages from the message sequence.

In the SDO Timeout (dec.) input field, the timeout for the SDO protocol is specified.

The value within the input field SDO Block size (dec.) determines the SDO block size to be used for SDO block upload transfers.
