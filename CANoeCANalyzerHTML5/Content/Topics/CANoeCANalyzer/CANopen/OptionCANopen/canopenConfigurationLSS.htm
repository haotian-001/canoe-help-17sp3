LSS

Ribbon: Simulation Tab|CANopen Configuration

On the LSS page you can interactively execute various LSS master functionalities for configuring the node ID and bitrate of an LSS slave according to CiA 305. The sequence of the individual commands must basically comply with the procedure described in CiA305.

The page includes the following areas:

Switch Mode

In this area you can set one or all LSS slaves to the Waiting or Configuration state via the corresponding operating elements.

Selective

Via this button you set the slave, corresponding to the LSS numbers of the table, into the state Configuration.

Take over from node

Via this list you can select a CANopen node from the configuration. Then the values from its object dictionary (object entry 0x1018) are taken over into the table. Via the Selective button you can set this node to the state Configuration.

Global Configuration

Via this button you set all slaves to the state Configuration.

Global Waiting

Via this button you set all slaves to the Waiting state.

Fast Scan

Via the Start Scan button you can start the Fast Scan mechanism starting from the values stored in the table in the Start Value column. During the scan, the notification Performing LSS Scan appears in a status text field. If one of the LSS numbers has been uniquely identified, it is then displayed in the Found Value column in the corresponding LSS number row.

When all four LSS numbers have been found, the notification LSS Slave found is displayed accordingly in the status text field.

If the scan was not successful, the notification No LSS Slave found appears.

The time interval in which the master expects a response from the slaves you can specify in ms in the Timeout input box.

Identify

In this area you can identify a slave via the Slave button with the help of the values specified in the table. Here you can specify value ranges for Revision Number and Serial Number in the corresponding input boxes. All slaves whose values match the specified values or value ranges then respond accordingly.

You can use the Non Configured Slave button to inquire whether there are any slaves in the network that have not yet been configured, and they will respond accordingly.

Configure

In this area you can configure the node ID and bit timing for the node that is in the Configuration state.

Set Node ID

Enter the node ID in the corresponding input box and then click the Set Node ID button.

Set Bit Timing

Select the bit timing from the list and then click the Set Bit Timing button.

Store Configuration

This button allows you to save the configuration of the node that is in the Configuration state.

Activate Bit Timing

You can activate the bit timing via this button. area Therefor you have to set all nodes to the state Configuration before (e.g. via the Switch Mode and the Global Configuration button).

Inquire

In this area you can ask for the node ID and the four LSS numbers of a node that is in the Configuration state.

Click one of the five buttons in the first column to display the values in the second column.
