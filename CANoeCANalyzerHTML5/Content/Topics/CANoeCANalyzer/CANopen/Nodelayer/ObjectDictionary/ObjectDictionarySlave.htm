Object Dictionary for Slave (deprecated)

 

In order to configure the node layer as slave, the object dictionary must contain several objects:

Synchronous Window Length

Time of the sync window for PDOs

Index

0x1007

Type

Unsigned32

Description

This object contains the time of the sync window for PDOs in microseconds.

 

Guard Time

Time for guarding (optional)

Index

0x100c

Type

Unsigned16

Description

This object contains the time for the guarding in milliseconds. If guarding should be supported, then this object is required.

 

Life Time Factor (optional)

Index

0x100d

Type

Unsigned16

Description

The 'life time factor' multiplied by the 'guard time' produces the time for the life guarding. If guarding should be supported, then this object is required.

 

Producer Heartbeat Time (optional)

Time for heartbeat

Index

0x1017

Type

Unsigned16

Description

This object contains the time for the heartbeat in milliseconds. If heartbeat should be supported, then this object is required.

 

PDO Parameter

Index

0x1400..0x15FF for RPDO
0x1800..0x19FF for TPDO

Type

Structure

Description

Parameter for PDOs. A slave requires at least one PDO.

 

PDO Mapping

Index

0x1600..0x17FF for RPDO
0x1A00..0x1BFF for TPDO

Type

Array of Unsigned32

Description

The object contains the mapped objects. A slave requires at least one PDO.

 

A slave node is in the pre-operational state after the boot-up. If the node should go automatically into the operational state, the object 0x1F80 (NMT Startup) must be present and the bit 2 must contain the value 0.
