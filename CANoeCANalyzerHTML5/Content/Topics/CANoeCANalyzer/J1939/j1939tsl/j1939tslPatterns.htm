Test Pattern

 

Note

The following test patterns are part of the J1939 Test Service Library (J1939TestServiceLibraryNL.dll), which is no longer being developed and should therefore no longer be used. Instead, you can use the test commands described on page Testing in a J1939 Network.

Test patterns are small test procedures. The J1939 test pattern can be used in CAPL and XML test modules. Test patterns are active, this means messages are sent to create a response.
