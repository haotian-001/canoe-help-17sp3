Test Service Library

 

Note

The J1939 Test Service Library (J1939TestServiceLibraryNL.dll) is no longer being developed and should therefore no longer be used. Instead, you can use the options described on page Testing in a J1939 Network.

The J1939 Test Service Library (short J1939 TSL) extends the Test Feature Set with J1939 specific functions. The functions can be used in CAPL and XML test modules.

In difference to the standard functions of the Test Feature Set (TSL), the J1939 TSL uses parameter groups and PGN. The functions of the standard Test Feature Set can also be used with J1939, but J1939 transport protocols are not supported and complete extended CAN identifiers are used. This means for example, if the priority is changed, the standard TSL may not recognize the message.
