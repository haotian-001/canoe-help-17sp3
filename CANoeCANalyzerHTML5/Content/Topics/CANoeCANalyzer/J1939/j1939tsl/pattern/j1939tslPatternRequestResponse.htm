Request/Response

 

Name

J1939 Request/Response Test Pattern

Description

This test pattern is suitable for validating the behavior of a node, if it receives a J1939 request (EA00h). The request Parameter Group is sent be the test module. Then it waits for the response.

The response must be sent before timeout. The response can be the requested message or the acknowledgment Parameter Group (E800h). Should the request be observer only passive, the Request Check can be used.

The XML test pattern is able to validate the signal values of the response.

The request test pattern differs between global request, which are addressed to all nodes and specific request which are addressed to a single node. Depending on this the response must be sent global or specific.
