J1939 DTC Monitor

Ribbon|Diagnostics & XCP Tab|J1939|DTC Monitor|View / New

General

With the J1939 DTC Monitor in the Measurement Setup you can easily display and request error codes (DTC – Diagnostic Trouble Codes) and diagnostic messages. With the support of Freeze Frames (extended information from fault memory) you can use the J1939 DTC Monitor for emission related or power train diagnostics.

The J1939 DTC Monitor is divided into three parts:

The Upper Part

The upper part of the J1939 DTC Monitor displays the state of the warning lamps. If an ECU transmits a diagnostic message DM1 on which a warning light is activated, then this is indicated in the corresponding display field.

Warning Lamp

Description

MIL

Malfunction Indicator Lamp

RSL

Red Stop Lamp

AWL

Amber Warning Lamp

Protect

Protection Lamp

The Middle Part

The middle part of the J1939 DTC Monitor lists the ECUs from which messages are received. The display of the ECUs depends on their state in the network:

  • ECUs without active error code (DTC - Diagnostic Trouble Code) are displayed with a gray icon .
  • ECUs with at least one active error code (DTC) are displayed with a red icon .
  • ECUs sending the diagnostic messages DM1 with a cycle time less than 1,1 s are displayed in black letters.
  • ECUs displayed in gray letters are in one of the following states:
  • The ECU has not sent a DM1 yet.
  • The ECU has not sent a DM1 since a while.
  • The ECU sends the DM1 with a cycle time more than 1,1 s.

The columns show the following information:

Column

Description

Node

name of the ECU

Address

address of the ECU

CAN

CAN channel on which the ECU sends and receives

MIL/RSL/AWL/Protect

state of the warning lamps

The Lower Part

The lower part Diagnostic Trouble Code displays the error codes of all selected ECUs. The following pages are available:

  • DTC History
  • Active DTC
  • DTC
  • Freeze Frame
  • Configuration
