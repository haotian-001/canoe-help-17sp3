Data Export

 

The J1939 DTC Monitor offers the possibility to copy the displayed data into the clipboard for further processing or archiving. If you copy data, no diagnostic messages are requested. So you can use the copy function if the measurement is stopped or if CANoe is not connected to a network.

To copy the data of a single ECU, select this ECU and press <CTRL>+<C>. Alternatively you can open the context menu of the ECU and select the entry Copy. To copy the data of multiple ECUs, select the ECUs by holding the <CTRL> key pressed. Then press <CTRL>+<C> or select the entry Copy of the context menu.

You can also select one or more DTCs (Diagnostic Trouble Code) in the lower part of the J1939 DTC Monitor. Follow the instructions described above for the ECUs to select the DTCs.

If you copy data, the current column selection and sorting as well as the number format is considered. The data in the clipboard are available for text, rich text (RTF) and CSV. So you can use the data within software for word processing or spreadsheets.

Format

Description

Text

For text format standard spaces are used to format tables. Use a non-proportional font, e.g. Courier New.

RTF

For RTF format tabulators are used to format table. Eventually you must modify font size or the tabulator space to get an optimal table format.

CSV

Numeric values are always displayed in decimal format for the CSV export and separated from measuring units. That allows a comfortable processing within spreadsheet programs.

Note

The values FMI and SPN are always copied numeric and with symbolic names (if available).
