Remote

Tx Request
  • yes: messages are filtered regardless if direction is Rx, Tx or TXRQ
  • no: messages with direction TXRQ are not filtered

J1939

Direction
  • Tx: messages with node address as source address are filtered
  • Rx: messages with node address as destination address are filtered
  • Tx/Rx: messages with node address as source address or destination address are filtered

NMT Activated

Network Management

  • deactivated: the address is used to identify network nodes
  • activated: the Device Name in ACL messages is used to identify network nodes

The ACL messages are evaluated how they enter the Measurement Setup. Modifications that are made later, e.g. by CAPL nodes, are not considered.

Note

This configuration setting can be overwritten by the global settings in the J1939 options.

J1939 Node

Name

name of the node: name of a database node or *

Addr

address of the node

  • NMT not activated: enter any address or *
  • NMT activated: dynamic (NMT) is displayed, the field is not modifiable

AAC

Arbitrary-Address-Capable of the node

  • NMT not activated: n/a is displayed, the field is not modifiable
  • NMT activated
  • Name contains node name from a database
  • If the appropriate field of the node has a specific value in the database, enter this value or *.
  • If the appropriate field of the node has no specific value in the database, enter any value or *.
  • If the value differs from the value in the appropriate field in the database, a * is prefixed to the node name in Name.
  • Name contains *: enter 0 (no), 1 (yes) or or *
