Configuration

 

The Configuration dialog you open via the context menu command with the same name or via the Configuration button in the toolbar:

In this dialog, you can change the settings of the GNSS Monitor. There are the following pages available:

  • Graph
  • Units
  • Message Filter
  • Logging
  • Optimizations

Note

Help for the selected page can be started with the [Help] button.
