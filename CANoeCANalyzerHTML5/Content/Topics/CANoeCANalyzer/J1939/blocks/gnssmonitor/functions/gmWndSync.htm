Window Synchronisation

 

If the measurement was stopped, the function Synchronize windows can be activated in the context menu of the GNSS Monitor window or with the icon in the CANoeHome ribbon tab. The measurement cursor is shown with the left mouse button and makes simultaneous scrolling in all Trace, Graphics and Map Windows available. This allows a direct assignment of the displayed data in all windows.

Note

The window synchronization only jumps to the point of time in the trace or Graphics Window if the data is contained in the window buffer.
