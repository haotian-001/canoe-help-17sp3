Toolbar

 

The toolbar provides the following symbols for controlling the GNSS Monitor:

Button

Meaning

Deletes the Graphics Window and all saved positions.

Adjusts the zoom factor and moves the graphic so that the entire graphic is visible.

Resets scrolling so that the scroll bars are located in the center position. The graphic is not redrawn.

Resets moving the graphic and reset the scrolling so that the scroll bars in the center position. The graphic is redrawn with the saved positions.

If this check box is selected, the zoom factor and the way the graphic is moved after the new position is received will be adjusted so that the entire graphic is visible in the Graphics Window.

If this check box is selected, scrolling will be readjusted so that the last position received is always visible. If the position is outside of the graphic area, however, it will no longer be displayed.

Reduces the size of the graphical display

Enlarges the size of the graphical display

Switches on/off logging

Opens a dialog to configure the GNSS Monitor
