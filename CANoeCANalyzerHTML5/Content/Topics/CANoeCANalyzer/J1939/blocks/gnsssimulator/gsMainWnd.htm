Simulation and Speed Area

Element

Meaning

[Start]

Starts the simulation. The simulation can not be started unless CANoe measurement has been started.

[Stop]

Stops the simulation

[Pause/Continue]

After the simulation has been started, the button [Pause] can be activated. This reduces the speed to zero. Data for the current position continues to be sent and the label for the same button now reads [Continue].

After the button is activated again, the speed is set back to the value that is displayed and the process of traversing the selected route continues.

Start with measurement

When this check box is set, the GNSS simulation start takes place together with the start of the measurement.

Speed

The speed at which the route of the selected model will be traversed can be entered in the input box.

The slide control located under the input box can be used for the same purpose. The unit of speed as well as the minimum and maximum value for speed can be changed in the settings.

If the current speed is calculated with time information of the positions for playback of a file, the input box and the slide control will be deactivated.
