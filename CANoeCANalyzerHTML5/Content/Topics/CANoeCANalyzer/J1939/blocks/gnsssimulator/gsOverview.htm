General

The GNSS Simulatorcan be used to simulate a GNSS receiver. To do this, a geometric figure or a position file is used to determine a path that will be traced at a variable speed when the simulation is started. GNSS positions thereby generated are transferred cyclically with configurable messages.

Window

Functions and Control

The main window consists of the following parts:

  • Main Window/Toolbar
  • Graphics Window for graphic display of positions to be sent and labeling for them
  • area for controlling the simulation

Pressing <F1> allows the user to start help.
