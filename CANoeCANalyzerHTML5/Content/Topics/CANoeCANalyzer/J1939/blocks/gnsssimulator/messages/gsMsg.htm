Notifications

 

The different messages of the GNSS Simulator are displayed in dialogs:

Notification

The file … was not closed.

Possible Cause

It is not possible to close the file selected in the model File after reading.

Remedy

No.

Notification

Invalid file line(s) … in file … .Data of these line(s) is not used.

Possible Cause

While reading the file specified in the model File one or more lines with invalid format are found. These lines are ignored and are not used by the GNSS Simulator.

Remedy

Check with a text editor whether the specified file contains GNSS position data with the correct format.

See also: Playing back logging files

Notification

No GNSS position data found in file ...

Possible Cause

The file specified in the model File contains no GNSS position data.

Remedy

Check with a text editor whether the specified file contains GNSS position data with the correct format.

See also: Playing back logging files

Notification

No GNSS position with latitude information found in file … . The altitude is set to 0.

Possible Cause

The specified file in the model File contains only GNSS position data without altitude information.

Remedy

To calculate the simulation data the longitude and latitude as well as the altitude of the positions is necessary. If the altitude is missing for all positions, the altitude 0 is used for the calculation.

Notification

No GNSS position data with time information found in file … .

Possible Cause

The file specified in model File contains only GNSS position data without time information, so that a calculation of the speed is impossible.

Remedy

If the model File with option Use route and speed is selected on page Model, the time of the positions in the file is used to calculate the speed. Does the file contain no time information, playing back is only successful if the option Use route is selected an the speed is controlled by the main window.

See also: Playing back logging files

Notification

Cannot open file ... .

Possible Cause

It is not possible to open the file selected in the model File.

Remedy

If you have entered an invalid path for the file, it must be corrected.

Notification

Please enter a floating point value between 0.0 and 90.0 followed by a N (North) or S (South).

Possible Cause

The format of the entered latitude is invalid.

Remedy

Enter a latitude with the valid format (e.g. 48.46N).

Notification

Please enter a floating-point value between 0.0 and 180.0 followed by a W (West) or E (East).

Possible Cause

The format of the entered longitude is invalid.

Remedy

Enter a longitude with a valid format (e.g. 9.11E).

Notification

Please enter a maximum value bigger than the minimum value.

Possible Cause

A maximum value is entered which is smaller than the minimum value.

Remedy

Enter a maximum value that is bigger than the minimum value.

Notification

Please enter a radius which is smaller than the half width and smaller than the half height.

Possible Cause

The radius of the rounded off corners of a rectangle or a meander loop on page Model is bigger than the half height or half width of the rectangle or meander loop.

Remedy

Make the radius smaller.

Notification

Please enter a position name.

Possible Cause

One or more positions in the table on page Positions have no name.

Remedy

Every position requires a unique name. Enter a name for every position.

Notification

Cannot open reference position file … . Error: … .

Possible Cause

It is not possible to open the file with reference positions specified on page Positions. The error description provides more exact details.

Remedy

Depended on the error description.

Notification

Cannot write to reference position file … . Error: … .

Possible Cause

It is not possible to store the reference position file specified on page Positions. The error description provides more exact details.

Remedy

Dependent on the error description.

Notification

The same position name is already assigned to another position. Please enter new name.

Possible Cause

The name of a position on page Positions is already assigned to another position.

Remedy

Every position requires a unique name. Enter another name or delete the position with the identical name.
