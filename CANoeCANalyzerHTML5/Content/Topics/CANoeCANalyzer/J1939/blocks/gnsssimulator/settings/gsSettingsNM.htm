Network Management

 

Here you can select the channel through which the messages of the GNSS Simulator are sent and received. If the GNSS Simulator is used in CANoe, the channel is already defined by the Simulation Setup.

Also defined here is the J1939 address that the GNSS Simulator should use and the device name that is used in the address claimed message.

The GNSS Simulator sends the address claimed message when the measurement is started. However, it does not evaluate address claimed messages from other participants.
