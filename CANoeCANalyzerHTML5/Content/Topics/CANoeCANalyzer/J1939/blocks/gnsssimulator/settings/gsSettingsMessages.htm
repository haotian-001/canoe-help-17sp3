Network Management

 

Here you can select parameters of messages (Parameter Groups) of the NMEA2000® and J1939 protocol with which GNSS positions will be transferred. The selection of the message to be sent is made in the list box.

In the upper part of the dialog there is a list of possible messages. Only the activated messages will be sent. Down below this list are the message-specific parameters. There are displayed the properties of the currently selected message.

Used Messages

List of different Messages

All messages that are activated are sent with the set cycle time.

The message Vehicle Direction/Speed (J1939, PGN 65256) is not sent cyclically per default. If this message is activated it can be required with a J1939 request.

Priority

J1939 priority of the selected message.

Cycle Time

Time in ms at which the selected message is sent cyclically.

[More data...]

With this button you can open a dialog in which you can enter further message-specific data. This is possible for the following two messages:

  • GNSSPositionData (NMEA2000)
  • COG&SOG, Rapid Update (NMEA2000)

The button is deactivated for all other messages, as these messages cannot transmit any further data.

Information: The message Position Delta, High Precision Rapid Update (NMEA2000) uses the sequence ID (SID) of the last GNSS Position Data message sent and sends the differences to this reference message.

Note

The NMEA2000:GNSSPositionData message is sent with the Fast Packet protocol.
