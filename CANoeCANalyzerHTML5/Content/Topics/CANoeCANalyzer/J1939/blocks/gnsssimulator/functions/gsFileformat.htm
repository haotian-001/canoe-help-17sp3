File Format

 

A GNSS logging file is in ASCII format so that positions can be inserted or changed manually.

One position is saved per line, in the following format:

ID:PGN:TIME:LON:LAT:ALT

with

ID:

The format ID of the file line (previously, exclusively 'A', which stands for NMEA2000®)

PGN:

Number of the Parameter Group (decimal number)

TIME:

Time stamp of the GNSS receiver (floating point number, in seconds starting from midnight, UTC)

LON:

Degrees longitude (floating point number, in degrees)

LAT:

Degrees latitude (floating point number, in degrees)

ALT:

Altitude (floating point number, in meters)

Note

Since altitude is not supported by all messages, this field may contain a minus sign (-) instead of a number. If the time stamp is of no interest, a zero (0) should be entered there. Commends must be preceded by an asterisk (*) or a semicolon (;) in the first column of the line. '.gns' is recommended as the filename extension.
