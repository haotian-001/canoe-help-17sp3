Generate GNSS Data with Models

 

Generating GNSS data is performed in two steps:

  1. Calculation of waypoints:
Various models are available for calculating waypoints. A model can be selected or parameters can be set for it on page Model of the Configuration dialog. When the dialog is closed, a number of waypoints will be calculated and displayed in the Graphics Window. The waypoints are identified by a small cross and are connected by a straight line.
 
  1. Generating and sending position data
After the CANoe measurement is started and the simulation starts, the route defined by the waypoints is traversed at the current speed. Position data is also generated and sent with the Parameter Groups configured on page Messages. The speed is controlled for most models in the main window. The only case in which the speed cannot be changed is with the model File when the Use route and speed option box is activated.
 
Whereas all other models use a geometric figure to determine the target route, a file position information can be replayed with the model File.
