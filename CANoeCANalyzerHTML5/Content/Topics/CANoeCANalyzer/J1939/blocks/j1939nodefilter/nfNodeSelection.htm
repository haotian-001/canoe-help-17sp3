Node Selection

 

Rules can be defined for the filter in this dialog. The following possibilities are available for setting the source address and the destination address:

  • Ignore: The setting will be ignored. If this setting is selected for the source address, for example, the sender address will not be taken into consideration during filtering. The Ignore setting is displayed in the main window by an asterisk (*).
  • Range: A range can be specified. For example, if you want to allow all PGs to pass through that are sent by nods with addresses 5-10, this can be specified by using a range.
  • Node address: The address of an individual node is specified: If you want to allow all PGs to pass through that are sent/received by a specific node, the address of the node can be specified under this setting.
  • Global: (recipient address only): Filtering for PGs whose recipient address is GLOBAL (255). This setting is displayed in the main window by "GLOBAL".

Example

All nodes that are sent by the node with Address #10 will be filtered.

  • Source address: Select Node address and enter Address 10.
  • Destination address: Select Ignore.

Example

All nodes whose recipient address is “GLOBAL" (255) will be filtered.

  • Source address: Select Ignore.
  • Destination address: Select Global.

Example

All PGs that are sent by nodes 2, 3, 4, and 5 to node 10 will be filtered.

  • Source address: Select Range and enter Start=2 and End=5 (see also Range selection).
  • Destination address: Select Node address and enter Address 10.
