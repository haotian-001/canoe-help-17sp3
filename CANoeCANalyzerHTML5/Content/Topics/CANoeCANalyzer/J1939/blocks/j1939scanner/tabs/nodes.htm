Nodes

Configuration

 

This view shows the nodes in a list and provides the following information:

Column

Meaning

Name

name of the J1939 node

Address

address of the J1939 node

Channel

channel on which the node sends and receives messages

First Change

time stamp on which the node appears on the bus for the first time

This time stamp is recognized by any message with a valid address. If that message is an ACL message, the device name of the node is displayed in the column Device Name.

Last Change

time stamp on which the address of the node has changed for the last time

This time stamp is recognized by an ACL message of the node.

Device Name

internal device name of the J1939 node
