Diagnostic Messages

 

Diagnostic Message

Purpose

DM10
Non-Continuously Monitored Systems Test Identifiers Support

It reports the list of non-continuously monitored system tests supported by the ECU.

This information is used in the command section. If the test type is set to Manufacturer Specific Test, the content of the combo-box in the column TID shows if the selected test is supported or not supported.

DM5
Diagnostic Readiness 1

It reports

  • the number of Active DTCs
  • the number of Previously Active DTCs
  • the OBD compliance of the ECU (visible in the node view of the monitor)
  • the support and state of some continuously and non-continuously monitored systems

DM21
Diagnostic Readiness 2

It reports

  • the distance traveled while MIL is activated
  • the distance since DTCs were cleared
  • the minutes run by engine while MIL is activated
  • the time since DTCs were cleared

These information have to be used in relation with those from DM5. The response to request for DM21 is visible in Trace Window.

DM26
Diagnostic Readiness 3

It conveys the pending status of OBD system monitors for the current drive cycle.

This is useful in determining whether the OBD system has a defect or not. This DM plus the DM5 and DM21 formulate the OBD system readiness. The response to request for DM26 is visible in Trace Window.

Cross Reference
