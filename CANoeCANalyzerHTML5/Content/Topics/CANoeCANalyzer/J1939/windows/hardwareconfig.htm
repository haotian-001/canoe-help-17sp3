The J1939 option offers the possibility to automatically set the baudrate for J1939 according to the standard. You can choose between J1939 based on CAN Classic and J1939 CAN FD (J1939-17). In both cases execute the following steps. To distinguish between J1939 and J1939 CAN FD, select the corresponding setting in step 3:
  1. Select Hardware ribbon tab|Network Hardware|CAN. in CANoe.
  2. Choose Setup in the dialog Network Hardware Configuration.
  3. Select the mode: CAN or ISO CAN FD.
  4. In the tree view of the dialog select J1939 Configuration.
  5. Select the appropriate baudrate. The settings that can be selected depend on the mode selected.
  6. Close the dialog with [OK].

If the baudrate is not properly set for J1939, the message Invalid bit timing for J1939. is displayed. To use correct values select for CAN Classic baudrate 250 kBaud (specified in J1939-11) or baudrate 500 kBaud (specified in J1939-14).

Of course, you can also enter all configuration parameters for a J1939 CAN FD network (J1939-17) directly on the Setup page. First configure the baudrate. For the arbitration phase select 500.0 kBaud. For the data phase select 2000.0 kBaud. Select the remaining values by clicking the [...] button to the right of the Sample Point [%] input box to open the Bit Timing Configuration dialog. Select the following settings there:
