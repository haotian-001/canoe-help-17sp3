System Variables

 

The J1939 IL creates three system variables for each node, the shows the current state and address of the J1939 IL.

  • ILAddress
  • ILState
  • ILEnable

These system variables can be used to request and change the address and the state of the J1939 IL. Each system variable can be found in the namespace <NetworkName>::<NodeName>, e.g. J1939TestBus::Node1.

System Variable

Function

ILAddress

displays the current address

If this system variable is changed, the J1939 IL is triggered to use the new address.

ILEnable

displays if the J1939 IL is active

  • 1: active
  • 0: passive

If the initial value is set to 0, the node will remain in passive state after measurement start. If the current value is set to 0 resp. 1 during the measurement, the node will change its state to passive resp. active.

ILState

displays the current state of the J1939 IL

  • -1: Uninitialized
  • 0: Initialized
  • 1: Claiming
  • 2: Active
  • 3: Stopped
  • 4: Suspended
