Panels

 

This page explains how to modify signal in panel, which are assigned to message of the J1939 IL.

The Tx messages must already be configured, see Setup Tx messages.
  • Edit field, sliders, switches and buttons can be inserted into panels. In the configuration dialog of theses panels, signals can be assigned.
  • A signal, which is transmitted with J1993 IL, can be assigned to a control. If the signal value is changed during the measurement with the control, the J1993 IL sends the message depending the configured send type of the signal.
  • It is also possible to use a Message Control. In the configuration dialog of this control, the node which uses the J1993 IL must be assigned. The control shows all Tx message of the node. If the signal values are modified in the control, the J1993 IL send the message of the signal depending the configured send type.
