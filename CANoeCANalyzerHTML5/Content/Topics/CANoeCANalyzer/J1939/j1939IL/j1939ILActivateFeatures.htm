Activation of J1939 IL Functions

 

The default configuration of the J1939 IL does not use network management (Address Claiming) and does not respond to request with a negative acknowledgment, if a message is not supported. This can be enabled by assign several messages.

Use Network Management

To enable the Address Claiming it must be activated first. If it is not activated the J1939 IL send its messages from a fix configured address. To learn how to activate network management, see the following page: J1939/ISO11783 Interaction Layer Network Management.

Use Acknowledgment

To enable the acknowledgment of the J1939 IL, the acknowledgment Parameter Group (E800h) must be assigned to the node as Tx message. If the J1939 IL finds this message in the Tx list of the node and the requested PGN is not in the Tx list of this node, the request will be responded with a negative acknowledgment.
