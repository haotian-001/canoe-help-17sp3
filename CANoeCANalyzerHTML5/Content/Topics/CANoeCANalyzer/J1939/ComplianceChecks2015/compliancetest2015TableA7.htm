  1. Inconclusive, if no requestable non cyclic PGNs with DLC>8 are configured.
  2. Fails, if DUT does not respond properly within Tr (200 ms) to any request.
  3. Fails, if the TP's do not interfere. (R2 does not reside within [R1; E1]).

CTS/RTS Transport: Ensure the ability to send the Component ID PGN for more than one device
  1. DUT address.
  2. Tool address (defaults to 0xF9).
  3. At least one requestable non cyclic PGN with DLC>8.
  1. DUT has to support and answer a J1939 Request (0x00EA00).
  2. DUT has to support Transport Protocol RTS/CTS (0x00EB00, 0x00EC00).
  1. Request the Component ID PGN from the DUT.
  • After receiving the data:
  • Check the content to contain 4 bytes with asterisks in ASCII (0x2A).
  • Expectation: If DUT responds with data, 4 ASCII bytes 0x2A can be detected.
  1. Inconclusive, if DUT does not repond (timeout) OR responds with NACK.
  2. Fails, if DUT responds with data that does not contain 4 ASCII bytes with value 0x2A.
