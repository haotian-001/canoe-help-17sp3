Extended CAN Identifier

 

The extended CAN identifier used for J1939 receives, in addition to the PGN also the priority, source address, and DataPage information. This is depicted schematically below:

If the value of PDU format < 240 (peer-to-peer, PDU Format 1), then PDU Specific receives the recipient address of the Parameter Group. The GLOBAL address (255) can also be entered as a valid recipient address.

If the value of PDU format >= 240 (broadcast, PDU Format 2), then PDU format together with PDU specific forms the PGN.
