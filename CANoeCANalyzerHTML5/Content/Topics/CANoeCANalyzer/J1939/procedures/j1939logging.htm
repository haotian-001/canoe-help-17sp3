Replay Block and Offline Mode

 

J1939 Parameter Groups are recorded as normal CAN messages. The Replay Block and the offline mode replay CAN message with extended CAN identifier as Parameter Groups.

Transport Protocol

There is a feature, if transport protocol Parameter Groups are recorded in an ASC logging file. The assembled Parameter Groups with a DLC greater than 8 are stored as Parameter Groups in the logging file. The lines are marked with J1939TP.

The several Parameter Groups of a transport protocol transmission are written additionally to the logging file, if they are not filtered by a filter.

J1939 Interpretation

A comment line is inserted before each line of a J1939 Parameter Group in an ASC logging. This comment line contains the source address, destination address and the PGN of the J1939 Parameter Group. The symbolic display must be activated in the Home ribbon tab to use the J1939 interpretation.

Replay

J1939 Parameter Groups with DLC greater than 8 which a contained in a logging file are ignored on replay. The transport protocol Parameter Groups are handled and an assembled Parameter Group is created.

Import in Trace Window

When importing an ASC logging file into the Trace Window, also the Parameter Groups with a DLC greater than 8 are read. An interpretation of the transport protocol (as in the Replay Block) is not done.
