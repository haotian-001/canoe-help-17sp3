Send and Receive J1939 Frames

 

J1939 Frame

A new frame class named J1939Frame was added to .NET-API. It derives from CANFrame and thus has all the properties of this class.

Additionally the following properties can be set and requested:

  • Characteristic
  • DA
  • DA_TP
  • DLCEx
  • DP
  • EDP
  • PF
  • PGN
  • Priority
  • PS
  • SA

If the properties are used as write (set), the bit-fields are set directly in the CANId.

Sending J1939 Frames

The sending is shown here in a NET MeasurementScript by means of an example. The length (DLC) for the J1939Frame can be set directly in the constructor or via the property DLCEx.

Example

[OnKey('s')]
public void SendJ1939Frame(char c)
{
  J1939Frame frame = new J1939Frame(0xCFEF7F3, 10);
  frame.PGN = 0xFEF7;
  frame.SA = 0xF3;
  frame.Priority = 3;
  frame.DA_TP = 0xFF;

  var data = new byte[] { 0, 0, 0, 0, 0, 0, 0xFF, 0xFF, 0, 0 };
  frame.Bytes = data;
  frame.Send();
}

But the frames are also generated for the used J1939 database, these are then derived from the class J1939Frame.

Example

[OnKey('s')]
public void SendBatteryPotentialSwitched (char c)
{
  var frame = new NetworkDB.Frames.VEP1_ACS();
  frame.BatteryPotentialSwitched.Value = 2;
  frame.ElectricalPotential.Value = 10;
  frame.Send();
}

In addition to the parameterless method Send() , which transmits on the first channel, the SendTo(channel) method can be used to transmit on a specific channel.

Reception of J1939 Frames

To receive frames in callback methods the attribute class OnJ1939FrameAttribute was provided in the API.

The callback methods must be provided with this attribute. The following example code shows how frames can be received globally, for certain channels and CANIds and for certain channels and PGNs.

Example

// Output in Write Window
private void ToWriteWindow(string prefix, J1939Frame frame)
{
  string output = String.Format(prefix + " : t {0} | ID {1:X} | PGN {2:X}",
    frame.TimeNS, frame.ID, frame.PGN);

  Output.WriteLine(output);
}

// Reception of J1939 Frames
[OnJ1939Frame]
public void OnJ1939FrameReceived(J1939Frame frame)
{
  ToWriteWindow("J1939 Frame", frame);
}

[OnJ1939Frame(0, 0xCFEF7F3)]
public void OnJ1939FrameReceivedForId(J1939Frame frame)
{
  ToWriteWindow("J1939 Frame special ID", frame);
}

[OnJ1939Frame(0, PGN=0xFEF7)]
public void OnJ1939FrameReceivedForPGN(J1939Frame frame)
{
  ToWriteWindow("J1939 Frame special PGN", frame);
}

The modification of signal values can be implemented as follows:

Example

[OnChange(typeof(NetworkDB.DM30.TestLimitMinimum))]
public void OnMySignal()
{
  Output.WriteLine("My signal| value : {0}", NetworkDB.DM30.TestLimitMinimum.Value);
}
