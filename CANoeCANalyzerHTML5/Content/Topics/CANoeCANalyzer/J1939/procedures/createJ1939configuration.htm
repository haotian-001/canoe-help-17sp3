Create J1939 Configuration

 

When you create a new configuration from a J1939 template, it is already prepared to work in a J1939 network:

  • A J1939-specific baudrate is set.
  • For J1939 helpful windows are configured.
  • Possibly the J1939 standard database is already assigned.

To build an enhanced J1939 configuration that includes a standard J1939 database, follow the steps below:

  1. In File Ribbon Tab select the New command.

    The available templates are displayed.
  2. Select one of the a J1939 templates, whose icon has a yellow folder symbol, and click on Create Configuration From Template.
  3. Select a folder where the new configuration will be stored. Confirm with [OK].
