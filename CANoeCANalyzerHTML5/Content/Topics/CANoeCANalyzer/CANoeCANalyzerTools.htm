Vector Tool Launch

 

Note

  • The availability of the listed tools depends on your actual installation.
  • Click the blue link to open the respective tool.

Tool

Short Description

Support Assistant

Using Vector Support Assistant you can assemble all necessary system files and configuration files into a report (VSUPPORTREP file). The necessary files are sent compressed and password-protected to the Vector Support.

See also Vector Support Assistant File Descriptions.

 

Aerospace Message Editor

Using Aerospace Message Editor (AME) you can create and modify messages, signals and virtual links for aerospace bus systems (incl. ICD import, DBC export as well as a library with standardized ARINC-429 words).

ASAP2 Updater

Using ASAP2 Updater you can update the address and data type information in an ASAP2 file based on the entries of a Linker MAP file.

ASN.1 Import Wizard

Using ASN.1 Import Wizard you can convert an Â»ASN.1 module into a Car2x database for further usage in CANoe.

CANdb++ Editor

Using CANdb++ you can create, visualize and edit CAN databases.

CANeds

Using CANeds you can create and edit CANopen Electronic Data Sheets (EDS) and Device Configuration Files (DCF).

Car2x Network Explorer

Using Car2x Network Explorer you can visualize Car2x databases. Furthermore you can edit basic node and message properties.

Scenario Editor

With the Scenario Editor you can create a traffic scenario that can be played in CANoe.

Vector GreenPHY Configurator

The Vector GreenPHY Configurator is a dedicated tool for firmware update and configuration of the HomePlug Green PHY chips from Qualcomm (VT7870 module on a VT7970) and MediaTek (VT7870 module on a VT7971). It requires Npcap.

Vector Security Manager

Using Vector Security Manager you can manage and configure security profiles (e. g. SecOC, diagnostics, authenticationâ¦) for usage in Vector tools.

VT System Application Board Configurator

Using Application Board Configurator you can easily create and adapt application board configurations. The configurator contains information about the hardware used on the application board.

VT System Calibration Manager

Using VT System Calibration Manager you can read and write the calibration parameter of VT System modules. Another purpose of the tool is to start the automated self-calibration process.

VT System Firmware Updater

Using VT System Firmware Updater you can update the firmware of VT System modules.

VT System FPGA Manager

Using VT System FPGA Manager you can manage, compile and download your projects to VT System modules.
