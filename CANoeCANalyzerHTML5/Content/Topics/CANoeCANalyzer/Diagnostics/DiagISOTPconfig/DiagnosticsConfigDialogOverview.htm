Diagnostics/ISO TP Configuration

Ribbon: Diagnostics & XCP Tab|Diagnostics/ISO TP Configuration

Tree View / Functions Level 1

On the left in this dialog you find a tree view via that you can access supported diagnostic functions. E.g. you can add diagnostic descriptions to one or more networks and modify dedicated parameters.

This dialog lists the connected networks with their ECUs and diagnostic functions. If you click on left on a tree element the assigned diagnostic functions and possibilities for configuration appear on the right.

On level 1 of the tree view (see above) all assigned networks are listed (when available and configured in the Simulation Setup, with their diagnostic descriptions). With a double click on a row you open the corresponding sub dialog for the configuration. That way you automatically switch on the corresponding subordinated element in the tree view.
