ECU Control (Diagnostic Feature)

Ribbon|Diagnostics & XCP Tab|Diagnostic Window|ECU Control (Diagnostic Feature)

General

The ECU Control is a Diagnostic Feature that you can use within the Diagnostic Window. The ECU Control is used to change the diagnostic session for an ECU, to enable the security access in the ECU and to authenticate the tester at the ECU. Security aspects from any configured security sources are taken into account. Furthermore, ECU resets can be executed, DTC setting services and communication services can be sent.

Basics

The ECU Control can be configured as a Diagnostic Feature within the Diagnostic Window. The window functionality is only available if the corresponding diagnostic description contains suitable diagnostic services.

Note

If the variant of an ECU configured in the Diagnostics/ISO TP Configuration dialog does not contain any suitable diagnostic services, the window functionality is not available, even if there are other variants in the diagnostic description that contain such diagnostic services.

Window

The Diagnostic Feature ECU Control is only available in online mode and consists of two areas:

  • ECU Cockpit

  • ECU Control Services

    This area contains parameters that are specified by a security source. These can differ depending on the security source or configured security profile.

Note

The functions displayed under ECU Control Services depend on both the configured security profile and the services defined in the diagnostic description.
