Diagnostic Window Configuration

Ribbon|Diagnostics & XCP Tab|Diagnostic Window|Configure

General

In this dialog you can assign the available diagnostic features to one Diagnostic Window or distribute the features to several of these windows.

Currently you can configure the diagnostic features:

  • Variant Coding
    The diagnostic feature Variant Coding you can use in the Diagnostic Window.
    The diagnostic feature Variant Coding is used to read, write and compare variant coding data of an ECU. Security aspects from possibly configured security sources are taken into account.
  • ECU Control
    The ECU Control is used to change the diagnostic session for an ECU, to enable the security access in the ECU and to authenticate the tester at the ECU. Security aspects from any configured security sources are taken into account. Furthermore, ECU resets can be executed, DTC setting services and communication services can be sent.
  • Diagnostic Console
    The Diagnostic Console is a diagnostic feature, that you can use in the Diagnostic-Window.
    In the Diagnostic Console it is possible to directly send individual diagnostic requests of an ECU, display and evaluate the corresponding diagnostic responses.

Basics

The window functionality is only available if at least one diagnostic description has been configured in the Diagnostics/ISO TP Configuration dialog.

Note

Depending on the type and content of the diagnostic descriptions, it is possible that some Diagnostic Features are not available for the diagnostic description.

Window

The dialog consists of two areas:

  • On the left side the Diagnostic Features are listed, which are already assigned in the respective Diagnostic Windows.
  • On the right side you will find Diagnostic Features that have not yet been used and therefore can still be distributed to new or existing Diagnostic Windows.

You can assign the Diagnostic Features to an existing or a new Diagnostic Window using drag and drop from right to left.

You can remove a Diagnostic Feature already configured for a Diagnostic Window by dragging it from the left back to the right. If you drag all configured Diagnostic Features of a Diagnostic Window to the right, the corresponding Diagnostic Window is deleted. Alternatively, you can also drag the corresponding window row to the right.

You can group the Diagnostic Features list on the right side by dragging the relevant column heading into the area marked above the list.

You can change the name of a Diagnostic Window by clicking on the relevant window name on the left side and renaming it.

The Clear all button allows you to reset the configuration of all Diagnostic Features.
