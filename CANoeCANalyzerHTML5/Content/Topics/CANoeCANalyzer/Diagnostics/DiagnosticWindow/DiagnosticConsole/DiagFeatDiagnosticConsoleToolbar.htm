Toolbar (Diagnostic Console - Diagnostic Feature)

Ribbon|Diagnostics & XCP Tab|Diagnostic Window|Diagnostic Console|Toolbar (Diagnostic Console - Diagnostic Feature)

Symbol

Funktion

Expande/collapse nodes

Expands the nodes of the target level and all hierarchically higher nodes. Clicking again collapses all nodes regardless of their level. The options of this button allow you to set the target level. You can access the options from the button's context menu or from its list.

History of executed requests

Display the history of executed requests. A request can be selected from the list and navigated to the corresponding service.

Best fit all columns

Optimizes the column width based on the length of the respective data to be displayed.

Show qualifier

If this option is activated, the elements in the ECU / Service / Diagnostic Class / Parameter column are displayed with their qualifier or short name.

Create raw message

The button is active when an ECU, a service or a user-defined message is selected. It creates a new raw message according to the selection. If a service or a user-defined message is selected, it will be copied and its PDU data will be taken over. If the ECU is selected, an empty raw message is created.

The raw message can be deleted again with the button next to the qualifier.

Note

If no values are specified under PDU / Raw value for a raw message, it cannot be sent. This is displayed with a red frame and a corresponding tooltip.

Create user-defined message

The button is active when a service or user-defined message is selected. It creates a new raw message according to the selection. Parameters, identifiers and values are taken over. The user-defined message can be modified according to the rules of the original message. The user-defined message can be deleted again using the button next to the qualifier. can be deleted.

Display Vehicle System Groups

Switches between the Diagnostic Classes and VSG view (VSG: Vehicle System Groups, system groups of the connected ECU).

Vehicle system groups are a different representation of the available diagnostic services. Various system groups can be freely defined in the CANdela description file. A link to the individual diagnostic instances can be assigned to these system groups. This allows a logical grouping to be represented.

Note

The option is only active if vehicle system groups are defined in the ECU. Such groups can only be built and modified in CANdelaStudio. The option is only available for databases of type CDD or PDX, not for ODX databases.

Filter search results

If a full text search is active, this button can be used to specify whether the search results should be displayed or filtered.

Search

Opens or closes the full text search. Multiple search terms can be separated by a space. Search terms containing spaces must be enclosed by quotation marks.

If only a specific column is to be searched, the column name is separated from the search term with a colon, e.g. SID:10 searches for 10 but only in the SID column.

Show only selected items

If this button is active, only the requests selected in the Diagnostic Console Configuration are displayed.

Open Diagnostic Console Configuration

Opens the Diagnostic Console configuration dialog.

Clear Diagnostic Trace content

Clears the Diagnostic Trace area.

Show only failed

Shows only failed messages.
