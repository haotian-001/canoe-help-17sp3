Basic Diagnostic Editor

Ribbon: Diagnostics & XCP Tab|Basic Diagnostic

General

Using the Basic Diagnostic Editor, you can describe simple diagnostic services (UDS & KWP) for control units on CAN, LIN, FlexRay and K-Line.

After adding a basic diagnosticECU to the diagnostic configuration, the Basic Diagnostic Editor opens automatically for this ECU. You can then also open the editor in CANoe via the Diagnostics & XCP ribbon tab|Basic Diagnostics.

While the measurement is in progress, editor functions for editing basic diagnosticECUs are disabled.

Functions and Control

General Operation Information
  • Menu
  • Tree View
  • Edit Window
  • Error Messages

In the tree view, you can add and delete services using the Edit menu or the context menu.

In the Edit window, you can add, delete, copy, cut and paste services and parameters via the Edit menu or the context menu. The selected parameters can be edited directly in the tables.

You can navigate between fields within a table with the keyboard (arrow keys, PageUp, PageDown). To restore the original value in a field while entering values (e.g. in case of an incorrect entry), press <ESC>.
