Toolbar

Ribbon|Diagnostics & XCP Tab|Variant Coding

Symbol

Function

Read selected parameters

Reads the variant coding data from the connected ECU and compares it with the specified values.

Read selected parameters and use their ECU values as new specified values

Reads the variant coding data from the connected ECU. After successful reading, the ECU values read are transferred as new specified values.

Write selected parameters

Writes the specified values for the variant coding data to the connected ECU. It is assumed that the write was successful if the corresponding write services return positive responses.

Write and read selected parameters

Writes the specified values for the variant coding data to the connected ECU and then reads them back from the ECU to ensure that they have been written successfully.

Select parameters with differences

Selects parameters with different values in the Selection column. Parameters with the same values are deselected.

Select last imported domains

Selects the last imported domains. Is available only if domains have been imported.

Reset specified values to default values

Resets the specified values to the default values stored in the diagnostic description.

Import variant coding data

Imports the desired variant coding data (specified values) from a VVCD file.

Export variant coding data

Exports the desired variant coding data (specified values) to a VVCD file.

Expand/collapse nodes

Expands the nodes of the target level and all hierarchically higher nodes. Clicking again reduces all nodes regardless of their level. The options of this button allow you to set the target level. You can access the options via the context menu of the button or via its list.

Show qualifier (short name)

When this option is activated, the elements are displayed in the ECU/Coding Domain/Parameter column with their qualifier or short name.

Show selected parameters

Filters the Coding Data area by rows selected in the Selection column. This filter has a higher priority than any column filters you may have set.

Show differences / all

Filters the Coding Data area by lines where ECU value and specified value differ. This filter has a higher priority than any column filters you may have set.

Search

Opens or closes the search function, which can be used to search the variant coding data using a full-text search. Multiple search terms can be separated by a space. Search terms containing spaces must be enclosed by quotation marks.

Best fit all columns

Optimizes the column width in the Coding Data area based on the length of the data to be displayed.

Clear Diagnostic Trace content

Deletes the Diagnostic Trace area.

Show only failed

Shows only failed messages.
