Toolbar

Ribbon|Diagnostics & XCP Tab|Configuration|Diagnostic Parameters|Open Window

Symbol

Function

Switching the Filter Area on/off

When it is switched on, you see the requests associated with the corresponding response parameter, as well as the ECU qualifier of the associated diagnostic description, in the left window area. The display of the parameters in the right window area can be filtered by activating/deactivating the check box to the left of the associated request.

Switch on / Switch off Tester Present

Tester present requests are sent regularly to the ECU if no other diagnostic communication between the tester and ECU has occurred for some time. Tester present is a message that is sent cyclically and maintains the communication with the ECU. If the filter area is switched on, you can use this button to switch on/off sending of tester present for the associated diagnostic description. This status also applies to other windows that are assigned to the same diagnostic description (e.g., the Fault Memory Window and Diagnostic Console).

Note

Tester present requests can only be sent if the ECU is online.

Reset all Parameter Values

You can use this button to reset the values and raw values, as well as the display of the warning and error icons, together with the underlying counter values, for receipt and send errors.

Best Fit All Columns

Adjusts the column width to the contents.

Configuration

Opens a dialog for configuration of the Auto-Read function. You can specify the maximum number of consecutive errors after which the Auto-Read function should be deactivated.

CANoe Display Settings

Opens Name Display page (CANoeOptions dialog). You can adjust the display of the parameter name from the underlying diagnostic description (with or without path, with or without ECU qualifier) there in the Signals and Environment Variables area.

Search

Entering a search term in the search field allows you to search for the first occurrence of this search term in the list. The displayed parameter names are taken as a basis for the Name column. In the search field options, which can be reached with the arrow button, you can specify whether the entire word should be searched for or whether capitalization should be taken into account.

Search Forwards/Backwards

These buttons allow you to jump to the previous/next occurrence of the search term in the list.

Select All

This button selects all the parameters in which the search term was found.

Remove Parameters

This button removes all selected parameters from the list.
