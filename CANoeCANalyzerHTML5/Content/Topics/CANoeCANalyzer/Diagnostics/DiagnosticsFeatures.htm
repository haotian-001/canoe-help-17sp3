Diagnostics | Functional Overview

 

Glossary: »Diagnostics

Diagnostics describes the functional area in CANoe for:

  • Analyzing diagnostic communication between a diagnostic tester and ECUs.
  • Testing diagnostic functionality of ECUs interactively, semi-automated and automated.
  • Simulating diagnostic functionality of ECUs (Test the tester).

The Diagnostic Feature Set (DFS) of CANoe supports you in case of ad hoc analysis and troubleshooting on diagnostic level. It additionally provides support for customized semi-automated diagnostic sequences as well as for test-the-tester and quality assurance use cases in combination with the Test Feature Set (TFS).
