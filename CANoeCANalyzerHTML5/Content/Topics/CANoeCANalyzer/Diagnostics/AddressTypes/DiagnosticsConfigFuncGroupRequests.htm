Functional Group Requests

Ribbon: Diagnostics & XCP Tab|Diagnostics/ISO TP Configuration|Administrate Diagnostic Descriptions|Usage|Functional Group Request

To configure a functional diagnostic description in the relevant network you have to do the following steps:
  1. Open the Diagnostics/ISO-TP-Configuration dialog (Diagnostics & XCP ribbon tab).
  2. Select a node in the tree view left hand side (e.g. an ECU).
  3. In the Usage of the diagnostic description area select the Functional Group Request option.

Example

An sample configuration that demonstrates the usage of functional diagnostic description files please refer to: UDS Diagnostic System Configuration.
