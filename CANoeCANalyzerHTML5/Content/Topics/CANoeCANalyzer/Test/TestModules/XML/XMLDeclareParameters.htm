Declaration and Transfer of CAPL Test Case and Test Function Parameters

 

The usage of CAPL test cases and CAPL test functions is described on the following pages:

  • Call CAPL test cases
  • Call CAPL test functions

Here you can find some examples for the declaration and transfer of parameters:

Integers (byte, word, dword, int, long, qword, int64)

Note: All CAPL integer types are referred to as int in XML.

testcase TC1(long value)
{
  $signal = value;  // the value of the signal is set to value
}

<capltestcase name="TC1">
  <caplparam type="int" name="value">50</caplparam>
</capltestcase>

testfunction TF1(long value)
{
  $signal = value;  // the value of the signal is set to value
}

<capltestfunction name="TF1">
  <caplparam type="int" name="value">60</caplparam>
</capltestfunction>

Floating point numbers (float, double)

Note: All CAPL floating point types are referred to as float in XML.

testcase TC2(float value)
{
  $signal = value;  // the value of the signal is set to value
}

<capltestcase name="TC2">
  <caplparam type="float" name="value">44.4</caplparam>
</capltestcase>

testfunction TF2(float value)
{
  $signal = value;  // the value of the signal is set to value
}

<capltestfunction name="TF2">
  <caplparam type="float" name="value">55.5</caplparam>
</capltestfunction>

String (char [])

testcase TC3(char title[])
{
  TestCaseTitle("TC3", title);
  // ...
}

<capltestcase name="TC3">
  <caplparam type="string" name="title">"My Title"</caplparam>
</capltestcase>

testfunction TF3(char desc[])
{
  TestStep("My Test Function", desc);
  // ...
}

<capltestfunction name="TF3">
  <caplparam type="string" name="desc">"My Description"</caplparam>
</capltestfunction>

Environment variables — with indication of type and *

Possible types: Int, Float, String and Data.
Note: Always use envvar in XML.

Environment Variable Type

CAPL Parameter

Int

envvarint*

Float

envvarfloat*

String

envvarstring*
Data

envvardata*

testcase TC5(envvarint * ev)
{
  @ev = 1; // the value of the envVar is set to 1
  // ...
}

<capltestcase name="TC5">
  <caplparam type="envvar" name="ev">
    <envvar name="EnvAccelerate"></envvar>
  </caplparam>
</capltestcase>

testfunction TF5(envvarint * ev)
{
  @ev = 1;  // the value of the envVar is set to 1
  // ...
}

<capltestfunction name="TF5">
  <caplparam type="envvar" name="ev">
    <envvar name="EnvAccelerate"></envvar>
  </caplparam>
</capltestfunction>

System variables — with indication of type and *.

Possible types: Int, Float, String, IntArray and FloatArray.
Note: Always use sysvar in XML.

System Variable Type

CAPL Parameter

Integer

sysvarInt*

Long

sysvarLongLong*

Float

sysvarFloat*

String

sysvarString*

Integer Array

sysvarIntArray*

Float Array

sysvarFloatArray*

Data

sysvarData*

testcase TC6(sysvarint * sv)
{
  if (@sv < 0)
  @sv = 0;  // the value of the sysVar is set to 0
  // ...
}

<capltestcase name="TC6">
  <caplparam type="sysvar" name="sv">
    <sysvar name="SysVarInt" namespace="ns"></sysvar>
  </caplparam>
</capltestcase>

testfunction TF6(sysvarint * sv)
{
  if (@sv < 0)
  @sv = 0;  // the value of the sysVar is set to 0
  // ...
}

<capltestfunction name="TF6">
  <caplparam type="sysvar" name="sv">
    <sysvar name="SysVarInt" namespace="ns"></sysvar>
  </caplparam>
</capltestfunction>
