FlexRay AUTOSAR PDUs

 

Overview

CANoe supports two PDU concepts for FlexRay:

  1. frPDU
  • FlexRay specific PDU concept
  • Available since CANoe version 6.1
  • Focus of supported functionality: FIBEX and AUTOSAR 3.x
  1. ASR4 PDU
  • Generic PDU concept, independent of the bus system
  • Available since CANoe version 9.0 SP3
  • Focus of supported functionality: AUTOSAR 4.2 and following

  • CAPL data type frPDU and all functions that work with the data type frPDU are not supported

Configuration

When creating a CANoe configuration, the selection of the template determines which PDU concept is to be used for FlexRay PDUs.

  • ASR4 PDU concept: Template ASR4_FlexRay, ASR4_Ethernet and ASR4_CAN
  • frPDU concept: All other templates

Note: The definition of the PDU concept to be used applies to all FlexRay channels of the configuration. Subsequent changes to the PDU concept used are not supported. The used concept is shown in the configuration details (File ribbon tab|Configuration Overview).
