How to create a configuration for simulation? (FlexRay)

 

 

Step by Step Procedure

Assign Database (FIBEX)

  1. In File Ribbon Tab select the New command.
    The available templates are displayed.
  2. Select the FlexRay template, and click on Create Configuration From Template.
    A configuration with cluster FlexRay and channel FlexRay 1 is created.
  3. Open the Simulation Setup via the Simulation ribbon tab.

  4. Select the context menu command Add... of the Databases entry.

  5. In the Open dialog, select the desired database (e.g. z.B. PowerTrain.arxml, Open Folder) and apply it the cluster FlexRay by clicking [Open].

Add and Configure Nodes

  1. Select the context menu command Insert Network Node in the system view for the Nodes entry.
    The node ECU 1 is inserted in the system view and in the graphical overview.
  2. Select in the graphical view for node ECU 1 the context menu command Configuration....
  3. In the Node Configuration dialog, on the Common page, under Network node, select the desired node from the inserted database.
  4. Accept the settings for the node from the database with [OK].

  5. Repeat these steps for all nodes you want to include from the database in your configuration.

Notes

  • For remaining bus simulations, simply connect real FlexRay nodes to your FlexRay interface and deactivate the simulated version of these nodes in the Simulation Setup e.g. per double-click or context menu command.
  • The FIBEX database sample file can be found in the User Data directory, which was defined during installation (see Installation Directories).
