How to use CANoe as a Diagnostic Tester?

 

If you add a diagnostic description database (CDD, ODX/PDX, MDX) to a FlexRay bus in the Diagnostics/ISO TP Observer configuration dialog (Configuration|Diagnostics/ISO TP Configuration...), then you can

  • assign the Node or Functional Group that is target for the requests and
  • configure the underlying FlexRay Transport Protocol that is to be used.

You can assign more than one diagnostic description database (with optionally different targets) to one FlexRay bus.

With an assigned and configured diagnostic description you can send diagnostic requests

  • interactively with the Diagnostic Console or
  • automated with a CAPL Function or
  • using the vTESTstudioDiagnostic Service test command or
  • via the COM Interface.

The Fault Memory Window displays the ECU name on the left of every DTC of the ECU fault memory list.
