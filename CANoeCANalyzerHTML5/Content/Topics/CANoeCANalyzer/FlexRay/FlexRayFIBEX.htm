FIBEX Support

 

General info

To open a FIBEX database and add it to the active configuration, proceed as with dbc files: use the Simulation Setup to add a new database to the configuration.

A table lists all supported database formats.

Symbolic Interpretation

Additionally to the general information of nodes, frames and signals the following information are analyzed:

  • Cycle Multiplexing
  • In-Cycle Repetition
  • Signalgroups/PDUs
  • Sub-Frames

Note

PDU support is only available with FIBEX+ or FIBEX 3.0 databases (see also PDU Support).

Cluster Configuration of the CC Parameters

In addition to the symbolic message names and the signal mapping, the cluster configuration of the CC parameters is also read from the FIBEX database.

In this process, the system distinguishes between global parameters (allocated to the cluster) and local cluster parameters (allocated to a controller of an ECU) in the FIBEX file.

The tool calculates the local cluster parameters from global values or node values are taken from the database. From which database nodes the local cluster parameters shall be taken can be configured on the pages Controller or Protocol (redundant).

Troubleshooting

There may be errors, when opening a FIBEX file, or there may be no data in the file. Reasons can sometimes be found in the FIBEX file. Some general advice about FIBEX (XML) files and typical failures follows:

  • Make sure the namespace definitions are correct

Namespace definitions must match the FIBEX definition. Changing the namespaces to local paths does not work. Always use the network paths as defined by in the FIBEX standard, do not change them to local ones, e.g.:

xmlns:fx="http://www.asam.net/xml/fbx" is correct,

xmlns:fx="file://c://fibex/fbx" is not correct.

You don't have to be connected to the Internet to work with FIBEX files.

  • Always define the character encoding

It is not possible to use XML files without an encoding definition.

Example of a correct encoding entry:

<?xml version="1.0" encoding="ISO-8859-1"?>
