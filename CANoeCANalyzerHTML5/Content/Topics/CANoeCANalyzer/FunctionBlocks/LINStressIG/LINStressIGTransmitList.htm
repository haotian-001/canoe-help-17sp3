LIN Stress IG: Transmit list

 

Each frame to be sent by the LIN Stress IG block must be added to the transmit list.

Setup

  • Click the [New|Symbolic Frame] toolbar command and select the frame that should be inserted into the transmit list from the Symbol Selection dialog that appears.
  • Click the [New|Raw Frame] toolbar command and enter the identifier of the desired frame directly into the Frame Id column of the new row.
  • Select one or multiple rows and press the <Del> key to remove frames from the transmit list.
    Alternatively use the Remove toolbar command.
  • Click with the right mouse button in a row to display the context menu and to remove the row.

Fields

  • Serial (#)
    Serial number of the item in the transmit list.
    To modify this value use toolbar buttons Move Up / Move Down.
  • Action
    Push button to transmit the specified frame (or error) over the LIN bus.
    The title of the button indicates the currently configured functionality and updated dynamically on each configuration change.
    This button is enabled during the measurement only.
  • Frame Id
    LIN frame Identifier (6 bits).
  • Frame Name
    Shows the symbolic name matching FrameId value.
    If no matching name found this field remains empty.
  • Length Exp.
    Indicates the expected number of data bytes in the frame.
    If there is symbolic frame matching FrameId value this field becomes read-only, that is since the frame length specified in the database is used here.
  • Length Trans.
    Indicates the transmitted number of data bytes in the frame.
    Setting a value deviating from the expected number of data bytes leads to sending a frame error.
  • Header
    Indicates whether header part of LIN frame has to be transmitted.
    When not set only response update will be sent.
  • Response
    Indicates whether response part of LIN frame has to be transmitted.
    When not set only header will be sent.
  • Channel
    Indicates the LIN channel on which the frame is sent.

Note

  • The transmit list can be arbitrary modified also during the measurement.
  • If an error is configured to be sent the corresponding stressed fields are indicated with a special background color and a tooltip text.
  • The push button in the Action column can remain disabled also during the measurement.
