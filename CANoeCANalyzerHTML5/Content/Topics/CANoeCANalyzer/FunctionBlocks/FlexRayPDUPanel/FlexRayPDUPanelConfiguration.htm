FlexRay PDU Panel: Configuration

Context Menu: Configuration

Before the start of a measurement, PDUs are inserted into the transmit list in the FlexRay PDU panel's configuration dialog. The data components (signals and undefined payload areas) of configured FlexRay PDUs can be modified during the measurement.

The dialog consists of a PDU list and a signal list. In the PDU list, you can select a PDU and configure it. A signal list is allocated to each PDU, corresponding to the definitions in the database.

PDU List

One PDU must be listed for each data update that is required.

The following functions are available to you for configuring your FlexRay PDUs:

  • Click the [Add PDU...] button (or select this option from the function menu) and select the PDU to be transmitted from the Symbol Selection dialog that opens. Multiple selections are also possible.
  • Use the [Remove PDU] button or the [Del] key to remove selected items from the list.
  • The [Remove All] button removes all PDUs from the list.

List Structure

Column

Description

Update

Use this button to pass the signal value settings to the network interface.

PDU Name

Shows the symbolic name of the PDU.

Cluster

The name of the cluster on which the PDU is transmitted.

Tx Node

Name of the PDU's transmission node.

Initial Tx

The PDU is sent automatically when the measurement starts (in accordance with the send flags).

Update Bit

The PDU is sent with update bit enabled or disabled, in accordance with the settings.

Send Always

The PDU is sent continuously.

Update Counter

The PDU is sent as often as specified.

Signal List

You can modify the signal values directly in the signal list.

The following functions are available to you for configuring your signal values:

  • Use the [Set Defaults] button in the function menu to set the default values for all signals associated with the selected PDU to correspond with the database.
  • The [Set all bits to 0] button in the function menu sets the bits for all signals associated with the selected PDU to 0.
  • The [Set all bits to 1] button in the function menu sets the bits for all signals associated with the selected PDU to 1.

Multiplexors are handled as special signals. The icon that is used for multiplexors is different than those for signals.

List Structure

Field

Description

Signal Name

"Undefined", or signal name from FIBEX database

Bit Position

Bit position of the signal in the payload

Size

Signal size in bits

Raw Value

Raw value of the signal; can be modified

Physical value

Physical value of the signal; can be modified

Symbolic Value

Shows where applicable the appropriate definition from an optional text table

Note

You only need to enter either the raw value or the physical value, because the corresponding value is always calculated automatically using the signal's conversion formula.

Transformation of the raw value into the physical value, and vice versa, is based on the following conversion formula:
[Physical value] = ( [Raw value] * [Factor] ) + [Offset]

Note

An "undefined" signal name refers to a payload area for which a signal has not been defined. If this is the case, there is no Physical Value column.
