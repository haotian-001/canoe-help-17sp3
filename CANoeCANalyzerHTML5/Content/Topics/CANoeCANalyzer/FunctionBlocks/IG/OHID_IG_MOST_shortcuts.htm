MOST IG: Keyboard Operations

 

In addition to the keyboard operations for MOST windows, the MOST IG can be controlled as follows:

Send List:

Command

Key Combination

Moving the current cell

Arrow keys: <Up>, <Down>, <Left> and <Right>

Deleting the whole message

Note: The marked cell has to be in one of the columns before the Send button

<Del>

Opening a choice list

<Alt> + Arrow key <Down>

Delete, cut, copy, paste current message

Delete: <Ctrl>+<Del>
Cut:     <Ctrl>+<x>
Copy:   <Ctrl>+<c>, <Ctrl>+<Ins>,
Paste:  <Ctrl>+<v>, <Shift>+<Ins>

Delete, cut, copy, paste the content of the current cell

Delete: <Ctrl>+<Del>
Cut:     <Ctrl>+<x>
Copy:   <Ctrl>+<c>, <Ctrl>+<Ins>,
Paste:  <Ctrl>+<v>, <Shift>+<Ins>

Deleting the sequence (the messages, the sequence contained are not deleted)

<Del>, <Ctrl>+<Del>, <Shift>+<Del>, <Ctrl>+<x>

One can define single-key shortcuts for every message/sequence in the send list.

<a> .. <z>, <0> .. <9>

Parameter tree:

Command

Key Combination

Moving the current cell

Arrow keys: <Up>, <Down>, <Left> and <Right>

Opening a choice list

<Alt> + Arrow key <Down>

Selection of an element from a choice list

Arrow keys: <Up> and <Down>

Expanding a row (if '+' is displayed at the beginning of the row)

<Space>, <+>, <*>

Collapsing a row

<Space>, <->

Inserting a new array element

<Space>

Deleting an array element (Note: The current cell has to be in the column 'Parameter')

<Del>

Opening an edit field

<Space>
