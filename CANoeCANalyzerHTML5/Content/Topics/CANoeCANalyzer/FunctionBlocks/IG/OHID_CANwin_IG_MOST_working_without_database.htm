MOST IG: Working without a Database

 

Even if you have not assigned a function catalog to the CANalyzer/CANoe configuration, you can still send messages. However, since in this case there is no parameter structure, such messages can only be sent without parameters

Generating a new message

  • Generating an empty message by clicking on the bottom (highlighted in dark gray) row in the transmit list.
  • Manual entry of FBlock, Function, OpType and Instance, and Destination in the send list or the hexadecimal representation of the message.
  • Selection of the transmission channel; MOST channel 1 is set by default.

All values must be entered in the fields numerically. If an invalid value is entered in a field, a corresponding message appears in the status line. The FBlock, Function and OpType fields in the send list are highlighted in color, because no correspondence is found in a function catalog.
