IG: Configuration

Context Menu: Configuration

In the Interactive Generator's configuration dialog messages are placed in a transmit list and their contents and trigger conditions are defined.

The configuration dialog of the IG is subdivided into a transmit list and a signal list.

In the transmit list you can select individual messages and configure them. Assigned to each message is a signal list in which signal values can be configured.

The currently selected input boxes are explained in the status bar of the configuration window.

Special key combinations (Shortcuts) can be used for working without a mouse.
