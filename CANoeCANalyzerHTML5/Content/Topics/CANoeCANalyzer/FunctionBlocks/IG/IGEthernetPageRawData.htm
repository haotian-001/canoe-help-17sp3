Ethernet IG: Raw Data

 

Messages

Configuration

Raw Data

 

On this page you can enter the payload of the message by using the hex editor in hexadecimal and ASCII values.

Manual Input

To change the data bytes click on the byte position you want to change, then insert the hexadecimal value you want to insert. If you want to insert the values in ASCII format click in the ASCII column and enter your values. With <Tab> you can switch between the hexadecimal field and the ASCII field. The byte position of the cursor persists.

Changed values are highlighted in color. To commit the changed value press <Enter> or click in another window. To discard your input press <ESC>.

Copy and Paste Input

If you don't want to insert the values manually you can use the copy/paste mechanism. Copy the data in the clipboard, set the cursor to the position you want to insert the data and paste them with <CTRL>+<V>.

Note

If the cursor position is in the hex column, only hexadecimal numbers can be added. If there are other characters within the stream, the complete data will be discarded.
