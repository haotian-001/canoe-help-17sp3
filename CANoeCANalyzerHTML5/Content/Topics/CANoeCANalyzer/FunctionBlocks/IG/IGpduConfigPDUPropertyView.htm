PDU Properties

 

The PDU Properties view displays all properties of the currently selected AUTOSAR PDUs in the PDUs view.

Group

Column

Options

Description

Basic Properties

DB comment

—

Comment of the PDU from the database.

Comment

—

User-defined comment.

Length

—

Length of the PDU in bytes.

Name

—

Name of the PDU from the assigned database.

Timing

external

The timing of the PDU is not affected. An external timing e.g. by the interaction layer controls the transmission of the PDU.

  

event based

The PDU is sent immediately in the event of a payload change, taking into account the set debounce delay. The debounce delay specifies the minimum time in milliseconds that must elapse before the PDU can be sent a second time.

cyclic

The PDU is sent cyclically with the set period (in milliseconds). The first transmission takes place after the time period specified with Offset (in milliseconds).

Trigger

manual/ on key

A click on the button updates the payload of the PDU and triggers the PDU. Additionally a trigger key can be defined for the PDU. Pressing the key then has the same effect as clicking the button.

periodic

A click in the button updates the payload of the PDU with the set period duration and triggers the PDU.

PDU

When the specified PDU is received on any channel, the PDU is triggered.

system variable

When updating or changing the value (selectable) of the selected system variable, the payload of the PDU is updated and the PDU is triggered. The variable must not be marked as Use only for analysis.
