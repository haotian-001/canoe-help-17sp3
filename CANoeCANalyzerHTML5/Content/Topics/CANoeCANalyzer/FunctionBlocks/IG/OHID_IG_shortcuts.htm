IG: Keyboard Operations

 

The following key combinations relate to configuration of the Interactive Generator:

Command

Key Combination
  • Press the chosen button
  • Opens the Symbol Selection dialog to the message name fields
  • Makes a selection in the case of check boxes

<Spacebar>
  • Switches between the various buttons
  • Switches between the various input boxes
    An explanation of the selected box appears in the status bar.

<Arrow keys>

Incrementing or decrementing of the selected box

  • By the value 1 for timers and data bytes in the Transmit List
  • By the smallest possible raw value difference in the Signal List

<Ctrl>+<Up>
<Ctrl>+<Down>

Logarithmic adjustment of the slider in the transmit list

<Ctrl>+<Right>
<Ctrl>+<Left>

Incrementing or decrementing of the selected box by the preset step width (phys. step) in the signal list

<Page Up>
<Page Down>
  • Press the chosen button
  • Opens a selection list in the Transmit Lis

<Enter>

Exits the boxes to be edited, and any changes are canceled.
Attention: <Esc> stops a running measurement!

<Esc>

Starts the measurement

<F9>

Note

As long as the configuration dialog of the Interactive Generator is opened and active, all keyboard entries except <F9> and <Esc> are used to edit the dialog.
