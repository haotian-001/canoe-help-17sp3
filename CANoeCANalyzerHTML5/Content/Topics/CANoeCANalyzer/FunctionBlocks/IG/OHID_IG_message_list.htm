IG: Transmit List

 

Note

For displaying the transmit list the views Standard or J1587 are available.

Each message to be sent by the J1587 IG must be added to the transmit list.
The following procedures are available:

  • Click the [New] button and select the message that should be inserted into the transmit list from the Symbol Selection dialog that appears.
  • Double-click in an empty field in the Message Name column to open the symbolic selection dialog. Then select the message that should be inserted into the transmit list.
  • Enter the identifier of the desired message directly into the Identifier field in the Message Parameter column.

You can configure trigger conditions both before and during the measurement run.

Setup of the transmit list

Message name

Shows the symbolic name e.g. PowerTrain::EngineData.

If a symbolic name is not available, the identifier is displayed.

Message parameter

You can enter the following message parameters in the Transmit List:

Identifier

Serves to uniquely identify a message.

Channel

Indicates the CAN channel on which the message is sent.

Frame

Message format:

  • Data Frame: To transmit a message.
  • Remote Frame: To request a message.

DLC

(Data Length Code)

Indicates the number of data bytes in the message.

Triggering

With trigger conditions you can specify the point in time at which a message should be sent:

Send

Interactive triggering each time [now] is pressed.

Key

Initiates measurement by any desired key press.

This requires that you activate the check box to the left of t and specify any desired key.
E.G.: Transmit a message on key press a in test operation.

Note

As long as the configuration dialog of the J1587 IG is open and active, all keyboard entries except <F9> and <Esc> are used to edit the dialog.

Thus, the Generators are only triggered for transmission or the CAPL program nodes are only activated by the defined key if:

  • The CANoe main window is activated or
  • The test operation is explicitly switched on via the Test control field in the configuration dialog

The test operation is automatically deactivated upon exiting the control field.

Cycle time [ms]

Indicates the transmit interval for cyclic transmission of a message.

This requires that you activate the check box to the left of cycle time.

Data field

Contains max. 8 data bytes.

In each cell the specific byte value (hex/dec) is shown

Note

If defined values in the database are changed in the J1587 IG, the corresponding field is shaded orange (e.g. when changing the number of databytes (DLC column) of a message).
