MOST IG: Configuration

Context Menu: Configuration

In the configuration dialog of the MOST IG, the messages are collected in a send list and their contents and the transmission channel to be used are defined. In this dialog, the messages can be sent by pressing buttons.

The configuration dialog for the MOST IG is divided into a transmit list, a hexadecimal representation of the selected message, and a parameter tree.
