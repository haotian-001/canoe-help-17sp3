IG: Functional Overview

 

Using the Interactive Generator, you can configure messages and their signals before and during measurement.
The configuration dialog of the Interactive Generator can be used like a panel.

Insert messages

In the configuration dialog of the Interactive Generator, you can insert messages into the transmit list on the following way:

  • Enter manually
  • Insert directly from the existing database using the buttons provided to you

The signals of a selected message are displayed in the signal list.

If the message is entered manually, the signal list remains empty.

Configuring messages and signals

You can configure messages within the transmit list and signals within the signal list.

Sending messages

Various options are available in the transmit list to send messages:

  • Cyclical sending
  • Send via the defined button [now]
  • Send via any key

The IG as gateway

You can use the Interactive Generator as a gateway to transmit both individual messages as well as entire bus communication from one bus to another.

To do this, activate the option Gateway in the transmit list in the configuration dialog of the Interactive Generator.

Use of Signal Generators

You can define the signal chart and test the response of an ECU in the signal list in the configuration dialog with the help of the Signal Generator.

Generating a high load situation

To generate a high load situation, the next send command for the same message must follow immediately after a message is successfully sent.

To do this, activate the HighLoad option in the transmit list in the configuration dialog of the Interactive Generator.
