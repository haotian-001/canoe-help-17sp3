Value Generator Type: Random

 

This generator type sends random numbers between two values. Generating a random curve can be used, for example, to test system stress or responses to undefined events.

Because the signal chart is based on new random values for each transmission in this generator type:

  • This signal chart is only available under the Timing settings in not time bound sending mode
  • The signal chart is only automatically sent once (see Common settings)

A list of the calculated and output values as well as a graphic representation makes it easier for you to evaluate the defined signal chart.

Target settings

Here you can find information about signal values and cycle time.

  • Sample time
    Shown here is the time period at which the generator transmits.
    You can set this value under cycle time in the transmit list of the IG configuration dialog.
  • Default value
    The default value that is predefined for the selected signal is shown here.
    You can set this value in Raw value/Phys value in the signal list of the IG configuration dialog.
  • CANdb min
    This field displays the minimum signal value as defined in the Database
  • CANdb max
    This field displays the maximum signal value as defined in the Database
  • Type min
    Minimum signal type value calculated from the signal length, the offset and the factor defined in the database.
  • Type max
    Maximum signal type value calculated from the signal length, the offset and the factor defined in the database.

Common settings

The basic generator type settings are displayed and defined here.

  • Generator Active
    If this option is activated the generator is active.
  • Repetitive Sending Output Mode
    If this option is activated the generator continuously repeats the send operation (infinite loop).
    If the option is deactivated the signal is sent exactly one time.

Note

This option in this generator type is deactivated by default. Therefore, this option is not available (shaded gray).

Timing settings

  • Time bound sending
    If this control field is activated, the calculated signal values are only output at the time of the cyclical transmission. The cycle time determines the frequency of the sampling and therefore the accuracy of the signal chart.
    If this control field is deactivated, all the signal values calculated at the time the generator was defined are output at the cycle time intervals.

Note

This option in this generator type is deactivated by default. Therefore, this option is not available (shaded gray).

Specific Settings

You can make the following settings in this area:

  • Min
    Here you enter the lower limit of the value range for the random signal.
  • Max
    Here you enter the upper limit of the value range for the random signal.

Values / Graphical Output

In this area you find a list of the calculated values, the output values as well as a graphic representation.

The generator sends all values that are displayed in the Output column.

  • Value
    Values calculated by the generator.
    These values are displayed as red tags in the graphic representation.
  • Output
    Physical values sent from the generator depending on the signal duration and signal frequency. The factor and offset in the database are included in the calculation of the physical values.
    These values are displayed as green tags in the graphic representation.
  • With [Copy All] you can copy the values within CANoe and e.g. reuse and modify them in the User-defined generator type in the Values section by inserting them with the context menu command Paste.

Note

Values that are displayed in red (user may change the standard color via the context menu), are outside the permissible range (see Target settings 'CANdb min' and 'CANdb max').

If the green and red tags match, only the red tags are displayed.
