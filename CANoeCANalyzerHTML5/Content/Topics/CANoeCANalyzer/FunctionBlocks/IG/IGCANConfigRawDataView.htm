Raw Data

 

The Raw Data view shows the payload of the selected frame of the Frames view is transmitted on the bus. It is also possible to toggle between Raw Data and Signals view.

The view is a byte represented view of the frame. A row contains 8 payload bytes. The byte position within the frame can be calculated by the row number + 1 * column number + 1.

It is possible to insert values and navigate between bytes by keyboard. The editing mode of a byte must be enabled before the value can be changed. For entering the editing mode double click the byte value or use the <F2> key.

If the input mode is decimal the digits 0-9 are allowed for byte values. If the input mode is hexadecimal, additionally the hexadecimal digits A-F are allowed.

If a digit is entered, the editor moves to the next nibble of the byte. If the nibble is the last nibble within the byte, the next byte is selected for inserting values. It is also possible to use the arrow keys to navigate to the previous/next nibble or byte.

To show the context menu set a right click on a byte. The context menu shows following commands:

Copy (hexadecimal)
Copies the selected data to the clipboard.
The data is copied in the following format:
0x01 0x0B 0x7C

Paste (hexadecimal)
Inserts the copied data from the clipboard.
The formatting must correspond to one of the following variants:
- Variant 1: 0x01 0x02 0x03
- Variant 2: 01 02 03
- Variant 3: 010203
If the format in the clipboard is not correct or invalid values exist, the data from the clipboard is not copied and a warning message appears.

Reset Selection
Sets the current selected and focused byte to 0.

Reset All
Sets all byte values to 0.
