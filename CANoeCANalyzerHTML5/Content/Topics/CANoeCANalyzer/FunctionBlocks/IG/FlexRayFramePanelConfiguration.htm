Configuration

Context Menu: Configuration

The frames are added to the transmit list in the configuration dialog of the FlexRay Frame Panel before measurement. During measurement, the data content (payload) of the configured FlexRay frames can be modified. The frames are transmitted to the bus according to the transmission type (time-triggered, event-triggered).

Vector VN Interfaces

As well as changing the payload, Vector FlexRay interfaces can also be used to add and edit frames during the measurement period, as well as to end or restart transmission.

View

The dialog is divided into a transmit list for static and dynamic frames and in a payload list with raw data and signal interpretation.

You can select and configure a frame in the transmit list. Each frame is assigned a payload list.
