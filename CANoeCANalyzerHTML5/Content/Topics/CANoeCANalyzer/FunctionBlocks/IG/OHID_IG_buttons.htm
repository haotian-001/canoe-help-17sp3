IG: Buttons in the Configuration Dialog

 

The following buttons are provided to assist you in putting together the transmit list:

[New |▼]

Adds a new message to the transmit list. This button contains two parts and offers the following functions:

  • Insert a new message

If a database was associated, the Symbol Selection dialog is opened, in which you can select messages directly.

If no database was associated, a new message is opened in the transmit list, in which you must manually enter the following:

  • Identifier (ID)
  • Transmit channel
  • Number of data bytes (DLC)
  • Open the button's menu

In this menu you can open a message that is included in the send list. There are entries available for every allowed message type.

Additionally you can define the default settings for entries. This predefines the message type whenever you enter a message with no database associated.

[Duplicate]

Duplicates the message selected by ">" and its trigger conditions.

The duplicate is added at the end of the message list.

This function is very helpful if you want to send multiple messages with the same message identifier but with different data bytes.

[Special Frame  |▼]

  • CAN Error Frame

Enters an error message (Error Frame) in the transmit list.

You can use this function to intentionally send Error Frames onto the bus.

[Delete]

Deletes the message selected by ">" from the transmit list.

[Cut]

Transfers the selected message to the clipboard.

The trigger conditions of this line are lost in the process.

[Copy]

Copies the selected message to the clipboard without trigger conditions.

This function is very helpful if you want to send multiple messages with the same message identifier but different data bytes.

[Paste]

Pastes the clipboard contents at the end of the transmit list.

If the clipboard does not contain a valid message a warning appears.

[Arrange]

Automatically arranges the column widths in the transmit list and signal list so that their entries are as easy to read as possible.
