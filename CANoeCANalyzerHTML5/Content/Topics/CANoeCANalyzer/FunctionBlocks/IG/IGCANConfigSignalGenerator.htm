Signal Generator

 

The generator types can be defined by the Generator Type column of a signal of the Signals view. Possible Signal Generator types for the CAN IG are:

Sine

Ramps and Pulses

Value Range

Variable

 

To map values from the GUI (panel etc.) on a message without programming effort in CAPL

The properties of the generator types can be defined by the Generator Settings column of the Signals view.

Each generator type has the states:

  • running
  • paused
  • stopped
