FlexRay Filter Types

 

Frame Filter

There are two types of frame filters:

  • Frame Filter
    uses frames from a database or frames defined via Absolutely Scheduled Timing
  • Slot Range Filter
    uses frames that are defined via a specified slot range, base cycle and cycle repetition

You can insert a frame filter:

  • via the context menu (right mouse-click)
  • via the menu bar (Edit menu)
  • Add From Database…
    opens a symbol selection dialog that you can use to insert frames from a database
  • Add Frame Filter…
    opens a parameter selection dialog that you can use to configure a Frame Filter using the timing, FlexRay channel and application channel
  • Add Slot Range Filter…
    opens a parameter selection dialog that you can use to configure a Slot Range Filter using the slot range, timing parameters, FlexRay channel and application channel

The frame filter parameters view is divided into two areas.

  • the left side (main view) lists all of the filters.

Column

Description

Active

Activates/deactivates a filter

Name

Shows the frame name from the database (SHORT NAME) for frames that are present in a network description or database; otherwise only the timing is shown

NF

Null frame flag
(if * is selected, the Null Frame flag is ignored)

Direction

Direction of transmission
(if * is selected, the transmission direction is ignored)

Channel Mask

FlexRay channel number (A, B or AB)
(if AB is selected, frames on channels A and B are filtered)

Channel

Application channel number.
The selection options depend on the number of configured channels.
  • the right side displays the properties of the currently selected filter (properties view).

Property

Description

Schedule

Frame timing, or slot range for a Slot Range Filter

Frame Flags

Sync flag, start-up flag, payload preamble bit
(if * is selected, the flags are ignored)

Network

Shows the frame name, database name and service type
(not editable; available only for database frames)

PDU Filter

PDU filters can only be used with configurations that include PDU databases or network descriptions.
You can only insert PDUs from a database!

You can insert a PDU filter:

  • via the context menu (right mouse-click)
  • via the menu bar(Edit menu)

Column

Description

Active

Activates/deactivates a filter

Name

PDU name from database

Update Bit

Filters only on specified update bit states
(if * is selected, update bits are ignored)

Direction

Direction of transmission
(if * is selected, the transmission direction is ignored)

Database Name

Name of the database
(not editable)

Channel

The application channel number that the database and the PDU are assigned to
(not editable)

ECU Filter

You can only insert ECUs from a database!

You can insert an ECU filter:

  • via the context menu (right mouse-click)
  • via the menu bar(Edit menu)

Column

Description

Active

Activates/deactivates a filter

Name

ECU name from database

Direction

Direction of transmission
(if * is selected, the transmission direction is ignored)

Channel Mask

FlexRay channel number (A, B or AB)
(if AB is selected, frames and PDUs on channels A and B are filtered)

Service Type

Message service type (frames and PDUs)

Database Name

Name of the database
(not editable)

Channel

 The application channel number that the database and the PDU are assigned to
(not editable)

Other Filters

The following further filter types are available:

  • Start-Cycle Events
  • Status Events
  • Error Events
  • Error Frames or Invalid Frames
