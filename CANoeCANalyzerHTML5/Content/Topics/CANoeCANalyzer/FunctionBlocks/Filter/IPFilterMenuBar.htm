IP Filter: Menu Bar

 

From the toolbar, you can configure general settings for the IP Filter.

Setting

Description

(activated)

(deactivated)

This button is used to activate or deactivate the IP Filter.

In the active state, the filter conditions are evaluated, and only conforming packets are passed. If the block is deactivated, all packets are passed.

Type

You can select the operating mode of the IP Filter from the selection list.

In the Pass Filter operating mode, only those packets are passed that conform to the configured filter conditions. In the Blocking Filter operating mode, only those packets are passed to which the configured filter conditions do not conform.

View menu

You can group the configured filter conditions by channels here. In addition, you can hide the summary of filter conditions in the display.

Edit menu

You can add new filter conditions or deactivate and delete existing filter conditions here.

Note

Filter conditions that are added during a running measurement remain unconfigured and pass all packets. That is, these filter conditions are initially in a deactivated state.

This button is only available on the Packet Filter page.

You can use this button to reduce the output of the configured protocol fields to those protocol fields for which the values are entered.
