Logging Block / Logging File

Ribbon|Analysis Tab|Logging

General

The symbol for the logging file indicates that the data flow ends in a file.

The name of this file can be entered by clicking on the file icon or by equivalent keyboard input.
A file selection box then appears.

The automatic default extension of the filename can be set in the Options dialog

Note

  • From version 6.0 it is possible to use long symbolic names (> 32 characters). These are displayed for the ASCII-Logging if in the Export Settings the representation is set to symbolic.
    The produced ASCII files can be read in older versions of the program and, however, messages with long symbolic names are ignored.
    If you like to read ASCII files without exceptions in older versions, please, log numerically, see Export Settings.
  • The decimal or hexadecimal representation is set via the number format within the Export Settings.

You can startand pausethe data logging via

  • the symbols of the Logging Block
  • the state indicator in the status bar

Context Menu of the Logging Block

Configuration...

Comment...
Opens the Comment: Function Block dialog. Here you can insert a comment for the function block and choose the font settings of this comment.

Name
Opens the Name dialog. Here you can enter the name for the Logging Block.

Remove...
Removes the selected Logging Block.

Context Menu of the Logging File

Logging File Configuration...

Open Logging File...
Opens an editor to edit the logging file. You can select the editor in the Options dialog.

Open Destination Folder...
Opens the destination folder that is set in the Logging Block.

Add Logging File(s) As Offline Source(s)
Switches to offline mode and adds all logging files that are logged with that block to the list in the Offline Mode Window.

Replace Logging File(s) As Offline Source(s)
Switches to offline mode and replaces all in the logging files that are configured in the Offline Mode Window.

Export...

Comment...
Opens the Comment: Function Block dialog. Here you can insert a comment for the function block and choose the font settings of this comment.

Show Comment
Displays the comment in Measurement Setup. If this option is deactivated the comment is displayed as tooltip.
