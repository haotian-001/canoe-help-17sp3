Properties

 

Depending on the selection in the Schedule Tables ,Network view or the Frames view, the properties of the selected element are displayed here.

—

Configured LIN protocol version.

—

Current baudrate.

Break Field Properties

Break length

—

Length of the sync break in bits.

Delimiter Length

—

Length of the sync delimiter in bits.

Break Field Limits

Min. Break Length

—

Minimum permitted length of the sync break in bits.

The value is automatically rounded to 1/16 bits.

Max. Break Length

—

Maximum permitted length of the sync break in bits.

The value is automatically rounded to 1/16 bits.

Min. Delimiter Length

—

Minimum permitted length of the sync delimiter in bits.

The value is automatically rounded to 1/16 bits.

Max. Delimiter Length

—

Maximum permitted length of the sync delimiter in bits.

The value is automatically rounded to 1/16 bits.

Wake-Up Properties

Wake-Up Length

—

Length in microseconds of the wake-up signal sent by the Interactive Scheduler.

The value is automatically rounded to a multiple of 50.

Wake-Up Count

—

Maximum number of repetitions of the wake-up signal.

ttobrk

—

Time between two consecutive wake-up signals in milliseconds or bits (LIN 1.x).

Slot Properties

Group

Column

Option

Description

Slot Properties

Slot-Index

—

Index of the slots within the schedule table.

Active

—

Defines if the slot is active or not. Inactive slots are not sent during scheduling.

Note: The activity of the slot can only be defined, if the master simulation is activated by the LIN ISC node.

Type

Unconditional

The slot assigned to the frame is an unconditional frame.

Sporadic

The slot assigned to the frame is a sporadic frame.

Event-Triggered

The slot assigned to the frame is an event triggered frame.

Assign Frame ID

The slot is a master request of the type Assign Frame ID.

Assign Frame ID Range

The slot is a master request of the type Assign Frame ID Range.

Assign NAD

The slot is a master request of the type Assign NAD.

Conditional Change NAD

The slot is a master request of the type Conditional Change NAD.

Data Dump

The slot is a master request of the type Data Dump.

Free Format

The slot is a master request of the type Free Format.

Read by Identifier

The slot is a master request of the type Read by Identifier.

Save Configuration

The slot is a master request of the type Save Configuration.

Unassign Frame ID

The slot is a master request of the type Unassign Frame ID.

Delay [ms]

—

Delay of the slots.

Frame Properties

Name

—

Name of the frame.

Frame ID

—

ID of the frame.

DLC

—

Number of payload bytes.

Checksum model

Classic

The checksum is calculated exclusively on the basis of the payload.

Enhanced

The checksum is calculated on the basis of the payload and the PID.

Frame Properties

Group

Column

Option

Description

Frame Properties

Frame

—

Name of the frame. This input box is only editable for raw data frames.

Active

—

Determines whether the frame is sent or not.

Frame ID

—

ID of the frame.

DLC

0…8

Number of payload bytes This input box is only editable for raw data frames.

Send Type

Only Response

Only the response of the frame will be updated.

Only Header

Only the header of the frame is sent.

Header and Response

The response data is set and the header is sent.

Trigger

manual/ on key

After clicking the button in the Frames view, the frame is sent on the bus. You can configure a trigger key for the frame. Pressing the key, the frame is sent on the bus. Manual and OnKey trigger conditions can be used together.

periodic

After clicking the button in the Frames view, the frame with the defined time range is sent on the bus.
