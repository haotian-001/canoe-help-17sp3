Configuration

 

Toolbar

Icon

Description

Activate or deactivate filtering of CAPL commands for controlling the LIN scheduler for the assigned network.

If filtering is active, CAPL functions such as linStartScheduler or linStopScheduler are ignored. If a command was filtered, this is indicated by a corresponding notification in the Write Window.

Activate or deactivate filtering of CAPL commands that are used for controlling LIN Master functions for the assigned network.

If filtering is active, CAPL functions such as linGotoSleep are ignored. If a command was filtered, this is signaled by a corresponding notification in the Write Window.

During a running measurement, changes the master mode for the assigned network.

Note: This function is only active if the assigned hardware channel supports the changeover of the master mode during a running measurement.

Inserts a new schedule table in the Schedule Tables view.

Inserts a slot with a frame from the database into a user defined schedule table.

Inserts a slot with a raw frame into a user defined schedule table.

Add LIN frame from an assigned database to the LIN Interactive Scheduler.

Add LIN frame to the LIN Interactive Scheduler.

Remove selected LIN frame from the LIN Interactive Scheduler.

Undoes the last action.

Redoes the last action.

View: Network

Represent the network configuration in the form of a tree and offers functions for controlling the LIN scheduler and the state of the assigned network.

View: Schedule Tables

Displays the configured schedule tables and slots of the network that was selected in the Network view. Furthermore, the view offers functions for dynamically changing the schedule table.

View: Frames

Displays the configured Frames of the network that was selected in the Network view. The properties of the frames can also be configured in this view.

View: Signals

Displays the signals of the frame selected in the Frames view. If the selected frame is no LIN frame form an assigned database, the Signals view is deactivated.

View: Raw Data

To configure the data of a LIN frame by bytes you can use the Raw Data view. The Raw Data view is available for LIN Frames from an assigned database as well as for Raw Frames.

View: Properties

Displays the properties of the schedule table or of the slot or frame that was selected in the Schedule Tables view or in the Frames view.
