Schedule Tables

 

Displays the configured schedule tables and slots of the network that was selected in the Network view. Furthermore, the view offers functions for dynamically changing the schedule table. The configuration of the schedule tables is taken from the LDF database that is assigned to the network. In the context menu, existing tables can be duplicated via copy/paste and then modified.

The properties of the selected schedule table / of the selected slot are displayed in the Properties view.

Column

Options

Description

Table/slot

Name

Name of the table/slot

—

Click to send the table once.

Note: One-time sending is only possible if the measurement is running and master simulation is activated by the LIN ISC node.

—

Click to send the schedule table cyclically.

If the measurement is stopped, the function defines the start table of the scheduler.

Note: Cyclical sending is only possible if master simulation is activated by the LIN ISC node.
