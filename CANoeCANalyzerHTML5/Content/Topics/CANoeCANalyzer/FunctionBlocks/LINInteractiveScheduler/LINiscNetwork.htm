Network

 

The Network view displays the network configuration in the form of a tree. The Schedule table configuration of the last selected list entry is displayed in the Schedule Tables view.

The following functions are available for the network; the functions can be controlled using the buttons next to the network name:

The properties of the selected network are displayed in the Properties view.

Function

State

Description

Master Simulation

The LIN ISC performs the master simulation. Provided an LDF database with at least one schedule table is configured for the network, scheduling can be performed. Click to change the node to slave mode.

The LIN ISC network node is in slave mode. Scheduling is deactivated. Click to activate master simulation .

Channel state

The LIN network is in the awake state. Click to send a sleep mode frame.

If the measurement is stopped, the function defines the start state of the network. In this state, the network starts in the awake state.

Note: Sleep mode frames can only be sent if master simulation is active.

The LIN network is in the asleep state. Click to send a wake-up signal. If the measurement is stopped, the function defines the start state of the network. In this state, the network starts in the asleep state.

Scheduler state

The LIN scheduler is started. Click to stop the scheduler.

If the measurement is stopped, the function defines the start state of the LIN scheduler. In this state, the measurement starts with running scheduler.

Note: The scheduler can only be controlled if master simulation is active and the network is assigned an LDF database with at least one schedule table.

The LIN scheduler is stopped. Click to start the scheduler.

If the measurement is stopped, the function defines the start state of the LIN scheduler. In this state, the measurement starts with stopped scheduler.

Note: The scheduler can only be controlled if master simulation is active and the network is assigned an LDF database with at least one schedule table.
