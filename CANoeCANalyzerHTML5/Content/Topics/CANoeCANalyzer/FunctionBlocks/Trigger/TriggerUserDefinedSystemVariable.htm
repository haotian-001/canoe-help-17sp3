System Variable Condition

 

Here you can edit the properties of a system variable condition. To create a new system variable condition you have to select one of the items within the Symbol Selection dialog.

System variable

  • Name
    The symbolic name of the system variable as defined in the configuration or XML file.
  • Mode
    See set of user-defined conditions: Mode of the required value.
  • Value type
    The value type of the system variable as defined in the configuration or XML file.
  • Min/Max (visible only when the value type is numeric)
    The values of the boxes correspond to the Minimum/Maximum properties of the system variable in the database.
  • Relation

    You can use a logical operator to describe the relation of the notified and the required values.
  • Value
    Required value.
    Allowed input depends on the type of the system variable's value.
  • Numeric

    The entered number must correspond to the displayed type.

    If there is an associated value table the field is a list, which contains a list of the values from the table.
  • Textual

    Any text is allowed.

    Press the corresponding […] button to edit long texts.

Note

You can always reselect the current system variable by using the [Select…] button.

Timing

The text near the check box shortly describes the current timing settings of the condition.

The settings are active when the check box is activated.
Use the [Time settings…] button to define the Timing.
