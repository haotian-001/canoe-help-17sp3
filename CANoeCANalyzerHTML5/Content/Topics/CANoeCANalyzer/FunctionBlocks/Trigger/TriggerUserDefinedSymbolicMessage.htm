Symbolic Message Condition

 

Here you can edit the properties of a symbolic message condition.

To create a new symbolic message condition you have to select one of the items within the Symbol Selection dialog.

The suggested bus systems are configured via the Options dialog (Measurement|General).

Message

  • Name

    • The symbolic name and the identifier of the message as specified in the database.

  • Database

    • A name of the database where the symbolic message is defined.

  • Channel

    • The channel on which the message is expected to be received.
    The default channel is one to which the database is associated.

  • Attributes

    • See set of user-defined conditions: Attributes.

Note

You can always reselect the current symbolic message by using the [Select…] button. The new choice is restricted to the current bus system and protocol type. Be aware that Channel and Attributes can be re-set according to the newly selected item.

Timing

The text near the check box shortly describes the current timing settings of the condition.

The settings are active when the check box is activated. Use the [Time settings…] button to define the set of user-defined conditions: Timing.
