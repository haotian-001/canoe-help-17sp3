Set Raw A429 Word

 

Description

Defines the content and properties of the ARINC-429 word independent of the database.

Objects

Object

Description

gap

Overwrites the value of the minimum transmission interval (gap) that was previously set by the channel presetting for the selected ARINC-429 word.

Example

A4293::002.gap: addresses the gap for ARINC-429 word on channel 3 with Label = 002o

byte

This object type addresses an individual data byte of the ARINC-429 word; the other data bytes remain unchanged. The data bytes "byte 0…byte3" can be addressed. Byte3 is reserved; an assigned value has no effect.

Example

A4291::002.byte1: addresses byte 1 in ARINC-429 word 002o on channel 1

all data

This object type addresses all four bytes of the selected ARINC-429 word .

The last byte is reserved; its value has no effect on the data content of the ARINC-429 word.

Operators

Operator

Description

=

Assigns a constant or a byte string.

Operands

Operand/
with Operator

Description

gap

Gap value in nanoseconds, value range [0; 1/8 bit time…1 second]

The value 0 activates the gap value from the channel presetting.

byte

Value of an individual data byte

all data

The value of the ARINC-429 word in the form of a byte string can contain four bytes in hexadecimal notation. The last byte is reserved; its value has no effect on the data content of the ARINC-429 word.

Example

AA BB CC 00: Byte "00" is ignored; the rest is assigned to bytes 1 to 3 of the ARINC-429 word.
