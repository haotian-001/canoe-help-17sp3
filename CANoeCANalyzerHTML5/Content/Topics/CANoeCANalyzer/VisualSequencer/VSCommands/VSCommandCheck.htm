Check

 

Description

The Check command performs a simple check on a system variable or checks the current value of an ARINC-429 signal.

The result of the check is either displayed in the Write Window or saved to the logging file.

You can set the target path of the text output in the Advanced Settings dialog.

Operators

Operator

Description

=, !=, >, <, , <=

Checks the value of the object and the operand according to the specified operation.

in range

Checks whether the value of the object falls within or is at the limits of the specified range
(low limit <= value <= high limit).

outside range

Checks whether the value of the object falls outside of the specified range.

Raw value operators

(only with signals and members of system variables)

Operators with the addition [raw] compare the signal value without using the conversion formula from the database:
