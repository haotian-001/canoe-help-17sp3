Control Replay Block

 

Changes

This command replaces the obsolete command Replay!

Description

Starts/stops a configured Replay Block or changes the source file of the Replay Block. The Replay Block must be configured in the Simulation Setup.

On stop of the sequence all replay blocks started by the sequence will be stopped. Please ensure the desired replay time by using additional waiting times or by setting option Execute until Measurement Stop.

Objects

Object

Description

<name of the replay block>

Name of the Replay Blocks as it is in the Simulation Setup; content (value) of the Replay Name field in the configuration dialog of the Replay Block.

The name of the Replay Block has to be unique.

Operators

Operator

Description

start

Starts replaying the file.

stop

Stops replaying the file.

setfile

Changes the source file of the Replay Block to the path set in the operand. This is allowed even during running the previous file. In this case the replay is stopped, the previous file is closed, the new file is opened and the replay is started using the new file.

If the start operator shall be used immediately after this operator you have to set a wait time after setfile (in the Wait field of the setfile operator; 20 ms are recommend). To replay several files with the same Replay Block immediately in series, it is recommended to use a sequence of setfile operators (without start/stop).

Note: The setfile operator starts the replay automatically after loading. If the start operator is executed during playback, the replay is not restarted.

Operands

For the operators start and stop this filed remains empty.

Only for the setfile operator: The path to the new source file.

Relative path names are related to the path of the CANoe configuration file.

Absolute path names are possible but not recommended.

Example

The following sequence sets the source file of ReplayBlock 1 to Replay1.asc, replays the file for 2 seconds and stops the replay.

Half a second later this is repeated with source file Replay2.asc.

Finally it waits half a second.
