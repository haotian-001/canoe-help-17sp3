Set CAN Cyclic Frame

 

Description

Starts or stops cyclical transmission of a CAN frame. Use the Set command to set the signals of the frame.

The command is only available if the usage of the signal layer is switched off.

This command overwrites settings (for the selected CAN frame) for cyclical transmission of previous commands.
