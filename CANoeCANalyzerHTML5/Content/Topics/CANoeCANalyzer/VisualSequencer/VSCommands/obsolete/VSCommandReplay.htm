Command

Replay

Function

Starts or stops playback of a replay file.

A replay file can contain system and environment variable sequences as well as signal sequences.

Objects

Object

Description

File path of the replay file

Relative file paths are relative to the path of the CANoe configuration file. You can also use absolute file paths, but this is not recommended.

Operators

Operator

Description

start

Starts playback of the file.

stop

Stops playback of the file.
