Visual Sequencer Commands

 

Set Signals/Variables

Set

Sets the value of a signal or a system variable.

Map

Maps system variables to other system variables or signals.

Flow Control

Break

Allows premature exit from a loop.

Exit

Ends execution of the sequence.

If

Adds a conditional branch to the sequence.

Else If

Adds a conditional branch to the sequence.

Else

Adds a conditional branch to the sequence.

End If

Adds a conditional branch to the sequence.

Repeat

Defines a command block that is executed repeatedly.

Repeat End

Defines a command block that is executed repeatedly.

Wait Commands

Wait

Pauses the sequence for a specified time period.

Wait For

Pauses the execution of the sequence until a variable meets a certain condition.

Wait For CAN Frame

Stops the sequence execution until a certain CAN frame arrives.

Wait For CAN Raw Frame

Stops the sequence execution until a CAN frame arrives.

Wait For GMLAN Frame

Stops the sequence execution until a certain GMLAN frame arrives.

Wait For LIN Frame

Stops the sequence execution until a certain LIN frame arrives.

Wait For Key

Stops the sequence execution until a certain key is pressed.

Wait For A429 Word

Stops the execution of the sequence until reception of an ARINC-429 word that is defined in the database.

Wait For Raw A429 Word

Stops the execution of the sequence until reception of a specified ARINC-429 word.

Parameterizing and Sending Messages

Set CAN Raw Frame

Parametrizes a CAN frame without accessing a database.

Set CAN Cyclic Frame

Starts cyclical transmission of a CAN frame.

Set CAN Cyclic Raw Frame

Starts/stops cyclical transmission of a CAN frame that has been parameterized in raw format (without database support).

Send CAN Frame

Transmits a CAN frame.

Send CAN Raw Frame

Transmits a CAN frame that has been parameterized without database support.

Send CAN Error Frame

Sends a CAN Error Frame.

Send GMLAN Frame

Transmits a GMLAN frame.

Set LIN Resp

Activates/deactivates the LIN frame response.

Send LIN Frame

Transmits a LIN frame or LIN header once.

LIN Control Scheduler

Starts or stop the LIN Scheduler or switches the current LIN Scheduler table.

LIN Send Sleep

Sends a LIN Sleep Frame with an optional Wakeup Frame ID.

LIN Send Wakeup

Sends a LIN Wakeup Frame.

Send A429 Word

Sends an ARINC-429 Word that is defined in the database.

Send Raw A429 Word

Sends an ARINC-429 Word in raw format.

Set Raw A429 Word

Defines content and properties of an ARINC-429 word.

Set Cyclic A429 Word

Starts or stops the cyclic transmission of an ARINC-429 word that is defined in the database.

Set Raw Cyclic A429 Word

Starts or stops the cyclic transmission of an ARINC-429 word.

Output Commands

Check

Checks the value of a signal or a system variable.

Write

Outputs the value of a signal or a system variable.

Write Text

Outputs a given character string.

Other

Control Replay Block

Starts or stops a replay file.

Comment

Defines a comment row.
