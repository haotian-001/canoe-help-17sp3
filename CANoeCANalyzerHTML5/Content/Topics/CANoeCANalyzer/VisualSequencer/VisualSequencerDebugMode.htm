Debug Mode

 

The Visual Sequencer's Debug mode is useful for error search and step-by-step execution.
Press to activate the Debug mode.

Progress Marker

In Debug mode the next command to be executed is marked by coloring the relevant row.

Breakpoints

At so-called breakpoints the execution of a sequence is halted.

To insert and delete breakpoints, use the context menu or double-click the command icon. When the sequence pauses at a breakpoint, you can resume it using the [Start] button () or continue in [Step-by-Step] () mode.

Note

Inactive rows and comment rows do not affect the execution of a sequence. You cannot set breakpoints in these rows.
