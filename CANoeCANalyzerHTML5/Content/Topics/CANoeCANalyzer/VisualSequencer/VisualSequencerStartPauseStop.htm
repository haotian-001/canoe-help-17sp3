Start, Stop, Step-by-Step Execution

 

You can start the visual sequence using the run button () when the following conditions are met:

  • A measurement is started
  • The sequence does not contain any errors, i.e., there is no error symbol () in any of the rows.
    You can temporarily deactivate rows that contain errors using the context menu.

You can pause () or stop () the sequence at any time.

Repeated Execution

If the () button is activated in the toolbar, the sequence is executed repeatedly.

Start on Measurement Start

To start a sequence as soon as the measurement starts, activate the () button. In standalone mode this is the only way to start a visual sequence.

Execute until Measurement Stop

Here you can set whether the sequence shall run until measurement stop () after the sequence executed its last command. This setting equals a Wait command at the end of the sequence with the wait time set to infinite. This is especially useful with commands that are active as long as the sequence hasn't been stopped, like Set CAN Cyclic Frame and Map.
