Context Menu

 

The following context menu is available when you're working in the table:

Insert new row
Inserts a new command row above the selected row.

Delete
Deletes the selected row from the table.

Cut
Removes the selected rows from the table and places them in the clipboard.

Copy
Copies the selected rows to the clipboard.

Paste
Inserts the command rows in the clipboard above the selected row.

Activate
Activates all selected rows.

Deactivate
Deactivates all selected rows.
Deactivated rows are not included in the sequence execution.

Activate all
Activates all rows in the sequence.

Deactivate all
Deactivates all rows in the sequence.

Set/remove breakpoint
Inserts or removes a breakpoint.

Note

A sequence cannot be edited while it is running; all the menu commands are therefore deactivated.
