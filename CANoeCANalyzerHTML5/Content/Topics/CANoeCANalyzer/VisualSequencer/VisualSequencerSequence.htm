Creating a Sequence

 

You can insert new sequences using the Automation Sequences dialog.

To create a sequence in the Visual Sequencer, you define a series of commands. To do this, you insert new rows into the Visual Sequencer table and then populate them.

Depending on the command you've selected, the row will have up to five cells (columns).

The following support functions make it easier to populate the rows:

  • Selection lists for commands and operators
  • Symbol selection with text completion in the Object and Operator columns
  • Dragging & dropping symbols from the Symbol Explorer
  • Exporting () and importing () sequences to/from a file
  • After creating the command list you can validate the sequence with the button in the toolbar.
  • If the cell contents are invalid an error symbol appears ().
    If you hover the mouse pointer over the error symbol, a note about the cause of the error and how to eliminate it is displayed.
  • The status of the sequence is shown in the status line at the bottom of the window:
  • in progress (),
  • sequence executable (),
  • sequence faulty ().
