ADAS Feature Set in CANoe

 

ADAS Feature Set supports the basic concepts of CANoe and extends them.

ADAS objects are supported in the Feature Set based on ASAM OSI objects. A scene window is available for analysis, ADAS algorithms can be stimulated via CANoe SAB, and a specific ADAS Test Feature Set in CAPL / C# offers convenient test options.

Application area: Development, analysis and test of advanced driver assistance systems.
