User Interface

Ribbon|Ribbon Tab Environment|Group ADAS|Scenario Manager

Control

Description

Load / Unload / Open Scenario Editor

Loads/unloads a scenario or opens the Scenario Editor to create a scenario.

Scenario Start / Stop /Start at the start of measurement

Starts/stops a scenario during the measurement.

If the button Start scenario at start of measurement is activated, the scenario is automatically started when the measurement starts.

Open help

Opens the help for the Scenario Manager.

Views

Screenshot

Description

Scenario

When a scenario is loaded, the stations included in the scenario are displayed.

Car2x Database

If the stations are contained in a Car2x database, the station is displayed with a brown icon. The name of the database appears before the name of the station.

ADAS Database

If the stations are contained in an ADAS database, the station is displayed with a blue icon.
