Functionality

 

General

After measurement start, ADAS objects (Sensors, Detected Objects and Environment Objects) are automatically drawn into the Scene Window according to their position and size.

For sensors, the following fields are used by the Scene Window:

Field

Description

sensor_view_configuration.sensor_Id

SensorID must be unequal 0. Otherwise the sensor will not appear in the Scene Window.

sensor_view_configuration.field_of_view_horizontal

Opening angle of the sensor.

sensor_view_configuration.range

Range of the sensor.

sensor_data.mounting_position.position

Installation position of the sensor relative to the reference point.

sensor_data.mounting_position.orientaion.yaw

Rotation angle of the sensor.

For Detected and Environment Objects the following fields are used by the Scene Window:

Field

Description

baseInfo.dimension

Size of the object. For the Simple Mode, width and length are sufficient. For the Complex Mode, the height must also be set.

baseInfo.position

Position of the object.

baseInfo.orientation

Orientation of the object.

Highlighting objects

If you highlight entries in the Sensor, Detected Object or Ground Truth view, then the associated objects are highlighted in the Scene View.
