Option SmartCharging

 

Option SmartCharging supports analysis, test, and simulation of Smart Charging Communication (SCC). The certificate handling is based on the Security Manager.

Application area: Smart Charging Communication between electric vehicle (EV) and charge point (EVSE). Both vehicle and charging station can be simulated.

  • DIN 70121 / ISO 15118‑2 / ISO 15118-20: It requires Option Ethernet and a VT7870 module for the transfer to powerline communication.

  • GB/T 27930: It requires Option J1939 and Option CAN. The standard is mainly used in the Chinese market.

  • CHAdeMO: It requires Option CAN. The standard originally comes from Japan.

In addition, the communication between charging station and management system can be analyzed, tested and both participants can be simulated.

  • OCPP: It requires Option Ethernet.
