ISO 15118-20 Attributes

 

Check and Change Attributes

The attributes can be found, just like the DOs themselves, in the Smart Charging Configuraton dialog.

Additionally, some of the attributes can be changed during runtime with CAPL-Calls, see the following example.

Example

void UpdateAttributes()
{
  //Set shared attribute value for EV
  setAttribute(CCS::DistributedObjects::EV, CCSBinding::ProcessingLooping, CCSBinding::Enabled);

  //Set EV specific attribute
  setAttribute(CCS::DistributedObjects::EV, CCSBinding::EV::AttenuationTxEV, 5.5);
}

No

Double

7.0

No

Specifies the insertion loss of the transmitting path between the HomePlug Green PHY Modem and the charge coupler on EV side.

Match_SignalAttenuation_Direct

Double

10.0

No

Lower threshold for the signal strength measurement. EVSEs with lower attenuations are considered found. EVSEs with an attenuation between both “Match_SignalAttenuation” attributes are considered potentially found.

Match_SignalAttenuation_Indirect

Double

20.0

No

Higher threshold for the signal strength measurement. EVSEs with higher attenuations are considered not found. EVSEs with an attenuation between both “Match_SignalAttenuation” attributes are considered potentially found.
