Supported Protocol Versions

Mode of Functioning

The modeling libraries can be operated with various versions of the Vehicle2Grid communication protocol. Such a version consists of an XML schema, defined by a namespace string, and a two-digit version number. It is negotiated between vehicle and charge point using the SupportedAppProtocol message pair (Protocol handshake). If the vehicle module is part of the simulation, the desired protocol version can be configured using the vehicleâs configuration file, or selected online via the CAPL interface. If the charge point module is part of the simulation, it will accept the SupportedAppProtocol request if namespace and version are one of the entries listed below.

Note

When switching protocol versions online, switching of XML configuration section will most likely be necessary because of different parameter formats. In this case, please refer to the function LoadV2GConfig .

Protocol Versions

The following versions are available:

  • ISO 15118‑2
  • DIN70121
