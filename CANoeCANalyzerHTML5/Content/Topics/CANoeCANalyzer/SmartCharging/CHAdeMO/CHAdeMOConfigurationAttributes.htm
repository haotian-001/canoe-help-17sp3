CHAdeMO Configuration Attributes

 

Check and Modify Attributes

The CHAdeMO attributes can be accessed through the Smart Charging Configuration dialog.

Available attributes

Attributes marked as “mutable” in the tables below can also be modified via CAPL code.

Example

void UpdateAttributes()
{
  //Set a shared attribute value for EV
  setAttribute(EV, "CHAdeMOBinding::ProtocolVersion", 3);

  //Set an attribute value for EV
  setAttribute(EV, "CHAdeMOBinding::EV::MinimumChargeCurrent", "123");
}
