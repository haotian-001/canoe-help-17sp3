CHAdeMO System Variables

 

System Variables Configuration

The CHAdeMO modeling library uses system variables to receive (input) HW signals.

These system variables must be located in the namespace 'CHAdeMO::{Name}', where {Name} must match the name of the object created with the Smart Charging Configuration editor. The default names are EV and EVSE.

The system variables can be linked via mapping.

System variables for CHAdeMO 3.0 HW input signals

Please also refer to section 6.1.2 Sequence circuit of the CHAdeMO 3.0 standard.
