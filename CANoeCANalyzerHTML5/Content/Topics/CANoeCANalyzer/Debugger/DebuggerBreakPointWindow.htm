Debugger: Breakpoints

 

The Breakpoint area displays all breakpoints you have set in the Document view.

You can enable, disable, delete, and navigate to set breakpoints using the table view.

In the Name column the name of the program and the row of the breakpoint is displayed. The state of the breakpoint is visualized in the same way as in the Document view.

When a program reaches an active breakpoint during its execution, it is stopped. This breakpoint is highlighted in the overview (font: bold).

With Search you can search for breakpoints.
