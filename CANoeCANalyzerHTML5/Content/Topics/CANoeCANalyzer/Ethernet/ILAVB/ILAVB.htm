AVB Interaction Layer (AVB IL)

 

The AVB Interaction Layer (AVB IL) simplifies the simulation of an AVDEEC end station with primarily AVTP Listener and Talker endpoint functioniality. You can assign the AVB IL as a component to a node in the Simulation Setup. This gives the CAPL program access to the API provided by the AVB IL.
