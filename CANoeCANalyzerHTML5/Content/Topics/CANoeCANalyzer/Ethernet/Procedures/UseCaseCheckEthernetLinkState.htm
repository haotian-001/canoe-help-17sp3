How to Check the Ethernet Link State

 

To enable the transmission of Ethernet packets, a link between two participants must be established.

Note

If no Ethernet link is established, check settings in the Vector Hardware Config dialog (Windows start menu path: Control Panel|Vector Hardware). The bitrate or master/slave may be set incorrectly.

Bus Statistics Window

In the Bus Statistics Window, the Ethernet link status is displayed in the Connection status line. If Link up is displayed, the Ethernet link has been established successfully, the Ethernet packets can be exchanged.

You can use the Analysis ribbon tab to open an existing Bus Statistics Window or enter a new one.

Trace Window

The Trace Window displays an info event () for each change of the Ethernet link status. In the Protocol info column, Link up or Link down with the current bitrate is displayed. The detailed view can be used to display further information on the event.

Statistics System Variables

A statistics system variable is available for each configured Ethernet channel. It can be found in the _Statistics namespace, for example _Statistics::Eth1::ConnectionStatus.

These system variables can be used in the Data Window and Graphics Window and in panels.

Step by Step Procedure: Checking the link status

  1. On the Environment ribbon tab, press the button to open the Symbol Explorer.
  2. In the Symbol Explorer, select the view for the system variables .
  3. In the _Statistics namespace, search for the Ethernet channel and for the ConnectionStatus below it
  4. Drag the ConnectionStatus system variable using drag and drop on the Data Window or Graphics Window.
    If Link up is displayed, the Ethernet link has been established successfully.

CAPL

In CAPL, the Ethernet link status can be queried using the ethGetLinkStatus CAPL function or using Eth1.status.
