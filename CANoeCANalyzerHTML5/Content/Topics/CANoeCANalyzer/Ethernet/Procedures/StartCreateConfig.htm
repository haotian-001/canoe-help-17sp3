How to Create an Ethernet Configuration

 

Creating an Ethernet Configuration by Means of a Template

Procedure for Network-Based Access

VN Ethernet interfaces from firmware version 11.1 support a network-based access.

Step by Step Procedure

  1. Create a new configuration with File ribbon tab|New|Generate Configuration From Template.
  2. Use a template from the Ethernet area. An Ethernet network and the Trace Window are already preconfigured here.
  • Ethernet (Communcation Setup)
    Template prepared with the Communication Setup.
  • Ethernet (Simulation Setup)
    Template prepared with the Simulation Setup.
  1. Give a meaningful name to an Ethernet network so that its use is unique. A network defined in CANoe and a network defined in the Ethernet device configuration automatically find each other when they have the same name.
    You open the Ethernet Device Configuration via the Network Hardware Configuration/Vector Hardware Config dialog.
  2. A Vector Ethernet interface, for example VN5610 can be used as network hardware. Open the Network Hardware Configuration dialog via the Hardware ribbon tab|Bus Hardware.
  • How to Configure a Vector Ethernet Interface
  1. Start the measurement and observe the Trace Window.
    In the Trace Window, the Ethernet or IP protocols with the standard settings are interpreted. To be able to also evaluate signals, for example of AUTOSAR, a database is required.

Procedure for Channel-Based Network Access (legacy)

Precondition: The used Ethernet interface supports channel-based access (VN5610, VN5640 or a Windows network adapter).

Step by Step Procedure

  1. Create a new configuration with File ribbon tab|New|Generate Configuration From Template.
  2. Use template Ethernet (Simulation Setup) from the Ethernet area. An Ethernet channel and the Trace Window are already preconfigured here.
  3. Activate the channel-based access with the Network Access|Channel-based access option in Options dialog.
  4. A VN5610, VN5640 or a Windows network adapter can be used as network hardware. Open the Network Hardware Configuration dialog via the Hardware ribbon tab|Bus Hardware.
  • How to Configure a Vector Ethernet Interface
  • How to Configure a Windows Network Adapter
  1. Start the measurement and observe the Trace Window.
    In the Trace Window, the Ethernet or IP protocols with the standard settings are interpreted. To be able to also evaluate signals, for example of AUTOSAR, a database is required.

Using Several Ethernet Channels

The configuration templates contain an Ethernet channel or an Ethernet network. To use further Ethernet channels, proceed as follows.

Step by Step Procedure

Configure the number of Ethernet channels.

  1. Open the Options dialog via the Quick Access Toolbar ().
  2. On the General page, enter the number of Ethernet channels for Eth.
  3. Close the dialog by clicking [OK].

Configure the network hardware for each channel.

  1. Open the Simulation Setup via the Simulation ribbon tab by pressing the button.
  2. Select Networks in the System View.
  3. Use the Add... context menu command to open the Add Network dialog.
  4. Select Eth as a bus system and enter a name for the network.
  5. Close the dialog by clicking [OK].
  6. Select the Channels item under the new network.
  7. Use the Add... context menu command to open the Add Channel dialog.
  8. Select the desired Ethernet channel.
  9. Close the dialog by clicking [OK].

Configuring Protocols

Ethernet or IP protocols are interpreted on the basis of the Ethertype or the UDP/TCP port numbers. They are set by CANoe via the assigned database or can be manually set to required values.

Step by Step Procedure

  1. Open the Options dialog via the Quick Access Toolbar ().
  2. On the Bus Systems page in the Protocol Identification area, set the Ethertype and UDP or TCP ports.
  3. Select a protocol in the table and click [Edit...].
  4. Enter the port number in the edit dialog.
  5. Close the dialog by clicking [OK].

Assigning a Database

To enable CANoe to interpret Ethernet packets up to the signal level, a database must be used:

  • for AUTOSAR PDUs, an ARXML file is required
  • for SOME/IP, a Fibex or ARXML file can be used

Step by Step Procedure

  1. Open the Simulation Setup via the Simulation ribbon tab by pressing the button.
  2. Select Databases in the System View.
  3. Use the Add... context menu command to open the Open dialog.
  4. Select a Fibex or ARXML file here.
  5. Close the dialog by clicking [OK].
