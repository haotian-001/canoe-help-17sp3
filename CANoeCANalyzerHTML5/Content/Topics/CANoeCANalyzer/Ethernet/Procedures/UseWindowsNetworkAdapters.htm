How to use the Ethernet interface of your computer

 

You can use Ethernet network adapters provided by Windows as measurement interfaces. Restrictions apply for this kind of use because functionalities like the precision of timestamps may vary dependent on the hardware. If you require constant, reproducible measurement precision, use network interfaces provided by Vector.

The appropriate driver for accessing Windows network adapters in CANoe is not installed with the default CANoe setup program. However, it can be installed separately from the following location:
<InstallationDirectory>Installer Additional Components\Vector Network Driver\NetworkDriver.msi

Click Start Setup to install the driver.

Using Windows Network Adapters in CANoe

Step by Step Procedure

  1. Open the Network Hardware Configuration dialog via Hardware ribbon tab|Bus Hardware.
  2. If you select an Ethernet channel, for example Eth 1, information on the selected channel will be displayed on the right side.
  3. Close the dialog by clicking [OK].

Using Windows Network Adapter Exclusively for CANoe

You can configure the network adapter of your system such that it will be exclusively used for CANoe.Ethernet. In this way, you prevent Windows or other applications from using this interface and interfering with your embedded network. At the same time, your network cannot interfere with the Windows system.

You can use NetIsolator to make this setting quickly and easily.

Step by Step Procedure

  1. Open the context menu of the icon in the Windows task bar.
  2. Select the context menu commandInsulation and the network interface you want to insulate.
