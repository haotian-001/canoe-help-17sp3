Operate a VN4610

 

To put a VN4610 with CANoe .Car2x into operation, proceed as follows.

Step by Step Procedure

  1. Install the appropriate driver from the Vector driver CD.
    This is the prerequisite to be able to put a VN4610 into operation.
  2. Connect the VN4610 to the computer/laptop via USB.
  3. Launch CANoe.Car2x.
  4. Click ribbon tab Hardware|Channels|Network Hardware in the menu bar.
    The Network Hardware Config dialog opens.
  5. Click [Driver...].
    The Vector Hardware Configuration dialog opens.
  6. Assign the channels of the VN4610 under Application|CANoe by right-clicking on the interface device channels. To provide an accurate UTC-based time base, a GNSS channel should always be configured in real mode. The CANoe reference timer is then automatically set to the corresponding module:
  7. Close the dialog.
    You return to the network hardware configuration dialog. Here you are shown whether the device is physically connected.
  8. In the Setup area you can set various communication parameters if necessary.
    Details can be found on the help page of the network hardware configuration dialog (help: Setup tab).
  9. Click [Ok].
    The dialog closes.
  10. Make sure you are in real mode.
  11. Start the measurement.

With CANoe .Car2x you also have the possibility to change or read out transmission parameters at runtime. Use the CAPL functions C2xConfigureChannel, C2xSetSendParameters or C2xGetChannelConfiguration.
