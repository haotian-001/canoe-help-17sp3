Simulate and Stimulate ITS Stations

 

To simulate and stimulate ITS stations, proceed as follows.

Step by Step Procedure

  1. Create a new configuration for the simulation/stimulation of ITS stations.
    Depending on the standard to be used, we recommend that you use one of the supplied templates:
    • Car2xTemplate EN 302 637 (2014)
    • Car2xTemplate EN 302 637 (2018)
    • V2xTemplate SAE J2735 (2010)
    • V2xTemplate SAE J2735 (2016)
  2. Configure the database in the Simulation Setup with the Car2x Network Explorer:
    1. For example, change the simulation and stimulation behavior of the ITS stations.
    2. If necessary, add one or more application messages to a network node. The transmission behavior of these application messages can then be configured with the attributes (cyclic transmission, cycle time, use of a certificate, ...).
  3. Close the Car2x Network Explorer
  4. Now configure the Simulation Setup:
    1. Create a node in the Simulation Setup for each network node in the database.
    2. Then assign this node the network node from the database (DB node):
  5. The transmission behavior of the node from the Simulation Setup is now linked to the network node from the database.

In the CAPL code from the templates different callback functions like OnPreTxApplMsg and OnRxApplMsg are implemented. The OnPreTxApplMsg function is called before a frame is sent. At this point you have the possibility to change the content of the frame with the Car2x Packet-API functions. The OnRxApplMsg function is called when a frame is received. At this point you have the possibility to check the content of the frame with the Car2x Packet-API functions for e.g. correctness.
