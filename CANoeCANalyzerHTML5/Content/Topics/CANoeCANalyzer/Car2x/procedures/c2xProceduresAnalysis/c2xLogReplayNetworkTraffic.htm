Log and Replay Network Traffic

 

Log Network Traffic

To log frames, proceed as follows.

Step by Step Procedure

  1. Go to the measurement setup.
  2. Insert a logging block.
  3. With a right click you open the Logging File Configuration dialog.
  4. Configure the logging block.
    If necessary, specify the target path and the name of the logging file.
  5. At the start of the measurement, all frames you receive are recorded and saved in the logging file. Only BLF Frame Logging is supported as file format for CANoe .Car2x.

Replay Network Traffic

If you want to replay or analyze this log file later, you have various options. Always deactivate your logging block before doing this so that the original logfile is not overwritten.

Option 1: Replay of Logfiles

  1. Insert a replay block in the Simulation Setup.
  2. You are still in online mode, i.e. .Car2x will use the current system time as the time base and the Simulation Setup is still active.

Option 2: Replay of Logfiles in Offline Mode

  1. In offline mode, specify your previously recorded log file as the source.
  2. The offline mode gets its time base from the log file, the Simulation Setup is not active. In the offline mode you have the possibility to replay all frames immediately (as fast as possible) or as animation. For more information, see the Offline Mode help page.
