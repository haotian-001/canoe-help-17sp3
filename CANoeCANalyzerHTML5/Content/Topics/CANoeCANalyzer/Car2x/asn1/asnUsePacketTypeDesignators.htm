Initializing Several Protocol Layers

 

You can initialize a packet with the wanted underlying protocols by using combinations of Packet Type Designators. The single packet type designators are intercombinable and can be assigned as strings to the functions C2xInitPacket() and C2xInitProtocol(). The pipe character (|) is used as separator between the single packet type designators. For each protocol header only one packet type designator is allowed.

CAPL example

variables
{
  long packetHandleCAM = 0;
  long packetHandleDENMwithSH = 0;
  long packetHandleLocationService = 0;
}

on start
{
  packetHandle = C2xInitPacket("CAM", "BTP-A | TSB-SHB");
  packetHandleDENMwithSH = C2xInitPacket("DENM", "BTP-B | GBC | SH");
  packetHandleLocationService = C2xInitPacket("geo_eh", "LS-RQ");
}

Forbidden combinations

If one of the following combinations is used the error 46-0143 is reported:

  • BTP-A|BTP-B
  • multiple usage of Extended Header Packet Type Designators, e.g. BEACON|GUC|GBC
  • SH | SH_121

Packets based on GeoNetworking protocol according to ETSI TS 102 636

The following Packet Type Designators are available for these packets when a BTP packet is initialized:

Protocol Designator

Packet Type

Designator

Description

btp

btpOverCnh

Initializes the packet with BTP and GeoNetworking-CommonNetworkHeader (geo_cnh) according to ETSI TS 102 636.

btpOverGeo

Initializes the packet with BTP, GeoNetworking-ExtendedHeader (geo_ext)and GeoNetworking-CommonNetworkHeader (geo_cnh) according to ETSI TS 102 636.

To create an application message based on GeoNetworking protocol according to ETSI TS 102 636, a BTP packet with the packet type designator btpOverCnh or btpOverGeo is initialized first. In this packet the application protocol is then initialized.

CAPL example

variables
{
  // Application messages utilizing
  // Geo Networking protocol according to ETSI TS 102 636
  long packetHandleCAM = 0; // CAM with Extended Header
  long packetHandleDENM = 0; // DENM with Common Header only
}

on start
{
  packetHandleCAM = C2xInitPacket("btp", "btpOverGeo");
  C2xInitProtocol(packetHandleCAM, "CAM");
  packetHandleDENM = C2xInitPacket("btp", "btpOverCnh");
  C2xInitProtocol(packetHandleDENM, "DENM");
}

Forbidden combinations

The packet type designators btpOverCnh and btpOverGeo cannot be combined neither with each other nor with other packet type designators.
