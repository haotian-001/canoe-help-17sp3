Analysis Windows [top]

Screenshot

Short Description

Bus Statistics Window (Car2x)

The Bus Statistics Window shows WLAN specific data and values.

Car2x Certificate Explorer

With the Car2x Certificate Explorer you can analyze certificates that are known at runtime.

Car2x Network Explorer

With the Car2x Network Explorer you can load a Car2x database (e.g. ASNV_Configuration.xml) and display its content.

Car2x Station Manager

The Car2x Station Manager is the central unit to manage ITS Stations. With the help of the Car2x Station Manager, received messages are assigned to ITS stations that are managed in CANoe.

Car2x Protocol Analyzer

The Car2x Protocol Analyzer checks data content and timing of all Car2x frames against an extensible set of rules.

Data Window

With the Data Window for Car2x you can display signal values. The value of a signal is displayed by default as a physical value, as a raw value and with a progress bar.

Graphics Window

With the Graphics Window you can display Car2x signals graphically as curves on a XY diagram.

Map Window

With the Map Window you can display and analyze location coordinates of ITS objects and events received in a Car2x network on the map. In addition, you can use CAPL or .NET to draw your own objects on the map.

You can also display recorded position information of a GNSS receiver in this window.

Trace Window

With the Trace Window for Car2x you can specifically analyze WLAN packets.

State Tracker

With the State Tracker you can analyze states and state transitions of Car2x signals to display time-related dependencies.

Short Description

Car2x Network Explorer

With the Car2x Network Explorer you can load a Car2x database (e.g. ASNV_Configuration.xml) and display its content.

Scenario Editor

The Scenario Editor is an application with which traffic scenarios can be created via a graphical interface.

Scenario Manager

The Scenario Manager is the central unit for managing scenarios.

WLAN Packet Builder

With the WLAN Packet Builder you can easily create and configure WLAN packets and send them to the network.

Function Blocks [top]

Short description

Logging Block

You can log the WLAN bus traffic in BLF (Binary Logging Format) format to logging files.

Replay Block

You can replay logging files by using the Replay Block. Alternatively use the in the offline mode.

IP Filter

The IP Filter lets you filter out Ethernet packets , AFDX packets, WLAN packets and protocol-specific messages of the Application Layer, Automotive.
