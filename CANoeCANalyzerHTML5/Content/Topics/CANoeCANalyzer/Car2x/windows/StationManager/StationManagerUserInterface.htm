Measurement Start Action

  • Delete assignments
    At the start of measurement and activation of this mode, all contents are discarded. Assignments between certificates and MAC addresses to stations will not be saved. All assignments (also to database nodes) must be carried out during each measurement.
  • Save assignments
    If this mode is activated all assignments of certificates and MAC addresses to stations and database nodes will be saved and will not be rejected at the start of measurement. As long as no other mode is activated these assignments are still available even after reopening the configuration or after a restart of CANoe. You can manually assign or delete assignments of ITS Stations even outside the measurement.
  • Specific assignments
    If this mode is activated, the last exported image of the Car2x Station Manager is used at the start of (a) measurement. This mode is specially suited for test cases with defined initial state.

Import of the last image

Loads the last exported image of the Car2x Station Manager. This way it is possible to view and use the last exported image of the Car2x Station Manager.

Export of the current image

Saves the current image of the Car2x Station Manager. This way it is possible to reuse exactly this image at the start of measurement. Refer to Specific assignmentsabove.

Delete the content of the Car2x Station Manager

This proceduredeletes all assignments of certificates and MAC addresses to ITS stations. This is specially suitable if too many MAC addresses are available or if they were assigned wrongly and quick resolving is no longer possible.
