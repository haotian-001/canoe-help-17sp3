User Interface

 

The window is divided into several areass: the certificate overview, the certificate view with an upper and a lower area, the toolbar and the status bar.

Certificate Overview

On the left side of the window, the certificates are displayed in a tree view, which maps the signature hierarchy. Certificates with unknown signature certificate are attached at the highest level.

The following details are displayed:

Column/ Symbol

Description

Name

Name of the certificate that was assigned in the Car2x Certificate Manager. Validity status symbols are displayed to the right of the certificate name (see status column below).

HashedId8

HashedId8 of the certificate

Type

Certificate type (Root CA, Pseudonym CA, Pseudonym, etc.)

Signer HashedId8

HashedId8 of the signature certificate

Status

The validity status assigned to the certificate by CANoe. The validity status is also indicated by the icon to the right of the certificate name:

certificate is valid

certificate is invalid

validity of the certificate is unknown

Note

The certificates known at runtime are displayed in the Car2x Certificate Explorer.

Certificates and their private keys that are to be known at the start of measurement are managed in the Car2x Certificate Manager.

Certificate View

If you select a certificate on the left side in the certificate overview area, it is displayed on the right side.

Upper area: Here the certificate is displayed decoded in a tree view that conforms to the specification ETSI TS 103 097 respectively IEEE 1609.2. The columns field, value and raw value are displayed.

Lower area: Here the certificate is displayed coded as a hexadecimal byte sequence.

Toolbar

Symbol

Description

[]

With this, you can save certificates in order to, for example, import them later in the Car2x Certificate Manager of another configuration. In the Export dialog, you can limit which certificate types are to be saved: Root CA certificates, Pseudonym CA certificates, Pseudonym certificates, or other certificates. You cannot export private keys from the Car2x Certificate Explorer. Instead, use the Car2x Certificate Manager.

[]

With this, you can stop and continue the automatic update of the certificate list. With activated automatic update, the new certificates that are captured during measurement are added to the certificate overview.

Status Bar

The status bar at the bottom of the Car2x Certificate Explorer indicates the number of valid, invalid, and not yet analyzable certificates.
