Setup

Signal Protocol

 

You can set the communication parameter for an application channel on this page.

The dialog allows to configure the following parameters of the VN4610:

  • Channel Type:
    This entry allows -depending on the country code- to preconfigure some parameters to suitable default values.
  • Radio channel
  • Band width
  • Transmission power:
    See also transmission power restrictions applied in tolling areas
  • Data rate
  • Protocol Layout
    Defines whether EPD or LPD frames are used for transmission.

    The protocol type LPD is usually used for Car2x communication within the EU. In the USA, EPD is usually used. However, a correct protocol type must be used so that standard-compliant Car2x communication is possible.

    The protocol types EPD (EtherType protocol discrimination) and LPD (LLC protocol discrimination) allow the formal discrimination of the Car2x packets (IEEE 802.11p standard) from the other protocols and services of the IEEE 802 family. LPD and EPD headers define the structure of the beginning of a Car2x packet and thus how it is transmitted over the radio link:
  • LPD Car2x header is 8 bytes long and uses Ethertype 0x8947 and additional LLC/SNAP/... fields.
  • EPD Car2x header is 2 bytes long and uses Ethertype 0x88DC
See also IEEE 802.11-2016 and ETSI EN 302 663.
  • Tx-Antenna
    Allows selection of one of the two Tx-antennas on VN4610 device. The Auto setting is recommended for both the single-channel and the dual-channel operation. The settings 1 or 2 manually determine the antenna used for the channel. These should only be used if one of the following cases is present:
  • Single channel operation and only one antenna is connected.
  • Two-channel operation with unequally well placed antennas. Here the better placed antenna can be assigned to the more important channel.

The settings of the radio channel and the bandwidth are determined by the selection of the channel type; the channel type itself depends on the particular country code (e.g. US, EU).

Note

Fixed device property, specified on purchase.

The other parameters can be freely set, whereby the available maximum transmission power depends on the country code due to regulatory requirements. (10dBm EIRP for EU or 19dBm EIRP for US).

Note

The channel settings can also be modified and retrieved during measurement using the CAPL functions C2xConfigureChannel and C2xGetChannelConfiguration.

There various configuration parameters are described in more detail there.

Note

If the dialog shows Country Code: Unknown, no transmit permission, then there is an issue with the VN4610 (not connected or driver not installed).
