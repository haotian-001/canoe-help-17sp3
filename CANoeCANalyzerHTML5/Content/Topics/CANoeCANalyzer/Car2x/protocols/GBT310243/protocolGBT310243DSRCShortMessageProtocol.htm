Protocol Information

The DSRC Short Message Protocol (GB/T 31024.3) is a network protocol containing basic information for the C-V2X communication between individual ITS stations.

Protocol Dependencies [▲ top]

The data is interpreted as DSRC Short Message protocol if the protocol type is set to four in the previous Adaption Layer.

Optional Fields

The optional fields can occur in any order.

In the following table it was assumed for the offset determination that first the data rate and then the channel number appear. If the fields appear in a different order, the offsets change accordingly.

As soon as an optional field is available, the Count DSMP Information Elements field is automatically inserted. This field indicates how many optional fields are used.
