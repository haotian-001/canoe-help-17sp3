Protocol Information

The Adaption Layer (GB/T 31024.3) is a network protocol that determines the subsequent network protocol (IPv4, IPv6 or DSMP).

Protocol Dependencies [▲ top]

The data is interpreted as an adaptation layer if the following conditions are met

  • In the C-V2X General Information Layer, the Message Family is set to four.
