Protocol Information

Each packet that is evaluated by CANoe starts with the WLAN header.

Several tokens are available only if the used hardware support these tokens. If they are not supported, the detail view of the Trace Window shows --.

Protocol Dependencies [▲ top]

Each WLAN packet transmits an Ethernet header. Dependent on the use case the headers for CNH, BTP or GeoNetworking follow.
