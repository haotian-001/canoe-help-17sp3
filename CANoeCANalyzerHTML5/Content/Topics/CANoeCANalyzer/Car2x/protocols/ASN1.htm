Protocol Description Language ASN.1

 

Protocol Information

»ASN.1 is a standardized description language to define the fields of a protocol and the data description of Car2x application messages. The definition is done in an ASN.1 module.

Application messages that are specified by the ETSI as CAM, DENM, SPat and TOPO or by the SAE as J2735 (BSM, TIM, ...) are described in ASN.1 in the Car2x environment.

Protocol Dependencies

Application messages described with ASN.1 are transmitted with different protocols as their payload. Which protocol should be used for transmission must be defined during the import of the ASN.1 module.

Protocol and Packet Format

The protocol designator is set during the import with the ASN.1 Import Wizard. The token designators are defined in the ASN.1 modules.

  • Use Application Messages
  • Show Physical Values with ASN.1 in Trace Window
  • Initializing Several Protocol Layers
