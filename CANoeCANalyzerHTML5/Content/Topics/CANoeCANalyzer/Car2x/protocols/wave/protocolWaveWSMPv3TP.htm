Protocol Information

The WAVE Short Message Protocol (IEEE 1609.3 - 2016) is a network protocol that contains basic information for communication between individual ITS stations.

This protocol contains the Transport Header of the WAVE Short Message Protocol. This Header can never occur without a WSMP Network Header.

Protocol Dependencies [▲ top]

The protocol interpretation depends on the Transport Protocol ID of the WSMP Network Header.

Transport Protocol ID

Description

0

Address type PSID. No WAVE Information Elemente

1

Address type PSID. WAVE Information Elemente are available

2

Address type ITS Port Number. No WAVE Information Elements

3

Address type ITS Port Number. WAVE Information Elements available

Packet API [▲ top]

Note

During the initialization of the WSMP - Transport Header, the WSMP - Network Header is initialized as well. The Packet API sets the field Ethertype = 0x88DC (WAVE Short Message) in the Ethernet Header during the initialization of the WSMP - Network Header.
