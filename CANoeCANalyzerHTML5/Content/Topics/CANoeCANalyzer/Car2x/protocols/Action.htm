Protocol Information

The frame body of the WLAN Management Frame with sub type Action contains specific information (see IEEE 802.11, IEEE 802.11p and IEEE 1609.12) that are represented by the Action protocol layer in CANoe.

If the type is set to Management and the sub type is set to Action in the Frame Control field of the WLAN header, the frame body of the Action frame is interpreted after the Sequence Control field. If the fields contain the values specified below, the following is interpreted as WAVE Security Services (WSS) layer and then a WAVE Service Advertisement (WSA) layer.
