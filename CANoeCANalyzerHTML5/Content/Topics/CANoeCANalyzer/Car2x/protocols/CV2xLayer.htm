C-V2X Layer

 

Protocol Information

In the C-V2X layer, the Message Family is determined according to 3GPP TS 24.386 (2017-12).

Protocol Dependencies

The data is interpreted as a C-V2X layer if the following conditions are met:

  • The Data is transferred in a C-V2X frame.
