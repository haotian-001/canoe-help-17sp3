How to Manipulate LIN Signals

 

Using CAPL signal API

  1. Create a LIN configuration.
  2. In the Simulation Setup, add the Door.ldf database from the System Configuration (LIN) sample configuration. in the Simulation Setup CANoe adds a LIN ISC (Master), as well as several network nodes (RVM_Right, RVM_Left, DWR_Right, DWR_Left, DWF_Right, DWF_Left, GWI).
  3. Select a network node (e.g. DWF_Left) in theSimulation Setup (Simulation ribbon tab|Simulation Setup) and choose Configuration… from the context menu.
  4. Enter a filename for your CAPL program and then press [Edit] to start the CAPL Browser.
  5. Now use the CAPL function SetSignal to manipulate the LIN signal FrontLeftSwitchStatus.
    To get the signal's latest value use the function GetSignal.
    Here is a simple example assuming the LDF database (e.g. door.ldf) defines a signal DWFL_WinPos:

on key ’+’
{
  int val;
  val=$DWFL_WinPos;    // reading signal
  $DWFL_WinPos=val+1;  // writing signal
}

  1. Now save and compile the CAPL program e.g. by pressing <F9> and start the measurement.
  2. Open the Trace Window (Analysis ribbon tab|Trace) and expand the corresponding LIN frame (e.g. DWFL_WinPos).
    On pressing the key <+>, the signal should now increase its value by one.

Using the CAPL function "output()"

You can also update signals using the CAPL function output. Using this method, you can update multiple frame signals simultaneously. To try out this method, add a new event handler for the key<-> to your CAPL program with the following code:

on key ’-’
{
  linFrame DWFL_WinPos myframe;
  int val;
  val=myframe.FWL_WinPos;
  myframe.FWL_WinPos=val-1;
  output(myframe);
}

Now save and recompile your CAPL program and start the measurement again. With key<-> you should be able to decrement the signal's value.

Using the LIN Interactive Scheduler (LIN ISC)

  1. Create a LIN configuration.
  2. In the Simulation Setup, add the Door.ldf database from the System Configuration (LIN) sample configuration. in the Simulation Setup CANoe adds a LIN ISC (Master), as well as several network nodes (RVM_Right, RVM_Left, DWR_Right, DWR_Left, DWF_Right, DWF_Left, GWI).
  3. Open the ISC e.g. via double click.
  4. To update the signals of a LIN frame, select the Frames node, click on Add frame and insert a LIN frame e.g. DWFL_01.Select the appropriate send type.
  5. During the measurement, you can interactively change the frame signals, e.g. DWFL_WinPos, using editing buttons in the Signals view of the dialog. By clicking on [Send] you can send the frame.

Using Panels

You can also create your own panels using the Panel Designer (Tools ribbon tab|Panel Designer), which can also be started from the main toolbar .

  1. Select the Track Bar and insert it on the panel via drag and drop from the Toolbox.
  2. In the Properties Window of the track bar you can assign a LIN signal.
  3. Now save your panel File|Save.
  4. To configure CANoe to use this panel, open the menu path Home ribbon tab|Panel Configuration and use the [Add…] button to add the panel to your configuration.
  5. To display your panel on the currently active desktop, press the button [Display] and then close the dialog with [OK].
  6. After measurement start it should now be possible to change the signal’s value using the slider.
