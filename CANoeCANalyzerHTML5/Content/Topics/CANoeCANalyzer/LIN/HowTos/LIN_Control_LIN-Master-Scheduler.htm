How to Control a LIN Master's Scheduler

 

Using the Interactive Master

Step by Step Procedure

  1. Create a LIN configuration.
  2. In the Simulation Setup (Simulation ribbon tab|Simulation Setup) insert an Interactive Master block using the context menu.
  3. Open the Interactive Master block e.g. per double-click.
  4. Configure per context menu, which schedule the Interactive Master should start on measurement start. Per default the first schedule defined in the LDF will be started.
  5. Start the measurement and change the schedules as desired interactively.

Note
