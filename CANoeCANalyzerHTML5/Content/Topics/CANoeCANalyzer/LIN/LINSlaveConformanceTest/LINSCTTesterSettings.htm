Tester Settings

Test Setup for Test Modules|Slave Conformance Test Module|Configuration…|Tester Settings…

Here you configure the test system settings.

On test case fail

  • Add Trace snapshot to the report
    After each failed test case a Trace snapshot is added to the report.
    See test report: window capture. for further information on trace snapshots in the test report.
  • Name of the Trace Window
    Name of the Trace Window from which the snapshots are taken.
  • Trigger Scope hardware(Scope option is required)
    USB connection to the Scope hardware will be established automatically on starting the tests. After that on each failed test case the Scope hardware will be triggered automatically.
  • Add Scope snapshot to report
    Scope Window snapshot will be added to the report each time the Scope hardware triggers.
