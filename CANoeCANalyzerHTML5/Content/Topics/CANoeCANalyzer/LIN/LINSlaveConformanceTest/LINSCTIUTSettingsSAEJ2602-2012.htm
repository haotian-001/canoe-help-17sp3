IUT Settings SAE J2602:2012

Test Setup for Test Modules|Slave Conformance Test Module|Configuration…|IUT Settings…

Here you can configure IUT settings that are not specified in the database.

Common Settings

  • Sends wake-up signal
    Please deactivate this option only if the IUT is unable to send a wakeup sequence. All test cases for wakeup sequences will be deactivated.
  • Deviant Sleep timeout
    Here it is possible to set it or to override the sleep timeout value defined in the assigned database.
  • Deviant Initial Node Address (NAD)
    Initial node address value is a mandatory parameter for performing conformance tests. Here it is possible to set it or to override the value defined in the assigned database.
  • Deviant Variant ID
    Here it is possible to set it or to override the Variant ID value defined in the assigned database.
  • Deviant Supplier ID
    Here it is possible to set it or to override the Supplier ID value defined in the assigned database.
  • Deviant Function ID
    Here it is possible to set it or to override the Function ID value defined in the assigned database.
  • Supports autobauding
    Please activate this option only if the IUT has an autobauding clock.
  • Supports hardware DNN selection
    Please activate this option only if the IUT supports DNN selection via hardware, e.g. using a deep switch.
  • Supports software DNN selection
    Please activate this option only if the IUT supports DNN selection via software, i.e. if the DNN can be programmed somehow.

Configuration Services

  • Supports AssignNAD (0xB0) service
    Please deactivate this option only if the IUT does not support this optional service.
  • Supports AssignFrameID (0xB1) service
    Please deactivate this option only if the IUT does not support this optional service.
