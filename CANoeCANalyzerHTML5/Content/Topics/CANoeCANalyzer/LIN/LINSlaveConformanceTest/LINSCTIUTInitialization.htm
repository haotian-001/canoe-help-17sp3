IUT Initialization Settings

Test Setup for Test Modules|Slave Conformance Test Module|Configuration…|IUT Initialization…

There are several possibilities to reset a slave node. It is also possible to specify the times needed for a reset.

Reset Type (prior to each test case)

  • HW-reset using VH11xx
    This is the recommended reset method and conforms to the LIN conformance test specification. The hardware reset is performed using Vector's test hardware VH1100 or VH1150/VH1160.
  • HW-reset using VT7001
    The hardware reset is performed using VT System power module VT7001.
  • HW-reset manually
    The hardware reset must be performed manually. Each test case will wait for confirmation of the manual reset.
  • SW-Reset (waiting for BusIdle)
    Only a software reset is performed by waiting for the time specified for bus idle timeout (see separate edit box) to occur prior to each test case.
  • No reset
    No hardware reset is performed. This method can be used for initial tests if no hardware reset is easily possible.

Reset Parameters

  • Power off time
    Time the IUT needs for an hardware reset. For this time period the IUT supply is disconnected.
  • Wait for BusIdle time
    Time the bus has to be idle, so that the IUT performs an software reset.
  • IUT init time
    Time the IUT needs to start up after a reset.
  • Supply voltage
    Supply voltage of the IUT.

VT7001 Settings

  • Module name
    Name of the VT7001 module to be used. With automatic resolution the first found module is used.
  • Output channel
    Output channel to be used.
    Important: Transmission of the AvgCurrent measurement value has to be activated on the used output channel – for details see VT System configuration.

Automatic Sleep/Awake Detection

  • Activate
    Select this option if the IUT's transition between sleep and awake states can be detected automatically.
    If not selected the correct state transition must be confirmed by the user.
  • Maximum current consumption in sleep mode
    Maximum current consumption [in mA] of the IUT in sleep mode. This threshold is used to detect a transition from normal mode to sleep mode.
  • Minimum current consumption in normal mode
    Minimum current consumption [in mA] of the IUT in normal (awake) mode.
    This threshold is used to detect a transition from sleep mode to normal mode.
  • Time to low power state after sleep command
    The time the IUT needs to reach the low power state after receiving a sleep command. After this time period, the current consumption of the IUT is checked using the values defined above. The minimum possible value is 150 ms.

Automatic Adjustment of Recessive and Dominant Bus Level (only for J2602)

For J2602 conformance test cases which require adjustment of the recessive or dominant bus level, automatic adjustment via Vector's VH1150/VH1160 test hardware is available.
