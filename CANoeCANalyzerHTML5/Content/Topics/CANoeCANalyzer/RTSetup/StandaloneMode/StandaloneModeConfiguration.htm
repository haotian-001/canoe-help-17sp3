Configuration and Operation

 

You can access the configuration and control for standalone mode in CANoe via Home ribbon tab|Group Measurement|Vector Tool Platform|CANoe/CANalyzer|Standalone Mode.

CANoe is shipped with accessories called Standalone Manager and Vector Platform Manager that you can also install on any other computer. You can use it to control standalone mode on any VN8900 devices connected to that computer.

Note

An installation package for the Standalone Manager can be found in the CANoe installation directory in the subfolder Installer Additional Components\StandaloneUtilities and as a download package on the Vector website.

The Vector Platform Manager is delivered as part of a Vector driver package (available for download on the Vector website).

To set up standalone mode, please take the following steps:

Configuration of the Standalone Mode

  1. Check whether the configuration currently opened in CANoe is suitable for standalone mode (see Configuration Requirements).
  2. Open the Vector Tool Platform.
  3. Select the device to be configured (see Device Selection).
  4. Download the configuration to the device (see Device Configuration).
  5. Activate standalone mode on the device (see Device State).
  6. Start the standalone measurement (see Device State).

Note

For information about errors, see Error Messages.

As an alternative you can export the realtime specific part of a configuration in a standalone configuration file (RTCFG). You can load it later on the selected or an identical device using CANoe, the Vector Platform Manager or the Standalone Manager.
