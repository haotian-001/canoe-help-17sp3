Prerequisites

 

License

To use the CANoeOption For EtherCAT, the corresponding license must be available on your system.

Network Adapter

For the CANoeOption For EtherCAT, a network adapter of the type Intel® PRO/1000 is required (e.g. Intel I210 or Intel I219LM).

Driver

To be able to use the CANoeOption For EtherCAT, you have to install a special driver for your network adapter. It is delivered together with CANoe but it must be installed subsequently.

For the driver installation you can choose between a

  • Manual installation for all approved devices
  • Partially automated installation with the EtherCAT® Driver VTP-App for approved VT6000 and VN8900 devices

Manual Installation

Note

Make sure to install the suitable driver for your operating system.

  • If you use Windows 64 bit, you have to install the driver from the x64 directory.

  • For Windows 32 bit, you have to install the driver from the x86 directory.

Installation

For this, proceed as follows:

  1. Open the WindowsDevice Manager.
  2. Select under Network adapter the network adapter you want to use for the option For EtherCAT.
  3. Open the context menu for this network adapter and select Update Driver Software.
  4. Then choose the following commands:
  • Search for the driver software on the computer
  • Select a driver from a list of device drivers on this computer
  • Data carrier...
  1. Select the appropriate driver (32- or 64-bit) suitable for your operating system.
  2. <CANoe_Installation_Folder>\Installer Additional Components\EtherCAT\Driver — Open folder
  3. Select in the Action menu of the Device Manager the command Search for modified hardware.

The network adapter should now appear in the list under the name ECAT Driver for network adapter. This means that it is ready for the usage of the CANoeOption For EtherCAT.

Deinstallation

For this, proceed as follows:

  1. WindowsDevice Manager.
  2. Select under Network adapter the network adapter you want to uninstall.
  3. Open the context menucontext menu for this network adapter and select Uninstall driver.
  4. Select in the Action menu of the Device Manager the command Search for modified hardware.

In the list, the device should now be displayed again as a regular network adapter.

Partially Automated Installation with the EtherCAT® Driver VTP-App

The EtherCAT® Driver VTP-App enables the installation of the EtherCAT® driver for specific Ethernet adapters (Intel I210 network adapter).

Note

The use of the EtherCAT® Driver VTP-App is only allowed for the following devices with a 64-bit operating system:

  • VT6020
  • VT6060
  • VN8911
  • VN8914

Installation

  1. Connect the device to the computer via Ethernet (VT6000/VN8900) or USB (VN8900 devices only).

  2. Install the EtherCAT® Driver VTP-App via the Vector Platform Manager|Tool Platform|Applications with the [Add...] button.

    3. The EtherCAT® Driver VTP-App is stored in the CANoe installation in the following directory:

    <CANoe_Installation_Folder>\Installer Additional Components\EtherCAT\Vector.EtherCatDriver.vtpapp — Open folder

    Notice!

    For VT6000 devices, do not activate the EtherCAT® driver for the Ethernet port (EtherCAT® Driver|Installation/Activation page) that is used for the connection to CANoe or the Vector Platform Manager. Otherwise it will no longer be possible to establish a connection to the device!

    Note

    If the EtherCAT® Driver VTP-App is not loaded or displayed, please close the Vector Platform Manager and restart it.

  3. On the EtherCAT® Driver|Installation/Activation page, select one of the supported network adapters from the list and activate or deactivate the driver using the corresponding button.

    Notice!

    • Please do not switch off the devices during the activation or deactivation process!
    • During the activation/deactivation of the EtherCAT® driver, the devices are restarted several times. Due to the resulting disconnection, the EtherCAT® Driver VTP-App is automatically closed. Depending on the device, this process can take up to several minutes. The current state is displayed in the output window on the EtherCAT® Driver|Installation/Activation page.
    • The Ethernet adapter configuration may enter an unexpected state on VN8900 devices with a Windows 7 image after deactivating the EtherCAT® driver when manually assigning a new IP address (Tool Platform|Network Settings page). The Ethernet port can then no longer be used normally. In this case, the original Ethernet adapter configuration can be restored by installing an operating system image.

    4. Note

    If you want to deactivate the EtherCAT® driver for an adapter of a device with a Windows 7 image and then set a new IP address for the resulting Ethernet adapter via the Tool Platform|Network Settings page, please wait 2 minutes, so that all adapters can initialize completely.
    The completed initialization is displayed on the EtherCAT® Driver|Installation/Activation page by the IPv4 status Up or Down. If this is not the case, please restart the Vector Platform Manager.
