Communication Concept

 

 CANoe as simulation and test environment provides an uniform and cross-network communication concept. This communication concept enables application-related modeling of classic and new communication patterns. The communication concept provides consistent decoupling of logical communication on the application layer from protocols of the network level.

This means that CANoe provides a network-independent interface for application models and test scripts. Separately from this, the actual access or transmission is configured via a specific network.
