Safety Functions

 

The safety functions are used to prevent faulty setting of the VT System and thus to protect the VT System and its hardware from damage.

You can prevent the following, for example, using the safety functions:

  • Inputs of your control units from being stimulated with excessive voltage
  • A short-circuit being created by a relay while excessive currents are flowing

To implement such requirements, you must define so-called constraints.

Constraints

Constraints can be defined for most settings of the VT System.

Proceed as follows:

  1. Open the VT System Configuration dialog.
  2. Select the channel concerned there.

  3. Select the Constraints page.

    Constraints can be displayed, edited and created here. The constraints cannot be changed during a measurement. It's best to specify the constraints for a configuration before the first measurement is started.

For setting values with which relays are switched, a condition under which the relay can be closed or opened can be specified with the constraints. In most cases, the condition checks the value of any desired long or float system variable. Thus a short-circuit can be prevented from switching if the measured voltage on a channel exceeds a certain value, for example. The opening or closing of a relay can also be forbidden in general.

Note

The constraint is only checked before the relay is switched. Continuous checking is not carried out.

Conditions can be specified directly for the remaining setting values. For example, a stimulated voltage can be constrained to a value ≤ 12V.

If an attempt is made to set a setting value that violates a constraint, an error message is output in the Write Window and the invalid setting value is not transferred to the VT System.

Note

The standard value of setting values cannot be changed.

This means that setting values could violate constraints when a measurement is started.

For this reason, assign valid values to relevant setting values when starting the measurement to ensure that constraints are always complied with.
