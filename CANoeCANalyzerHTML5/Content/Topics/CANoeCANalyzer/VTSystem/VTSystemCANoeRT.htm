Notes for Usage with Distributed Mode

 

In principle there are no restrictions when running VT System along with distributed mode.

The VT System is connected to the RT server, i.e. the computer on which the CANoe realtime component is executed and to which the network interfaces are connected.

Note on EThernet

A separate Ethernet interface must be used for the VT System on the RT server.

In principle it is often possible to use the same Ethernet interface also for connections to the RT server and the user computer.

However, the two systems (internal CANoe communication and communication with the VT System) will steal bandwidth from one another, which greatly limits the practicable running of the system.

Note on Interface Settings

The VT System interface settings are saved in the user computer configuration file.

If you intend to use it without the RT server, you must adjust the network interface settings accordingly in the Options dialog (Extensions|VT System).
