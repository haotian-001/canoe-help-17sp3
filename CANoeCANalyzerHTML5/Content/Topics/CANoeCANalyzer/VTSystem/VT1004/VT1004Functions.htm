These settings can be adjusted in XML test modules using the vtsystem_configure command.

Example — XML Test Module

<vtsystem_configure title="change settings">
  <reset_min_max channel="LowBeamRight" />
  <integration_time channel="LowBeamRight" sysvar="RMS">0.1</integration_time>
</vtsystem_configure>

Example — CAPL Test Module

// reset min/max values of "LowBeamRight" channel
sysvar::VTS::LowBeamRight.ResetMinMax ();

// change transfer cycle time of current value to 10ms
sysvar::VTS::LowBeamRight::Cur.SetTransferCycle (0.01);
