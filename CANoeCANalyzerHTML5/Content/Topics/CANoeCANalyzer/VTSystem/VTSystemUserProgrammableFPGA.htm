User FPGA

 

The User FPGA feature is used to extend the standard functionality of a VT System module with additional functions, that are defined by the user. For this purpose an additional FPGA is provided on the VT System module, that can access the inputs and outputs of the module and communicate with the CANoe configuration via system variables.

Note

To use this feature the relevant VT System module has to be available in the User FPGA variant.

Programming and managing the User FPGA is done with the VT System FPGA Manager, which you'll find under Tools in the CANoe folder.

If there is a User FPGA on the VT System module, depending on the configuration additional system variables are available for this module. They are managed by the VT System FPGA Manager.

  • Is it about measurement values you have to activate them like other variables in CANoe's VT System Configuration dialog, before they can be used.
  • If the User FPGA is used to stimulate values, the Stimulation Mode of the used module has to be set to Voltage and the Curve Type has to be set to User FPGA.
