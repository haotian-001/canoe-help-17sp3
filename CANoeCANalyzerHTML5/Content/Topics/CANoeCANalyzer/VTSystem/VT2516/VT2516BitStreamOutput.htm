Bitstream Output

In addition to outputting digital values and generating PWM signals directly, the VT2516 can also output bitstreams. The bitstream is defined in a file, and can have a maximum of 4096 bits.

You can load the bitstream from the file to the VT System module via the LoadWFBitStream CAPL function or the vtsystem_configure command. In addition, you can set the time increment, pause and number of repetitions parameters via the SetWFParams CAPL function or the test primitive.

  • The time increment parameter specifies the interval between two bits.
  • The number of repetitions parameter specifies how many times the bitstream is repeated.
  • If the bitstream is output repeatedly, the pause parameter specifies the length of the pause before each repetition.

File Format

The file format is very simple. It contains the bitstream in the form of a string composed of the numbers 0 and 1. You can insert spaces and line breaks between the numbers to structure the bitstream. Comments can be inserted after //. All characters on this line are interpreted as comments and are ignored, up to the end of the line.

Example 1:

The following file defines the bitstream 001100110010101100:

// Sample bitstream
0011
0011
0
0  // bits may be defined in one line
1 0 1 0
11 00  // definition in groups

Example 2: Outputting a bit sequence (bitstream)

In this example, a bit sequence that constitutes a digital signal with increasing pulse durations and pauses is output. The bit length is set to 100 Âµs, and the sequence is output twice with a pause of 5 ms in between.

CAPL

// set high and low state states
sysvar::VTS::DOut.SetPWMVoltageHigh (12.0);
sysvar::VTS::DOut.SetPWMVoltageLow (0.0);
// set output mode to "voltage from bitstream"
sysvar::VTS::DOut.SetCurveType (eVTSCurveTypeBitStream);
sysvar::VTS::DOut.SetWFParams (0.0001, 0.005, 2);
sysvar::VTS::DOut.LoadWFBitStream ("bitseq.txt");
sysvar::VTS::DOut.SetStimulationMode (eVTSStimulationModeVoltage);
// start bitstream output
sysvar::VTS::Switch.StartStimulation ();
