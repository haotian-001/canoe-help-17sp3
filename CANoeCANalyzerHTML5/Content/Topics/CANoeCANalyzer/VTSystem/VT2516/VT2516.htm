VT2516 – Digital Module

 

General

The VT2516 digital module has 16 identical channels, each of which can be connected to an ECU input or output.

Each channel is connected using one line (single-ended). The signals pertain to the ECU ground, which is connected separately to the VT2516 along with the ECU power supply (Vbatt). The voltage between the signal line and ECU ground can range from -40 V to +40 V.

Notes on Module Series VT2516

The term VT2516 and the belonging description also cover variants of the module, e.g. VT2516A.

If there are differing details for a variant it will be described explicitly in particular cases. There are no visible differences in CANoe between the VT2516 and the VT2516A. Please refer to the VTSystem Manual (PDF) for data about differences in the accuracy of analog voltage measurement.

The VT2516 provides circuitry for ECU inputs and outputs, which essentially have two states, i.e. they operate digitally. Switch inputs are an example – the ECU detects whether or not the switch at this input is connected to a reference potential (ground or Vbatt). The input has no other states. External circuitry is also simple; the switch itself can generate a short circuit to ground or Vbatt. An example in the opposite direction is connecting simple LEDs. To turn on the LED, the ECU switches the output to Vbatt or ground. The ECU recognizes only two states in this case as well: the output is either switched or not.

Switching Different Signal Paths

Usage, Functions, Technical Data

Relays can be used to switch a channel's ECU line to an original component (depending on whether it is used as an input or an output, this is either a sensor or an actuator/load), to a load resistor, to ground, to Vbatt or to a bus bar.

The relays are switched using each channel's system variables.

All signal paths can carry a constant current of up to 800 mA.

Relays to ground and Vbatt can be used for two purposes:

  • To simulate the original sensor, especially if this is a switch or similar item that switches to ground or Vbatt (opens/closes) or switches between the two potentials.
  • To short circuit the ECU line to ground or Vbatt.

Notice!

Correct grounding is absolutely essential when operating VT modules!

Additionally the ECU ground and Vbatt must be connected to the VT2516!

For more information, see VT System Ground Connection.
