The following settings can be adjusted via the member functions of the VT2516 system variables or the corresponding system variable namespaces in CAPL.

Setting

System Variable

CAPL Function

Start and stop stimulus output

Namespace of a VT2516 channel

Stimulation mode

SetCurveType

Voltage in Low state

Load digital output curve (bitstream)

LoadWFBitStream

(for groups of channels)

Namespace of the VT2516 module

SetThreshold1_8 / SetThreshold9_16

Set integration time

System variable Avg

Set cycle time

All VT2516 read system variables

SetTransferCycle

You can modify these settings in XML test modules using the vtsystem_configure command.

Example: Test Module

CAPL

// set high/low voltages to 10 V and 2 V
sysvar::VTS::DoorSwitch.SetPWMVoltageHigh (10.0);
sysvar::VTS::DoorSwitch.SetPWMVoltageLow (2.0);

// start stimulating "DoorSwitch" channel
sysvar::VTS::DoorSwitch.StartStimulation ();

// set 5.0 V as threshold for channels 1-8 of "DoorIOs" module
sysvar::VTS::DoorIOs.SetThreshold1_8 (5.0);
