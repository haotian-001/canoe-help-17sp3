Voltage Stimulation

 

To simulate an equivalent sensor, the module can generate a voltage that is applied at the ECU input. The voltage to be issued is specified via a system variable and can therefore also be varied during the course of a test. A new voltage value can be specified every 1 ms.

There are two variants of the module VT2004. The older variant (label VT2004) supports only an output voltage of …27 V, while the newer variant (label VT2004A) supports a voltage range of 0…40 V. The following section lists the values for the new variant VT2004A; the values for the old variant are in brackets.

An output voltage of 0 to 40 V (27 V) can be specified. The resolution is 2.5 mV. The voltage stimulus output can drive or receive a maximum of 150 mA.

The specified voltage is delivered as voltage to line a in respect to line b. Line b does not need to have ground potential in this case. The voltage output at line b must always be balanced against the ECU ground within a range of 0…40 V (27) Volt.

Notice!

Since the output voltage on line a is always referenced to line b, line b must always be set to a fixed potential, e.g. the ECU ground, when voltage is output.

Outputting voltage, even if it is only for testing, only works if e.g. the ECU ground is connected to line b.

The stimulation can occur with fixed values or as a result of a PWM signal directly from CANoe via the corresponding system variable. However, the module can also output a saved wave form. See Stimulation with arbitrary wave forms.

Example 1: Create Constant Voltage

The following example creates a constant output voltage of 12 V.

CAPL

// Set voltage stimulation mode
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeVoltage);
// Set curve type constant value
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypeConstant);
// Set voltage to 12V
@sysvar::VTS::TempSensor::Voltage = 12;

.NET (#C)

Example 2: Create PWM Signal (Voltage)

The following example describes the creation of a PWM voltage signal with a frequency of 100 Hz and a duty cycle of 40%. The high level of the signal is 14 V and the low level is 10 V.

CAPL

// Set voltage stimulation mode
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeVoltage);
// Set curve type PWM
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypePWM);
// Set the low voltage of the PWM signal to 10V
sysvar::VTS::TempSensor.SetPWMVoltageLow(10);
// Set the high voltage of the PWM signal to 14V
sysvar::VTS::TempSensor.SetPWMVoltageHigh(14);
// Set duty cycle to 40%
@sysvar::VTS::TempSensor::PWMDC = 40;
// Set frequency to 100Hz
@sysvar::VTS::TempSensor::PWMFreq = 100;
// Start stimulation
sysvar::VTS::TempSensor.StartStimulation();

.NET (#C)

Set Sidebar | General | Curve Type to PWM
