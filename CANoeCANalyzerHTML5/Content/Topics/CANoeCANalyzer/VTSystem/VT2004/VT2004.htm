VT2004 – Stimulation Module

 

General

The Stimulation Module VT2004 has four identical channels, each of which can be connected to an ECU input.
All channels have a two-wire layout, so the ECU inputs do not need to work against a fixed potential such as the ground. Line b typically carries the smaller potential, e.g. ground.

Notes on Module Series VT2004

The term VT2004 and the belonging description also cover variants of the module, e.g. VT2004A.

If there are differing details for a variant it will be described explicitly in particular cases.

There are two variants of the module VT2004. The older variant (label VT2004) supports only a voltage range of ±32.7 V, while the newer variant (label VT2004A) supports a voltage range of ±40 V. The following section lists the values for the new variant VT2004A; the values for the old variant are in brackets.

Each line must bring a voltage to bear against the ECU ground ranging from at least -40 V (-32 V) up to +40 V (+32 V). A voltage between -40 V (-32 V) and +40 V (+32 V) can be applied between lines a and b.

Switching different signal paths

Usage, Functions, Technical Data

The relays are used to switch the ECU lines (for each channel) to the original sensor, the internal decade resistor or one of the two bus bars, or to generate a short circuit between the two lines.

The relays are switched via the corresponding system variables of the individual channel or module (bus bar switch relay).

Notice!

  • Correct grounding is absolutely essential when operating VT System modules.

    In particular, the ECU ground needs to be connected to AGND, and AGND to DGND.

    If these connections are missing, the VT System module may be damaged!

  • For more information, see VT System Ground Connection.
