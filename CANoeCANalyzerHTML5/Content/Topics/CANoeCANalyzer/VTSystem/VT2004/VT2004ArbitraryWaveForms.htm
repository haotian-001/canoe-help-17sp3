Arbitrary Wave Forms

 

Stimulation of Wave Forms

Not only can the VT2004 stimulation module be used to stimulate constant values and PWM signals, it also allows the use of arbitrary wave forms for stimulating. A maximum of 512 interpolation points are defined in a file for this. In addition to the stimulated resistance or voltage value, the number of intermediate points following an interpolation point must also be specified.

An example of such a wave form is shown in the following figure.

A curve can also be loaded from a file using CAPL command LoadWFVoltage or using the vtsystem_configure command in a channel of a VT2004 module. In addition, the SetWFParams command or the primitive can be used to set the following parameters: Time increment, Pause, Number of repeats, Starting point, and Start delay . Time increment and Pause can also be changed using system variables. The change has a direct effect on an active output for this module.

  • The Time increment specifies the time increment between interpolation nodes or intermediate points. This parameter can also be changed during the output of the curve using a system variable. In addition, this value can be negative in order to change the direction of the output. The output can also be frozen at a point by setting the time increment to 0.0. If a different value is set afterwards, the output is resumed at the point at which it was frozen.
  • The Number of repeats specifies how many times in a row the curve is to be stimulated.
  • The Pause specifies for multiple stimulation that length of the waiting period after the complete stimulation until the curve output is restarted. This parameter can also be changed during the output of the curve using a system variable.
  • The Starting point specifies the interpolation point of the curve at which the stimulation is to be start.
  • The Start delay specifies the time delay after which the stimulation of the curve is to start.

Additional parameters are available for curves with voltage values. They have no effect on the output of curves with resistance values.

  • The value specified in the file is multiplied by the Factor. It is set using system variable WFFactor. The default setting is a value of 1.0.
  • The Offset is added after the multiplication by the factor. (Output value = (Voltage value from file * Factor) + Offset). It is set using system variable WFOffset and its default setting is a value of 0.0.
  • The Minimum can be used to specify for a curve that a certain value will never be fallen below. This means that output values that are less than the Minimum after applying Factor and Offset are replaced by the Minimum. The Minimum is set using system variable WFMin and its default setting is a value of 0.0.
  • Accordingly, the Maximum is used to specify that a certain value will never be exceeded. This means that output values that are greater than the Maximum after applying Factor and Offset are replaced by the Maximum. The Maximum is set using system variable WFMax and its default setting is a value of 27.0.

File Format

Example

Structure of a file containing a wave form:

1. interpolation point value [; number of intermediate points]
2. interpolation point value [; number of intermediate points]
...
n. interpolation point value [; number of intermediate points]

Each line defines one interpolation point with a value that is interpreted either as a voltage in volts or a resistance in ohms. A period "." must be used for the decimal point separator. A total of three decimal places will be considered here. In addition, the number of intermediate points can also be indicated, preceded by a semicolon. As shown in Figure 1, an additional intermediate point extends the stimulation of the current by one time increment. Generally, the number of intermediate points between interpolation points can range from 0 to 65534. An exception to this is the resistance stimulation on channel 4, in which a maximum of only 254 intermediate points is possible. If the number of intermediate points is not specified, it is set to 0. That is, the next interpolation point follows directly after one time increment. Line comments can be inserted in the file, preceded by "//". Lines that are empty or contain only comments are ignored.

Example

File that stimulates the wave form in the figure above:

// Example of an arbitrary wave form
2     // Next value is an interpolation point
3; 2  // Two output points lie between this and the next interpolation point
5; 1
4; 1
2.8;  // Also possible

During stimulation the stimulated voltage wave form is not directly represented by system variables. For example the system variables Voltage and Resistor serve for control and do not change during a wave form stimulation.

Example: Create Voltage Wave Form

The following example describes the stimulation of a voltage wave form with a time increment of 100 Âµs. The Wave form is stimulated 3 times with 2 seconds pause between the wave forms.

CAPL

// Load curve from file
sysvar::VTS::TempSensor.LoadWFVoltage("C:\\ECU_Test\\WaveForms\\TempCurve.txt");
// Set voltage stimulation mode
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeVoltage);
// Set curve type Wave Form
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypeAnalogWaveform);
// set parameters: time increment 100Âµs, pause 2s, 3 repeats
sysvar::VTS::TempSensor.SetWFParams( 0.0001, 2.0, 3);
// Start stimulation
sysvar::VTS::TempSensor.StartStimulation();
