Resistor Stimulation

 

To simulate a sensor, the VT2004 can generate a resistance via an internal decade resistor. The resistance value is specified by the test program via the corresponding system variable. A new resistance value can be applied at most every 1 ms because of relay switching times.

There are two variants of the module VT2004. With the newer variant (label VT2004A) the decade resistor supports a larger voltage range than the older variant (label VT2004). The following section lists the values for the new variant VT2004A; the values for the old variant are in brackets.

There are two resistor stimulation modes that differ in their switching behavior:

  • R>
    While switching between a resistance value R1 and a value R2, interim values must fall within or above the value range of R1/R2, i.e., values must be greater R1 and R2.
  • R<
    While switching, interim values must fall within the range of R1/R2 or below this range, i.e., values must be less than R1 and R2.

The stimulation can occur with fixed values or as a result of a PWM signal directly from CANoe via the corresponding system variable. However, the module can also output a saved wave form. See Stimulation with arbitrary wave forms.

Example 1: Create Constant Resistance

The following example creates a constant resistance output of 120 Ω.

CAPL

// Set resistance stimulation R> mode
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeResistanceGreater);
// Set curve type constant value
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypeConstant);
// Set Resistance to 120Ohm
@sysvar::VTS::TempSensor::Resistor = 120;
// Start stimulation
sysvar::VTS::TempSensor.StartStimulation();

.NET (#C)

Example 2: Create PWM Resistance Signal

The following example describes the creation of a PWM resistance signal with a frequency of 100 Hz and a duty cycle of 40%. The high level of the signal is 240 Ω and the low level is 120 Ω.

CAPL

// Set resistance stimulation R> mode
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeResistanceGreater);
// Set curve type PWM
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypePWM);
// Set the low resistance of the PWM signal to 120Ohm
sysvar::VTS::TempSensor.SetPWMResistanceLow(120);
// Set the high resistance of the PWM signal to 240Ohm
sysvar::VTS::TempSensor.SetPWMResistanceHigh(240);
// Set duty cycle to 40%
@sysvar::VTS::TempSensor::PWMDC = 40;
// Set frequency to 100Hz
@sysvar::VTS::TempSensor::PWMFreq = 100;
// Start stimulation
sysvar::VTS::TempSensor.StartStimulation();

.NET (#C)
