PWM Switching

 

For periodically switching of matrix paths, each channel can be operated in PWM mode.
To run PWM switching, the following steps are required, either manually or programmatically:

  • set Curve Type to PWM,
  • select all rows which shall be connected to the desired channel,
  • set the desired parameters for PWM switching,
  • start the stimulation.

After start of the stimulation, all connected rows are switched synchronously with the configured PWM parameters.

Note

While a PWM stimulation is running, the static setting of each matrix node is overruled if and only if the corresponding row is selected. After stop of the stimulation, the last state of the mode is maintained. The last static configuration of the node is set when Curve Type is changed to Constant.

Example

In this example, the matrix rows 2 and 4 are switched periodically with 500 Hz, a duty cycle of 40% and with 10 repetitions on matrix channel 3. The selection of rows 2 and 4 corresponds to the bit pattern I0I0, therefore the argument for the SetOutputMask method has an integer value of 10.

CAPL

// This calls the method SetCurveType on VTS::MyMatrixChannel
sysvar::VTS::MyMatrixChannel.SetCurveType(eVTSCurveTypePWM);

// This calls the method SetOutputMask on VTS::MyMatrixChannel
sysvar::VTS::MyMatrixChannel.SetOutputMask(10);

// Set the system variable VTS::MyMatrixChannel::PWMOutputFreq to 500
@sysvar::VTS::MyMatrixChannel::PWMOutputFreq = 500;

// Set the system variable VTS::MyMatrixChannel::PWMOutputDC to 40
@sysvar::VTS::MyMatrixChannel::PWMOutputDC = 40;

// This calls the method SetPWMRepeats on VTS::MyMatrixChannel
sysvar::VTS:: MyMatrixChannel.SetPWMRepeats(10);

// Start stimulation
sysvar::VTS:: MyMatrixChannel.StartSimulation();

.NET (#C)
