System Variables

 

System variables can be used to read the following output values for each VT2832 channel.

The namespace is the name of the channel specified in the VT System configuration:

Value/Setting

System Variable

R/W

Value Semantics

Current value of the total current on this channel

  • Positive
    current flows from matrix column to matrix row/switch,
    i.e. the matrix column is connected to the "+" pole
  • Negative
    current flows from matrix row/switch to matrix column,
    i.e. the matrix column is connected to the "–" pole

CurCurrent

R

DOUBLE, in [A]

Current value of the voltage of this channel against DGND.

CurVoltage

R

DOUBLE, in [V]

System variables can be used to read or write the following settings for channels 1 to 8 of a VT2832.

The namespace is the name of the module in the VT System configuration:

Value/Setting

System Variable

R/W

Value Semantics

Connection of matrix row 1 to the matrix column
which corresponds to this channel

RowSwitch1

W

INTEGER

(0 = open, 1 = closed)

Connection of matrix row 2 to the matrix column
which corresponds to this channel

RowSwitch2

Connection of matrix row 3 to the matrix column
which corresponds to this channel

RowSwitch3

Connection of matrix row 4 to the matrix column
which corresponds to this channel

RowSwitch4

Status of the stimulation

StimulationRunning

R

INTEGER

(0 = off, 1 = running)

System variables can be used to read or write the following settings additionally for channels 5 to 8 of a VT2832.

The namespace is the name of the module in the VT System configuration:

Value/Setting

System Variable

R/W

Value Semantics

Connection of the matrix column (C) of this channel
with the corresponding switch port (S).

The following assignment is implemented:
C5–S1, C6–S2, C7–S3 and C8–S4.

DirectSwitch

W

INTEGER

(0 = open, 1 = closed)

The namespace is the name of the module in theVT System configuration:

Value/Setting

System Variable

R/W

Value Semantics

PWM output, frequency

PWMOutputFreq

W

DOUBLE, in [Hz]

(0.02…10000.0)

PWM output, duty cycle

PWMOutputDC

W

DOUBLE, in [%]

(0…100)

Pause between repetitions when a bitstream is switched

WFPause

W

DOUBLE, in [s]

(0.0…4294.0)

When a bitstream is switched, this specifies how long
the value of a sampling node is maintained before
moving on to the next sampling node.

If the system variable is set to 0.0,
the replay will be frozen until another value is set.

WFTimeIncrement

W

DOUBLE, in [s]

(-0.065…+0.065)
