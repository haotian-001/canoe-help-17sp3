Internal Power Supply Unit

 

The VT7001/VT7101 provides a third power supply unit, which is an internal power supply. This is fed by the VT System's power supply unit. It provides up to 2 A, but does not have an adjustable current limiter. The output voltage can be set between the minimum and maximum voltage as shown in the table below. You can do this directly via a system variable or an arbitrary curve, as with the external power supply units. The rise time of the output voltage is quite moderate.

The internal power supply unit is especially useful for static powering less power-intensive ECUs when there are no special power supply requirements (e.g. regarding rise time).

Example: Using the Internal Power Supply

The internal power supply unit delivers power and the ECU is connected to OUT1 (clamp 30), OUT2 (clamp 15) and GND1 (clamp 31). In this example, the channel for the internal power supply unit is named IntSupply, the two output channels are named Clamp30 and Clamp15, and the VT7001 module is named PowerSupply.

CAPL

// set mode to one power supply only -> internal power supply
sysvar::VTS::PowerSupply.SetInterconnectionMode (eVTSInterconnectionModeSupInt);

// set voltage to 12.0 V
sysvar::VTS::IntSupply.SetRefVoltageMode (eVTSRefVoltageModeConstant);
@sysvar::VTS::IntSupply::RefVoltage = 12.0;

// switch both outputs on
@sysvar::VTS::Clamp30::Active = eVTSOutputModeActive;
@sysvar::VTS::Clamp15::Active = eVTSOutputModeActive;
