The functions for operating the RS232 interfaces on the VT7001/VT7101 module are listed on page RS232 API.

You can modify these settings in XML test modules using the vtsystem_configure command.

Example Test Module

CAPL

// set interconnection mode to internal power supply only
sysvar::VTS::PowerModule.SetInterconnectionMode (eVTSInterconnectionModeSupInt);

// set output voltage to 12.0 V and activate output OUT1 (named "clamp15")
@sysvar::VTS::IntSupply::RefVoltage = 12.0;
@sysvar::VTS::Clamp15::Active = eVTSOutputModeActive;
