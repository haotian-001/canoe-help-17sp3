VT7001/VT7101 – Power Module

 

General

The VT7001/VT7101 power module is used to control the power supply lines of the ECU being tested.

The actual power for the ECU is delivered by an internal or external power supply. The VT7001/VT7101 controls this power supply, measures voltage and current, and allows for a variety of connections between the power supply unit and the ECU. The module can also be used to generate line breaks and short circuits on the lines connected to the ECU.

Notes on Module Series VT7001/VT7101

The term VT7001/VT7101 and the belonging description also cover variants of the module, e.g. VT7001A.

If there are differing details for a variant it will be described explicitly in particular cases.

The VT7001/VT7101has three RS232 interfaces for communication with external devices, especially PSUs and displays. The interfaces are assigned to the two channels for external power supplies and the channel for the entire module. A more precise description can be found on page RS232 API Functions (VT7001/VT7101).

Switching Different Signal Paths

Usage, Functions, Technical Data

The following image illustrates the VT7001/VT7101's different inputs and outputs.

You can reverse the polarity of the input of power supply unit 2. This lets you run reverse polarity tests even with a power supply unit that can only supply positive voltages.

When power supply unit 2 has reversed polarity, the corresponding voltage for controlling the power supply unit will automatically have reversed polarity, too. If the power supply unit is controlled via VT7001/VT7101, you should first set the control voltage to 0 V, then reverse the input polarity and finally set the voltage that applies a negative voltage at the ECU.

The various connection options (mode switches) are explained in Interconnection Modes.

AGND and DGND are automatically connected in the VT7001/VT7101 power module (GND1 to AGND, AGND with DGND). If you are using a VT7001/VT7101, you do not need to connect AGND and DGND at the power supply plug on the backplane. Adding a further connection at the backplane will not damage the system, but it affects the precision of the measurements (watchword: ground loop). If more than one VT7001/VT7101 is used in a system, the ground only needs to be connected in the first VT7001/VT7101.

Notice!

Correct grounding is absolutely essential when operating VT System modules!

For more information, see VT System Ground Connection.
