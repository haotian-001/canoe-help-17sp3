The signal lines are labeled High and Low because with CAN, these two lines carry the CAN High and CAN Low signals. With LIN and Single Wire CAN, only the High line is used.

The RLC relays are located on an optional supplemental circuit board (RLCpiggy). You can use them e.g. to switch a capacitance between bus lines. The exact use of the relays and switching of signal paths depends on the actual supplemental circuit board used.

You can use system variables to read or write the following settings for a VT6104/6204 module.

The settings are valid for the entire module. The namespace is the name of the module in the VT System configuration:

System Variables

This variable indicates if this channel/module operates in offline or online mode.

IsOffline

R

Boolean

It is not possible to read back output values from an output system variable. When the various system variables are read, only the most recent value is returned. However, this value does not always represent the current state of the hardware.

The system variables can be used in CAPL, XML and .NET test cases. They can also be used in analysis windows and panels.
