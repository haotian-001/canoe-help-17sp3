Standalone Mode

 

Effective with CANoe 8.2 SP2 you can use VT System also in CANoe's standalone mode. For the most part the needed files are automatically taken for the standalone configuration, as long as the VT System integration is already activated at that time when you create the configuration.

An additional step is only necessary for files that are loaded on the VT System modules when outputting Arbitrary Curves and Bitstreams. You have to register these files as so-called user files in CANoe's Options dialog (Extensions|User Files) to have them being found during execution in standalone mode. Thereby these files are also included in the configuration of the standalone mode. No further modifications are necessary; e.g. it is not necessary to adjust the path in CAPL test programs when loading arbitrary curves.

Note on Using Systems with Lower Performance

If you use a VT System in standalone mode the system requirements are higher than in normal mode. On systems with lower performance problems may occur that normally not occur in normal mode.

In contrast the performance of the realtime modules VT6051/VT6051A/VT6060 are sufficient. Generally for the use of a VT System in standalone mode a system with at least two processors is recommended.
