Generation of VT System Documentation

 

In addition to manual manipulation of the VT System, this dialog also provides the option for generating various types of documentation. This includes:

  • Text lists of all modules and channels of the VT System including their connections.
  • Tabular overviews in CSV format.
  • Graphic representations of the configured VT System in various views (front view, rear view, and a schematic representation).
  • Graphic representations as PDF vector graphics, which are particularly well-suited for printouts.

The dialog for creating this documentation can be opened via (Create documentation) in the toolbar.

The various documentation types can be selected, configured, and then created within the dialog. To obtain useful results, it is important here to assign labels and specify the layout of the modules beforehand.

Labels

All connections of the VT System modules have labels both in the interactive circuit diagram representation and in the connection representation.

Each of these labels represents the connection of an external hardware that is connected to the VT System at this point. These labels make the connections easier to view, because they are kept consistent in all views, thereby facilitating the recognition of important connections. In addition, they are incorporated into the generated documentation.

To edit the text of a label, you simply click it. An editor then opens where you can enter the new text for the label. You confirm this input by pressing the <Enter> key or by clicking outside the editor.
