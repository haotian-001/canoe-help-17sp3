Configuration

 

You can open the settings dialog with at the top right of the toolbar of the VT System control. This dialog is divided into the following categories:

General

On this page you configure general settings of the VT System control.

Schematics

  • Update cycle time
    This value specifies the time interval at which the states of the relays in the circuit diagrams are updated.

Sidebar

  • Update cycle time
    This value specifies the time interval at which the measurement values in the sidebars are updated.
  • Autoexpand groups in the sidebar
    If this option is selected, all groups in the sidebars (e.g., measurement value arrays and wave form parameters) are expanded automatically.

Labels

  • Clear all labels
    This can be used to erase all labels entered in the circuit diagrams and connection views.

Code Generator

On this page you configure the code generator of the VT System control.

General

  • Add comments to generated code
    A short comment is generated for each line of code.
  • Insert blank line after each code block
    A blank line is inserted after each generated block of code to make the code easier to read.
  • Capture while the code window is closed
    If this option is selected, operations are also recorded when the code generator window is not open.

CAPL Settings

  • Use function/method syntax
    Depending on which option is selected, CAPL commands are generated in the function syntax (e.g., vtsSetLoadMode(…)) or method syntax (e.g., channel.SetLoadMode(…)).

.NET Settings

  • Generate initialization code
    If this option is active, initialization code is also generated in addition to the actual commands for the operations. This includes the initialization of the required VT System variables.

Documentation (PDF)

You configure the PDF output of the documentation generator on this page.

Page size

In this area you configure the size of the output PDF file. You can specify the paper format, the orientation, and the margins here.

Content

  • Add page captions
    If this option is selected, a caption is added to every page of the PDF document. This contains the name of the represented module, among other things.
  • Detailed documentation
    This option can be used to add a title page containing information about the CANoe configuration to the PDF document.

Documentation (Listing)

You configure the list output of the documentation generator on this page.

Format

In this area you specify the encoding as well as the characters to be used for indention and for line breaks.

Content

  • Include connectors and labels
    If this option is cleared, only the modules are specified but not their individual connectors.

Documentation (CSV)

You configure the CSV output of the documentation generator on this page.

Format

In this area you specify the encoding as well as the characters to be used for separation and for line breaks.

Documentation (Image)

You configure the graphic output of the documentation generator on this page.

Pixel Size

In this area, the length of the long edge can be specified in pixels (the graphic is then scaled so that its longer side corresponds to the specified length).

Printing Size

The resolution (DPI) of the graphic and the resulting image size in cm (or mm or inches) can be specified here.

Color Settings

  • Grayscale
    If this option is selected, the graphic is created in grayscale, i.e., without color information.
