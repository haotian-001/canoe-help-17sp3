VT System Control

Ribbon: Hardware Tab|VT System Control

In this dialog you can manually manipulate a previously configured VT System during an active measurement. Outside of the measurement, the channels of the VT System can also be configured and their states stored as start states. In addition, information about the hardware connected to the VT System can be entered in this dialog. This can be used to generate various types of documentation (such as a system overview).

The navigation area

The currently configured VT System is shown in a tree view on the left side. Here, it is possible to select the object to be displayed in the display area on the right side. You can also right-click to open a context menu that provides access to the various functions for managing the start states.

The display area

The display area on the right side graphically displays the currently selected object. The tabs at the top edge allow you to switch between an interactive circuit diagram representation and a representation of the connections. If the interactive circuit diagram is selected, a sidebar will also be displayed on the right edge of the display area, if applicable. This makes various functions of the selected channel available during the measurement.

Navigation in the display area

It is possible to zoom out of or zoom in on all graphic representations within the dialog. Likewise, the displayed screen segment can be shifted. Thus, you have the choice at any time to obtain an overview of the displayed object or to concentrate on certain subareas.

In order to zoom in on the view, move the cursor to any point within the graphic. If you now rotate the mouse wheel in the upward direction, you will zoom in on the graphic with the focus at this point. Similarly, you can zoom out of the graphic by rotating the mouse wheel in the downward direction.

As soon as the zoom level is such that the entire view can no longer be seen in the dialog, you can shift the view. This allows you to select the segment that is currently visible. To do so, click in the graphic and hold down the mouse button. If you now move the mouse, the visible screen segment will shift accordingly.

Manual manipulation of the VT System

In order to manually manipulate the VT System, interactive circuit diagram must be selected as the form of representation in the display area. In addition, the module to be controlled must be selected in the navigation area.

If the measurement is now started, all interactive elements of the circuit diagram will be highlighted in color. The correspond mainly to the relays available on the channel. To change the state of this element, it can be clicked (left-click). Simple relays with only two states are switched over immediately. For relays with several possible states, the desired state can be selected from a context menu.

If the active channel provides additional functions that are not displayed in the form of relays in the circuit diagram, they can be accessed via the sidebar in the display area. A description of the values and settings displayed there can be found in the help for the particular module. This help can be quickly accessed via the link to the corresponding button in the toolbar.

Specifying start states

The current state of a VT channel can be stored as a start state at any point in time. This state is then produced automatically in CANoe at the start of measurement.

Note

Start values for system variables configured in CANoe take precedence over the VT System start states. That is, if a system variable has been provided with a start value both in the VT System control as well as in the Start Values dialog, the value from the start values dialog is assigned at the start of measurement.

You can access the various options for managing the start states via a context menu in the navigation area. This menu offers the following settings:

  • Save current state of channel as start state
  • Delete start state (i.e., reset to default values)
  • Deactivate start state (start state is not produced at the start of measurement) or activate start state (start state is produced at the start of measurement)
  • Copy and paste start state (this is only possible on compatible channels)
  • Produce start state

Two or more channels can also be selected for all options. Thus, for example, a start state can be quickly copied and pasted on multiple channels.
