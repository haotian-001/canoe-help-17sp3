Layout of the modules

 

By default, the modules of the VT System appear in all graphic representations in the order in which they were created in the configuration dialog. However, free slots in the racks are not taken into consideration in this process. In order to adapt the virtual representation to a connected real VT System, it is therefore possible to specify the positions of the individual modules in more detail.

For this purpose, you must first change to the connection view. Now you can select the VT System or an individual rack in the navigation area. Then you can use the appropriate button in the toolbar to open the dialog for the layout of the modules.

In this dialog, you can use a drag-and-drop operation to move the modules to the correct positions. In doing so, you cannot change the order of the modules because this is specified by the configuration of the VT System. You can also add and remove racks by right-clicking in the graphic.

If modules that are not automatically displayed are used, you can add these manually. This pertains to VT6000 realtime modules and VTC8920 voltage supplies, for example. To add these modules to the configuration, simply right-click in the graphic. In the context menu, you can then open a dialog for creating user-defined modules. In contrast to the other modules, you can move these modules to any position in the racks.
