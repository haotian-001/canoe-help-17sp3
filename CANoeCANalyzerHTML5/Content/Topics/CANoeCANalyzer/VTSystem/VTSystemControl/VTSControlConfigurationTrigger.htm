Trigger Configuration

 

You can open the Trigger Configuration dialog via the channel functions (splitter on the right) of a selected VT System module:

This dialog allows configuring the triggers of the currently selected VT System module.

Trigger Overview

The list on the left hand side shows all available triggers. The check box next to the trigger indicates whether the trigger is currently active. To edit a trigger's configuration simply select it in the list box. The panel on the right hand side then provides several controls to alter the trigger's configuration.

Basic Trigger Settings

Here you can specify which channel of the current module this trigger is attached to. Also the kind of edge the trigger reacts to can be set here. Via Start Trigger you can start the trigger. If the trigger is already running it gets restarted.

Advanced Trigger Settings

The advanced trigger settings allow configuring the trigger for special purposes. Please see the trigger documentation for details on these advanced settings.

Linked Triggers

The check boxes in this section allow you to specify what triggers are used as a precondition for the current trigger to be startable. You can also define which triggers actually start the current trigger. The indices below the check boxes directly correspond to the indices of the triggers in the selection list.
