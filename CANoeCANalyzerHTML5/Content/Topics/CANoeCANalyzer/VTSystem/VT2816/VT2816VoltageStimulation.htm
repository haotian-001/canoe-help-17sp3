Voltage Stimulation

 

You can switch the voltage output range between ±10 V and 0…28 V using the corresponding function. Each channel can carry up to 200 mA. You use a system variable to specify the voltage that is output, which means you can also vary it during the test. However, the next voltage value cannot be set until at least 1 ms has passed.

You can also output the specified voltage from line a to line b. Line b doesn't necessarily need to be connected to ground in this case. You can also apply the specified voltage to a ground potential, e.g. the ECU ground. In addition, you can use relays to connect line b either to DGND or to another suitable external potential attached to the VT2816.

Note

When voltage is output to line b, it should always be connected to a fixed potential, e.g. the ECU ground or DGND.

Furthermore, you can only test the voltage output when line b is connected to a fixed potential.

You can stimulate with fixed values directly from CANoe using the corresponding system variables. The module can also output a saved curve independently; see stimulating with arbitrary curves.

Example: Generating a fixed value voltage signal

The example below generates a voltage signal fixed at 12 V.

CAPL

// Set output range to 0-28V
sysvar::VTS::TempSensor.SetOutputRange(eVTSOutputRangeZeroTo28V);

// Switch line b to Ground
@sysvar::VTS::TempSensor::RelayGnd = 1;

// Activate voltage stimulation
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeVoltage);

// Set curve type constant value
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypeConstant);

// Set voltage to 12V
@sysvar::VTS::TempSensor::Voltage = 12;

.NET (#)

IVTSystem vts = VTSystem.Instance;

// Get the VT2848channel
IVT2816StimChannel tempSensor = vts.GetChannel<IVT2816StimChannel>("TempSensor");

// Set output range to 0-28V
tempSensor.OutputRange.Value = OutputRange.ZeroTo28Volts;

// Switch line b to Ground
tempSensor.RelayGnd.Value = true;

// Set curve type constant value
tempSensor.CurveType.Value = CurveType.Constant;

// Set voltage to 12V
tempSensor.Voltage.Value = 12.0;

// Start stimulation
tempSensor.StartStimulation();

VT System Control

Select TempSensor

Output a constant voltage of 12V
Set Sidebar | General | Output Range to 0..28V
Set Schematic | RelayGnd to closed
Set Sidebar | General | Stim Mode to Voltage
Set Sidebar | General | Curve Type to Constant
Set Sidebar | Constant Output | Voltage to 12.0
Click Sidebar | General | Start Stimulation
