Arbitrary Wave Forms

 

Stimulation of Wave Forms

With the VT2816, you can output constant values on channels 13 and 14, and you can also stimulate with arbitrary wave forms. With the VT2816A this functionality is also available on channels 15 and 16. To do this, you define a maximum of 512 interpolation points in a file. For each interpolation point, the file defines the value of the stimulated voltage and the number of intermediate points that follow the interpolation point.

You use the LoadWFVoltage CAPL function or the vtsystem_configure command to load a wave form you have defined in a file to a channel on the VT2816 module. In addition, you can use the SetWFParams command or the primitive to set the time increment, pause, number of repetitions, start point and start delay parameters. Time increment and pause can also be modified via system variables; changes immediately influence the output of this module.

  • The time increment parameter specifies the interval between the interpolation points and intermediate points. You can use a system variable to change this parameter during wave form output. In addition, you can make the value negative in order to reverse the direction of the output. The output can be frozen at a certain point by setting the time increment to 0.0. If a different value is set afterwards, the output is continued from the point where it has been frozen.
  • The number of repetitions parameter specifies how many times the wave form stimulation is repeated.
  • If the wave form is stimulated repeatedly, the pause parameter specifies the length of the pause before each repetition. You can use a system variable to change this parameter during wave form output.
  • The start point parameter specifies the interpolation point or intermediate point at which the stimulation starts.
  • The start delay parameter specifies a time delay before the start of the wave form stimulation.

File Format

Example 1: Structuring a Wave Form File:

1. Interpolation point value [; number of intermediate points]

2. Interpolation point value [; number of intermediate points]

...

nth interpolation point value [; number of intermediate points]

Each line defines an interpolation point with a value that is interpreted as voltage in volts. Note that you need to use a period (full stop) "." as the decimal separator. Three decimal places are supported. You specify the number of intermediate points after the semicolon. Each additional intermediate point prolongs stimulation of the current value by one time increment. You can have 0…65534 intermediate points between two interpolation points. If you do not specify any intermediate points, this value is set to 0. In this case the next interpolation point follows as soon as the specified time increment has elapsed. You can insert comments in the file starting with "//". Lines that are empty or that only contain comments are ignored.

Example 2: A File that Stimulates the Wave Form Shown in the Figure Above:

// Example of an arbitrary wave form
0     // Next value is an interpolation point
2
3; 2  // Two output points lie between this and the next interpolation point
5; 1
4; 1
2.8;  // Also possible

Note

Because the module needs some time until the value specified in the stimulation is reached, a time resolution of 1 µs is not always practicable.

For this reason there should be at least 10 µs between interpolation point.

Example 3: Generating a Voltage Wave Form

The following example shows the stimulation of a voltage wave form with a time increment of 100 µs. The wave form is stimulated three times with a pause of 2 seconds between the wave form.

CAPL

// Load curve from file
sysvar::VTS::TempSensor.LoadWFVoltage("C:\\ECU_Test\\WaveForms\\TempCurve.txt");

// Set output range to 0-28V
sysvar::VTS::TempSensor.SetOutputRange(eVTSOutputRangeZeroTo28V);

// Switch line b to Ground
@sysvar::VTS::TempSensor::RelayGnd = 1;

// Activate voltage stimulation
sysvar::VTS::TempSensor.SetStimulationMode(eVTSStimulationModeVoltage);

// Set curve type Wave Form
sysvar::VTS::TempSensor.SetCurveType(eVTSCurveTypeAnalogWaveform);

// set parameters: time increment 100µs, pause 2s, 3 repeats
sysvar::VTS::TempSensor.SetWFParams( 0.0001, 2.0, 3);

// Start stimulation
sysvar::VTS::TempSensor.StartStimulation();
