Step by Step Procedure: Setup VT System

To use VT System with CANoe, please proceed as follows:

  1. Connect VT System to the computer running CANoe via an Ethernet interface
    (see Notes for use with distributed mode).
  2. Now activate VT System in the VT System configuration dialog (Home ribbon tab|Hardware|VVT System|Configuration).
    Choose the network interface (IntelÂ® 82574L Gigabit Network Connection) on the VT6000 that is connected to the VT System backplane.
    Configure the VT System modules and channels
    (Hardware ribbon tab|VT System Configuration).
    When you add a new VT System configuration, it is created automatically based on the VT System that is currently connected.
    You can work immediately with this configuration, though you may need to activate the measurement values to be used.

The VT System configuration is saved to the same folder as the CANoe configuration with the same name but using VTCFG as the filename extension. If this file exists, it is automatically used for the CANoe configuration. However, you must make sure that if you move or copy the CANoe configuration, you move or copy this file along with it, otherwise CANoe will not be able to find the file. You can also copy this file to other CANoe configurations or import it via the VT System configuration dialog.

Activation of VT System and setting of the network adapter are settings that are specific to a particular CANoe installation, because they are dependent on the circumstance on the relevant computer and the connected hardware. You must therefore execute steps 1 to 3 on every computer on which you want to use the VT System. This is also necessary if you only want to load a CANoe configuration in which the VT System is used.
