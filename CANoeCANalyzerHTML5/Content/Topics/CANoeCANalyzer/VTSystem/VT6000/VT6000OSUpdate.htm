Updating the Operating System

 

In certain rare cases, you may need to update the Windows operating system running on the VT6000. This is done by loading a memory image (provided by Vector) to the internal flash memory of the device.

Notice!

To avoid damage to the module, different preparation steps for the operating system update must be performed depending on the module type.

Therefore, perform the correct preparation for the respective module type.

Preparation for VT6020/VT6060

  1. Disconnect the VT System from the power supply.

  2. Connect the USB Type B port on the VT6000 front panel with the USB port on the user computer using an appropriate USB cable.

  3. Reconnect the VT System to the mains to supply power to the VT6000.

Preparation for VT6011/VT6051A

  1. To prepare for this operation, unplug the VT System from the power supply.
  2. Remove the VT6000 from the rack.
  3. Connect the USB Type B port on the VT6000 front panel with the USB port on the user computer using an appropriate USB cable.

Updating the Operating System

Start by locating the executable VN89Flash.EXE in the Vector CANoe installation directory.

  1. Double click the executable.

  2. In the upper part of the window, enter the path to the image file provided by Vector.

  3. Under Device storage medium, select the system drive of the connected VT6000; it is labeled VT6000Sys.

  4. Now start the update by clicking the [Update] button.
    You can observe the update progress in the lower part of the dialog.
    Please note that the update will take a few minutes.

    Notice!

    • If you select another USB drive connected to the computer, instead of selecting the VT6000 drive as the storage medium, all files on the selected drive will be overwritten.

    • Do not disconnect the device during the update.

    • When the device is connected to the computer via its internal USB port, the VT6000's internal flash memory drives are also visible in Windows Explorer.

      If you delete or modify files via this connection, the device may no longer operate correctly.

  5. After the operating system update has been successfully completed, disconnect the VT6000 from the power supply again.

  6. Remove the USB cable.

  7. If necessary, put the VT6000 back into the rack.

  8. Reconnect the VT System to the power supply so that the VT6000 is supplied with power.

  9. Wait until the VT6000 has restarted properly.

    You can now use the VT6000 in the normal way.

Note

First start-up may take several minutes.

This is because some final installation steps are performed when the system is booted the first time.
