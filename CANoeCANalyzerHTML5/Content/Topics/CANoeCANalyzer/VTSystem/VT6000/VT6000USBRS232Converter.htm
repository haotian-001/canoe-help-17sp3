USB RS232 Converter

 

General

You can use serial interfaces with the VT6000 by means of a RS232 USB converter (serial port).

Built-in drivers are utilized to support chipsets from the following manufacturers:

  • Prolific (PL2303)

  • FTDI (FT232)

  • Silicon Labs (CP210x)

Note

Vector has successfully tested a range of RS232 USB converters that use the abovementioned chipsets.

However, there may be isolated instances when faults occur with the serial interface, caused e.g. by the specific hardware features of a given USB RS232 converter.

  • For example, there may be problems with certain Prolific PL2303 adapters.

    In such instances, the Device Manager correctly recognizes the adapter, but there are problems sending and receiving data.

    With some adapters only the send functionality, with others only the receive functionality works.

  • No problems have been identified with FTDI FT232-based RS232 USB adapters to date.

    However there is still a slight possibility that some of these adapters may also cause problems.

Setup

To set up the adapter, create a remote desktop connection to the VT6000.

Once you have connected the RS232 USB adapter to the VT6000, it should be displayed in the Device Manager of the VT6000.

Open the Device Manager on the VT6000

  1. Open the System Control Panel in the Control Service Console.

  2. Select the Device Manager in the Control Panel.

    If the adapter is recognized correctly, it will appear in the Ports (COM & LPT) node of the Device Manager.

    This will also show you the corresponding COM port with which you can address the adapter.

    You can change the COM port via the settings; however this setting is lost when the module is rebooted.

    To persist adapter settings, please follow the step-by-step instructions Changing advanced system settings.

Note

  • Due to problems with certain adapters, you should test whether the RS232 interface is working correctly before using it.

    This can be done using the CANoe RS232.CFG sample configuration: Load Configuration

  • The RS232.API is used to control the serial interface in CANoe.
