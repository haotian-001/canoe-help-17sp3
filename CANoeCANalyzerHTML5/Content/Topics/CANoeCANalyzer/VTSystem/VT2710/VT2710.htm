VT2710 – Serial Interface Module

 

General

The Serial Interface Module VT2710 enables the test of ECUs and sensors who communicate via serial protocol interfaces. The VT2710 can be operated as sender or receiver which allows for sensor or ECU simulation. Furthermore, the communication between real sensor and real ECU can be analyzed and logged. The configuration and analysis of the buses and bus components is established with the CANoe Option Sensor. The VT2710 can provide up to four PSI5- or SENT channels with one PSI5/SENT piggy per channel.

The PSI5/SENT piggy extends the basis module VT2710 to respectively one PSI5 or SENT channel. Sensor frames and protocol relevant electrical parameters like synchronization pulse forms, bitrates, current levels as well as the sensor supply can be configured and analyzed with the CANoe Option Sensor. Dedicated relays allow for shortcuts between the bus lines. The PSI5 or SENT channels can be interconnected or provided externally with two independent bus bars.

The module VT2710 also supports further serial protocols like SPI, UART (incl. RS232, RS422, RS485) and I2C.

Switching Different Signal Paths

Usage, Functions, Technical Data

The connecting cables of the serial interfaces on the bus bars and the short-circuits between cables can be switched via relay.
