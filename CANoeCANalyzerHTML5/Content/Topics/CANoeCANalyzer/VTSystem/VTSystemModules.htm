Modules and Extensions

 

VT System – Modules

Photo

Short Description

VT1004/VT1104 – Load and Measurement Module

The VT1004/VT1104 Load and Measurement Module has four identical channels, each of which can be connected to an ECU output. The module has an internal electronic load for each channel. The VT1004/VT1104 measures and pre-processes the ECU's output signal.

VT2004 – Stimulation Module

The VT2004 Stimulation Module has four identical channels, each of which can be connected to an ECU input. To simulate a sensor, the module can apply voltage at the ECU input and generate a resistance via an internal decade resistor. For potentiometer stimulation, a voltage is generated that is proportional to a specified factor and a set reference voltage.

VT2516 – Digital Module

The VT2516 Digital Module has 16 identical channels, each of which can be connected to an ECU input or output. The VT2516 measures and preprocesses the voltage level of each channel's signal line. Also the digital data stream of each channel's signal line is captured. The VT2516 can output a digital voltage signal on each channel. The ECU interprets this as a sensor signal. High and Low levels can be set separately for each channel.

VT2710 – Serial Interface Module (VT System)

The VT2710 serial interface module contains four slots. Each one can be equipped with a piggyback (see below) for the serial automotive interfaces PSI5 and SENT. The channels offer many possibilities for error simulation (e.g. short circuits) and for the connection to bus bars.

The module is prepared for further serial protocols like SPI, UART (incl. RS232, RS422, RS485) and I2C which will be activated with future releases of CANoe.

VT2808 – Current Measurement Module

The VT2808 Current Measurement Module features eight identical channels for current and voltage measurement. It provides the momentary current/voltage as well as average, minimum and maximum values.

VT2816 – General-Purpose Analog I/O Module

The VT2816 General-Purpose Analog I/O Module has 16 channels of which 12 channels can be used for signal measurement and 4 channels can be used for signal output. It is possible to output static voltage as well as arbitrary wave forms.

VT2820 – General-Purpose Relay Module

The VT2820 General-Purpose Relay Module has 20 channels, of which 12 are normally open contacts and eight are changeover contacts. The module is laid out to carry and switch high currents as well as transmit higher-frequency signals.

VT2832 – Switch Matrix Module

The VT2832 Switch Matrix Module allows arbitrary switching of 8 columns (channels) with 4 rows (8×4 switch matrix). Additional 4 paths can be switched directly. By using Solid State Relays it is possible to switch currents up to 16 A under full load at voltages up to 60 V on each channel.

VT2848 – General-Purpose Digital I/O Module

The VT2848 General-Purpose Digital I/O Module has 48 channels, each of which can be used as input or output. The module can generate and measure PWM signals as well as output bitstreams.

VT6000/VT6011/VT6020/VT6051A/VT6060 – Realtime Module

The realtime module handles the execution of the realtime test and simulation part of CANoe and drive the VT System hardware. The user computer is connected to the realtime module by Ethernet and does not affect the realtime behavior of the system.

VT6104/VT6204 – Network Interface Module

The VT6104 four-channel CAN/LIN network module is a high-performance interface module for the VT System. As on other VT System modules, faults can be fed onto the signal lines. A PCI-Express cable is used to connect the network interface to the VT6000 realtime module or a computer with CANoe.

The VT6204 has the same characteristics. This module type can additionally operate a FlexRay piggyback on channel 1.

VT6306 – Ethernet Network Interface Module

The VT6306 Ethernet Network Module supports six automotive Ethernet channels on a specific plugin board (separate article). Various signal paths and faults such as short circuits and line interruptions can be connected via relays.

Furthermore, the module has two additional 100/1000 Mbit/s Ethernet channels, which do not permit error connection. These channels can be used for media conversion, for example.

VT7001/VT7101 – Power Module

The VT7001/VT7101 power module is used to control the power supply lines of the ECU being tested.
The actual power for the ECU is delivered by an internal or external power supply. The VT7001/VT7101 controls this power supply, measures voltage and current, and allows for a variety of connections between the power supply unit and the ECU. The module can also be used to generate line breaks and short circuits on the lines connected to the ECU.

VT7900/7900A – Extension Module

The VT7900 Extension Module is used to extend the VT System by adding modules with task-specific circuits. The VT7900 serves as the main board, on which an application board is inserted. The input and output signals of the application board can be set and read by CANoe.

VT8006 – Backplane (short)

The VT8006 backplane has 6 insertion slots for VT modules. It connects the modules in the rack via EtherCAT® with the computer running CANoe. Additionally is contains the power supply lines for the modules.

VT8012 – Backplane (long)

The VT8012 backplane has 12 insertion slots for VT modules. It connects the modules in the rack via EtherCAT® with the computer running CANoe. Additionally is contains the power supply lines for the modules.

PSI5/SENTpiggy

The PSI5/SENTpiggy extends the basis module VT2710 (see above) to respectively one PSI5 or SENT channel. Sensor frames and protocol relevant electrical parameters like synchronization pulse forms, bitrates, current levels as well as the sensor supply can be configured and analyzed with the CANoe Option Sensor. Dedicated relays allow for shortcuts between the bus lines. The PSI5 or SENT channels can be interconnected or provided externally with two independent bus bars.
