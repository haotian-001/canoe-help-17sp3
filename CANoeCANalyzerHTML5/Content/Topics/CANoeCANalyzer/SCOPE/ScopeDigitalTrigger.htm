Digital Trigger

Devices and Triggers Page|Context Menu|Add Trigger Condition|Digital Trigger…

Requirements

  • The MSO-Trigger can be used with all scopes of PicoScope 6000 series. For example the device PS6403E-034 provided by Vector.
  • A special trigger cable (Y-cable, same for I/O Trigger) is required. For details contact Vector Support.
    The cable connections are presented in the Scope manual.

Adding Digital Trigger

If a corresponding scope device is integrated and configured on the Devices and Triggers (Scope) page, a digital trigger condition can be added using the context menu of the scope device.

Configuring a Digital Trigger

In the Digital Channel and Trigger Condition Configuration dialog you can configure digital ports and setup trigger conditions on each digital channel contributed to a port.

Ports Configuration

The digital port configuration is equal to Hardware Configuration for scopes.

Digital Trigger Configuration

Here you configure all digital channel capabilities for a selected port. (Port 0 or Port1). The port settings Threshold and Hysteresis are common for each digital channel. The entire configuration of a digital port is a complex trigger condition where channels D0 to D7 are logically ORed to each other.

Channel

Fix logical channel names (D0…D7).

Label

User defined description.

Enable Trigger

Activates or deactivate trigger

Trigger Condition

Trigger on levels (High (1) or Low (0)) or transitions (Rising or Falling). The selected condition refers to configured port threshold. For level triggers values above threshold are High otherwise Low. For transitions triggers the direction from below to above threshold value represents a Rising edge and the inverse direction a Falling edge.
