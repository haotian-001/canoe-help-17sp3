I/O Trigger

Devices and Triggers Page|Context Menu|Add Trigger Condition|I/O Trigger…

Requirements

  • The I/O-Trigger can be used with PicoScope 6403E and 5444D.
  • A special trigger cable (Y-cable) is required. For details contact Vector Support.
    The cable connections are presented in the Scope manual.

Adding I/O Trigger

If a corresponding scope device is integrated and configured on the Devices and Triggers (Scope) page, an I/O-Trigger condition can be added using the context menu of the scope device.

Configuring an I/O Trigger

You can make the following settings in the Trigger Condition Configuration dialog:

Select Signal Source

Here you must select a logical signal source that has been assigned to a physical oscilloscope channel beforehand in the device configuration.

Logical signal sources are either assigned bus signals (e.g., CAN_H, CAN_L, LIN, FlexRay_BP, Flexray_BM) or originate from any external signal source.

External signals are assigned as logical I/O channels initially to an oscilloscope channel.
Thus, trigger conditions can also be defined for I/O signals.

You can assign a maximum of 4 logical I/O channels to an oscilloscope with 4 physical channels.
You can define only one trigger condition for each signal source. If no signal source is selected, the trigger condition is invalid.

Oscilloscope Setting for Signal Source

The device settings are displayed here.
The trigger conditions have a direct relationship to the settings for each oscilloscope channel.

  • Channel Here you see to which physical channel the current signal source is assigned.
  • Voltage range: Display of the voltage measurement ranges of the signal and the oscilloscope
  • Probe multiplier: Multiplication factor of the probe.
  • Label: The label indicates whether you have added a description for the measurement channel.

Edge Trigger Condition

Note

The edge trigger is a component of the pulse trigger. It is used for falling or rising pulse edges at the end of a pulse. For example, a positive pulse has a falling edge at the end and a negative pulse a rising edge.
  • Edge direction: You can also trigger on rising and falling edges. With setting Rising/Falling the first condition to be met is the triggering condition.
  • Threshold value: Threshold value of the voltage at which the oscilloscope triggers. The default setting is 50% of the voltage measurement range.

Note CANstress/FRstress

If you want to trigger on the rising or falling edge at the trigger output of the stress hardware, you must select 3V as the threshold value and the setting Rising/Falling as the edge direction.

Pulse Trigger Condition

  • Condition (time-related): Here you set the trigger condition for the selected pulse width. With the setting Time greater than or Time less than, triggering occurs on the first pulse that is greater than or less than the pulse width, respectively. The condition for the edge detection value must be met.
  • Pulse width: Duration of a pulse. The trigger time always lies at the end of the pulse.
  • Pulse direction: Criterion whether triggering is to occur on a positive pulse (low->high->low) or negative pulse (high->low->high).
