Diagram

 

General

The diagram shows the following information:

  • Signal curves
  • The chart control has a multi pane concept. Analog and digital signals are shown in separate panes (see Scope Configuration dialog).
  • Highlight information based on chronological values as displayed in the Trace View.
  • The ground bar with corresponding ground levels.
  • The time axis
  • You can set the time unit and the display of time values in the Scope Configuration dialog.
  • The grid
  • Y grid (in mV) is configurable,
  • X grid is implicitly changed depending on the X zoom factor.
  • Trigger time-stamp information regarding
  • trigger request (trigger condition has been recognized)
  • trigger acknowledge (trigger pulse has been created)

Note

The difference between these two time stamps depends on the trigger settings.

  • Choose Pre-Trigger, if you don't want to trigger during the pre-trigger stage. With this setting no trigger condition will be evaluated during pre-trigger time. The trigger acknowledge follows immediately after the trigger request.
  • The setting of Min. Post-Trigger enables evaluating the trigger condition within the pre-trigger stage. Here the time difference between the two trigger time stamp values may add up to the pre-trigger stage length.
    Triggering within the pre-trigger stage is currently supported by the Scope hardware type 4227.

Analysis the Diagram

Measurement Cursor/Difference Cursors

First you have to start a measurement. Then you can select a cursor via the toolbar menu of the Scope measurement chart. You can select Measurement Cursor (one) or Difference Cursors (two). All cursor data is shown in a special cursor legend which is docked on the bottom of the Scope Chart view:

You can set a cursor via the following context menu commands:

  • Position cursor at current mouse position
  • Adjust cursor to frame field highlighting
    (also available in context menu of the Scope Trace view)

Logical Interpretation

Logical interpretation is shown via the background color.

To make the logical interpretation visible, you need to zoom in until you can see individual bus events.

  • A time segment that is marked corresponds to the bus event (or bus event field) currently selected in the Trace View
  • Depending on the context, bus events or even/odd bits are highlighted differently.
  • You can navigate from one logical unit to another by:
  • Moving the measurement points with in the toolbar
  • Navigating in the Trace View

Working in Diagram

  • Dragging, Scrolling and Selecting
  • To drag the entire image, click any point in the diagram outside of the signal curve, hold down the left mouse key and move your mouse in the desired direction.
  • To drag a signal curve, click any point on the signal curve, hold down the left mouse key and move your mouse in the desired direction.
  • You can customize the diagram colors.
  • You can copy the contents of the diagram to the clipboard (context menu command: Copy bitmap to clipboard)
  • You can export measurement values to a CSV file (context menu command: Export Signals(s) to CSV...).
    The CSV export contains the signal data of the currently shown time range. This is useful when analyzing the edge steepness.
  • You can compare time segments from one or more signal curves.

Y-Axis Display Mode

All signals are scaled by the same Y-Axis.

The signals have a common X-axis and their own Y-axis. All Y-axes are displayed to the left of each other.
