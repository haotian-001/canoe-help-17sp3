Window Layout

 

Toolbar

The Devices and Triggers page is used to:

  • Manage and configure Scope devices
  • Map Scope device channels to application channels
  • Manage trigger conditions

The Eye Diagram page is used to configure the Eye Diagram.

The Measurements page is used to view and navigate the captured data.

The Diagram is used to analyze data visually.

 

Using the marker bar, you can set markers for marking significant points of a measurement.

Eye Diagram

Legend of Eye Diagram

You can open the legend via the toolbar of the Eye Diagram.

The Trace view is used to:

  • Display bus events that occur during data acquisition
  • Navigate to individual components of a bus event
  • The diagrams and the Trace view are synchronized with each other.
