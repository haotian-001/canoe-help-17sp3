Transport Protocol

 

The transport layer provides for the dividing of information into multiple messages when it does not fit into a single message (segmenting and assembling of data blocks).

A DLL is integrated in CANoe for this purpose and can be replaced by OEM-specific modeling libraries (DLLS).

In the help you can fin further information here:

  • LIN transport layer
  • J1587 transport protocols
  • Basics of the ISO TP Observer
  • OSEK TP sample configuration OSEK TP Multi Channel

You can also find further information about transport protocols in the document library (DOC installation folder).
