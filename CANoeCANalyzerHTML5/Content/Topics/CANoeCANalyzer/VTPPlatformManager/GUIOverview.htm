Overview

 

Symbol

Description

Connection

On the page Connection you can select, among other things, the device you want to access in order to make settings on the other pages.

You can find information about supported devices here.

CANoe/CANalyzer

The page CANoe/CANalyzer is only available if you have connected a device on the Connection page that installs the CANoe plugin. The CANoe plugin is available for all devices.

The setting options vary slightly depending on whether you call the plugin via CANoe in the Vector Tool Platform dialog or via the Vector Platform Manager.

Tool Platform

The page Tool Platform is only available if you have connected a device on the Connection page.

The setting options depend on the connected device.

Status Bar
  • Connected: Display of the currently connected device and its serial number.
  • Disconnected: No device was connected.
  • Status indicator of the application execution on the computer and the connected devices.
    With a click on the symbol you open a window with the corresponding status message.
  • : Application execution OK.
  • : Error in application execution.

Note

Further information about other plugins can be obtained by opening the separate help by pressing F1 in the respective area.
