The Vector Tool Platform allows you to manage the certificates on an intelligent network interface.

Via the selection list Certificate store, you can switch between the different Windows certificate stores. Only standard certificate stores are supported. It is not possible to add or remove custom certificate stores. All certificates shown are valid system-wide.

Certificates

The table shows all certificates for the selected certificate store:

Issued To

Name of the entity associated with the public key contained in the certificate.

Issued By

Name of the certification authority that created and signed the certificate.

Expiration Date

The date until which the certificate is valid.

With [Add…] you can install certificates to the selected certificate store. If you add a password protected certificate, a password prompt will appear.

With [Remove...] you can remove the selected certificate from the certificate store.

With [View…] you can view the selected certificate on your local machine. Alternatively, you can double click the certificates in the list.
