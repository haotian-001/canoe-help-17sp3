Admin Settings

Note

This section is not available on all devices supported by the Vector Tool Platform. It is only available on devices that allow WLAN access.

Certain configuration settings can only be made after prior legitimation with the help of an administrator password. Via Change password you must first define an administrator password. This requires a device-specific master password, which can be obtained from Vector Support upon request. Later you can change the password with the currently valid password or with the master password.

After entering the administrator password that you have specified, you will be given administrator access via [Legitimize]. This allows you to unlock certain features. Currently, setting up a WLAN Access Point is the only feature that requires activation. Once you have completed this process, click [Lock] to exit the administrator access and thus complete the activation of the feature.

Extended Real Time (ERT)

Note

This section is not available on all devices supported by the Vector Tool Platform.

With the feature Extended Real Time as a part of the Vector Tool Platform the latency and determinism of Vector tools like CANoe and CANape are improved. To achieve this, the device is logically divided into two areas. A new area provides Extended Real Time in which predefined functions can be executed under realtime conditions.

Extended Real Time is supported by the Vector hardware VT6051A, VN6060, VN8911, VN8914 and RT Rack CS246E.

Click [Activate] to configure ERT on the currently selected device.

All ERT-supported network interfaces of the currently selected device are assigned to the ERT. You can change the assignment manually in the Configuration|Network Interfaces menu.

Power Up/Down

Note

This section is not available on all devices supported by the Vector Tool Platform.

Enter a value under Shutdown delay to define the time between the detection of a shutdown request on the Power Up/Down line and the actual initiation of the system shutdown. Click [Apply] to use the value.

Network Share

Network share allows you to remotely access a removable disk connected to the VTP Device.

To set up the access via the network, first select one of the available removable disks and click [Create...]. In the following dialog, enter the network share parameters and confirm them with [OK].

To remove an existing network share click [Delete].

Note

  • If no removable disk is displayed for selection, please make sure that the disk is connected to the VTP Device and formatted with a supported file system (FAT, NTFS).
  • Access from a remote Windows system can be set up by calling net use in a command prompt, specifying the share address and configured user ID. Use net use /help to get detailed information about this command.
  • A maximum of one removable disk can be accessed at a time.
  • The root directory of the selected removable disk is always shared.
  • The network share is always set up with read and write permissions.
