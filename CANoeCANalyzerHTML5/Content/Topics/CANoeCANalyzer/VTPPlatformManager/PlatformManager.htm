Overview

Vector Tool Platform

 

The Vector Tool Platform is an environment installed on different devices (e.g. VN8800, VN8900, VT6000, RT Rack), on which applications with increased realtime requirements run, especially in standalone operation. Configuration and control of devices with VTP from a computer can be done using the Vector Platform Manager, a tool supplied with various Vector products that you can install on any other computer.
