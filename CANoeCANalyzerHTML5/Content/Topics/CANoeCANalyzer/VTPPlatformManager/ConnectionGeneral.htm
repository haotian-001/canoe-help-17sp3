Device Selection

Selected Device

Via this list you can select a device. This device list contains all devices that are found and supported by the Vector Tool Platform.

  • Devices connected to the computer via USB
  • IP network devices found in the local network
  • All IP network devices, added by the user manually ([Edit...] button)

With [Edit...] you open the Edit Connection Parameters dialog. Here you can add and delete IP network devices. These settings are retained even after you exit the program, which means these network devices will always be listed, regardless of whether they are found in the network or not.

With [Advanced Search…] you open the Advanced Search dialog. In this dialog you can find devices that are not in the same address range and configure their IP address.

After selecting a device from the list, the connection is established. The current state of the device is displayed in the status bar. Now you can make other device settings under Tool Platform.

Vector Tool Platform and Drivers

Update Components

With [Update…] you can update the Vector Tool Platform and drivers on the connected device.

Installation Source Path

This is where you enter the current location (on your computer) of the drivers you want to install. You can download an installation package with the current drivers from the Vector Homepage (http://www.vector.com). With [Browse…] you open a search dialog to navigate to the driver location. The selected path is then inserted into the field.

Once you have entered a path to the driver package, the Component selection is updated. Drivers that can be updated with the selected driver package are automatically preselected for installation. This preselection can be adjusted by expanding the component selection area.

Component Selection Area

This list shows the driver versions that are currently installed on the VTP Device and the versions contained in the selected driver package. The components of the Vector Tool Platform group can only be installed together.

The list displays the following information:

  • Component
    This column contains the name of the hardware or component, e.g. VN1640.
  • Installed version
    This column contains the version number of the driver currently installed on the VTP Device.
  • Available version
    This column displays the version number of the driver contained in the driver package selected via Installation source path.

Note

  • You can only install drivers provided by Vector. Third party drivers cannot be installed.
  • Please note that USB devices must be connected to the VTP Device before you switch on the device.
  • If a USB device has its own power supply, you should connect it to an external power source.
  • Driver update may fail in some situations. In this case, switch the VTP Device off and on again and check whether the driver was installed correctly anyway.

Operating System

Preparation

In certain rare cases, when a software or driver update doesn't suffice, it can be necessary to update the operating system (including all pre-installed drivers) running on the device (e.g. VN8900). This is done by loading an image (provided by Vector) to the internal flash memory of the device.

To prepare for this operation, unplug the device (e.g. VN8900) from the power supply. Open the device (for details, see the device manual) and remove the plugin module.

Remove the CFast card from the drive of the device and connect the card to the user computer using the provided card reader.

Updating the Operating System

To install an image, click on the Update button in the Standalone Mode dialog under Vector Platform Manager|Connection|Operating System. This button takes you to the Vector VN8900 Flash Program dialog.

Proceed as follows:

Updating the Operating System

  1. In the upper part of the window, enter the path to the image file provided by Vector support.
    Files with the extensions VTPIMG and BIN are supported. VTPIMG files are compressed files that also allow more detailed information about the image they contain (e.g. version number) to be displayed in the dialog.
  2. Under Storage medium of the device, select the system drive of the connected VN8900; it is labeled VN8900Sys.
  3. Now start the update by clicking the [Update] button.
    You can observe the update progress in the lower part of the dialog. Please note that the update will take a few minutes.

Notice!

  • If you select another USB drive connected to the computer, instead of selecting the VN8900 drive under Storage medium of the device, all files on the selected drive will be overwritten.

  • Do not disconnect the device during the update.

  • When the device is connected to the computer via its internal USB port or CFast-card, the VN8900's internal Flash memory drives are also visible in Windows Explorer.

    If you delete or modify files via this connection, the device may no longer operate correctly.

First Usage After Update

Once the operating system is updated successfully, re-insert the CFast card in the device's drive and re-mount the plugin module (for details, see the device manual). You can now use the device in the usual way with your computer using the external USB port. Depending on the exact contents of the image that you loaded and the installed CANoe version on the user computer, you may first need to execute a software or driver update on the device.

Note

First start-up may take up to two minutes while the D1 (or status) LED keeps blinking orange.
This is because some final installation steps are performed when the system is booted the first time.

Remote Desktop

With [Connect...] you can establish a remote desktop connection to the connected device.

Note

Remote Desktop is not available if the following applies:

  • The connection to the selected device is a USB connection.
  • The image of the VTP device does not support remote desktop.
