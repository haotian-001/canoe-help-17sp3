Adapter Information

General information about the selected adapter is shown here.

Physical Address

Is the hardware address of the adapter and is also referred to as the MAC address.

IP Settings

Here you can change the IP configuration of the selected device. If multiple adapters are available on the device, you can select the adapter from the list. The configuration is made for the selected network adapter. You have the following configuration options now:

  • Obtain IP address automatically 
    The adapter participates in the Dynamic Host Configuration Protocol (DHCP) based IP assignment service configured for the network.
  • Use fixed IP address 
    Enter the IP address to be used (e.g. 192.168.0.1). You will also need to enter the subnet mask and the IP address for the standard gateway in the corresponding fields.

WLAN-Settings

Note

This page is only available for certain devices or adapter types.

Mode

Some VTP devices can be configured and controlled via a wireless connection:

  • On devices of type VN8911 (from image version 3.0) and VN8914 (from image version 3.0) the following WLAN adapters can be used, which are connected to the device via USB:
  • LM007 – LM Technologies
  • LM808 – LM Technologies

In general, there are two different options available to configure a WLAN connection.

Infrastructure

You must configure the device to connect to an external access point.

How to configure WLAN to connect to an external access point

  1. Select Infrastructure in the area Mode.
  2. Possible access points are displayed in the Infrastructure area. If necessary, edit the list of automatically detected access points via [Add] and [Remove].
    Double-click on an access point in the list to open the WLAN dialog.

  3. Make the following settings in the WLAN dialog:

    1. Select a suitable security type.
    2. Select a suitable encryption type.
    3. Enter the password of your wireless access point.

    Note: The security type and encryption type are predefined for automatically detected access points.

  4. Exit the dialog with [OK].
  5. In the IP Settings section, change the IP configuration for this adapter. Use DHCP, if available, or enter the address information manually.

WLAN Access Point

Use this mode when the device itself is to serve as a WLAN Access Point. You can then connect to this access point from an external device (such as your computer).

Note

You can only set up a WLAN Access Point if you have previously activated it on the General page under Admin Settings after entering the administrator password. After the setup has been completed, it can also be used without administrator access. Please note that the Access-Point functionality is currently not available on a VN8911 (from image version 5.0) and VN8914 (from image version 5.0).

How to configure WLAN for internal WLAN Access Point connection

  1. In the Mode area, select the WLAN Access Point as the access mode.
  2. Click [Edit] in the WLAN Access Point area to open the WLAN dialog to edit the WLAN Access Point settings.
  3. Make the following settings:
    1. If necessary, overwrite the default SSID.
    2. Select the encryption type.
    3. If necessary, change the WLAN standard frequency and the channel.
    4. Select a suitable password that must always be entered when connecting to the device.
    5. Select whether one or more users are authorized to connect to the device.
  4. Exit the dialog with [OK].

Advanced Settings

Note

This page is only available for certain devices or adapter types.

Jumbo Fame (MTU)

Allows the selection of non-standardized, extra large Ethernet packets. The PlatformManager currently supports the following frame sizes:

  • Disabled (1500 Bytes): The Jumbo frame support is deactivated for the adapter. The maximum transmission unit is 1500 bytes (standard Ethernet packet).
  • Enabled (4074 Bytes): The Jumbo frame support is activated for the adapter. The maximum transmission unit is 4074 bytes.
  • Enabled (9000 Bytes): The Jumbo frame support is activated for the adapter. The maximum transmission unit is 9000 bytes.

Interrupt Moderation

This enables the interrupt moderation for the adapter to be activated or deactivated.
