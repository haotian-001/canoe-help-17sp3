Panel Configuration: Panels

Ribbon: Home Tab|Panels|Configuration

Definition

  • Panels
    Panels are graphical elements via which values of signals and variables can be changed and displayed using so-called controls.
  • Referenced Panels
    Panels that are not assigned to the CANoe configuration and are configured using the Panel Control button.

Configuration

Programmed Panels

Settings

 

In the Panels list you can select the panels which you'll work with in CANoe. The selected panels will be added to the configuration and can be displayed on different desktops.

The following columns are available:

Panel

Desktops

Referenced Panels

Displays the panel names of the added (configured) panel.

A tooltip indicates where the displayed panel is saved.

Displays the desktop on which the configured panel is displayed.

Panels can be displayed on various desktops. You use the icon to open the Desktop Display dialog where all available desktops are listed. Here, you can select the desired desktops on which a panel is to be displayed.

A tooltip indicates the various desktops on which a panel is displayed.

Display of the referenced panels.

Symbols in the first column:

All referenced panels are displayed one after the other in the appropriate line.

All referenced panels are displayed in alphabetical order in an extra line.

A tooltip indicates all referenced panels.

You can configure the list with the following buttons:

  • [Add…] opens a dialog where you can select an existing panel. If you close this dialog with [OK], the panel name is indicated in the table and the panel is displayed on the currently selected desktop

You can set the window type of the panel (MDI or standard window) on the Settings page of this dialog.

  • [Remove] deletes a panel from the list and, thus, from the configuration. The panel file itself is not deleted by this action.

Referenced panels can neither be added nor removed. The panel configuration file itself is not deleted by removing panels.

  • [Edit...] opens the panels selected in the list (XVP format) in the Panel Designer. Here, you can edit the selected panels.

Create panels

Panels can not only created using the Panel Designer but also with:

  • Visual Basic.NET
  • Visual C#

These panels can also be inserted to the configuration.

Condition

  • The panel programmed in VB .NET or C# must be derived from System.Windows.Forms.UserControl. It is not possible to display a control derived from System.Windows.Forms.Form. 
  • Only one control per assembly (DLL) can be loaded. That is, an assembly should contain only one control. Otherwise, CANoe takes the first control it finds.
