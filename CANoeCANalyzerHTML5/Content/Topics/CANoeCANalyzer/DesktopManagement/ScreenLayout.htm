Screen Layout

 

Main Window

  • Title bar

In the title bar are displayed:

  • The filename of the configuration (*.cfg)
  • The window name of a maximized MDI window
  • The working mode (Simulated/Real)

On the left of the filename you can find the Quick Access Toolbar.

  • Ribbon

The ribbon is located in the main window below the title bar. This is used for fast selection of important functions.

  • Desktop tab

You use the tabs at the bottom of the main window to switch between the desktops. You can create desktops for your choice of topics and display your choice of windows inside them.

  • Status bar

The status bar at the bottom edge of the main window displays information on the current configuration and measurement according to the situation:

  • Load indicator
  • Keyboard inputs during the active measurement
  • Configuration of the key combinations
  • Name of the database
  • Write Window state indicator
  • Animated state indicators
  • Time period of the measurement

Child Window

Below the toolbar is the content area of the main window. The main window can contain several child windows. There are various options for displaying these windows. (See Window Management for more information). Since many windows are often used in parallel, it is useful to distribute them among desktops structured by topic.
