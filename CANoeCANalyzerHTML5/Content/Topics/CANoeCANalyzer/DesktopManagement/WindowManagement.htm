Window Management

 

Context Menu of Windows

The context menu of the child windows provides not only the window types MDI, Standard and Docking but also other window management options:

  • Show in Favorites
    With this command you can add windows and dialogs to the favorites on the Home ribbon tab.
  • Copy Screenshot
    Thereby you can easily create a screenshot of a window (Graphics Window, statistic windows,Data Window, Trace Window, Write Window, Panel or State Tracker) and copy it to the clipboard.
    See also: Window Capture

    		•

    Note

    The Copy Screenshot command reproduces only the content of the window. If you want a screenshot of the entire window including its title bar, select the Standard Window window type and drag the child window out of the main window. Then, use the key combination <Alt> + <Print>.
  • Move to
    You select the name of the desktop to which the window is to be moved in the submenu.
  • Show on all Desktops
    Thereby the current window is made visible on all desktops.
    If the window wasn't so far visible on a certain desktop it will be shown at the same position as on the current desktop. Otherwise it keeps its position on other desktops.

For panels the following menu commands are additionally available:

  • Edit...
    Opens the panel in the Panel Designer.
  • Start
    Starts the measurement.
  • Stop
    Stops the measurement.
  • Signal Generators

Ribbon

Depending on the window or panel some of menu commands listed above are also available via the ribbon.

If you move the mouse over a window or panel name in the ribbon, the [...] button is shown at the end of the line. If you then move the mouse over the button, a submenu is shown immediately.

Working with Multiple Monitors

If you are working with two or more monitors, windows/dialogs may appear outside the visible range.

Note the following:

  • When the primary monitor is set up on the right and the secondary monitor on the left, CANoe configurations are always opened on the primary monitor because the configuration is loaded with its saved position information. 
  • If you want to open CANoe configurations on the secondary monitor, you must maximize the CANoe main window before you open the configuration. The saved position information of the configuration is not taken into consideration in this case.
  • If a configuration is loaded and then saved on the secondary monitor, this configuration will always be opened on the secondary monitor.

Moving Dialogs and Windows to the Visible Area

Key combinations can be used to move dialogs to the visible area. The following steps are necessary:

  1. Open the dialog menu for the active dialog or the window menu for the active window respectively:<Alt>+<Space>+<m>
  2. a) The active dialog/window can now be moved with the arrow keys <Up Arrow>/<Down Arrow>/<Left Arrow>/<Right Arrow> or
    b) move the mouse to move the dialog into the visible area.
