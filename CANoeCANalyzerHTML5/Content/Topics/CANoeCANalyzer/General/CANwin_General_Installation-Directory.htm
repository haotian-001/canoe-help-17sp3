Installation Directories

 

General

With CANoe 7.0, you can install user data and application data separately from program files. This means you can use CANoe as a default user even if you have chosen the Windows program folder as the CANoe installation folder.

You will need administrator rights for administrative tasks and installations (especially hardware installations and configuration of application channels in the Vector driver applet).

Installation

The user data and application data are installed in different folders, depending on the selections made during installation.

User Data

During installation, items such as sample configurations and templates are saved to the user data folder.

Application Data

Settings for programs (e.g. CAN.ini) and add-on components are saved to the application data folder.

Program Folder

The program files and add-on components are saved to the program folder. This folder also contains INI files that are read-only or that are changed only during administrative tasks (e.g. CAN driver configuration).

Note

You can open the user and application data folders using the Windows Explorer via General|File Locations in the Options dialog.
