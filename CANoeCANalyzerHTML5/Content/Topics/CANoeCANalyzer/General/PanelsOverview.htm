Panel

 

Definition: Symbols

System variables, environment variables, signals, and diagnostic parameters are referred to as symbols in CANoe. In the panel environment, symbols can be used to display values or initiate actions.

Screenshot

Overview

Panels are graphical elements via which values can be changed and displayed using so-called controls.

In terms of the controls, a distinction is made between controls and display elements, where certain controls can serve as both control and display elements.

The panels are saved in XVP (extended vector panel) format and can then be loaded in CANoe configurations.

You can create panels with the Panel Designer.

Cross Reference

Detailed information regarding the creation of panels can be found in the Panel Designer help.

For the stimulation, CANoe provides you with ready-made panels that you can use to change values.

.NET panels can be integrated in CANoe.

Control Elements

Display Elements

With controls, you can change values of assigned system variables and environment variables.

If you are working with transmission models, e.g., as a result of integrating an interaction layer, you can also change values of assigned signals.

With display elements you can display changes of signals, system variables and environment variables, and diagnostic parameters.

Notes

  • With CANoe run, only display elements can be created and configured. Combined control and display elements can only be used for display purposes.
  • Supported symbols depend on the CANoe/CANalyzer program variant.
