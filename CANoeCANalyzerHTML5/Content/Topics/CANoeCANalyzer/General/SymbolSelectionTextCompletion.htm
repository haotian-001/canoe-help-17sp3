Symbol Selection: Text Completion

 

Symbol Selection with Text Completion

  • An easy way of making entries in table cells is to use the list that appears for the cell.
  • This list contains all values that are available for this cell.
  • The upper part (Suggested Symbols) of the list shows your previous entered symbols. When using the list for the first time this part is hided.
    The lower part (All filtered Symbols) shows all available values.
  • Your previous entry is used to filter the list (both parts).

Symbol Selection with Selection Dialog

  • Use in a table cell to open the Symbol Selection dialog and choose the desired symbol.

Opening the List

  • To open the list, start entering the name of the symbol (signal, environment variable or system variable) into the cell.
    The list then shows all available symbols that contain the text you have entered.
  • Alternately, you can open the list by pressing <Alt>+<Arrow down>.

Selection

  • You can select the desired symbol by double-clicking it with the mouse, or
  • by navigating to the symbol using the arrow keys and then pressing <Return>, <Enter> or <Tab> to make your selection.
  • If you want to enter a value that is not in the list you can close the list with <ESC>.

Hints

  • The familiar icons that appear before the symbol name indicate the symbol type (signal, environment variable or system variable).
