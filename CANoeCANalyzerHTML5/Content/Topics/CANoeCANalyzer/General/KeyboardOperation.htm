Keyboard Operation

 

In the following areas you can use the keyboard to control the program:

Shortcuts

» General Commands (always active)

» Main window: keyboard operation

» Standard Windows Commands (always active)

» Graphics Window: keyboard operation

» Dialogs (general)

» Trace Window: keyboard operation

» Tables (in dialogs)

» Data Window: keyboard operations

» Non-Measurement Commands (only when no measurement is running)

» Write Window: keyboard operation

» Measurement Commands (only when measurement is running)

» Statistics Window: keyboard operation

» Offline Commands (only in offline mode)

» Interactive Generator (IG): keyboard operation

» Activating menu commands

» Visual Sequencer: keyboard operation

» Calling Help

» MOST windows: keyboard operation

» Other keys

» Measurement Setup: keyboard operation

 

» Simulation Setup: keyboard operation

Dialogs (general)

Function

Key Combination

Moves the insertion point to the next field.

<Tab>

Moves the insertion point to the previous field.

<Shift>+<Tab>

Moves the cursor to the field whose name contains the underlined character.

<Alt>+Character

Executes the current command or accepts the changes (like [OK]).

<Enter>

Closes the dialog without executing any actions (like [Cancel])

<Esc>

Tables (in dialogs)

Function

Key Combination

Moves the insertion point to the previous/next editable cell.

<Arrow key left>/<Arrow key right>

Moves the insertion point to the previous/next editable row.

<Arrow key up>/<Arrow key down>

Starts editing the selected cell.

<F2>

Cancels editing the selected cell without changes.

<Esc>

Ends editing the selected cell; changes are accepted.

<Tab> or <Enter>

Marks the selected rows for bulk editing.

<Shift>+<Arrow key up>/
<Shift>+<Arrow key down>

Select rows for bulk editing.
Use <Space bar> to mark/unmark a row.

<Ctrl>+<Arrow key up>/
<Ctrl>+<Arrow key down>

Activating Menu Commands

Function

Key Combination

Selects the ribbon and activates the selection of the tabs.

After that the following functions are available:

<Alt>/<F10>

Opens the tab for the selected character.

<Character>

Opens the menu/window for the selected character on the selected tab.

<Character>

Opens the tab to the left or right of the active tab.

<Arrow key right>/
<Arrow key left>

Moves the selection of menu commands within the opened menu.

<Arrow key up>/
<Arrow key down>

Executes the command of the selected menu command.

<Enter>

Calling Help

Function

Key Combination

Opens help.
If a menu has been selected, then help is opened for the active menu command.
In dialogs, help is displayed for the particular dialog.
If Windows help has already been opened, then the use of the help system is explained.

<F1>

Other Keys

Function

Key Combination

Opens the system menu of the main window.

<Alt>+<Space bar>

Opens the window menu of the active window.

<Alt>+<->

Ends the program.

<Alt>+<F4>

Switches between the active Windows programs.

<Alt>+<Tab>

Closes the active window.

<Ctrl>+<F4>

Switches between the main windows within an application.

<Ctrl>+<Tab>/<Ctrl>+<F6>

Switches between the main windows of different applications.

<Alt>+<Ctrl>+<Esc>

You can now move the main window/the active dialog with the arrow keys (up/down/left right).
With the key combination on the right you can activate this feature.

<Alt>+<Space bar>, afterward <v>
