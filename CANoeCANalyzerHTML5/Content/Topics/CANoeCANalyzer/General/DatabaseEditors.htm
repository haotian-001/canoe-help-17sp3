Database Editors

 

Editors/Viewers for Bus Systems

AUTOSAR Explorer

Using AUTOSAR Explorer you can visualize and edit the communication aspects of AUTOSAR system description files and ECU extracts of system descriptions.

CANdb++

Using CANdb++ you can create, visualize and edit CAN databases.

Car2x Network Explorer

Using Car2x Network Explorer you can visualize Car2x databases. Furthermore you can edit basic node and message properties.

FIBEX Explorer

Using FIBEX Explorer you can visualize and edit FIBEX files for FlexRay.

LDF Explorer

Using LDF Explorer you can create, visualize and edit LDF files and NCF files.

Editors/Viewers for Diagnostic Descriptions

Basic Diagnostic Editor

Using the Basic Diagnostic Editor, you can describe simple diagnostic services (UDS & KWP) for control units on CAN, FlexRay, LIN and K-Line.

CANdelaStudio View

CANdelaStudio is a tool that has been developed for recording, editing and presenting diagnostic data for control units in the automotive sector. It offers a wide range of functions to assist you in your daily work with diagnostic data.

ODXStudio View

The ODXStudio View provides a view on ODX data. ODX is a complex data format used to store information relevant for diagnostics in a structured way. ODXStudio allows you to easily navigate through the ODX model, just like a web browser.
