Command Line Parameters

 

Following command line parameters are supported by CANoe:

a

CANoe starts the measurement automatically

f

CANoe loads the defined configuration

m

CANoe starts with the specified application name.

Note

The application name is used for channel assignment in the Vector Hardware Config dialog. Especially if multiple instances of CANoe are operated on one computer, these instances must be started with different application names. One use case for this is Multi CANoe.

r

CANoe automatically loads the last configuration at startup.

c

CANoe starts in distributed mode using a designated remote address.

Example

Command Line Parameter f

CANoe64.EXE /f "D:\CANSystemDemo\CANSystemDemo.CFG"

Command Line Parameter m

CANoe64.exe /m MasterCANoe

Command Line Parameter c

CANoe64.exe /c 192.168.0.2:2809

To limit CANoe to a certain variant, the following arguments are available to you:

  • CANoe64 /limitto:<variant> with <variant>=pex or <variant>=run

Example

CANoe64.exe /limitto:pex
