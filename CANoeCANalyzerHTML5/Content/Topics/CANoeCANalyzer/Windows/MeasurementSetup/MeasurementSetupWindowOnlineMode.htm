Online Mode

 

Select Online Mode

You can switch between online and offline mode in the following ways:

  • By double-clicking the switch symbol
  • Using the context menu of the switch symbol
  • By pressing the <Space> with the switch symbol selected
  • By using the Home ribbon tab

Data Source in Online Mode

In online mode, the data of a current measurement are used.

Online represents a connection to the real hardware (working mode: Real Bus) or a simulated measurement (working mode: Simulated Bus). You can select the working mode in the Home ribbon tab.

Context Menu of Online Data Source

Simulation...

Simulink Analysis Models...

Comment...
Opens the Comment: Function Block dialog. Here you can insert a comment for the function block and choose the font settings of this comment.

Show Comment
Displays the comment in Measurement Setup. If this option is deactivated the comment is displayed as tooltip.

Control Online Mode

You can control the process of a measurement via the Home ribbon tab.

Symbol

Function


Start

Starts, or resumes, the measurement.

Stop

Stops the measurement.


Break

Pauses the simulation.

This function is only available in working mode Simulated Bus.


Step

Starts or resumes the stepwise simulation.

This function is only available in working mode Simulated Bus.

Step-length of time step in ms

In online mode you can enter the length of the time step in ms. The largest possible step is 60000 ms.

This function is only available in working mode Simulated Bus.
