Measurement Setup

Ribbon|Analysis Tab|Measurement Setup

General

The data flow is displayed graphically in the Measurement Setup.
All options for parameterizing the Measurement Setup are selected in this window.

Description of the Measurement Setup

The data flow plan is displayed in the measurement configuration window. It contains data sources, basic function blocks, hot spots, inserted function blocks and data sinks. Connecting lines and branches are drawn between the individual elements to facilitate understanding of the data flow.

Either the bus connected via the hardware (Online) or a logging file (Offline) can serve as the data source.

Square points (hot spots) exist between basic function blocks in the data flow diagram. Additional function blocks can be added at these points, or data flow can be blocked at these points. If the data flow is not blocked, the hot spots permit all data to pass unhindered.
