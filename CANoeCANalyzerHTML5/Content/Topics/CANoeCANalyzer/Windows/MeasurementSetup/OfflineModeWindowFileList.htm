Configuring the Offline Sources List

You can add files via

  • the toolbar
  • the context menu of the offline sources list
  • drag and drop (invalid formats cannot be added, a corresponding message is displayed)
  • the context menu of a logging file in the Measurement Setup

You can delete files via

  • the toolbar
  • the context menu of the offline sources list
  • the keyboard (<Del>)

You can replace all files via

  • the toolbar
  • the context menu of the offline sources list
  • drag and drop (with <Ctrl> key pressed)
  • the context menu of a logging file in the Measurement Setup

Context Menu of the Offline Sources List

Add Offline Source...
Adds a new offline source to the list.
Multiple files can be selected.
New files are always added at the end of the list.

Replace All Offline Sources...
Removes all offline sources and adds new offline sources to the list.

Remove Offline Source
Deletes a selected offline source from the list.
Multiple files can be selected.

Remove All Offline Sources
Deletes all offline sources from the list.

Open Source Path
Opens the target directory in Windows Explorer and the file is selected.

Multiple Selection

  • Left mouse button + <Ctrl>: Selection of individual offline sources
  • Left mouse button + <Shift>: Selection of a offline source range

Configuring Columns

For each added file, metadata is detected in the background. This metadata can be displayed in the columns of the offline source list.

If the table cells remain empty, the corresponding data cannot be detected (e.g. the file format does not support this data or the file does not contain it).

You can use drag and drop to

  • modify the sequence of columns
  • remove columns from the view

This does not apply to the Active and Offline Source columns.

Context Menu of the Column Headline

Hide This Column
Removes the selected column.

Column Chooser
Opens the Customization dialog which allows you to configure the view of the columns. You can add columns in the offline source list with drag and drop or with a double click.

Best Fit
The column width of the selected column is optimized for the contents of all cells of the column.

Best Fit (all columns)
The column width of all columns is optimized for the contents of the corresponding column.

Columns

Active

If an offline source is marked as active, then it is played for offline analysis.

Offline Source

Filename.

Icons

  •   : Valid imported logging file
  • : Valid imported video file
  • : Valid imported image series
  • : Corrupted or no longer available offline source. The file is ignored at the start of a measurement in the offline mode.
  • : Group of split video files

If you add video files to offline analysis, the system automatically checks whether the video files already exist and whether they

  • are from the same measurement
  • are parts of a split measurement (e.g. video files separated by triggers, or video files split by a certain size).

If several files belong together, they are united in one group. They are named with Measurement_{date}, where {date} stands for the time of the start of a measurement.

Channel Mapping

This column displays the currently valid channel mapping. The selection list allows you to change the channel mapping. The standard set is marked with <Standard>.

Start

Start of recording

End

End of recording

Offset

Offset added to the beginning of the file (measurement start). Negative values are allowed.

Format Version

Format version of the file

Selected Target

In this column, the target window for video files and image series is selected. For logging files, the Bus Logging value is predefined and cannot be changed.

Video Window

In the selection list of the column, all configured Video Windows are displayed. Windows which cannot be selected are grayed out.

You can assign a video file, a group or a separate image series to each window. A mixture of video files and image series is not possible. You can define only a common target for the files of a group.

Notifications

If video files from different measurements are assigned to one Video Window, a note appears, indicating that synchronization files, which may be present, will be ignored during the offline analysis. The playback sequence is then determined by the start of the corresponding measurement (from the metadata) or by the filename, if no metadata is available.

If video files assigned to one target window overlap and the offline analysis is carried out according to date/time (absolute time stamp) a warning in the corresponding lines appears. If offline analysis is carried out according to time stamp (relative), an error message appears.

Index

The column displays the determined index within a measurement.

Under certain conditions, the index cannot be determined from the metadata, e.g.:

  • for ASCII files
  • for BLF files up to and including CANoe 10.0 SP2
  • for videos with Videosync files up to and including CANoe 10.0 SP2

In such cases the measurement end is interpreted in 1/100 nanoseconds (if available) as an index.

By default, the Index column is not visible. You can add it via the Column Chooser in the context menu of the column overview.
