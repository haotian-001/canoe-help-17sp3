Toolbar

Ribbon|Analysis Tab|Offline Mode

The toolbar provides the following icons for the configuration.

The functions are as follows:

Symbol

Function

Adds a new logging or video file to the list.

Multiple files can be selected.

New files are always added at the end of the list.

Adds new files from a selected directory or one of its ZIP archives to the list.

Use this icon to add image series for offline analysis. You can also drag and drop a directory to the list.

Logging files and video files located in the selected directory can also be added.

If there is more than one image series, or a mixture of different source types (logging files, video files, image series) in the selected directory, a selection dialog opens.

Here you can select individual image series for import which are represented by their prefix (e. g. CAM3_*). In addition, you can select if existing logging files or video files should also be added. Press [Select All] to select all entries in the list. Another click cancels the selection. Use [OK] to add the selection of the offline analysis.

The individual offline sources have different symbols for better differentiation:

  • : Video file (possible formats: AVI, WMV, MPG, MPEG, MP4)
  • : Image series

Notes

  • All popular pixel-based graphic formats are supported (JPG, TIFF, GIF...).
  • Camera-specific formats such as DNG are not supported.
  • Nested ZIP files (ZIP in ZIP) are taken into account.
  • Nested folders are ignored.

Deletes a selected offline source (e.g. logging file) from the list.

Multiple files can be selected.

Deletes all offline sources from the list.

Replaces all offline sources in the list.

Multiple files can be selected.

Opens the Offline Mode Configuration dialog.

Adapts all columns' widths to optimally accommodate their contents.

Multiple Selection

  • Left mouse button + <Ctrl>: Selection of individual offline sources
  • Left mouse button + <Shift>: Selection of a offline source range
