Measurement Setup: Online/Offline Mode

 

A differentiation is made between online mode and offline mode in the Measurement Setup.

In online mode, the data of the current measurement (hardware or simulation) is used.

In offline mode, a logging file (e.g. BLF, ASC, MF4) with recorded measurement values is used.

In the Measurement Setup, both data sources are displayed. The mode is displayed via the switch symbol and on the status bar.

You can switch between online and offline mode in the following ways:

  • By double-clicking the switch symbol
  • Using the context menu of the switch symbol
  • By pressing the <Space> with the switch symbol selected
  • By using the Home ribbon tab

CANoe Versions

As of version 7.5, the Measurement Setups in online and offline modes remain identical.

You can no longer save different Measurement Setups for online mode and offline mode in one configuration. Instead, you must save the Measurement Setup in online mode and the Measurement Setup in offline mode in two separate configurations.

If you save a configuration in 7.5 or later in an older version (File ribbon tab|Save As), the Measurement Setup is saved twice in the configuration to ensure that it can be opened in older program versions.
