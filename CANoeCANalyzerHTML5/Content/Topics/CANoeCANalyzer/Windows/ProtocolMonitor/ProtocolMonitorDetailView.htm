Detail View

 

The Detail View shows more information about the currently selected element.

If a property refers to another object, this can be selected directly in all views via the context menu command Select (depending on the network selection, grouping and filter settings).

Note

The fully qualified name is displayed for:

  • Networks
  • Endpoints (in tooltip)
  • Connections (in tooltip)

Element

Category

Property

Description

Endpoint

General

Protocol

Protocol to which the endpoint belongs.

ID

Protocol-specific ID of the endpoint.

Network

Network in which the endpoint is located.

Network Node

Network node to which the endpoint belongs.

Parent Endpoint

Endpoint on which the current endpoint is built.

Child Endpoints

Endpoints that build on the current endpoint.

Connections

Connections of the current endpoint.
Status

Status

Endpoint status:

  • Up: the existence of the endpoint has been actively confirmed (e.g., by an event sent by the endpoint).
  • Down: the existence of the endpoint has been explicitly denied (e.g. by a management protocol)
  • Pending: the existence of the endpoint is unknown (e.g. because it has only appeared as a receiver so far)

Time [s]

Last time when the endpoint was observed.

Last 'Up' [s]

Last time when the endpoint was observed in the Up status.

Last 'Down' [s]

Last time when the endpoint was observed in the Down status.

# Events

Number of all events that affected the endpoint.

# 'Up' Events

Number of events at which the endpoint was Up.

# 'Down' Events

Number of events at which the endpoint was Down.

# 'Pending' Events

Number of events at which the endpoint was Pending.

Validation Results

List of validation results that exist for this endpoint.

Protocol-Specific

Contains a list of protocol-specific information about this endpoint.

Note: Protocol-specific information is not displayed for every protocol.

Connection

General

Protocol

Protocol to which the connection belongs.

Initiator

Initiator of the connection, i.e. the initial source endpoint.

Addressee

Addressee of the connection, i.e. the initial destination endpoint.

Network

Network in which the connection is located.

Parent Connection

Connection on which the current connection is built.

Child Connections

Connections that build on the current connection.

Status

Status

Connection status:

  • Up: the connection is established
  • Down: the connection has been terminated
  • Pending: the connection is not yet completely established or terminated

Time [s]

Last time when the connection was observed.

Direction

Directions with which the connection has been used so far.

Last [s]

Last time when the connection was used from the initiator to the addressee.

Last [s]

Last time when the connection was used from the addressee to the initiator.

# Events

Number of all events that affected the connection.

# Events to Addressee

Number of events at which the connection was used from the initiator to the addressee.

# Events to Initiator

Number of events at which the connection was used from the addressee to the initiator.

Validation Results

List of validation results that exist for this connection.

Protocol-Specific

Contains a list of protocol-specific information about this connection.

Note: Protocol-specific information is not displayed for every protocol.

Network Node

General

Name

Name of the network node.

Networks

Display of all networks in which endpoints of this network node are located.

Root Endpoints

Display of all endpoints (of the lowest protocol layer) that belong to this network node.

Network

General

Name

Name of the network.

VLAN ID

VLAN IDs of the network.

Network Nodes

Display of all network nodes that have endpoints in this network.

Root Endpoints

Display of all endpoints (of the lowest protocol layer) in this network.

Root Connections

Display of all connections (of the lowest protocol layer) in this network.

Parent Network

Network in which the current network is located.

Child Networks

Display of all sub-networks of the current network.

Protocol

General

Name

Name of the protocol.

OSI Layer

OSI layer to which the protocol is assigned.

Validation Result

—

Description

Description of the result with more detailed information.

Severity

The severity of the result:

  • Information
  • Warning
  • Error

Time [s]

Last time when the validation result was observed.

Endpoints

List of affected endpoints to which the validation result applies.

Connections

List of affected connections to which the validation result applies.
