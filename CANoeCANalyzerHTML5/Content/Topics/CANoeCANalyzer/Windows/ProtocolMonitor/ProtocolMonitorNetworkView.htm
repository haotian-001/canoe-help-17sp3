Diagram View

 

The central Diagram View provides a graphical summary of the network communication. The connections of the endpoints are symbolized by arrows. The endpoint hierarchy is represented by colored nested boxes.

You can expand and collapse the view of the layers within a box either by using the arrows in the box or by using the context menu. Using the context menu commands Show All Endpoints and Hide All Endpoints you can toggle the visibility of all endpoints at once (see also Filtering & Hiding).

You can adjust the spacing between adjacent boxes by using the Spacing Mode context menu command.

You can zoom in the view with <Ctrl>+mouse wheel. Use the mouse wheel to move the view vertically and <Shift>+mouse wheel to move it horizontally. Further display functions are available in the toolbar.

If you have a large network overview by zooming out (reduced display of the individual elements), then you get an enlarged display of the endpoints, network nodes or networks within a tootip.

When displaying a very large network overview, not all endpoints and connections are displayed graphically anymore. Selected endpoints and connections are always displayed.

Several connections that link the same elements are graphically combined into a single, solid arrow (= multiconnection, orange). A number is used to indicate how many connections have been combined.

A selected connection is highlighted in red. All elements except the connection, its initiator and its addressee are displayed gray.
If you select a multiconnection, the corresponding connections are highlighted in yellow in the Connections View.
An unselected connection is only displayed if at least one of its endpoints is displayed in the overview.

A selected endpoint is highlighted with a red border. All elements except its connections and the corresponding remote endpoints are displayed gray.

You can move the entire view with click and hold.

As long as you hold the <Shift> key, all connections are hidden. This can make it easier to select endpoints, networks and network nodes.

With <ESC> or a click on a free area, you cancel a selection.

Filtering & Hiding

You can configure the view of the endpoints and their connections using the filter selection in the Endpoints View and Connections View.

You can use the context menu to adjust the visibility of the elements. You can also create filters for a Trace Window.

You can view the endpoints grouped by networks ( in the toolbar) or grouped by network nodes ( in the toolbar).

Example

Example: Filtering & Grouping

  • In the toolbar of the Endpoints View the filters for all protocol layers and Only Relevant Layers have been activated.
  • In the toolbar of the Connections View, the filters for all protocol layers and Only Highest Layers have been activated.
  • If an Ethernet packet with the protocols IPv4 and UDP is observed, the two UDP endpoints appear in the Diagram View.
  • The UDP connection of the two UDP endpoints is shown as a line. The parent IPv4 and Ethernet connections are not shown according to the selection of the connection filters.

If Node is selected as the grouping, the two UDP endpoints are assigned to the nodes Node1 and Node2.

 

If Networks is selected as the grouping, both UDP endpoints are assigned to the network Network1.
