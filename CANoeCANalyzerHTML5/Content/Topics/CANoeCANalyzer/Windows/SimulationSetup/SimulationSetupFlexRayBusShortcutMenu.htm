Simulation Setup: Context Menu of the FlexRay Bus

 

Insert Network Node
Inserts a new Network Node in the Simulation Setup.

Insert Test Module
Inserts a new CAPL test module in the Simulation Setup.
In order to keep a clear separation between tests and simulations we strongly recommend to insert test modules in the Test Setup.

Insert Test Module XML
Inserts a new XML test module in the Simulation Setup.
In order to keep a clear separation between tests and simulations we strongly recommend to insert test modules in the Test Setup.

Insert FlexRay Frame Panel
Inserts a new FlexRay Frame Panel in the Simulation Setup.

Insert FlexRay PDU Panel
Inserts a new FlexRay PDU Panel in the Simulation Setup.

Insert Replay Block FlexRay
Inserts a new Replay Block in the Simulation Setup to play previously recorded/logged data.

Insert…
Opens the file selection dialog to insert a CANoe block file (CAN, XML, VXT).

Paste
Inserts blocks that you previously cut out of a different network.

Paste Shortcut
Inserts blocks that you previously copied from a different network.
With Paste Shortcut you create a Gateway between these two networks.

Switch All Blocks to Simulation
Thereby you switch the status of all nodes in the Simulation Setup over to simulation mode simultaneously.
During a measurement the functionality of nodes is simulated by the assigned CAPL programs.

Switch All Blocks to Realtime Mode
Thereby you switch the status of all nodes in the Simulation Setup over to real mode.
The network node models in the simulation configuration have no effect on the measurement in this case.
