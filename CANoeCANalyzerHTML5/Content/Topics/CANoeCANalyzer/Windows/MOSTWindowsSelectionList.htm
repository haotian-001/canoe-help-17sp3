MOST Windows: Selection List

 

Behavior of selection lists in MOST IG, MOST Application Filter and MOST custom event highlighting.

Setup:

  • The selection list consists of an entry box and the actual selection list.
  • Each entry in the selection list contains a symbolic value and an ID (in brackets)

Basic use:

  • The selection of entries from the list can be carried out in the usual way using the mouse (open, scroll, select).
  • It is possible to exit the edit mode at any time by means of the ESC key, without any change to the value in the field.

Activating edit mode in a selection list:

  • If the selected field in the message list has a selection list associated with it, the list can also be opened by means of the key combination <Alt> + <Down>. The focus is then on the selection list.
  • If the field selected in the message list has a selection list associated with it, a selection can be made by simply typing a letter or number. The focus is then on the entry field.

Working with the entry field:

  • Everything the user enters in the entry field limits the selection in the selection list. The entry in the field is interpreted as a partial string that may occur anywhere in the entries in the selection list.
  • <Ctrl> + <Del> clears all the characters to the right of the cursor position
  • To move from the entry field to the selection list, press the <Down Arrow> key or the key combination <Ctrl> + <Down Arrow>
  • If the <Enter> key is used to exit the entry field, the current text contained in the field is matched with the selection list if possible. If there is no match for the text entry, edit mode is deactivated but no change is made to the value in the field.

Valid entries:

  • IDs in the format described in the note in the status bar.
  • Symbolic values as they are listed in the selection list. Only the part of the entry up to the first space is evaluated.
  • If the <Left> or <Right> arrow key or the <Tab> key is used to move away from the entry field, the entry selected in the selection list is accepted as the value.

Working with the selection list:

  • The <Up> and <Down> arrow keys can be used to scroll up and down in the list.
  • Pressing <Ctrl> + <Up> moves the focus directly from the selection list to the entry field. If the very first entry in the selection list is selected, simply pressing the <Up Arrow> key is also sufficient.
  • Pressing <Alt> + <Down Arrow> closes the selection list and deactivates edit mode without effecting any change to the value in the field.
  • If the <Enter> key, the <Left Arrow> or <Right Arrow> key or the <Tab> key is used to move away from the selection list, the entry selected in the selection list is accepted as the value.
