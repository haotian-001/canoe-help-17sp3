Start Values Window: Configure Start Value List

 

Groups

You can add groups to the Start Values Window via the toolbar or the context menu.

With a double click on the group cell you can directly change the group name.

By activating the Persist column, the start values of these symbols are updated outside and after a measurement and can be retained throughout the runtime of CANoe.

Symbols

Enter Symbols

You can insert symbols in the list as follows:

  • Keyboard
    In the Symbol column, enter a portion of the symbol name.
    A list of all available symbols containing the typed text string is displayed. You can select the desired symbol using the up and down arrow keys. The symbol is inserted with the <Tab> or <Enter> key.

    For list display, note the settings for array elements of distributed objects in the Options dialog.

Note

You can also drag messages to the Start Values Window. All signals of the message are thereby entered into the list of start values.

Enter Start Values for Symbols

You can enter the start value using the Physical Value or Raw Value column. If a value table is assigned to the desired symbol, you can also select the start value symbolically using the Symbolic Value column.

If the value is changed in one of the three columns, the other columns are automatically synchronized with the newly entered value. For signals, the conversion factors defined in the data source are used for the conversion between physical value and raw value.

When several rows are selected and you enter the start value in a selected cell (Physical Value or Raw Value), this new value is automatically set in all selected rows.

Configure Rows

With drag and drop you can change the order of the lines. You can shift whole groups or single start value rows.

Context Menu of the Table Rows

Insert Application Layer Objectsâ¦
Adds a distributed object to the selected group.

Insert Signals...
Adds a signal to the selected group.

Insert Variables...
Adds a system variable to the selected group.

New Group
Adds a new group above o below the selected group.

Expand/Collapse

  • Expand Selected
    Expands all selected value groups.
  • Expand All
    Expands all value groups.
  • Collapse Selected
    Collapses all selected value groups.
  • Collapse All
    Collapses all value groups.

Set Start Value to

  • Data Source Values
    Sets the start values of the selected symbols to the values stored in the data source.
    See also Action 1 in function overview.
  • Current Values
    Sets the start values of the selected symbols to the currently measured values.
    See also Action 3 in function overview.

Set Symbol to

  • Individual Start Values
    Sets the selected symbols to their individual start value.
    See also Action 2 in function overview.
  • Data Source Values
    Sets the selected symbols to the values stored in the data source.
    See also Action 4 in function overview.

Export Selected Start Values
Exports the selected start values into a file.

Delete
Removes the selected groups and symbols from the start value list.
The removal occurs without a further prompt and can be undone with <Ctrl>+<Z>.

Cut
Removes the selected groups and symbols from the start value list. The removed objects are available in the clipboard.

Copy
Copies the selected groups and symbols to the clipboard.

Paste
Pastes the objects from the clipboard into the selected line.

You can remove rows from the list as follows:

  • Keyboard
    The <Del> key is used to delete all selected lines from the list.
  • Context Menu
    You can either delete all selected lines or all lines from the list.

Note

You can use the <Ctrl>+<a> key combination to select all lines in the list.

You can use the <Ctrl>+<z> key combination to reverse the last executed action.
