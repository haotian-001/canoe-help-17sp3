Start Values Window: Functionality

 

In the Start Values Window, values that are set on measurement start can be preassigned for symbols.

Actions

  • Action 1
    The start values of the selected symbols are reset to the initial values from the data source (context menu).
  • Action 2
    The symbol value of the selected symbols is set to the configured start value. This is also the standard action at the start of the measurement (context menu).
  • Action 3
    The start values of the selected symbols are set to the current symbol value (context menu).
    This action is also available during a measurement.
  • Action 4
    The symbol value of the selected symbols is reset to the initial values from the data source (context menu).
    This action is also possible from the Options dialog.

Behavior during Measurement

During measurement the following functions are not supported:

  • Import of start values
  • Append start values
  • Export of start values
  • Settings in Options dialog
  • Set symbol to:
  • Individual start values
  • Data source values

During measurement the following functions are supported:

  • Activating/deactivating of groups and start values.
  • Editing group and symbol names in the start value list.
  • Editing of the start values (physical value, raw value, symbolic value, comment) in Start Value Window.
  • Creating and removing of groups and symbols.
  • Copying and pasting of groups and symbols.
  • Searching of texts in Start Value Window.
  • Setting of the start values to data source values or current values.
