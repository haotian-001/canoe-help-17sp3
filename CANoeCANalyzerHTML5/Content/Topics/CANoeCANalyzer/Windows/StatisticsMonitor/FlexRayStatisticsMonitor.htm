FlexRay Statistics Window

Ribbon|Analysis Tab|Statistics|View / New 

General

The FlexRay Statistics Window displays statistics about bus activities during measurement.

This Statistic Window can be inserted into the Measurement Setup via the context menu of the relevant function block (bus statistics etc.). To show the window use the Show Window context menu command.All FlexRay bus events that arrive at the input of the Measurement Setup are evaluated for this purpose.

If a database is available on a FlexRay channel, statistics can also be evaluated individually for each available FlexRay node.

FlexRay statistics can also be evaluated by using statistics system variables.

Various statistic CAPL functions are available.

In the Options dialog you can activate/deactivate the calculation of all bus statistics values.

Displays the current POC state of the communication controller in the interface (is not supported in the "Asynchronous Mode")

Offset Correction

Displays the Offset Correction of the communication controller in the interface (is not supported in the "Asynchronous Mode")

Rate Correction

Displays the Rate Correction of the communication controller in the interface (is not supported in the "Asynchronous Mode")

An available FlexRay channel can be selected from the FlexRay channel selection list. The selection list shows the current FlexRay channel, followed by the corresponding name of the FlexRay network in the Simulation Setup. Selecting the FlexRay All option in the channel list provides an overview of the Current/Last columns for all available FlexRay channels.

The Current/Last column shows the current or most recently derived statistical value. The Min, Max and Avg columns show the corresponding minimum, maximum and average statistical values. All channels can be reset when FlexRay All is selected; otherwise only the selected FlexRay channel is reset.

In the configuration dialog (context menu/right mouse button) you can adjust the windows refresh rate. In the Available Statistics area you control via the check boxes which statistics are shown and which not.

If a measurement is running, the currently displayed values can be paused by clicking the pause toolbar button or by the context menu command Pause Statistics.

Via the context menu or the toolbar you have further functions:

Configuration
Opens the configuration dialog.

Collapse All
Collapses all statistics.

Expand All
Expands all statistics.

Reset All Statistics
Resets all statistics.

Pause Statistics
Freezes all statistics during a meassurement.

Adapt Columns Width
Adapts the widths of all columns to the current values.

 

Sort
Sorts statistics Alphabetically or Logically.

 

Copy to Clipboard
Copies selected values to the clip board.
