FlexRay Cluster Monitor: Views

 

The FlexRay Cluster Monitor offers the following views:

View

Description

Statistics (general)

The global network statistics are displayed in frames/second (fr/s). However, with respect to nodes or frames, the display refers to frames per 64 communication cycles. This display allows for an exact comparison of the data with the communication model without rounding errors.

Network View

The page with the network data is divided into three areas: Parameter, counter, and statistics.

The parameter area displays the statistical network parameters:

  • Cycle length in ms
  • Number of statistical slots
  • Length of payload in the static area
  • Length of payload in the dynamic area
  • Number of nodes (configured in the application)
  • Number of nodes (from the database)

The statistics area displays the running statistics of the network:

  • Frames (dynamic + static) in fr/s
  • Dynamic frames in fr/s
  • Static frames in fr/s

The number of frames is included in the counter area:

  • Number of cycles
  • Frames (dynamic + static)
  • Static frames
  • Dynamic frames
  • Null frame

Node View

The node view displays all the nodes of the FIBEX communication model. All frames that are either not assigned to any node in the FIBEX database or that are not defined at all, but that are received, will be assigned to the entry "Not assigned frames".

The information of the statistics (all frames, static + null frames, dynamic frames) is each divided into channel A and channel B. The desired values are displayed in square brackets, the actual values are placed on the left them. The desired value depends on the current filter setting.

Frame View

The frame view shows all frames from the FIBEX database and all frames that were transmitted on the bus during the measurement.

For frames from the FIBEX database that are assigned to a node, detailed data are displayed from the database:

  • Name (designator)
  • Slot ID
  • Channel (name of the FlexRay channel)
  • Offset (cycle offset, a timing parameter)
  • Repetition (cycle repetition, a timing parameter)
  • Duration (time interval in ms between individual transmissions)
  • Invalid frames (counter for frames whose DATAVALID flag is not set)
  • Missing frames (frame counter indicates the number of frames that were not transmitted contrary to the FIBEX schedule)
  • Null frames (counter for Null Frames)
  • Frame type (indicates the FIBEX frame type)
