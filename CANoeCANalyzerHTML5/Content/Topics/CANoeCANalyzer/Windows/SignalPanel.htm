Signal Panel

 

General

The signal panel makes it easy to parameterize CAN, LIN and FlexRay signals at measurement time using a graphical interface. The following signal panels are available:

  • Node Panel
    A node panel contains all signals associated with a given Tx node.
  • Network Panel
    A network panel contains all Rx signals associated with a given network.

Node Panel: Configuration

Network Panel: Configuration

Node panels are available for CAN, LIN and FlexRay.
To configure a node panel…

  1. Add a network node in the Simulation Setup.
  2. In the configuration dialog, assign a Network node from the assigned database.
  3. Select Open Node Panel... from the network node's context menu.

The node panel that opens contains all Tx signals associated with the node.

Network panels are available for CAN, LIN and FlexRay.
To configure a network panel…

  1. In the Simulation Setup, select Open Network Panel... from the network node's context menu.
  2. Select the nodes to be stimulated from the list of Rx nodes.

The network panel that opens contains all Rx signals associated with the selected node (default setting: no node selected).

Use

Display of Signals

  • The signals are shown in a different sort order depending on the view selected (message view, signal view, Rx node view). Signals can also be grouped using the View menu. Depending on the view and the grouping scheme selected, additional nested tables will appear. You can expand these with the "+" symbol and collapse them with the "-" symbol.

Moving Columns

  • You can move columns to different positions by dragging & dropping the column heading.

Hiding and Showing Columns

  • You can show or hide additional columns using the context menu of the column heading. Initially, only the most important columns are shown. You can use the context menu of the columns to open the Choose Field... dialog, which shows all available columns. Then add columns to the table using drag and drop.

Sending Signals

  • In message view, signal values are transmitted exclusively via the [Update] button in the corresponding row. All signals in the message are transmitted at the same time (group update).
  • In all other views, signal values are transmitted as soon as any signal value is modified.
  • You can modify signal values in the Physical Value, Raw Value and Symbolic Value columns.

Deactivating Messages

  • When you select a row in the table, you can activate/deactivate the associated Tx message via the context menu.

Tx Ack Column

  • The Tx Ack column shows when a signal update was carried out successfully. During the signal update, a "Tx Pending" icon appears. Once the signal value is transmitted successfully, the icon disappears again.
