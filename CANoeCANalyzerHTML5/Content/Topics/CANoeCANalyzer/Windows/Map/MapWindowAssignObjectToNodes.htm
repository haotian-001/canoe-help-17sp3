Assign ITS objects to database nodes

 

With the Car2x Network Explorer you can create send nodes for WLAN packets in the database, which are defined with a fixed MAC address. These send nodes can be used to display signals in analysis windows, such as the Data or Graphics Window.

During a certificate change the MAC address of a send node can change and the database node is no longer detected. For this case you can assign a received ITS object to a database node during the measurement. With this assignment the received data can always be displayed as signals, regardless if the MAC address of an object has changed or not.

To assign an object select the appropriate database node in the Node column. The displayed name is built by the schema Database name::Node name.
