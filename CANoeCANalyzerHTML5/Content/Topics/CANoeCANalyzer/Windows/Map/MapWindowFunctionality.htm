Window Synchronization

After measurement stop or replaying a logging file you can use the windows synchronization (CANoeHome ribbon tab, button (Window Synchronization) to find and analyze specific traffic scenarios.

To find the scenarios more quickly you can use the measurement point in the Graphics Window or the cursor in the State Tracker. These windows have a continuous time line. In the next step you can do iterative and more detailed analysis in the Trace Window.

Please note that the windows must be available at the start of measurement. The Graphics Window and the State Tracker should have at least one object included (e.g. a bus statistics signal).

Automatic Fading

GNSS objects, ITS objects and ITS events from which no GNSS signals or Car2x WLAN frames are received within three seconds are automatically faded out in the map view. These objects/events remain in the Detail view and are marked with a gray background as inactive.

If WLAN frames are received from inactive objects/events again, these objects/events are displayed in the map view again and the gray background in the Detail view is not shown any longer.

Highlighting of selected objects/events

If you select entries in the Detail view the appropriate objects/ events are highlighted with a frame in the map. Vice versa, entries are selected in the detail view if you select an object/events in the map with the left mouse button.
