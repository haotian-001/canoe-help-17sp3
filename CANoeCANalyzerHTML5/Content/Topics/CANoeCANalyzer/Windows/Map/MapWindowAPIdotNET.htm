Map Window API .NET

 

With the Map Window API for .NET you can draw into a CANoe map window during measurement. On this help page a C# MeasurementScript is used as an example.

CANoe Configuration

To use the Map Window API, a map window must first be inserted into the measurement setup of the CANoe configuration.

A .NET MeasurementScript can then be added to either a network node or the communication setup.

It is recommended to use Visual Studio as .NET file editor. You can specify this in the Options dialog|tab External Programs|page Tools|area List of Tools.

For the .NET File Editor, select the appropriate script from the {CANoe_Exec64}/Scripts directory that matches your installed version of Visual Studio:

  • Edit_NET_Source_with_VS_2005.vbs
  • Edit_NET_Source_with_VS_2008.vbs
  • Edit_NET_Source_with_VS_2010.vbs
  • Edit_NET_Source_with_VS_2012.vbs
  • Edit_NET_Source_with_VS_2013.vbs
  • Edit_NET_Source_with_VS_2015.vbs
  • Edit_NET_Source_with_VS_2017.vbs
  • Edit_NET_Source_with_VS_2019.vbs

You can then, for example, add a new C# file (.cs) to a node in the Simulation Setup via Configuration in the context menu of the node.

Classes

The following classes are available in the Map Window API:

Class

Description

MapWindowAPI

Central class with methods to create, delete and draw map objects.

MapRectangle

Object for displaying rectangles.

MapEllipse

Object for displaying ellipses.

MapLine

Object for displaying lines.

MapTriangle

Object for displaying triangles.

MapText

Object for displaying text.

MapCross

Object for displaying a cross.

MapBitmap

Object for displaying bitmaps (e.g. in the formats bmp, jpg, png)

MapVehicle

Object for displaying vehicles (Default display of vehicles in the map window)

The individual methods and properties are described in an API reference.

General Procedure

Step by Step Procedure

  1. The objects to be drawn can be defined as member variables in the MeasurementScript:

using Vector.Tools;
using Vector.CANoe.Runtime;
using System.Drawing;

public class Map_API_Demo : MeasurementScript
{
  private MapRectangle mRect = null;
}

  1. In the Start method they are created once:

public override void Start()
{
  mRect = new MapRectangle(new GeoPosition(48.825095, 9.09604));
  mRect.LineColor = Color.DarkBlue;
  mRect.LineWidth = 2;
  mRect.FillColor = Color.CornflowerBlue;
}

  1. If the objects are to be drawn periodically and moved, for example, a timer can be used:

[OnTimer(100)]
public void DrawObjects()
{
  mRect.Offset = new GeoPosition(0.0000005, 0);
  MapWindowAPI.DrawObject(mRect);
}

If you have changed properties of map objects via the API (e.g. Position or FillColor), these changes become visible after the DrawObject method is called for the respective object.

Examples

  • Create a map object dynamically
  • Draw a bitmap
  • Delete Map Objects
  • Navigate to a position on the map

Create a map object dynamically

After pressing the C key, a new rectangle is created at position 48.825095, 9.09204 and drawn with DrawObject.

private MapRectangle mDynRect = null;

[OnKey('c')]
public void CreateObjectOnKeyDown(char key)
{
  if (mDynRect == null)
  {
    mDynRect = new MapRectangle(new GeoPosition(48.825095, 9.09204));
    mDynRect.LineColor = Color.Black;
    mDynRect.LineWidth = 2;
    mDynRect.FillColor = Color.Red;
    mDynRect.Width = 40;
    mDynRect.Height = 40;
  }
  MapWindowAPI.DrawObject(mDynRect);
}

The name of the method is variable, but the signature must contain a parameter of type char.

The query mDynRect == null can be used to prevent the object from being created again when the key is pressed again.

With a slight modification, an object could either be recreated or, if already existing, moved to the right:

[OnKey('c')]
public void CreateOrMoveOnKeyDown(char key)
{
  if (mDynRect == null)
  {
    mDynRect = new MapRectangle(new GeoPosition(48.825095, 9.09204));
    mDynRect.LineColor = Color.Black;
    mDynRect.LineWidth = 2;

    mDynRect.FillColor = Color.Red;
    mDynRect.Width = 40;
    mDynRect.Height = 40;
  }
  else
  {
    mDynRect.Offset = new GeoPosition(0, 0.00005);
  }

  MapWindowAPI.DrawObject(mDynRect);

}

Draw a bitmap

After pressing the B key, a new bitmap is created at position 48.825095, 9.09204 and drawn with DrawObject. The bitmap car.png is located parallel to the CANoe configuration in the images directory.

private MapRectangle mBitmap = null;

[OnKey('b')]
public void CreateBitmapOnKeyDown(char key)
{
  if (mBitmap == null)
  {
    mBitmap = new MapBitmap(new GeoPosition(48.827095, 9.09404));
    mBitmap.Filepath = @"images\car.png";
    mBitmap.Width = 40;
    mBitmap.Height = 60;
  }
  MapWindowAPI.DrawObject(mBitmap);

}

Delete Map Objects

If map objects are no longer needed, they can be deleted using the DeleteObject method. In this code example, an object is dynamically created or deleted by pressing the T key.

[OnKey('t')]
public void ToggleCreateOnKeyDown(char key)
{
  if (mDynRect == null)
  {
    mDynRect = new MapRectangle(new GeoPosition(48.825095, 9.09204));
    mDynRect.LineColor = Color.Black;
    mDynRect.LineWidth = 2;

    mDynRect.FillColor = Color.Red;
    mDynRect.Width = 40;
    mDynRect.Height = 40;
    MapWindowAPI.DrawObject(mDynRect);
  }
  else
  {
    MapWindowAPI.DeleteObject(mDynRect);
  mDynRect = null;
  }
}

Navigate to a position on the map

This code is used to navigate to a position (latitude, longitude) on the map by pressing the N button:

[OnKey('n')]
public void NavigateOnKeyDown(char key)
{
  MapWindowAPI.SetMapPosition(48.825095, 9.09204);
}
