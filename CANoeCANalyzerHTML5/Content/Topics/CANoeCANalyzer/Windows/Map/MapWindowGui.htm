The context menu of the stations/events provides the following commands:

Auto-Follow

Automatically scrolls the map with the station so the station is always shown in the displayed map area. The station with an active Auto-Follow option is highlighted in color.

This option can be selected for a single station or event only. If an station or event already uses the option it is deactivated on activating the option for another station or event.

An active Auto-Follow option is overwritten by the CAPL functionSetMapPosition.

Pin

Pins a station to the upper part of the list so that it is stored in the configuration.

Show

Defines whether the station is shown in the map or not.

Move to

The map is moved so that the selected station is shown in the center of the map area.

With this command the position of an station is detected. In contrast to the Auto-Follow option the position is not connected with the station but is executed once on selection.

Remove

Removes the selected station from the Detail view.

If this station transmits position data during measurement it is added to the Detail view again.

Remove unpinned

Removes all station from the Detail view that are not pinned.

If position data are received during measurement the stations sending these data are added to the Detail view again.

Trace

Only active when an ITS event is selected. Selects in the trace window the last message that caused this event to be triggered. Can also be triggered by double-clicking the event in the detail view.

GNSS Display Settings

Only active when a GNSS station is selected. Opens the display settings for the selected GNSS station.
