Checks of J1939 Parameter Groups

 

Several checks of J1939 parameter groups are processed to detect violations against the J1939 specification. If a violation is detected, the appropriate row is highlighted in color and marked with an error icon. The column J1939 Violation displays an error message.

Note

During measurement start error messages about the transport protocol are possible. This is caused by measurement start during a transport protocol transmission so CANoe cannot evaluate the start of the transport protocol.

Note

If an import into the Trace Window was executed not all checks are processed. No checks are processed if a sequence of parameter groups has to be evaluated, e.g. the transport protocol.
