Trace Window

Ribbon|Analysis Tab|Trace|View / New

The purpose of the Trace Window is to record activities during measurement. All the messages received at the input to the Trace Window in the Measurement Setup are displayed as lines of text in the Trace Output Window.

A range of further events is output to the Trace Window, for example:

  • Error events
  • System and environment variables
  • Transport protocol messages
  • Diagnostic services

The Trace Window provides various functions for online and offline bus traffic analysis, such as:

  • Display and column filters
  • Various search functions for finding texts, conditions or patterns
  • Standard sorting functions for the individual columns
  • Import/export of Trace data to/from logging files

Tips & Tricks
  • During measurement signals or messages can be dropped from Trace Window into the Graphics and Data Window (Condition: Fixed position mode).
  • For each message there is a global message counter that can be displayed as an entry in the Trace, Data and Graphics Windows.
  • During measurement the Trace Window is paused automatically when scrolling up the vertical scroll bar.
  • Trace Window columns can be sorted (also during measurement) using drag and drop.
  • Event Fading
  • Working with Multiple Monitors
  • Unsorted events (Message, Error Frame,...) are displayed with a red exclamation mark: .
  • The symbolic interpretation offers you an easy way to interpret the information. You can get a symbolic display of the source and destination addresses as well as the packet data. You can use this in the detailed view too. To toggle between symbolic and numeric interpretation, use the buttons [sym/num] in the CANoe Home ribbon tab.

ISO11783

With the Option ISO11783 the Trace Window is extended by many functions, which prepare and represent ISO11783-specific information comfortably.

  • More than 30 different columns are available for all ISO11783-specific information, which you can select according to your needs (e.g. parameter group number, priority, source and destination address as well as the protocol and content interpretation in text form).
  • Individual parameter groups are highlighted according to their topics.
  • With predefined analysis filters you can show or hide certain groups of PGs with one click and thus significantly increase the clarity in the Trace Window.

A compact summary of all important functions can be found in a video on YouTube.
