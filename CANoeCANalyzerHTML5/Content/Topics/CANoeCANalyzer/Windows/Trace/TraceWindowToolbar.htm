Toolbar

 

The toolbar provides the following symbols for controlling the Trace Window:

The functions are as follows:

Symbol

Function

Detail view in Trace Explorer.

Statistics view in Trace Explorer.

Difference view in Trace Explorer.

Predefined filter in Trace Explorer.

Analysis filter in Trace Explorer.

Sequence filter in Trace Explorer.

Search view in Trace Explorer.

Clear Trace Window contents.

Faded events/data can be deleted from the Trace Output Window.

Suspends/Resumes Trace Window update.

Toggles the Trace Window time mode (absolute/relative).

Toggles the Trace Window display mode (fixed/chronological):

  • Chronological mode (activated symbol)
    The window content is updated with each newly arriving event.
  • Fixed position (deactivated symbol)
    Events are only cyclically updated.

Activates/deactivates analysis filters.

Activates/deactivates column filters.

If you have not defined a column filter, the function is inactive.

Field to insert a string to be searched for

Search backwards in the Trace Window

Search forwards in the Trace Window

Navigates to previous navigation time point in the navigation history of the analysis area.

Navigates to next navigation time point in the navigation history of the analysis area.

Go to marker (only available in chronological mode)

Opens the Trace Window configuration dialog.

Global settings for all Trace Windows:
  • Decimal places time
  • Decimal places float
  • Opens the font settings in the Options dialog.
  • Opens the Trace highlighting settings in the Options dialog.
  • Opens the swap file settings in the Options dialog.
  • Other Trace Settings
  • Opens the Trace Window settings in the Options dialog.

Opens a second view for the Trace Output Window:
  • Trace splitter horizontal
  • Trace splitter vertical
  • Hide Trace splitter

Changes font size:
  • Increase font size
  • Decrease font size
  • Default font size

 

Note

The font size can also be changed with a scroll wheel of a mouse and simultaneously pressed <Ctrl> key.

The selected font size will not be stored in the configuration.

Trace Explorer position:
  • Trace Explorer left
  • Trace Explorer bottom
  • Hides Trace Explorer

Column layout: Default view (Default)

Column layout: User-defined view
