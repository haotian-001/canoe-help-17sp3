ISO11783 Sequence Diagram

Representation of the message flow according to the Sequence Diagram type:

  • <-130->
    A node with the address 130 sends a broadcast message.
  • 38->129
    A node with address 38 sends a message to a node with address 129.
  • 38<-130
    In order to get a correct display, you should set a fixed width font in the Trace Window concerned, e.g. Courier New.
