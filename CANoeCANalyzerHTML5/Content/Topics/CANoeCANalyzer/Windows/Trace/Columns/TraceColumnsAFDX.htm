Columns for AFDX

Context Menu Path: Configuration|Column Configuration|AFDX

Standard AFDX Trace Window layouts

There are various predefined layouts for the AFDX Trace Window.

Layout

Description

AFDX Raw

This view is especially advantageous, if one is especially interested in the Frame Header (e.g. in routing) or addressing errors or if no suitable DBC information is available.

AFDX VL

This view shows key VL-relevant information. For optimal display representation, the relevant DBC information should be available.

AFDX Message

This view shows the key attributes of the individual messages.

AFDX FDS

This view shows the key attributes of the individual FDS
