Context Menu Path: Configuration|Column Configuration|ARINC 825

Standard Trace Window Layouts

There are various predefined layouts for the Trace Window.

 

View

Description

1.

ARINC 825 Basic

This layout shows the all parts of the CAN identifier for peer-to-peer and one-to-many communication.

2.

ARINC 825 Services

This layout is preferred for presenting node services. The columns Services and Interpretation are shown.

3.

ARINC 825 DMC

This layout displays the addressing parts of DMC (direct message communication) messages.

4.

ARINC 826 SDL

This layout corresponds to "ARINC 825 Services" without displaying the columns LCC and SMT.

17.

Specific18

FSB

Functional Status Bit: This bit together with the DLC (data length code) represents the functional status on message level.

  • 0 (DLC != 0) normal operation: "NO"
  • 1 (DLC != 0) functional test: "FT"
  • 0 (DLC == 0) no computed data: "NCD"
  • 1 (DLC == 0) fail/warn: "FW"
