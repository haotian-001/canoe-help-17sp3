Context Menu

 

Detail View / Statistics View / Difference View

Copy
The marked line is copied to the clipboard.

Copy All
All lines are copied to the clipboard.

Expand/Collapse (Detail view only)

Expand All Selected

Collapse All

Find Event with Minimal Value (Statistic view only)

Find Event with Maximal Value (Statistic view only)

Analysis Filter / Sequence Filter

Filter Group / Sequence Filter

Add Stop Filter
Adds a new Stop Filter.

Add Pass Filter
Adds a new Pass Filter.

Add Filter Group (only for analysis filter)
Adds a new filter group.

Remove Filter Group (only for analysis filter)
Removes the filter group. This menu command is only available with minimum two filter groups.

Rename Filter Group (only for analysis filter)
Opens a field to rename the filter group.

Filter Group / Sequence Filter Active
Activates/Deactivates the filter group / sequence filter.

Specific Filter

Add Condition

Add Events...
Opens a dialog to select messages, PDUs and signals.

Add Variables...
Opens a dialog to select system variables.

Add Application Layer Objects...
Opens a dialog to select application layer objects.

Add Diagnostic Service...
Opens a dialog to select diagnostic services.

Add MOST Events...
Opens a dialog to configure MOST filter.

Bus or protocol-specific entries
ISO11783 analysis filter, J1939 analysis filter, Ethernet filter

Paste Condition
Pasts an event that was copied to the clipboard.

Remove Filter
Removes the filter. This menu command is only available with minimum two stop or Pass Filters.

Filter Active
Activates/Deactivates the filter.

Specific Event

Edit Condition
Opens a dialog for editing the filter conditions like setting Rx and Tx.

Configure Value Filter...
Opens a dialog to configure a value filter.

Cut Condition
Cuts the event and copies to the clipboard.

Copy Condition
Copies the event to the clipboard.

Remove Condition
Removes the event as filter.

Empty Space

Add Filter Group
Adds a new filter group.

Search View

Edit (Conditions search)
Opens the Set of user-defined conditions dialog to define the search conditions.

Remove Event (Pattern search)
Removes the event from the search.
