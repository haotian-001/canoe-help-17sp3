Edit Column Layout

Here you can configure a column layout for the current Trace Window. You manage the column layouts on the Column Layouts page.

Name of the column layout that will be edited.

Source

Via the list field Source you can select fields from predefined bus-system-specific column layouts or from other trace column layouts. The first column indicates the Field in question and the second column shows a recommended Column title for this field.

Displayed Fields

Here the selected column layout for the Trace Window is displayed. Only the visible columns are listed, indicating for each column the title, field and available filter configurations. You can edit the column titles, change the sequence of columns and remove the available filter configuration.

Symbol

Description

Indicates that a column filter is configured for the field.

Moves a selected field down.

Moves a selected field up.

Deletes the configured column filter of the selected field.

You can also change the sequence of columns with drag and drop.

Please note that only one column per information field is displayed in the Trace Window. For example, you cannot add a time column for each bus system. If you attempt to select the same information field from different options, these fields are merged into a single column. In the case of different column titles, a new combined title is created. You can manually edit the column titles at any time.

Note

  • Trace Window columns can directly be inserted/removed in the Trace Window using the table head context menu Field chooser… (also during measurement).
  • You can save your individual column configuration on the Column Layouts page as a Standard layout.
    In this case, each new Trace Window initially has this configuration.
