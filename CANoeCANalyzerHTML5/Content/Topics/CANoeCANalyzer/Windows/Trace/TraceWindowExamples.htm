Creating user-defined column layouts for the Trace Window

To use the same column layout whenever you generate a new Trace Window, you can proceed as follows:

  1. Open the Trace Window configuration dialog for an existing Trace Window and go to the Column Configuration page.
  2. Select a column layout with the list field Layout.
  3. Create the column configuration you want to use by adding, removing and sorting the relevant column entries.
  4. Switch to the Column Layouts page.
  5. Select under available layouts the currently configured view.
  6. Select [Save as Standard Layout].
  7. Newly created Trace Windows will automatically be displayed with this layout.

You select a column layout for an open Trace Window using the toolbar ().

Take over created column layouts to other Trace Windows.

If you want to take over a column layout of a Trace Window to another Trace Window, you can proceed as follows:

  1. Open in a existing Trace Window the configuration dialog (Column Configuration page).
  2. Select in the list filed Source the layout to be applied.
    The user-defined layouts are listed at the very bottom.
  3. Move either all columns with or individual columns with into the displayed fields.
    Configured column filters will be taken over as well.

Tips for Working with Statistics and Different Views

Statistics view

  • Open the Statistics view in the Trace Explorer.
  • Select the events for which you want to generate statistics.

Difference view

  • Select the top row (starting point).
  • Set this as the reference event by choosing Set as Reference from the context menu.
  • Open the Difference view in the Trace Explorer.
    When browsing through the entries in the Trace Window you will now see the relevant Difference view displayed, without any (irrelevant) foreign objects (other messages) displayed in between.

Custom Trace Windows with filters

You can create special Trace Windows for different evaluations.

  • Add a Trace Window to the Measurement Setup.
  • Now use suitable filters in the Measurement Setup, e.g., Channel Filter.
  • You can use the predefined filters in the Trace Window to filter out events from the data flow (no event storage in the buffer) that are not of interest in the respective window.
