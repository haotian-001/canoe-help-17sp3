Set Markers

Context Menu: New Marker

In the Trace Window, you can set markers for quick location and commenting of events. The marker is assigned an event, and therefore its time stamp as well.

Note

Markers which were set in the Trace Window can be displayed, but not moved, in the Test Trace Window, Graphics Window and State Tracker. These markers are indicated by a small lock .

If a marker is set to a time stamp which does not fit any time stamp in the Test Trace Window, Graphics Window or State Tracker, the marker is then displayed on the next fitting event in the Trace Window. Thus there can be multiple markers for a single event. A tooltip displays the markers.

Only those markers which are in the analysis area are displayed. If you navigate to another marker outside this area, the trace content is reloaded.

Set Markers, Navigate to Markers

You can set markers via the context menu of the Trace Window or via a double-click on the marker column.

Navigation to the individual markers is possible either using the context menu of the marker column, shortcuts or the toolbar .

Note

Navigating between markers and also adding, editing or removing markers is only possible after the measurement or during the pause. The comments column is only then visible.

In the fixed position mode not all marker actions are available.

Visualization

For visualization of markers, a marker column is available to you on the left side of the Trace Output Window:

Symbol

Meaning

No marker available.

User-defined marker available.

If you move the mouse over this symbol, the comment is shown as tooltip.

System comment available.

System comments can't be edited.

User-defined marker and system comment available.

Export

Any markers entered but not hided will be exported during the export (ASCII and BLF). After the export from the Trace Window and subsequent import or loading as an offline source, the markers entered will be available again.
In the Export dialog you can select in the advanced settings, if you want to export all markers of all windows of the current configuration.
