Status Information for MOST Messages

 

Status information for control and AMS messages is displayed in the State column (in the Trace Window) and can also be called up by means of the MOST_State CAPL selector of mostMessage, mostAMSMessage or mostRawMessage variables.

Bit

Description

Message
type

1: Status information applies to the AMS messages (bits 0,1,4,5,6,8 and 9 are relevant)

0: Status information applies to the Control message (bits 0, 1, 4, 5 and 6 are relevant)

AMSExt
Error

1: A timeout occurred while waiting for a Tx receipt for a control frame of the AMS message (AMSExtErr).

0: No timeout occured.

AMSSeg
Error

1: The AMS protocol was violated. An error message was sent to the sender (cf. MOST Specification).

0: The AMS protocol was not violated.

Tx Result

Transmission result:

1: OK. In this case the Ack bit is set too, however, the NAck bit is not set.

0: Transmission error (TxF).

NAck

1: The recipient did not receive the message and acknowledged it negatively. In case of group/broadcast addressing, at least one recipient did not receive the message and acknowledged it negatively. Negative acknowledgment means that either the receive buffer is full or a CRC error was detected (see AckNack column).

0: Message receipt was not negatively acknowledged. This may be due to the following causes:

  • Message was received. If this is the case, the Ack bit is set.
  • Dir = Tx: Messages whose recipient address is not present in the ring. In this case, the Ack bit is also not set.

Ack

1: The recipient received the message and acknowledged it positively. In case of group/broadcast addressing, at least one recipient received the message and acknowledged it positively.

0: Receipt of the message was not positively acknowledged. This can occur with messages with Dir = TX, whose recipient address is not present in the ring. In this case, the NAck bit is also not set.

Unlock

1: Unlock event during transmission (Unl).

0: No unlock event during transmission.

Bus active

1: Bus active.

0: Bus inactive (NAc).
