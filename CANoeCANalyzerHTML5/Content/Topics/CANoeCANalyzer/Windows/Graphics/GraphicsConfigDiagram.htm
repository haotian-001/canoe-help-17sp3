Display Settings

  • Show/hide grid
  • Show/hide marker bar
    If you hide the marker bar, set markers will not be deleted.
  • Show symbolic values
    Select this option if the labeling of the Y-axis should be taken from the symbol's defined value table.
  • Show Screen Tip of measurement cursor
    If this option is activated, the measurement time, the symbol name and the absolute symbol value of this measurement point are displayed in a tooltip, if the measurement cursor is displayed and moved with the mouse.
  • Configure the tooltip length.
  • Show/hide measurement point (sample) drawing. The color of measurement points corresponds to the preset symbol color.
  • Mark symbols bold
  • Set display mode for symbols
  • Show all graphs with color
  • Gray unmarked graphs
  • Show only marked graphs

Color Settings

Background Color

You can choose among the classical Black (scope) and the modern White.

  • Gray/White sets the symbols legend’s background color to gray and the window’s background color to white.
  • Gray/Black sets the symbols legend’s background color to gray and the window’s background color to black.

Grid Line Color

You can chose separate grid colors for the two different background colors.

Note

The color settings affects all Graphics Windows of the configuration.

Font

With the [Font...] button you open the Options dialog. Here you can set the font for the diagram.
