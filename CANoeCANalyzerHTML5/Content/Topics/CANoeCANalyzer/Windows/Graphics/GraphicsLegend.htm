Graphics Window: Legend

 

You can open and close the legend via in the toolbar or directly with the splitter .

You can set the position of the legend within the window via the toolbar.

Functions of the Legend

Via the context menu you can

  • add symbols
  • change symbol values
  • select the line and sample type
  • select the display mode of the symbols
  • delete symbols
  • copy symbols to the clipboard
  • select the display of the Y-axis
  • create a common axis/group for symbols
  • set the selected symbol as x-axis

You can also add symbols via drag and drop.

Via the column view you can

  • select and mark symbols
  • change the symbol color
  • change the symbol name for the diagram display

In the configuration dialog of the Graphics Window you can make different settings for the axis (x-axis, y-axis, common axis).

Columns

In the list of the legend all symbols are displayed that are selected with drag and drop or via the context menu.

  • In the first column you can select the symbols that should be displayed in the diagram.
  • In the second column you can select the color of the symbol.
  • In the Name column you can enter a short descriptive symbol name. This is then also displayed in the Graphics Window's Configuration dialog. The display considers the global settings from the Options dialog

With the context menu command Column Chooser of the table head you open the Customization dialog. You can add the following columns to the legend via drag and drop.

Column

Description

Bin

Binary display of the current measurement value

dy

Value difference (when using difference cursor)

Hex

Hexadecimal display of the current measurement value

Max

Maximum measurement value

Min

Minimum measurement value

More Object Info

Fully qualified name of the symbol

Status

Security status for SecOC

x-Offset

This is the time interval in seconds by which the symbol is shifted in the display. A symbol may only be shifted to the right (positive time). By specifying a X-Offset multiple symbols with different start times can be compared directly with one another.

y

Current measurement value

y[sel]-y

Value differences between the selected symbol and symbols of the same type (when using measurement or difference cursor)
