Graphics Window: Diagram

 

In the diagram the curve of a »symbol is displayed graphically.

Functions of the Diagram

Via the context menu you can

  • add symbols
  • add measurement cursor/difference cursors
  • search for values or minimum and maximum of symbols
  • set specific time range
  • set markers
  • log a symbol curve
  • import /export a symbol curve
  • copy a screenshot of the Graphics Window to the clipboard
  • delete measurement values from the Graphics Window
  • synchronize data in analysis windows
  • display measurement data statistically
  • switch the Graphics Window on/off in the Measurement Setup

You can also add »symbols via drag and drop.

Via the toolbar you can

  • add measurement cursor/difference cursors
  • select the display of selected symbols
  • set, if you want to work with one or all selected symbols
  • select the axis mode
  • zoom in and out in the diagram
  • fit curves in the diagram
  • save and load the time period and the display value range
  • move the curves
  • scroll in the diagram

In the configuration dialog of the Graphics Window you can make different settings for the axis (x-axis, y-axis, common axis).
