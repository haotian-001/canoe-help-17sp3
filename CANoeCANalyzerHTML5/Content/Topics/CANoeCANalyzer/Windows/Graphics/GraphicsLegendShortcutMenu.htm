Common Axis/Group
Depending on the symbol selection inside the legend, one of the following commands is available in the context menu:

  • Create Common Axis/Group
    Creates a new common axis/group if you mark at least two symbols in the legend.
    The New Common Axis/Group dialog will be opened to enter the name of the axis or group. After closing the dialog with [OK] the marked symbols are grouped below the common axis/group.
    You can edit the common axis/group in the Graphic Configuration dialog.
  • Remove Common Axis/Group
    If you mark a common axis/group, you can remove the complete common axis/group with this menu command. The contained symbols will be moved to the top of the legend.
  • Remove from Common Axis/Group
    Removes selected symbols from a common axis/group. They are moved to other symbols that are not part of a common axis.
    The common axis will be removed completely if it contains less than two symbols afterwards.
