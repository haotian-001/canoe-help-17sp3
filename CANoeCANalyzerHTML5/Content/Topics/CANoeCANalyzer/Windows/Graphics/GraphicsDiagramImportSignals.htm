Graphics Window: Symbol Assignment in Import

Context Menu Path of the Diagram: Import/Export|Import...

In the Import dialog, you can import measurement series of symbols and assign them to a symbol.

For Source file, choose the file that contains your measurement series. The signal-based formats (MDF, MF4) and different message-based formats (e.g. ASC, BLF, MF4, PCAP) are available for the source file.
You can also move a file to the Graphics Window using drag and drop. The Import dialog is automatically opened.

Note

  • The import of large unsorted files can take a very long time.
    A dialog opens automatically when such files are imported where you can specify whether you want to sort the data before importing it.
    If you select [Yes], your data will be overwritten.
    If you select [No], the data will be sorted only temporarily for the import.
    You can activate and deactivate the display of the dialog in the Options dialog.
  • The maximum number of symbols must not be exceeded during the import. See Hint in the dialog.

After the extraction of the symbol information of the imported file, the dialog is initialized with the Default Mapping:

  • If the imported file contains symbols already configured in the Graphics Window, the action under Signal Information is set automatically to Replace.
  • If the imported file contains symbols that are not yet configured in the Graphics Window, the action under Signal Information is set automatically to Add.

Note

In State you are shown how many symbols have been extracted and how many of these are already configured in the Graphics Window.

Signal Information

All extracted symbols are shown in the table.

In the Import column, you have the option of selecting the measurement series of symbols to be read in. All symbols are selected initially by default.

In the Operation column, you have the following options:

  • Replace

You can use this action to overwrite measurement series of symbols already configured in the Graphics Window.

In the Existing / New Signal column, you can select which symbol sequence is to be overwritten from a list of all symbols configured in the Graphics Window.

Note

  • When replacing symbols the connection to the database remains active. Raw values read from the MDF file will be converted to physical values via the database.
  • When replacing symbols a comparison with the database is executed.
  • The replacement may result in loss of existing data of the symbol.
  • You can use each existing symbol for a maximum of one measurement series.
  • Add

You can use this action to add a newly defined symbol in the Graphics Window.

In the Existing / New Signal column, you can adapt the name of the new user-defined symbol. If you select the Add action for a previously configured symbol in the Graphics Window, a suffix will be automatically added to the existing name.

Note

When adding symbols no comparison with the database is executed. Raw values read from the MDF file will be converted to physical values via the MDF file.

With the Select / deselect all option, you can quickly select all symbols for the import or cancel the selection of the symbols.

With the [Replace All] button, you can select that all symbols already configured in the Graphics Window will be replaced. New symbols are unaffected by this.

With the [Add All] button, you can add all symbols as new symbols. For symbols already configured in the Graphics Window, a suffix will be automatically added to the name.
