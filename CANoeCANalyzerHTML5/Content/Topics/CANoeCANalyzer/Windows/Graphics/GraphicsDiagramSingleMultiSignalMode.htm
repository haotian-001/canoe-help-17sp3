Graphics Window: Single-Graph Mode/All-Graphs Mode

 

The following functions are available via the toolbar:

Symbol

Function

Single Graph

All Graphs

Over a longer period of time, you can work with one selected symbol or with all symbols. To do this, use the single-graph mode/all-graphs mode. Depending on the selected mode, commands such as Zoom In, Zoom Out , Get Scalings effect the selected symbols or all the symbols in the Graphics Window.

Select All Graphs to work with all symbols. All symbols in the legend are automatically selected. 

Select Single Graph to work with a particular symbol. Click on the symbol in the legend or diagram using the mouse to select the symbol.

In single-graph mode, you can work with several selected symbols. To do this, select the symbols in the legend or in the diagram. Selecting multiple symbols this way is helpful to quickly complete a few actions; cancel it by clicking anywhere in the Graphics Window.

Note

  • At any given time, either Single-graph mode or all-graphs mode will be active.
  • Disabled symbols are not included in the measuring functions Zoom In, Zoom Out, etc.
  • Operations which change time axis scaling are always executed for all symbols (independent of the option single-graph/all-graphs mode), since there is only one time axis for all symbols in each Graphics Window.
