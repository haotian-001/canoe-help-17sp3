Graphics Window: Configuration - Common Axis/Group

Context Menu Path: Configuration...|Signals/Axes
Context Menu Path of the Legend: Create Common Axis/Group

In the Graphics Window, you can group several symbols by assigning them to a common Y-axis or group in the.

Create a common axis for symbols using the same unit. Matches and deviations between curves of the same type can be determined. The measurement functions such as Zoom and Fit work together for all curves, so that the curves stay comparable in the view.

Create a group to enable or disable all symbols of the group at once.

Create/remove a Common Axis/Group

Common axis/groups can be created, edited and deleted in the Graphic Configuration dialog or the legend.

To create a common axis/group, mark several symbols in the legend and select the Create Common Axis/Group context menu. In the New Common Axis/Group dialog you can select whether you want to create a axis or a group and give it a name.

The common axis/group is visualized by a colored bar with the axis/group name.
Via drag and drop you can move symbols in and out of a common axis/group.
You can also remove symbols via the context menu form the common axis/group.

In the Graphic Configuration dialog you can change a common axis to a group and vice versa via the context menu of the axis/group. Here, you can also remove a axis/group.

In the Graphic Configuration dialog, you can use the context menu of the axis/group to change a group to an axis and vice versa or to remove it.
