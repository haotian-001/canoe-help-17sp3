Graphics Window: Search

Context Menu of the Diagram: Search...

You can search for values as well as the Minimum and Maximum of symbols in the Graphics Window.

Symbol Name

Shown in the selection list are the names of all symbols that are displayed in the diagram.
Disabled or hidden symbols do not appear in the selection list.

Default value: The symbol name that is selected in the legend

Relation

The following relations are possible:

  • =
  • ca.
  • hits
  • Min, Max
  • <
  • >
  • passes (up)
  • passes (down)

Minimum and Maximum are global values referenced to the Graphics Window's data buffer. When selecting Min and Max the Value and Format fields cannot be edited. Hits searches for points where the curve passes the given value even if no measure point has exactly this given value. Passes up and Passes down are similar but they only find points, where the curve passes the value from below (Passes up) and from above (Passes down). Ca. searches for points where the measurement value approximately equals the searched value. The tolerance is 1% (one percent) of the current scaling of the curve in the diagram.
Default value: hits

Value

This field is used to enter the value to be searched. This field is not available for the relations Min and Max.
Default value: 0

Format

This field indicates the value format. This filed is available if symbolic values are provided for the selected symbol. The following values are available for selection:

  • Phys for physical value
  • Symb. for symbolic selection (only if a value table has been defined for the symbol)
  • Raw(dec) for the raw value as a decimal value
  • Raw(hex) for the raw value as a hexadecimal value

Default value: Phys or Raw(dec)

Buttons

[Find Next]

With [Find Next] the value is searched in the Graphics Window, and the measurement cursor is positioned at the value found in the Graphics Window.

Activated difference cursors are shown to the left and right of the search value. The display of the local Minimum/Maximum is updated.

If [Find Next] is activated again, the search resumes for the next possible value in the Graphics Window.

If the search value does not exist in the entire data buffer, an error message appears:

"Search terminated in the Graphics Window. Unable to find this value."

[Cancel]

The dialog is closed by [Cancel]. However any search process that is still running is not aborted.

[Expand]

With [Expand] you have access to further option for the search. Here you can define

  • whether you want to search forwards or backwards
  • where you want to start the search: at the beginning (at 0 s), at the end, at the measurement cursor or at the border of the currently visible range
  • which data shall be searched: all stored data, only data of the currently visible range, data between the difference cursors or data of a predefined time range (you can define this time range).

Note on Editing Time

The time is displayed in days, hours, minutes and seconds.

  • XX d XX h XX min XX.XXXX s
  • XX d — only days
  • XX d XX h — days + hours
  • X.XXXX — seconds + milliseconds
  • X:XX.XXXX — minutes + seconds + milliseconds
  • X:XX:XX.XXXX — hours + minutes + seconds + milliseconds
  • XX:XX:XX:XX.XXXX — days+ hours + minutes + seconds + milliseconds

Even if the start time of the search is outside the defined time range, the search will only be done in the defined time range.

Note

  • Synchronization with the Graphics Window
    If the Graphics Window function Synchronize views is activated, the Trace Window display jumps to the desired values in synchrony with the Graphics Window display.
  • Global Minima/Maxima are always searched, even if the difference cursors are shown and result in local Minimum/Maximum being displayed in the legend.
