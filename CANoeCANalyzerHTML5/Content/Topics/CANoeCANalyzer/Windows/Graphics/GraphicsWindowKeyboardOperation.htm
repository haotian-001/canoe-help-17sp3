Graphics Window: Keyboard Operation

 

Copying, cutting and pasting of symbols can also be performed with the help of the following windows-conformant keyboard operations:

  • <Ctrl>+<c> for copying,
  • <Ctrl>+<x> for cutting,
  • <Ctrl>+<v> for pasting.
