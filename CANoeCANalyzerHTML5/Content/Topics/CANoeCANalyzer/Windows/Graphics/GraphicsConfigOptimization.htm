Optimization

 

Local Optimization

Here you can optimize the following measurement parameters of the Graphics Window.

User-Defined Refresh

Here you specify how often the Graphics Window display should be updated.

If you do not activate the option box, the program automatically sets a favorable value.

You should only activate this option box if your measurement configuration places special demands on the Graphics Window (high busload, simultaneous display of many symbols, etc.).

  • Small values result in a nearly continuous display of the curve, but on the other hand they require a lot of computing power, and with slow computers this could lead to performance problems.
  • High refresh values reduce the computational demand substantially, especially when many symbols are being displayed. However they result in a less smooth display of the curve.
  • As long as no load problems occur during the measurement you should not change these options.

Accuracy

The display accuracy setting is used to optimize the Graphics Window display. Using Pixel or Peak settings a delayed display can be avoided by a high amount of data.

Display accuracy: Pixel

If multiple measurement points fall within the time range of one screen pixel, only those measurement points at the ends of this range are displayed. This leads to faster output when there are many measurements in a small space. However, individual peaks in a curve might not be displayed if the measurement point with the extreme value lies within this range.

Display accuracy: Peak

If several measurement points lie within the time range of one screen pixel, at the most four of these are displayed:

  • the first
  • the last
  • the maximum
  • the minimum

This action is not as fast as would be the case with Pixel display accuracy, but much faster than with Full display accuracy, and all peaks are normally evident. Some peaks may still not be displayed if another stronger peak exists within the range.

Display accuracy: Full

All measurement points are output, even if they lie within the time range of the same screen pixel at the active scaling. When there are many measurement points in a small space, this leads to slower output speed, but all extreme values of the curve will be displayed.

Note

This option does not affect measurement value acquisition in any manner.

Note

These settings are not available in graphic mode XY display.

The title of the Graphics Window/configuration dialog indicates whether the XY display is activated.
