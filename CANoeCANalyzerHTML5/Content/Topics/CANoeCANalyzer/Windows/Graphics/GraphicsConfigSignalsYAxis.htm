Scaling

In the symbol list all in the Graphics Window configured symbols are displayed. The symbol name, the color of the curve and the activity state are identical to the display of the legend in the Graphics Window.

Columns of the Symbol List

  • In the first column you can select the symbols that should be displayed in the diagram.
    If the XY display is selected as the graphic mode, the symbol that was selected as the X-axis is marked here ().
  • In the second column you can select the color of the symbol.
  • In the Name input box you enter a short descriptive symbol name. This is then also displayed in the Graphics Window's legend. The display considers the global settings from the Options dialog
  • You define the displayed symbol value range by entering a Minimum and Maximum.
  • In the Y-Axis Caption column you enter a short descriptive name for the y-axis.

Via the context menu you can

  • reset the scaling to the default
  • make the scaling equal

Common Axis/Group

You can assign several symbols to a common Y-axis/group in the Graphics Window.
Matches and deviations between curves of the same type can be determined. The measurement functions such as Zoom and Fit work together for all symbol curves, so that the curves stay comparable in the view.

Options

Incoming measured values can exceed the current Y-axis scaling. To adapt automatically the complete curve or the value margin area, the function Expand Y-range by ... %, if the current range is too small can be activated.
