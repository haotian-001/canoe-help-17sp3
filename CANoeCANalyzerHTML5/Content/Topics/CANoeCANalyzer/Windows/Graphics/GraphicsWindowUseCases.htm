Graphics Window: Use Cases

 

Configure fewer signals in the Graphics Window

Task: Few signals should be displayed in the Graphics Window. The signals are defined in a database.

From the Add Signalsâ¦ context menu, insert two signals from the database into the Graphics Window. The signals are displayed in the legend. The associated Y-axis appears in the diagram, labeled with the signal name. The Y-axis value ranges are taken from the database.

To change the Y value range of the signals, select the Configurationâ¦ context menu and select the Symbols/Axes page. Determine the Y value range using the Minimum and Maximum columns.

To display all Y-axes, set the Y-axis view to All Y-axes using the toolbar.

The Y-axis of a signal is displayed on the right side if you select the signal in the legend and open the Y-axis left/right context menu.

To save space, you can hide the legend via the toolbar.

Configure and evaluate signals after measurement

Task: You would like to evaluate the signals of a message in the Graphics Window after a measurement.

Select the message in the Trace Window and drag it into the Graphics Window. This displays all the message signals in the legend as well as their curve in the Graphics Window diagram.

To display all curves completely in the diagram display area, select the Fit all graphs function in the toolbar.
If you prefer to display all curves with their associated Y-axis one on top of the other, select the Y-axis view Separate Views using the toolbar.

To display the curve measurement values, switch the measurement cursor on via the toolbar. In addition to the measurement cursor, a [y] column appears in the legend, which displays the measurement value at the position of the measurement cursor for every curve. The time of the measurement cursor signal is displayed in the time field.
Depending on the single-/multi-signal mode, you can move the measurement cursor either along a curve or move the cursor back and forth between the curves. In single-signal mode, the selected signal in the legend determines the selected curve.

If you would like to display the value difference between two curve times, switch the difference cursor on via the toolbar. In addition to the difference cursors, two columns ([y] and [dy]) appear in the legend. Position the difference cursors at the desired measurement points. The reference value [y] and the difference [dy] between the two reference values and the second value are displayed.

A subset of signals can be concisely evaluated using the Show only marked graphs mode because all unmarked signals are hidden. You can change the selection of displayed signals by marking the desired signals in the legend.

Note

You can set the measurement cursor/difference cursors with the left mouse button. The measurement cursor/difference cursors that are next to the mouse cursor are set.

Evaluate signals  during measurement

Task: You would like to evaluate curves during the measurement.

To display measurement values during measurement, you must select the y, Min and Max measurement columns from the Column Chooser.

Start the measurement. The current measurement values for the individual signals are displayed in the [y] column. The Min and Max columns display the minimum and maximum value of the total measurement value for the respective signal.

To more closely analyze a curve during the measurement, use the Pause function in the diagram or toolbar context menu. The continuous scrolling of the Graphics Window is stopped without stopping the measurement. Future measurement values continue to be written in the signal buffer in the background. All measurement values up to the time you selected pause are available for evaluation.

Zoom in, if necessary, on the section of the curve using the toolbar icon . If you would like to scroll back to the beginning of the measurement in one step, select the toolbar icon Go to Start . You can also scroll to the start gradually by selecting Scroll Left.

You can set up a desired section of the diagram in the Go to Xâ¦ context menu. In the dialog, enter the start and endpoint under Time Period, which limits the section of the diagram.

To display measurement values during the pause, switch the measurement cursor on via the toolbar. You will find a more detailed description of the measurement cursor in the example above.

Use the Searchâ¦ function to determine, among other parameters, whether a certain value will cycle through. To do this, open the dialog using the Searchâ¦ context menu. Select the signal that you would like to evaluate. Under Relation, select hits and enter the value to be cycled through. Select Search next to jump the measurement cursor to each position at which the value is cycled through. Under Expanded options you can define the following:

  • Search Direction: Defines whether the search should be conducted forward or backward.
  • Search Starting Point: Defines where the search should start
  • Search Area: Defines whether all signals or only one certain range should be searched.

Note

You can also activate the Pause function by zooming in on a section of the diagram. To do this, open a rectangle in the diagram with the left mouse button. Make sure that the X or XY axis mode is set.

Determination of matches and deviations between curves of the same type

Task: Matches and deviations between several curves of the same type should be determined. The measurement functions such as Zoom and Fit should work together for all curves, so that the curves stay comparable in the view.

You have the option of assigning / grouping several signals in the Graphics Window to a common Y-axis.

Mark the desired signals of the same type in the Graphic Configuration dialog and select the Create Common Axis/group context menu. All signals are then assigned to a common Y-axis. The signal group is labeled in the legend of the Graphics Window with a vertical color bar at the end of the name column.

The measurement functions such as Zoom and Fit work together for all curves.
