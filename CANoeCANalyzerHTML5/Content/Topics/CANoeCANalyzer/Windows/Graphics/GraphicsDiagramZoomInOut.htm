Graphics Window: Zoom In/Zoom Out

 

Depending on the single graph/all graphs mode and the axis mode, the marked symbols or all symbols are zoomed in/zoomed out. For example, if you select Zoom Out, the displayed Y-axis and/or X-axis value range will be enlarged.

The following zoom functions are available via the toolbar:

Icon

Function

Zoom in

Zoom out

Zoom in until no compression

Zoom in until no compression zooms in until the displayed values no longer exceed the buffer limit. The display then shows only uncompressed symbols.

Mouse

Activate the Drag Zoom mode to zoom into a segment with the left mouse button. The single graph/all graphs mode and XY mode settings are retained when you use this function.

Deactivate the Drag Zoom mode to zoom with your mouse wheel. The single/multiple signal and XY mode settings are retained when you use this function.

Cursor

Mouse Position

Function

X axis

Zooms all symbols in X direction.

Y axis

Zooms all symbols of the axis in Y direction.

in the diagram

Zooms dependent on the single graph/all graphs mode and XY mode settings.

Note

  • The commands that change the scaling of the time axis always apply to all symbols (regardless of the settings in the single graph/all graphs mode) because there is only one time axis for all symbols in the Graphics Window.
  • Hold the button down to automatically repeat the function.
  • The axis scaling can also be set separately for each symbol in the Graphic Configuration dialog.
  • It is scaled around the average value of visible measuring points.
