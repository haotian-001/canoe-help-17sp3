Graphics Window: Fit Symbols

 

The following functions are available via the toolbar:

Symbol

Description

Considered Symbols

Scale Direction

All

Marked

X

Y

Fit all graphs

Independent of the preset axis mode, the symbols are scaled so that they are completely visible. This involves determining the actual Min/Max values of each symbol as well as the time range for all symbols and setting the scaling accordingly.

•

—

•

•

Fit all graphs Y

Symbol scaling in the diagram is set up so that all symbol values in the visible time period are displayed. The actual minimum and maximum values of each symbol are defined in the visible time period.

Fit marked graphs

This command is identical with command Fit all graphs but considered only marked symbols.

Fit marked graphs Y

This command is identical with command Fit all graphs Y but considered only marked symbols.

Fit all graphs to special line

All symbols in the diagram are arranged one on top of the other and completely displayed. The actual minimum and maximum values of each symbol as well as the time period of all symbols are defined.

•

•

•

Fit all graphs to special line Y

All symbols in the diagram are arranged one on top of the other and all symbol values in the visible time period displayed. The actual minimum and maximum values of each symbol are defined in the visible time period.
