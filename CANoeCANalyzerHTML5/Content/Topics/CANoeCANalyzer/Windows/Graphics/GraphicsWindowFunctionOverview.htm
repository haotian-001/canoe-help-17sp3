Graphics Window: Function Overview

 

The following functions are available in the Graphics Window:

Configure Symbols
  • You can insert or remove Graphics Window symbols using drag and drop or using the clipboard.
  • Symbols can be inserted directly into the Graphics Window using the Add Symbol... context menu command. You can also insert MOST signals, system variables and diagnostic parameters.
  • Enter the symbol selection using the Configuration... context menu. On the Symbol/Axes page you define which symbols should be displayed in the Graphics Window.
  • Define the Y value range of the symbols in the Graphic Configuration dialog.
  • You can assign several symbols to a common Y-axis in the Graphics Window.
    Matches and deviations between curves of the same type can be determined. The measurement functions such as Zoom and Fit work together for all curves, so that the curves stay comparable in the view.
  • You can insert all or any symbols in a series of measurements (MDF file, signal-based, MDF version 2.0â4.1) into the Graphics Window using Import.

Configure the Graphics Window Display
  • Y-axis view in the diagram
    Depending on the number of symbols and duration of the measurement, you can fit the Y-axis view in the Graphic Configuration dialog.
    If only viewing a few symbols, you can display the Y-axis of all symbols next to each other. If there are many symbols in the Graphics Window, it is often best to display only the Y-axis of the selected symbol. Another option is to arrange the symbols with their Y-axes in separate diagrams.
  • Display Y-axis left/right
    You can display the Y-axis left/right in the Graphics Window. Particularly when viewing only two configured symbols, a nice layout results if one axis is displayed left and the other right.
  • Common Axis
    You can assign several symbols to a common Y-axis in the Graphics Window.
    Matches and deviations between curves of the same type can be determined. The measurement functions such as Zoom and Fit work together for all curves, so that the curves stay comparable in the view.
  • Groups
    You can create a group to enable and disable all symbols of the group at once.
  • Display symbolic values on the Y-axis division
    Defined discrete values in the database value tables can be displayed on the Y-axis and in the legend if you select the Show symbolic values option in the Graphic Configuration dialog.
  • XY display
    You can configure the time or a symbol as the X-axis. In most cases the display is configured to show a variable or symbol value with respect to time. For frequency analyses or similar evaluations, the selection of an symbol as the X-axis is helpful.
  • Display the symbol names in the diagram
    You can display the symbol names in the diagram to more easily and quickly associate symbol names and curves. Select between the symbol name display on the upper edge or on the Y-axis. Select the display in the Graphic Configuration dialog.
  • Symbol value representation in the legend
    You can display symbol values in the legend in the Binary representation and/or in the Hexadecimal representation.
  • Automatic Y-axis scaling adjustment
    The Y-axis scaling can be adjusted automatically if incoming measurement data cross or remain the current Y-axis scaling.

Measurement and Evaluation
  • Stop the Graphics Window during the measurement
    You can stop the continuous scrolling of the Graphics Window during long measurement processes using Pause, review a section of the measurement in any time resolution and analyze it using the measurement functions.
  • Logging of Graphics Window symbols during measurement
    To log symbols from the Graphics Window manually or automatically during measurement, you can use the context menu Logging Configuration...The symbols are extracted from messages and binary stored in signal-based MDF files.
  • Measurement cursor/difference cursors to evaluate curves
    After the measurement ends or in Pause mode, you can execute an absolute or relative measurement analysis using the Measurement Cursor or Difference Cursors.
  • Measurement columns in the Graphics Window legend
    You can display global and local minimums and maximums in the legend.
    If you would like to display the Y differences between symbols of the same types, select the Value Differences column in the legend.
  • Search Function to locate measurement values
    You can search for any value as well as the minimum and maximum of symbols in the Graphics Window.
  • Set the displayed time period
    If you would like to display a certain time period or a certain point in time in the diagram, call up the Go to X... function.
  • Scroll Functions
    When analyzing longer series of measurements, you can scroll to the oldest or newest measurement value with a click of the mouse.
  • Saving curves
    You can save curves charted in the Graphics Window to a file using Export. You can select whether you want to save the entire curve or only the displayed section.
  • Statistics Calculation
    Statistics data can be calculated from Graphics Window symbols, see Statistics... menu of the diagram.
  • Synchronizing the Graphics Window and the Trace Window
    If the Synchronize Window function in the Graphics Window is active for an active measurement cursor, the Trace Window display follows the Graphics Window display to the desired values.
  • Save and load symbol value ranges/Load default values
    You can save the Y-axis scaling displayed in the Graphics Window for all symbols. To do this, use the Store scalings function. After long zoom actions you can load the saved settings with Get scalings.
    You can set the symbol value ranges defined in the database in the Graphics Window with the Default all Y function.
  • Graphics Window toolbar
    The Graphics Window toolbar provides many functions.

Managing Many Symbols
  • Hide symbols
    Show only marked graphs     mode provides the option to briefly display and analyze only a few selected curves. Hidden curves are not displayed in the Graphics Window, i.e., not charted.
  • Symbols in the foreground
    Use the Gray unmarked graphs mode to bring selected curves briefly to the foreground for an analysis. Unselected symbols are shaded gray in the background.
  • Highlight symbols
    You can highlight individual symbols, in order to more easily follow the course of a measurement. Select the Marked graphs bold function. The lines of the selected curves are shown in bold.
  • Scaling functions
    The toolbar provides a number of scaling functions. For example, you can fit only the Y value range of the selected symbols in the displayed window of time for your series of measurements without modifying the Y value range of unselected symbols.
