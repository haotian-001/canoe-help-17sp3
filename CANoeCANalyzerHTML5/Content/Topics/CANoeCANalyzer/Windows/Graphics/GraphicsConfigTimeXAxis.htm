Scaling

  • Start and Length define the time range that will be displayed.
  • Display mode specifies the how the time is displayed in the Start and Length fields.
  • Measurement time [s]
    The time is displayed in seconds.

  • Measurement time [d hh:mm:ss]
    The time is displayed in days, hours, minutes and seconds.

    Note on Editing Time

    The time is displayed in days, hours, minutes and seconds.

    • XX d XX h XX min XX.XXXX s
    • XX d — only days
    • XX d XX h — days + hours
    • X.XXXX — seconds + milliseconds
    • X:XX.XXXX — minutes + seconds + milliseconds
    • X:XX:XX.XXXX — hours + minutes + seconds + milliseconds
    • XX:XX:XX:XX.XXXX — days+ hours + minutes + seconds + milliseconds
  • Date and time
    The time is displayed in days, hours, minutes and seconds.
    The start date can be selected in a date/time picker.

On Measurement Start

  • If you select the Set time axis to: n seconds option, at the measurement start the Graphics Window is set to a time range set by you (n seconds).
  • If you select the Get stored y scalings option, at the measurement start the Graphics Window is set to the Y-axis scaling of the symbols that you have previously defined it with Store scalings.
  • If you select no option, the last view settings of the Graphics Window are accepted.

Note

These settings are not available in graphic mode XY display.

The title of the Graphics Window/configuration dialog indicates whether the XY display is activated.

On Measurement Stop

  • If the Fit time axis option is activated, at the end of the measurement the curves are shown for the entire time range of the measurement duration.
  • If the Restrict data to: n seconds option is activated, at the end of measurement all data that is older than your selected time will be deleted. So it is easy for you to analyze data of your interest.

Scroll Mode

If the curves reach the right border of the Graphics Window sometime after the measurement start, the visible area (viewing area) automatically shifts with the curves, whereby the time axis is shifted to the left. You can configure this behavior referred to as Scrolling in the Graphics Window. There are two scroll modes.

  • The Autoscroll mode is a continuous scrolling mode. Each time the time axis is shifted to the left it is only shifted a minimal amount, creating the impression of continuously flowing curves.
  • To have the time axis move in jumps, thereby freezing the graphic diagram between these jumps, you should activate the option Shift graph ...%, when time axis is greater than ... seconds. You can configure the percentage of the displayed time interval by which the time axis should be shifted. The smaller the value the more uniformly the image will be scrolled, but the image will be shifted more frequently.
    If the time interval shown in the Graphics Window is less than the value you specified, the entire Graphics Window is redrawn, i.e. the image is shifted 100%. This prevents the Graphics Window from consuming too much computing power due to frequent scrolling with very small time intervals.

Note

With smaller time intervals the scrolling behavior becomes more and more computing intensive, since scrolling must occur more quickly. Therefore, there is a minimum refresh cycle for updating the window contents. If the displayed time interval lies within the magnitude of this cycle the curves are increasingly displayed again in jumps since the window contents cannot always be updated. You can set the minimum refresh cycle in the project file CAN.INI. To do this, set the following value in the [System] section:

GraphicWindowMinAutoRefreshrate = <Cycle time in ms>

Please note, however, that the smaller the value you select the greater the system load grows. Before changing the automatic refresh cycle it is therefore essential that you determine an optimum refresh rate with the help of the User-defined refresh function.

Note

These settings are not available in graphic mode XY display.

The title of the Graphics Window/configuration dialog indicates whether the XY display is activated.
