Y-Axis Display

With the Show axis option you can switch the display of the axis on and off.

You have the option of displaying one or more Y-axes. You can choose between:

  • Toggle Y-axes
    Only the selected symbol's Y-axis is shown.
  • All Y-axes
    Each symbol is shown together with its associated Y-axis.

Sorting of axes from right to left corresponds to sorting of symbols in the legend from top to bottom. The axis of the symbol selected in the legend is located on the far right in the diagram. Any changes made to the symbol layout will result in changes to the sequence of Y-axes.

In this view selected axes can be shown on the right side of the diagram. To toggle the position of the axes just mark the symbols in the legend and chose Y-axis left/right from the context menu.

  • Separate views
    All symbols are shown above one another with separate Y-axes.

To reset the height of the separate views to the initial height you have to select/mark all symbols in the legend and select from the toolbar.
You can also reset the height of a single symbol and of selected symbols. In this case the heights of the non-marked symbols are adjusted so that their total height uses the same space as their individual initial height while the ratio among each other remains.

You can also switch the Y-axis view type from the toolbar.

Note

  • No Y-axis is shown for disabled or hidden symbols.
  • The function Fit all graphs to a special line is not supported under Separate views.

Y-Axis Caption

The symbol names are displayed in the diagram. You can select whether

  • None display of the symbol name
  • The symbol name is displayed next to the axes
  • The symbol name is displayed above the diagram
