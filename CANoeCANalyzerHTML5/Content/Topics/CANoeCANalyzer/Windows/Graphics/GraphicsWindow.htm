Graphics Window

Ribbon|Analysis Tab|Graphics|View / New

General

In the Graphics Window symbols (signals, variables, and diagnostic parameters) are displayed graphically in an XY diagram. You can configure the time or any symbol as the X-axis.

Various functions are available for highlighting/hiding curves and their measurement points.

Many functions are available to measure and evaluate the curves. You can read measurement point values using the measurement cursor or difference cursors. The curve section can be changed using scaling and scrolling functions.

Measurement columns are available in the legend to display measurement point values according to certain evaluation criteria.
