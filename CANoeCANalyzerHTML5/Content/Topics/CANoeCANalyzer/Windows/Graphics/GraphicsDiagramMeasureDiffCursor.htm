Graphics Window: Measurement Cursor/Difference Cursor

Context Menu Path of the Diagram: Cursors|No Cursor/Measurement Cursor/Difference Cursor

The functions are also available via the toolbar:

Symbol

Function

Measurement cursor

Difference cursor

Execute absolute and relative measured value analysis with the Measurement Cursor and/or the Difference Cursors.

With No Cursor already set measurement cursors will be deleted.

Symbol values of measurement points are displayed in the Graphics Window legend (y column) and in the tooltip of the measure cursor(when moving the mouse) by activating the measurement cursor/difference cursors. In addition, the value different (dy column) between the two measurement points is displayed for the difference cursors.

The measurement cursor and the difference cursors are displayed as vertical lines.

When the measurement cursor is switched on, the measurement time, symbol name and absolute symbol value of this measurement point is displayed in the legend and in the tooltip of the measure cursor when moving the mouse. When the difference cursors are switched on, the second measurement time and the difference time with regard to the second measurement point is also displayed. The value difference (dy) is displayed for all symbols.

Positioning the Measurement Cursor/Difference Cursors

  • If the mouse pointer is moved over a measurement cursor/difference cursors, it changes to a horizontal double arrow. Click and hold the left mouse button to position the measurement cursor/difference cursors. While the button is clicked, a small square appears which indicates the next base point for the measurement cursor/difference cursor.
  • You have the option to set the activated measurement cursor or blue difference cursor with the left mouse button. This allows you to easily set measurement or difference cursor from the scrolled display area.
    The red difference cursor can be set with <Ctrl>+ left mouse button.

Note

  • The measurement cursor/difference cursors follow the mode Single Graph/All Graphs. In Single Graph mode, the small box only branches at the measurement points of all marked symbols; in All Graphs mode, the small box branches at the next measurement point for all symbols.
  • The measurement cursor is required to display the following values in the legend.
  • Value differences, y[sel]-y column
  • Binary data, Bin column
  • Hexadecimal data, Hex column
  • The minimum/maximum of all measurement data is displayed in the Minimum/Maximum column when the measurement cursor is activated. The minimum/maximum is displayed within the difference cursors when the difference cursors are activated.
  • If you forget to activate the mouse pointer over the measurement mark/difference mark, a rectangle is opened once the pointer is dragged. The contents of the rectangle are magnified (zoomed) when the mouse button is released.
  • If the measurement cursor/difference cursors are switched on, they are displayed at the last position selected, if this is in the display window. Otherwise, they are moved to the display area.
  • The bars can only be positioned in the display area.  However, the display area can be moved with the arrow keys.
