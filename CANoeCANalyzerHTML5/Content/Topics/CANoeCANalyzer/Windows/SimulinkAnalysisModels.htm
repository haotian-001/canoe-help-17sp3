Simulink Analysis Models

Ribbon: Environment Tab|Analysis Models

In this dialog, you can configure Simulink models in such a way that you can perform evaluations during the measurement.

Note

You can only access this dialog via the Environment Ribbon Tab if a Simulink analysis model already exists. You can configure or assign Simulink analysis models using the Simulink Analysis Models context menu command of the root element in the Measurement Setup.

In contrast to the models that are assigned for the node in the Simulation Setup, the analysis models have no effect on the simulation or measurement. For this reason, blocks that change signals or environmental variables cannot be used in these models. Analysis models are available in both online and offline mode and are started automatically when the measurement begins.

Configured Models

With the [Add] and [Remove] buttons you can select or remove analysis models.

Model Configuration

On the right side of the dialog with Module Description you can select the path to the module description file for the Simulink model built with the MATLAB Integration Package.

Note

There is a legacy configuration option available for this Dialog, which can be enabled by opening the Options dialog (External Programs|Simulink) and activate Legacy Simulink integration check box.

When the measurement begins, the Simulink analysis models are started automatically from CANoe.
