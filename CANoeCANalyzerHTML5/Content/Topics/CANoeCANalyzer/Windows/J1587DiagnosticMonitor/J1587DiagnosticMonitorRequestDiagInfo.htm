Request of Diagnostic Information

 

With the J1587 DC Monitor it is possible to request diagnostic messages from ECUs on the bus. Information for specifying the request can either be provided by clicking a context menu command of a node or diagnostic code, or by using controls on the right side of a page.

Received diagnostic responses (PID 196) are displayed in the view of the Data Requestpage.

Requests applying to a network node

Open the context menu of a node with the right mouse button and select one of the following entries to send a request:

Menu Command

Symbol

Description

Request all DCs

Selection of this entry results in a specific request of all diagnostic codes from the selected node.

Clear all OCs

Reset of all Occurrence Counts present on the selected device

Insert Node

To enable requests directed to an ECU which is not shown (because it has not send any messages since start of the measurement), use this command to add it to the list.

Requests applying to a specific diagnostic code

For a specific diagnostic code, you have the following entries to send a request:

Menu Command

Symbol

Description

Clear OC

Clears the Occurrence Count of the selected code

Req. Descr. Msg | ASCII

Request of an ASCII "Descriptive Message" for the selected diagnostic code

Req. Descr. Msg | proprietary

Request of a proprietary "Descriptive Message" for the selected diagnostic code

Requests using tab Data Request

Here the content of the message "Diagnostic Data Request/Clear Count" (PID 195) can be defined individually.

Menu Command

Symbol

DescriptionMeaning

Specify Request Data

This opens a dialog, in which all data of the request can be set manually and transmitted by clicking the Send button. Received diagnostic responses ("Diagnostic Data/Count Clear Response" - PID 196) are displayed in the view. Whenever a proprietary "Descriptive Message" is received its content is shown in hexadecimal format.

Request all DCs globally

For a global request of the diagnostic table (PID 194) right click on a blank area beneath the ECUs displayed in the nodes view and select this entry from the context menu.
