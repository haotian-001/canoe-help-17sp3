Configuration

State Tracker Toolbar: Show Configuration Dialog

Measurement Objects

Here you define the most important settings for the individual measurement objects (system variables, signals, messages, diagnostic parameters, ECUs, networks).

The page Presentation and Configuration of Measurement Objects gives a detailed description of the elements and settings.

Time Axis

Display

Here you set the Time unit (Microseconds/Milliseconds/Seconds/Minutes).

Axis Mode

The signal view can either be Smart scrolling or Fixed.
In smart scrolling mode you can set the range (percentage) the graph will be shifted.
In fixed mode the State Tracker can Draw label values as offset.

Optimizations

Memory Optimization

Here you adjust the Maximum size of reserved buffer for each signal (1 MB/5 MB/10 MB).
The buffer size defines the memory that will be reserved in RAM for each signal. Old data will be removed from the RAM buffer and cannot be displayed any more as soon as the buffer limit is reached.

If you Display value changes only the State Tracker logs and shows only changed values. This setting may increase the display range of a signal – depending on the update behavior of the signal/variable.

The State Tracker supports the writing of data to the system buffer (data history). For this purpose you have to activate Store signal values in history buffer.

Performance Optimization

To improve the performance on slower machines, you can adjust the Cyclic update of the State Tracker (30 ms/50 ms/100 ms/500 ms). Larger cycle times (higher values) cause less stress to the system.

Diagram

Diagram Settings

Here you set the appearance of the diagram.

Depending on your selection, the diagram will Show sampling points, Show grid, Show values and symbolic descriptions (signal label) and Show initial values for signals and variables.

Legend Panel

If you activate Show values in the legend panel the following will be shown:

  • during measurement: the last received values,
  • during pause: the value at the current cursor position,
  • after the measurement: the value at the current cursor position.

Colors

Main View Configuration

Here you set the Grid lines color, the State view background color and the Grid style (Solid/Dashed).

Trigger

Trigger Settings

Here you select the Post delay time for simple trigger. Possible values are: 500 ms, 1000 ms, 2000 ms, 5000 ms and 10000 ms.
