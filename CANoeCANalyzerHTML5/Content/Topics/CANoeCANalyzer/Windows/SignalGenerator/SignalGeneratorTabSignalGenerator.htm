Note

  • About pausing
    Signals that are allocated to an active Signal Generator cannot be modified via CAPL signal interfaces at the same time. Changing a signal value via CAPL is only possible if the corresponding Signal Generator is paused.
  • About configuration
  • Each signal can be configured multiple times with different Signal Generator types, but only one of them can be active.
  • System variables of type Array cannot be added.
  • For list display, note the settings for array elements of distributed objects in the Options dialog.
  • Any value can be used in the Signal Generator. The validity of these values is not checked against the database values, either during configuration or when transmitted on the bus.
  • The Signal Generators are stored in the configuration (*.cfg).
  • The central Signal Generators are not supported in CAPL or the interactive Generator.
