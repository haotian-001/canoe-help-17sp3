Signal Generators

Ribbon: Simulation Tab|Signal Generators and Signal Replay

Signal Generators are used to define a signal's sequence and measure an ECU's reaction.

Signal Generators can be assigned to signals, system variables and distributed objects.

Signal Generators and how they are controlled during a measurement are defined centrally via the Simulation ribbon tab|Signal Generators.

The supported Signal Generator types are similar to those of the interactive Generator.

About the Concept

In contrast to the generators of the interactive Generator, here it is the CANoe signal layer that transmits signal values on the bus (the CANoe signal layer is also used by the CAPL interface and by the panels).

The generators of the interactive Generator circumvent the CANoe signal layer completely. Messages sent from the interactive Generator are always assigned the signal values of the IG generator.

For this reason, we do not recommend using the interactive Generator and the CANoe signal layer in parallel for the same message.
Should you decide to use both in parallel for a single message, actual bus behavior may not conform to expected behavior.

Note

  • In CAN, the transmit behavior of the signal layer is determined via an interaction layer.
  • In LIN, the transmit behavior is determined exclusively by the scheduler.
  • If an interaction layer is configured in FlexRay, it determines the transmit behavior. Otherwise, the transmit behavior is determined by the scheduler. When replaying logging files, it is important to use the same scheduler that was used to record the logging file. Otherwise the transmission times may drift apart and affect the output results.

Configuring Signal Generators

To open the configuration dialog, open the menu path View|Signal Generators and Replay.

There are two ways to configure Signal Generators:

  • If you want to define the sequence for any one given signal, use the Signal Generators page.
  • If you want to replay several (or many signals) from a logging file, use the Signal Replay page.
