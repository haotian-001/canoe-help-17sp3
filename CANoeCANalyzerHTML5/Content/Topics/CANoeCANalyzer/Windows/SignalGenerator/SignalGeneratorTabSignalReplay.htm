Signal Generators

Signal Replay

 

You can add one or more logging files on this page and use them to configure signals/variables as Signal Generators.

A logging file that is already allocated to a signal in the Signal Generators page cannot be assigned in this page.

Context Menu

Add File...
Select here a new logging file. After the file selection the Add Signals and Variables from Logging File dialog is opened. Here you can search for available signals/variables.

Add Symbol...
Select a logging file in the list to configure symbols from this logging file as Signal Generators. The Add Signals and Variables from Logging File dialog is opened. Here you can search for available signals/variables.

Deletes the selected symbols or logging files.
