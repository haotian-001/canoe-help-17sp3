Signal Generator type: User-defined

 

You can use user-defined Signal Generators

  • to create signal sequences as needed,
  • and to modify logged signal sequences in a targeted way.

  • Signal sequences created in this way are written to the selected symbol (bus signal, system variable) during measurement.

FAQ: Terms

  • Sampling point
    A point in the signal sequence where the X value is a multiple of the sampling interval.
  • Sampling node
    Fulfills the same conditions as a sampling point, but is either explicitly defined (via a table or visual editor) or is created on read-in to describe the signal sequence.
    The following applies: Number of sampling nodes <= number of sampling points

Note

  • You can create a maximum of 512 sampling nodes for each user-defined generator (for memory/performance reasons).
    You can create a maximum of 20 user-defined generators in a CANoe configuration.
  • The first sampling node always lies at X = 0; it represents the start value of the generator.
  • If you zoom in very closely or make the grid spacing very small in the Editor view, the grid lines will be so close together that the area appears gray. You can return to the familiar view by zooming out or changing the grid spacing.
  • Increasing the zoom factor in the Editor View limits the scrolling area.
    The Editor can only display a limited number of pages. If you want to scroll through a larger area, you need to zoom out.
    You can only enter time-value pairs (in the table) that fall within the displayed working area based on the current zoom setting in the Editor View.
  • The generator uses the Interaction Layer (IL) to transmit bus signals.
    If you have not configured an IL, the generator is deactivated when measurement starts.

Data Entry

  • Visual Editor
    In the Visual Editor, you use your mouse to create the signal sequence.
    Use the left mouse button to insert sampling nodes in the grid. The sampling nodes are highlighted in blue in the Editor.
    The X axis represents the time and the Y axis the value of the symbol. You can only assign one Y value to each X value.
    You can move the sampling nodes.
  • Table
    You can also enter sampling nodes into a table.
    This is done by entering the sampling nodes as time-value pairs.
    New sampling nodes are always added to the end of the table; the entry is then sorted chronologically.

Settings

  • Sampling
    You can create signal sequences without sampling. The Signal Generator then writes the values only when the variable changes. You can create sampling nodes at each point of the X axis that is a multiple of the minimum value (1 ms). You can deactivate sampling for system variables, but it is always activated for bus signals.
    When sampling is activated, the variable is written cyclically. You specify this cycle via the sampling interval. You can only create sampling nodes at multiples of the sampling interval.
  • Sampling Interval
    Only available when sampling is activated. You use the sampling interval to specify the cycle time after which a new value is written to the symbol. Changing the sampling interval can lead to changes in the signal sequence. In the case of bus signals, the sampling interval is derived from the cycle time in the database. The sampling interval for bus signals normally corresponds to the signal's cycle time. The cycle time from the database is the default value for the sampling interval, but you can modify the sampling interval as required.
  • Interpolation
    Interpolation is not possible if sampling is deactivated, since values are written explicitly only when a change takes place. If this is the case, this option is disabled. Activating sampling makes constant or linear interpolation possible.
    With constant interpolation, the value of the most recent sampling node is written repeatedly, in accordance with the sampling interval cycle, until the next sampling node is reached.
    With linear sampling, the values of the sampling points that lie between the two sampling nodes are calculated, creating a linear sequence between the two sampling nodes.
  • Grid Spacing
    You use the grid spacing to select your working area. By modifying these values, you can configure the grid display in the Visual Editor. The grid spacing also determines where you are able to insert sampling nodes.
    The X axis grid spacing must always be a multiple of the sampling interval. If you have not configured the sampling, then this is a multiple of 1 ms.
    For the Y axis, you set the grid spacing e.g. to 0.1 if you want to specify current values with a maximum accuracy of 0.1V. The modifiable area depends on the data type of the selected symbol.
  • Periodic/Delay
    If you select the Periodic option, the signal sequence is run periodically until measurement ends or the generator is stopped.
    You can use the Delay option to configure the time interval between the last and first sampling nodes.
  • Start generator on measurement start
    If you select this option, the generator is started automatically when measurement starts.
  • Preview
    This creates a preview in the Editor showing how the values are written from the generator to the CANoe simulation environment. This gives you an impression of how the generator impacts the selected symbol. The small green points represent the sampling points and the larger green points the sampling nodes. You cannot make any modifications while viewing the preview.

Import/Export

  • Import
    Here you can import signal sequences that were created in other user-defined generators.
    The import supports all signal.-based logging formats of CANoe. E.g. you can export signal curves from the Graphics Window and import them in the user-defined Signal Generator.
    Logging File Import
  • Export
    Here you can export created signal sequences to a SIG file.
    You can import these exported files in other user-defined Signal Generators.
