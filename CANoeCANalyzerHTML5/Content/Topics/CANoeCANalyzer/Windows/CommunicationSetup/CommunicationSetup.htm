Communication Setup

Ribbon: Environment Tab|Communication Setup

The Communication Setup is the central starting point for communication configuration. You configure:

  • the application layer (application models)
  • the communication layer (communication, bindings)
  • the transmission medium (hardware, bindings)

Configuration

Steps

  • Load data sources into the CANoe configuration via the toolbar .
    This provides CANoe the contained information about elements participants, endpoints and distributed objects. After loading, they are displayed under Data Sources.

    		•

    Note

    Working with AUTOSAR extracts and system descriptions.
  • Add application models via the toolbar or application model properties that contain functions to be executed during the CANoe measurement.
  • You can also add a VTT model file (*.vttsut) that was generated from vVIRTUALtarget.

Publish Changed System Data

You must publish changes to system data explicitly so that they are accepted in CANoe and become visible, for example, in the Symbol Explorer.

An automatic publication is performed as soon as it is absolutely necessary for further action, e.g:

  • reload of data sources
  • start a measurement
  • save the configuration
  • close CANoe

The button is only displayed if you change something in the Communication Setup. As soon as you accept a change, the button is hidden again. You do not have to publish every change immediately. Despite the message, you can first enter all changes and then publish them all at once.
