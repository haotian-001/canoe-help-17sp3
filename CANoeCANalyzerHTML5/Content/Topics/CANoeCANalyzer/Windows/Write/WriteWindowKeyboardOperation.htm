Write Window: Keyboard Operation

 

Description

Keys

Opens the help for the Write Window.

<F1>

Opens the context menu.

<Shift> + <F10>

Copies the selected line from the Write Window to the clipboard.

<Ctrl> + <c>

Selects multiple selective lines individually.

<Ctrl> + mouse click

Selects multiple consecutive lines simultaneously.

<Shift> + mouse click

Jumps to the first line on the active page.

<Home>

Jumps to the last line on the active page.

<End>

Selects lines: in the Up direction starting from the focused line.

<Shift> + <Home>

Selects lines: in the Down direction starting from the focused line.

<Shift> + <End>

Moves the cursor one page up.

<Page Up>

Moves the cursor one page down.

<Page Down>

Select all lines on the active page.

<Ctrl> + <a>

Opens the search in the Write Window.

<Ctrl> + <f>

Continues searching.

<F3>
