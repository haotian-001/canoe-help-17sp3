Notification / Description

56-0016

AFDX <channel number> configured using <details>

Informative message of the AFDX driver which hardware adapter and Ethernet channels are used.

56-0017

Hardware Initialization failed (<error description>)

Error message of the AFDX driver, measurement is aborted.

56-0021

AFDX: Communication Unit: Transmission of packet failed <error description>

Error message of the AFDX driver about failed sending of a packet. The measurement is resumed if no further hardware error occurred.

56-0022

Invalid configuration encountered (<error description>)

The device configuration or requested channel assignment is not possible. The measurement is stopped. Check the channel assignment and the number of maximum available channels.

56-0023

AFDX: Communication Unit: Tx Buffer overflow, device could not process all Tx requests.

CANoe has tried to send more packets than the network adapter could process. AFDX packets are lost.

56-0024

AFDX: Communication Unit: Rx Buffer overflow, CANoe could not process all received packets.

More AFDX packets have been received than CANoe could process. AFDX packets are lost, but measurement is resumed.

56-0025

AFDX: Communication Unit: Device <channel number> is already in use. Abort measurement!

The device is already in use by another CANoe and cannot be used. The measurement is aborted.

56-0026

Internal error on VirtualLink-Management (<error description>)

An internal error occurred during management or assignment of the Virtual Links.

56-0027

Info: <information message>

Informative message of the AFDX driver, measurement is still possible.

56-0028

Warning: <warning message>

Warning message of the AFDX driver, measurement is still possible.

56-0029

Error: <error message>

The AFDX driver detected a problem.

56-0100

AFDX data model: attribute <attribute name> could not be found in database! Database not available!

An attribute with the given name could not be found in the database.

56-0101

AFDX data model: attribute <attribute name> of database object <object name> has invalid value <value>! Database not available!

The attribute value of the given object is invalid.

56-0102

AFDX data model: general database error <error message>! Database not available!

The database contains an error.

56-0103

AFDX data model: Database contains a non standard compliant Source IP address! The constant part of the address <address> is <address part>!

The non standard compliant address is reported but it is accepted.

AFDX data model: Inconsistent VL Source IP addresses for VL <VL>. This VL has an actual Source IP address of <IP address> but the actual message <message name> to add has a Source IP address of <IP address>.

The first source IP address of a sent frame/packet is used.

56-0105

AFDX data model: Warning (<warning message>).

The data model has been load, inconsistent data (e.g messages, virtual links) are discarded during loading. Measurement is possible.

56-0106

Echo Reply to an Echo Request was not received within <timeout>.
