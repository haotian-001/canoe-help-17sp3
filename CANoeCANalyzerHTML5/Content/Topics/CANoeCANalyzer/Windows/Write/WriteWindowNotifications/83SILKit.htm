Enter a valid filename for the API library of the integration on the SIL Kit page (Options dialog). The following characters are not allowed: < > : â \ / | ? *

83-0002

SIL Kit: The Registry URI is not specified.

Enter a Registry URI on the SIL Kit page (Options dialog).

83-0003

SIL Kit: Participant name is not specified.

Enter a participant name on the SIL Kit page (Options dialog).

83-0004

SIL Kit: Configuration file is not specified.

Enter the name and path of the configuration file for the SIL Kit on the SIL Kit page (Options dialog).

83-0005

SIL Kit: The specified configuration file <filename> does not exist.

Correct the name and path of the configuration file for the SIL Kit on the SIL Kit page (Options dialog).

83-0006

SIL Kit: Error while reading configuration file <filename>.

The mentioned configuration file could not be read in.

83-0007

SIL Kit: The encoding of the specified configuration file <filename> is invalid.

Invalid characters were detected in the mentioned configuration file. Two character encodings are supported:

  • ASCII

  • UTF8 with introductory UTF8-BOM (Byte Order Mark)

83-0008

A network channel is assigned to the SIL Kit, but the feature 'SIL Kit' is disabled.

Either activate the SIL Kit on the on the SIL Kit page (Options dialog) or remove the assignment of the channel to the SIL Kit (Application Channel Mapping dialog).

83-0100

Loading of SIL Kit API library <filename> failed: <error message of the operating system>

The used operating system could not load the API library into the CANoe runtime kernel.

83-0101

The SIL Kit API library is invalid, the required function <function name> is not exported.

The API library of the SIL Kit exports its services as C functions. The mentioned function is required by CANoe for integration of the SIL Kit, but is not exported from the library.

83-0102

The SIL Kit API library failed to create the participant: <error message or error code of API library>

CANoe could not create the simulation participant for the SIL Kit.

83-0103

The SIL Kit API library failed to destroy the participant: <error message or error code of API library>

CANoe could not delete the simulation participant for the SIL Kit.

83-0104

SIL Kit: The name for channel <bus type> <channel number> is not specified.

In the Simulation Setup of CANoe, please create a network for the named channel and give the network a unique name.

83-0105

The ingress queue for events from the SIL Kit exceeds the limit of <limit> entries.

If the SIL Kit continuously delivers more events than CANoe can process, these events accumulate in a queue whose length is limited.

83-0106

Invalid context parameter in callback function <function name> of SIL Kit.

Internal program error of the SIL Kit implementation. When calling back to the application, one of the context parameters is invalid .

83-0107

The SIL Kit API library operation <function name> for time synchronization failed: <error message or error code of API library>

The named operation to run the time synchronization failed.

83-0108

Time synchronization error of SIL Kit. Parameter 'now' of callback <function name> doesn't match the current simulation step.

Internal program error of the SIL Kit implementation. When calling back to the application, the time parameter is invalid.

83-0109

The currently selected working mode of CANoe does not support the feature 'SIL Kit'.

To use the SIL Kit, set one of the following working modes of CANoe on the General page (Options dialog):

  • Real Bus
  • Simulated Bus (Slave Mode); the animation factor must be set to the value 1.0

  • Simulated Bus (animated with factor); the animation factor should be set to the value 1.0, otherwise a warning will be displayed

83-0110

SIL Kit: System command 'Stop' received.

The measurement in CANoe was terminated due to a received stop command.

83-0111

SIL Kit Lifecycle Error: Unexpected call of callback function '<function name>'.

Internal program error of the SIL Kit implementation. The call of the mentioned function is not expected within the current state of the lifecycle of the SIL Kit.

83-0112

SIL Kit Lifecycle Error: Timeout while waiting for call of callback function <function name>.

The mentioned function was not called by the SIL Kit within the expected time span. The time duration can be configured on the SIL Kit page (Options dialog).

83-0113

Synchronization error of SIL Kit. The callback function <function name> was call before the previous simulation step was completed.

Internal program error of the SIL Kit implementation. Simulation steps must take place one after another.

83-0114

The currently selected working mode (Slave-Mode) only supports animation factor '1' while SIL Kit is enabled.

To use the SIL Kit, set one of the following working modes of CANoe on the General page (Options dialog):

  • Real Bus
  • Simulated Bus (Slave Mode); the animation factor must be set to the value 1.0

  • Simulated Bus (animated with factor); the animation factor should be set to the value 1.0, otherwise a warning will be displayed

83-0115

The currently selected working mode (Simulated Bus with animation factor) is set to use an animation factor not equal to '1'. This may lead to unexpected behavior while SIL Kit is enabled.

In Simulated Bus (Animated with Factor) working mode, unexpected states can occur in CANoe when communicating with nodes of a SIL Kit simulation if the animation factor is not equal to 1.

In particular, error 83-0105 can occur if an animation factor less than 1 is set.

(e.g. CAN identifier is outside the valid range, not allowed combination of protocol flags or payload lengths that are not provided for in the CAN protocol).

  • The second key slot must be configured to Off mode.
