Connectivity

 

[65] Connectivity

ID

Notification / Description

65-0002

Service '<ServiceName>': <Description> (Code: <StatusCode>).

This message is a general error message for a service. The description of the error and the status code are returned.

In this case a measurement is not possible.

65-0003

Active instance '<CertifcateID>' can not be used, because it's only available on <Side>.

A selected instance is either only available locally or in the cloud.

  • If the instance is only available in the cloud, reloading the certificate allows the instance to be used.
  • If the instance is only available locally, it can no longer be used and it is only possible to delete the certificate.
