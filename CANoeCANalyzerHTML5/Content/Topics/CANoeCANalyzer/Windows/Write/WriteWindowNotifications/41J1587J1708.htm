[41] J1587/J1708

ID

Notification / Description

41-0001

J1708 <channel number>: Driver: Init failed. Channel for application "CANoe" not assigned to any hardware.

The channel is not assigned to any hardware. Open the Vector Hardware Config dialog with Hardware ribbon tab|Network Hardware|[Driver] and assign the channel of your application to the hardware you want to use.

41-0002

J1708: Driver: ncdOpenPort failed. Are cabs/piggybacks connected to the used channels?

Check if J1708 compatible transceivers (Cabs, piggybacks) are connected to the hardware on the used channel.

41-0003

J1708: Driver: ncdSetNotification failed.

Internal error; contact Vector Support.

Internal error; contact Vector Support.

Internal error; contact Vector Support.

41-0007

J1708: Driver: ncdActivateChannel failed.

Plug in the transceiver and check if the channel assignment is correct. Use the Vector Hardware Config dialog (Hardware ribbon tab|Network Hardware|[Driver]).

41-0008

J1708 <channel number>: Driver: Reset J1708 transceiver failed with error <error code>.

Internal error; contact Vector Support.

41-0009

J1708 <channel number>: Driver: Ensure the right transceiver is connected: <transceiver name> on <hardware name> (<error code>).

Check if the appropriate transceiver is attached and if the channel assignment is correct. Use the Vector Hardware Config dialog (Hardware ribbon tab|Network Hardware|[Driver]).

41-0010

J1708 <channel number>: Driver: Activate J1708 Firmware failed (result <error code>).

Internal error; contact Vector Support.

41-0014

J1708 <channel number>: Driver: Hardware already in use (result <error code>).

The hardware that is assigned to the channel is already in use. Release the hardware connected to the channel or change the channel assignment to use the hardware.

41-0015

J1708: Driver: Port not init.

Internal error; contact Vector Support.

41-0016

J1708: Driver: ncdSetTimerRate failed.

Internal error; contact Vector Support.

41-0017

J1708 <channel number>: Driver: Tx message queue overflow (-> high busload?)

The busload might be to high. Reduce the amount of messages per time sent by CANoe and other bus participants.

41-0018

J1708: Driver: InitDriver <driver filename> failed.

Check your driver installation.

41-0019

J1708: Driver: Measurement Init: Multiple channels assigned to <transceiver name> on <hardware name>. Resolve conflicts in Vector Hardware Config.

Open the Vector Hardware Config dialog with Hardware ribbon tab|Network Hardware|[Driver] and make sure that each transceiver has a single J1708 channel assigned.

41-0020

J1708 <channel number> real bus with <baudrate> BPS.

A baudrate of <baudrate> BPS is used for the channel.

41-0090

J1708 Database: duplicate node definition for address <address> on channel <channel>: <name1> / <name2>

Multiple nodes with the same address are defined in the databases.

41-0091

J1708 Database: duplicate node definition for address <address>: <name1> / <name2>

Multiple nodes with the same address are defined in the databases that are assigned to multiple channels.
