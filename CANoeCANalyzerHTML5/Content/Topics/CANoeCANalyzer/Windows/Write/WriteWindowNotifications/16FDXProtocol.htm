FDX Protocol

 

[16] FDX Protocol

ID

Notification / Description

16-0001

FDX description file parse error - <filename><data group><data item>: file not found

The indicated FDX description file could not be opened. Please check the list of description files specified in Options dialog (Extensions|FDX Protocol).

16-0002

FDX description file parse error - <filename><data group><data item>: <XML syntax error>

A general XML syntax error occurred while reading the indicated FDX description file that was specified in in Options dialog (Extensions|FDX Protocol).

16-0003

FDX description file parse error - <filename><data group><data item>: undefined group id

A <datagroup> element without a specified groupID attribute was found in the indicated FDX description file.

16-0004

FDX description file parse error - <filename><data group><data item>: invalid group id "<ID>"

An invalid group ID was specified in the indicated FDX description file. Values in the range from 0 to 65535 are allowed.

16-0005

FDX description file parse error - <filename><data group><data item>: duplicate group id "<ID>"

A group ID that is already being used in another data group is specified in the indicated FDX description file.

Please note that group IDs must be unique on a global basis, i.e., including across multiple description files.

16-0006

FDX description file parse error - <filename><data group><data item>: undefined group size

The size attribute is missing in the indicated data group of the indicated FDX description file.

16-0007

FDX description file parse error - <filename><data group><data item>: invalid group size "<group size>"

An invalid value is specified for the size attribute in the indicated data group of the indicated FDX description file.

Please note that the group size must not exceed a value of 65535 bytes. Also note that the total transmitted IP packet must not exceed 65535 bytes.

16-0008

FDX description file parse error - <filename><data group><data item>: undefined item data type

The type attribute is not specified for the <item> element defined in the indicated FDX description file.

16-0009

FDX description file parse error - <filename><data group><data item>: invalid item data type "<data type>"

An invalid value was specified for the type attribute for the <item> element defined in the indicated FDX description file.

Valid data types are int8, uint8, int16, uint16, int32, uint32, int64, uint64, float, double, string and bytearray.

16-0010

FDX description file parse error - <filename><data group><data item>: undefined item size

The size attribute is not specified for the <item> element defined in the indicated FDX description file.

Please note that this specification is required for the string and bytearray data types, while the default size can be derived directly for the other data types.

16-0011

FDX description file parse error - <filename><data group><data item>: invalid item size "<item size>"

An invalid value was specified for the size attribute for the <item> element defined in the indicated FDX description file.

Valid values are between 0 and 65535 bytes. Note here that the total transmitted IP packet must not exceed 65535 bytes.

16-0012

FDX description file parse error - <filename><data group><data item>: undefined item offset

The offset attribute is not specified for the <item> element defined in the indicated FDX description file.

The offset defines the position of the element within the data group (distance in bytes from the start of the data group).

16-0013

FDX description file parse error - <filename><data group><data item>: invalid item offset "<offset>"

An invalid value was specified for the offset attribute for the <item> element defined in the indicated FDX description file.

Values between 0 and 65535 are valid, in principle. Note here that the total IP packet ultimately transmitted must not exceed 65535 bytes.

16-0014

FDX description file parse error - <filename><data group><data item>: item is overlapping with other items

The indicated <item> element in one FDX description file overlaps with other elements in the same group.

Please adjust the values for offset or size in the indicated element or in another element of the data group.

16-0015

FDX description file parse error - <filename><data group><data item>: item exceeds group size

The <item> element defined in the indicated FDX description file falls outside (in part or in whole) of the scope defined by the group size.

Please adjust the values for offset or size in the indicated element or the value for the group size.

16-0016

FDX description file parse error - <filename><data group><data item>: no signal or variable defined

The <item> element defined in the indicated FDX description file does not contain a definition of a signal, an environment variable, or a system variable.

16-0017

FDX description file parse error - <filename><data group><data item>: multiple signal or variable definitions in a single item

The <item> element defined in the indicated FDX description file contains several definitions of signals, environment variables, or system variables. Only one definition is permitted.

16-0018

FDX description file parse error - <filename><data group><data item>: undefined signal name

A signal name (name attribute) was not defined for the signal defined in the indicated FDX description file.

Please specify the name of a signal that is contained in one of the databases used in the CANoe configuration.

16-0019

FDX description file parse error - <filename><data group><data item>: undefined system variable name

A variable name (name attribute) was not defined for the system variable defined in the indicated FDX description file.

Please specify the name of a system variable contained in the CANoe configuration.

16-0020

FDX description file parse error - <filename><data group><data item>: undefined environment variable name

A variable name (name attribute) was not defined for the environment variable defined in the indicated FDX description file.

Please specify the name of an environment variable that is contained in one of the databases used in the CANoe configuration.

16-0021

FDX description file parse error - <filename><data group><data item>: invalid signal direction "<signal direction>"

An invalid value for the signal direction (direction attribute) was specified for the signal defined in the indicated FDX description file.

Valid values are auto and TXrq.

16-0022

FDX description file parse error - <filename><data group><data item>: undefined signal value type

The signal value type (value attribute) is not specified for the signal defined in the indicated FDX description file.

The signal value type distinguishes between a raw value (raw) and a physical value (phys).

16-0023

FDX description file parse error - <filename><data group><data item>: invalid signal value type "<signal value type>"

An invalid value for the signal value type (value attribute) was specified for the signal defined in the indicated FDX description file.

Valid values are raw (raw value) and phys (physical value).

16-0024

FDX description file parse error - <filename><data group><data item>: description must be included in a single "<name>" block

The root element <canoefdxdescription> within which all data groups must be defined is missing in the indicated FDX description file.

16-0025

FDX description file parse error - <filename><data group><data item>: undefined file version

A file version (version attribute) was not specified in the root element <canoefdxdescription> in the indicated FDX description file.

16-0026

FDX description file parse error - <filename><data group><data item>: incompatible file version ("<version number>")

A file version (version attribute) that is incompatible with the FDX protocol version implemented in CANoe is specified in the root element <canoefdxdescription> in the indicated FDX description file.

16-0027

FDX: conversion of data parsed from description file failed

An error occurred during conversion of the data groups read from the FDX description files.

Please contact Vector Support.

16-0100

FDX: environment variable "<environment variable name>" in group <ID> is not defined

The environment variable defined in the FDX description is not contained in any database used in the configuration.

16-0101

FDX: unable to determine type of environment variable "<environment variable name>" in group <ID>

An internal error occurred with the indicated environment variable when adjusting the data types from the FDX description and the database.

Please contact Vector Support.

16-0102

FDX: data type "<data type>" of environment variable "<environment variable name>" in group <ID> is incompatible to data type "<data type>" from FDX description

A data type that is incompatible with the data type defined in the database was specified for the indicated environment variable in the FDX description file.

Please note that an automatic conversion is performed in the case of different number types, but this conversion is not possible between numbers, strings, and byte arrays.

16-0110

FDX: system variable "<system variable name>" in group <ID> is not defined

The system variable defined in the FDX description is not contained in the CANoe system variable configuration.

16-0111

FDX: data type "<data type>" of system variable "<system variable name>" in group <ID> is incompatible to data type "<data type>" from FDX description

A data type that is incompatible with the data type defined in the CANoe configuration was specified for the indicated system variable in the FDX description file.

Please note that an automatic conversion is performed in the case of different number types, but this conversion is not possible between numbers, strings, and byte arrays.

16-0120

FDX: signal "<signal name>" in group <ID> is qualified using an invalid application channel "<channel number>"

An application channel (channel attribute) was specified during definition of the indicated signal for a more detailed specification. However, the name of the channel is invalid or the channel is not used in the configuration.

16-0121

FDX: signal "<signal name>" in group <ID> is not defined (specified bus "<bus name>" is not defined or inactive)

A network name (bus attribute) was specified during definition of the indicated signal for a more detailed specification. However, the indicated network is not contained in the CANoe configuration or has been deactivated.

16-0122

FDX: signal "<signal name>" in group <ID>: Specified bus "<bus name>" is not assigned to specified channel "<channel number>"

A network name (bus attribute) and an application channel (channel attribute) was specified during definition of the indicated signal for a more detailed specification. However, the specified values are not compatible.

16-0123

FDX: signal "<signal name>" in group <ID> is not defined (unknown database "<database name>")

The database (database attribute) specified in the FDX description during definition of the indicated signal is not contained in the CANoe configuration.

16-0124

FDX: signal "<signal name>" in group <ID> is not defined or ambiguous

The indicated signal is not contained in the databases of the CANoe configuration or the specified value is not unique.

Please note that a signal name can be assembled from the following components:
<Network>::<Database>::<Node>::<Message>::<Signal>.

The individual suffixes can be added in the FDX description using certain attributes during the signal definition.

16-0125

FDX: signal "<signal name>" (application channel "<channel number>") in group <ID> is not defined or ambiguous

The indicated signal for which an application channel (channel attribute) was specified in the FDX description for a more detailed specification is not contained in the databases of the CANoe configuration or the specified value is not unique.

Please note that a signal name can be assembled from the following components:
<Network>::<Database>::<Node>::<Message>::<Signal>.

The individual suffixes can be added in the FDX description using certain attributes during the signal definition.

16-0126

FDX: data type "<data type>" of signal "<signal name>" in group <ID> is incompatible to data type "<data type>" from FDX description

A data type that is incompatible with the data type defined in the database was specified for the indicated signal in the FDX description file.

Please note that an automatic conversion is performed in the case of different number types, but this conversion is not possible between numbers, strings, and byte arrays.

16-0127

FDX: data type "<data type>" of physical value in signal "<signal name>" in group <ID> is incompatible to data type "<data type>" from FDX description

A data type that is incompatible with the data type of a physical signal value (corresponds to double) was specified for the indicated signal in the FDX description file.

Please note that an automatic conversion is performed in the case of different number types, but this conversion is not possible between numbers, strings, and byte arrays.

16-0200

FDX: invalid datagram size (<datagram size> bytes)

A datagram whose size is incorrect was received in the CANoe FDX interface. This can be caused by any of the following:

  • The datagram is too small for the FDX header.
  • The datagram does not have enough space for the number of FDX commands specified in the FDX header.
  • The datagram contains surplus bytes after the last FDX command.

16-0201

FDX: invalid datagram signature

A datagram in which the FDX header contains an invalid signature was received in the CANoe FDX interface.

16-0202

FDX: version specified in datagram <version number> is incompatible to current version of protocol engine <protocol version number>

A datagram in which a version not supported by this CANoe version is specified in the FDX header was received in the CANoe FDX interface.

16-0203

FDX: invalid group id <ID> in <command name> command

A group ID that is not contained in any of the configured FDX description files was specified for the data exchange.

16-0204

FDX: group size (<group size> bytes) defined for group id <ID> exceeds upper limit (<limit value> bytes)

A data group size was defined for the data exchange such that the maximum size for the resulting UDP datagram is exceeded.

16-0205

FDX: data size (<data size> bytes) given in incoming DataExchange command for group id <ID> exceeds specified group size (<group size> bytes)

A DataExchange command received in the CANoe FDX interface contains a payload whose size is less than the size specified in the FDX description file.

16-0206

FDX: invalid command size (<size>bytes) in <command name> command

A datagram containing a command whose size is too small was received in the CANoe FDX interface.

16-0207

FDX: DataRequest is only allowed during measurement

A DataRequest command was received in the CANoe FDX interface, but CANoe is not ready to transmit data.

DataRequest commands can only be processed during a running measurement.

16-0208

FDX: sending datagram failed with WSA error code <error code number>

An error occurred while transmitting a datagram in the CANoe FDX interface. The specified error number refers to the Microsoft Windows socket error codes.

16-0209

FDX: receiving datagram failed with WSA error code <error code number>

An error occurred while receiving a datagram in the CANoe FDX interface. The specified error number refers to the Microsoft Windows socket error codes.

16-0210

FDX: receiving datagram with sequence number <sequence number>, but number <sequence number> was expected

The transmitted datagrams in the CANoe FDX interface are numbered consecutively in order to allow detection of data losses. On receipt of a datagram, an error was detected in the numbering of the datagrams.

16-0211

FDX: UDP connection reset by peer <peer name>

The receiver has closed the UDP port for the FDX interface unexpectedly. (Microsoft Windows socket error code WSAECONNRESET 10054 'Connection reset by peer').

16-0212

FDX: too many FDX errors have occurred, further FDX messages are skipped til end of measurement

To prevent flooding the CANoe Write Window with error messages from the FDX interface, if more than 50 FDX errors occur, the further output of FDX error messages is suppressed until the next measurement end.

16-0213

FDX: <error number> FDX errors occurred while measurement was running, <error number> of them are sequence number errors

Error occurrences in the FDX interface are numbered consecutively. At each measurement end in CANoe, if corresponding errors have occurred, the number of errors is displayed in the Write Window and the error counter is reset. The portion that are sequential number errors (see 16-0210) is listed separately here.

16-0214

Stop command received from <IP Address>

A Stop command from the denoted remote station has been received by CANoe FDX interface.

16-0215

FDX: DataExchange command is only allowed during measurement

A DataExchange command was received at the FDX interface, but no measurement is currently running in CANoe.

16-0216

FDX: Too many FDX warnings have occurred, further FDX warning messages are skipped until end of measurement

In order to avoid flooding the Write Window with too many messages, further error notifications for the FDX feature will be suppressed until the end of the measurement.

16-0217

FDX: The port for the FDX communication (UDP or TCP) is closed

Note that communication with the FDX interface is not possible.

16-0218

FDX: IncrementTime command is only allowed during measurement

An IncrementTime command was received at the FDX interface, but no measurement is currently running in CANoe.

16-0219

FDX: IncrementTime command is only allowed in slave mode

An IncrementTime command was received at the FDX interface, but CANoe is not configured for slave mode.

16-0220

FDX: accepting a tcp connection failed with WSA error code <Error-Code-Number>

An error occurred when establishing a TCP connection at the FDX interface of CANoe. The error number given refers to the Microsoft Windows Socket Error Codes.

16-0221

FDX: version specified in datagram <FDX version number> is invalid, version 2.1 or newer is required for FDX over TCP

The use of TCP as a transport layer requires FDX version 2.1.

16-0222

FDX: The configured <TCP|UPD> port <port number> is reserved for another service

The port number specified for the FDX feature is reserved for use by another service. In the CANoe Options dialog reconfigure the port number for the FDX feature.

16-0223

FDX: opening <TCP|UPD> port failed failed with WSA error code <Error-Code-Number>

An error occurred when opening the TCP or UDP port. The specified error number refers to the Microsoft Windows Socket Error Codes.

16-0224

FDX: binding <TCP|UPD> port failed failed with WSA error code <Error-Code-Number>

An error occurred when binding the TCP or UDP port to its address. The error number given refers to the Microsoft Windows Socket Error Codes.

16-0300

FDX Performance Warning: FDX-Frame-Update is used for one or more frames without registration from an up-to-date interaction layer dll. Usage of an up-to-date version of the interaction layer DLL can increase the performance of CANoe.

Note that a newer interaction layer DLL can improve performance.
