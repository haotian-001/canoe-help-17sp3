Simulink Model Viewer

Ribbon: Environment Tab|Model Viewer

General

The Simulink Model Viewer is used to display MATLAB/Simulink models. The Simulink Model Viewer provides fast access to a simulation model described with MATLAB/Simulink.
After installing the Simulink Interface add-on you can generate required information either from within CANoe or during the code generation process of MATLAB/Simulink.
The required information for the Simulink Model Viewer consists of one ini file and a set of *.emf files. These files are generated out of the Simulink model (*.mdl). The generation process requires a MATLAB/Simulink release 14 or later.

Functional Overview

Keyboard Operation

The following functions are available in the Model Viewer:

  • MATLAB/Simulink model is displayed.
    Subsystems are shown by double-click on the dedicated subsystem.
  • An explorer provides a hierarchical structure of the model with its subsystems and Simulink blocks.
    The context menu of such an item provides location functionality with the graphic part.
  • Search functionality for Simulink blocks and parameters is provided.
  • Signal values and parameter values can be shown in the model
  • Zoom functionality ease the usability of the Model Viewer.

Generate Model Viewer information out of Simulink model

The Model Viewer information can be generated:

  • in the configuration dialog of the Simulink tab, by clicking [Generate Info].
  • as part of the code generation process of the Simulink Coder, in case the CANoe target is selected. The Model Viewer information is automatically referenced in a vModule, if it is also generated.

Copy and Paste

Signal-, Environment- and System-variables are shown in the explorer within Simulink Model Viewer. These variables can be dropped (drag and drop) to any window in CANoe capable of working with these variables. Examples are Data Window, Graphics Window and Symbol Panel.

Edit Symbol values

You can open the Symbol Panel for each symbol that is shown in the Simulink Model Viewer with the context menu command Change Symbol value…. In the Symbol Panel you can view and edit/change the symbol's value.

Configuration

Notes

Using Simulink models

Simulink Model Viewer is displayed on Environment Ribbon Tab, if any Model Viewer information is assigned. To do this, open the Configuration dialog of a Node in Simulation Setup, switch to the Components page and add a vModule.

For using the Simulink Model Viewer you must enable Legacy Simulink integration in the Options dialog (File ribbon tab|Options|External Programs|Simulink). The option is enabled automatically at installation time of CANoe MATLAB Integration Package 2.0. Thereafter the configuration dialog for a node in the simulation view is extended by a Simulink tab.

The Simulink page provides the following functionality:

  • Assignment of the model to the node
  • Generation or direct assignment of information for the Simulink Model Viewer

Thereafter the Simulink Model Viewer can be used by double-click, selecting Show Simulink model in a (simulation) node's context menu, or select from the Environment ribbon tab.

To assign an analysis model or corresponding Model Viewer information, use the Simulink Analysis Models context menu command at the root element in the Measurement Setup, thus opening the dialog of the same name.

Supported MATLAB versions:

  • For generating information for the Simulink Model Viewer a MATLAB/Simulink release 14 or later is required.
  • The Model Viewer information generation process can take rather long for large and comprehensive Simulink models.
