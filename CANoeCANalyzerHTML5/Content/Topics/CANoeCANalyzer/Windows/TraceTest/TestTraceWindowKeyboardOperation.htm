Test Trace Window: Keyboard Operation

 

Copy events as text by means of the standard Windows key combinations

  • <Ctrl>+<c> for copy,
  • <Ctrl>+<v> for paste

Key combinations marked with a (*) are not contained in the context menu.

Other Key Combinations in the Test Trace Window

Description

Keys

Select all items

<Ctrl> + <a>

Activate the search field in the Trace toolbar (*)

<Ctrl> + <f>

Search forwards (*)

<F3>

Search backwards (*)

<Shift> + <F3>

Increase font size

<Ctrl> + <+>

Decrease font size

<Ctrl> + <->

Default font size

<Ctrl> + <0>

Note

The font size can also be changed with a scroll wheel of a mouse and simultaneously pressed <Ctrl> key.

The selected font size will not be stored in the configuration.
