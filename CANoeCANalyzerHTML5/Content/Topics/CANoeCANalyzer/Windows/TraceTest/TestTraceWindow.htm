Test Trace Window

 

General

The purpose of the Test Trace Window is to observe and analyze single or multiple tests during the test run. All executed actions of the tests are directly displayed in the Test Trace Window during test execution.

The Test Trace Window can be opened via the Test ribbon tab (only configured Test Trace Windows) or the Test Setup for Test Units.

The Test Trace Window provides various functions for analysis of the executed actions, such as:

  • Display and column filters
  • Various search functions for finding texts
