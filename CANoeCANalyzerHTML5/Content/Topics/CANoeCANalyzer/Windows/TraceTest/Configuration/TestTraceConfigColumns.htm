Test Trace Window Configuration: Columns

Shortcut Menu Path: Configuration|Columns

General

Columns

View

 

On this page you can select the columns to be included in the Test Trace Window and also specify the column titles and sequence.

Fields

Available Fields

Here the available fields for the column layout are displayed.

Displayed Fields

Here the selected column layout for the Test Trace Window is displayed. Only the visible columns are listed, indicating for each column the title and field. You can edit the column titles and also change the sequence of columns.

How to add/delete a column to the Test Trace Window:

  1. Select in the available fields the desired column with the mouse.
  2. Click on the button.

The new column is then inserted in the Displayed fields list (after the currently selected entry).

If no entry was selected, the new column will be added at the end of the list.
You can also select multiple columns and add them all in one step.
