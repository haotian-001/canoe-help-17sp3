Test Trace Window: Search

 

The Test Trace Window contains various search criteria in the Search page of the Test Trace Explorer.

The search is always started from the current cursor position in the display area. If the display area search has been completed, you can extend the search to the data history after the measurement. You can cancel the search at any time.

Note

  • All displayed columns are searched. Hidden columns are not taken into account in the search.
  • If the search is started during the measurement, the Test Trace Window goes into pause mode when the first event is found.

Text Search

With the text search, you have the ability to quickly and simply search for text within the Test Trace Window.

Enter the text or text part you would like to search for in the Search for this textfield.

Note

  • The wildcard (*) is automatically used for the search, e.g. *text*, text* and *text.
  • Wildcards are not automatically used if the Find whole words only option is selected.

You can also start the text search in the toolbar.

Search Options

Depending on the search criterion, the following options are available.

  • Match case
  • Find whole words only
  • Consider marker text
    Here you can specify whether the text of the set markers (marker name and comment) will also be searched during a text search.
  • Keep selection
    All search results remain selected and thus can be easily further processed.

Search Direction

Depending on the search criterion, you can search upwards or downwards.
