IL Configuration

Ribbon: Simulation Tab|Simulation Setup|Interaction Layers

General Information

The IL Configuration dialog (IL = Interaction Layer) offers you a straightforward way to display and edit transmission types.

You can also open the dialog with the IL Configuration context menu command of a node in the Simulation Setup.

The IL Configuration dialog can be used for AUTOSAR configurations up to and including AUTOSAR version 4.0.

Description

Application Example

The dialog evaluates the transmission type description of an interaction layer module (CANoe Interaction Layer, OEM Interaction Layer) from the database and displays it. The OEM-specific variants are taken into consideration here.

You can add and change transmission types without modifying the original database. All changes are highlighted in the IL Configuration dialog and saved in the CANoe configuration when you confirm the dialog with [OK].

Frame List / PDU List

Signal List

All messages or PDUs are displayed in this list. A line contains all information for a message or PDU that is relevant to the interaction layer (attributes, properties).

The signal properties of a selected message or PDU are displayed in this list. In some cases, the transmission of a message or PDU is tied to a signal condition.

The Frame / PDU list and the Signal list are handled in the same way:

  • You can add or remove columns using the Column Chooser command in the context menu of a column. The context menu also offers additional editing options for sorting and displaying.
  • The number of columns and the values result from the interaction layer that is active in each case.
  • You can change the individual values of the list or line.
  • Modified lines are identified by an asterisk * after the name (Name column); the text in this line becomes bold face.
  • You can reset the values from default values or to the values of the database using the context menu of a line. To reset the values of the currently selected line, open the context menu in an empty area of the list.

Toolbar

Properties

The following functions are available in the toolbar:

  • Enabled/Disabled
    You can use this button to activate or deactivate the configuration of the interaction layer on the tool side. In the case of deactivation, the original values from the database are applied.
  • Nodes Filter
    You can reduce the display of the selected nodes using the node filter.
  • Reset to
    These menu commands allow you to reset selected entries to the default values or the values of the database.

The right part of the dialog shows all properties of the selected object. These can be the properties of a frame, PDU, or signal.

You can also change values in this area of the window. The respective values are then also highlighted in bold face and the name of the object is marked with an asterisk.
