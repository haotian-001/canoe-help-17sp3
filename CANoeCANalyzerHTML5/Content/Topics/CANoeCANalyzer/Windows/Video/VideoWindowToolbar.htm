Video Window: Toolbar

 

The toolbar provides the following icons for operating the Video Window:

Icon

Function

Starts/stops a recorded or imported video after measurement stop.

Starts/stops the recording of a video file during a running measurement.

Displays the first frame of the video.

Displays the previous frame of the video.

Displays the next frame of the video.

Displays the last frame of the video.

Opens the Configuration dialog for the Video Window.

Imports a video file into the Video Window.

Displays/hides the navigation area.

Fits the video frame to the window

Fits the video frame to the window size.

Offset

Here, an offset can be specified for alignment of the displayed video file when window synchronization is activated.
The offset can also be entered in the configuration dialog.

Creates a Snapshot of the displayed video in format BMP. You find the images in the Snapshot folder that is configured in the Video Configuration dialog.
