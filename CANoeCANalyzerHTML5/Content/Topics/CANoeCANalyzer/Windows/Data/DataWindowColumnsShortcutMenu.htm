Columns Context Menu

 

Depending of the selected column different context menus are available.

Context Menu of the Value, Raw Value, Current Min and Current Max Columns

For the physical value, the raw data is extracted from the message and scaled with the (linear) conversion formula. The required data are derived from the database.

The raw data is extracted from the message only. It is not scaled by means of the conversion formula.

Binary
With this display type the value is shown as a binary value.

Decimal
The value is displayed as a floating-point number.

Hexadecimal
With this display type the raw value is shown as a hexadecimal value.

Science
The signal value is displayed as a scientific number, that is mantissa and exponent.

Symbolic
The value is shown symbolically.
The symbolic name is read from the database.
If no symbolic name is defined its value is shown decimally.

Text
The Value is interpreted as a sequence of ASCII characters.

Conditional Formatting...
Here you can define conditions to change the background color of the value cell depending on the current value of a symbol.

Decimal Places
You can configure the number of digits after the decimal point to be displayed.

The value display types available depend on the symbol type. If you configure the system variables in the Data Window, the same value is displayed for the physical and raw values. There is no conversion formula for system variables.

You can configure the display type for several signals simultaneously.
