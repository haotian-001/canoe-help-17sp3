Favorites

Favorites Column / Toolbar: Manage Favorites...

You can filter symbols of special interest via favorites.

Manage Favorites

You manage favorites in the Favorites dialog. You can open this dialog either via the Manage Favorites... command in the toolbar or in the Favorites column.

Here you can add (), edit, delete () favorites and change their order with drag-and-drop.
You can create a maximum of 16 favorites per Data Window.

Assign Favorites

You can assign individual symbols to a favorite using the menu in the Favorites column.

Activate Favorite Filter

By clicking on the favorites icon in the toolbar You can activate and deactivate the filter for a selected favorite.
Once you have selected a favorite via the toolbar, only the symbols assigned to that favorite will be displayed.

Notes about structured data types and user-defined groups

For structured data types it is possible that intermediate levels of other favorite assignments than the activated favorite filter are displayed to keep the data structure.

For structured data types, the set visibility level is not always reflected in a favorites filtered view.
The extension/reduction of a visibility level always takes place on the whole data structure, even if the whole data structure is not visible because of an activated favorites filter.

User-defined groups are always displayed as soon as at least one element of the group matches the activated favorite filter, even if the user-defined group itself is assigned to another favorite.
