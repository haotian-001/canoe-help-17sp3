Data Window

Ribbon|Analysis Tab|Data|View / New

The Data Window is used to display symbol values. The »symbol value is displayed as a physical value, as a raw value and with a progress bar by default.

Note

Up to two pages can be displayed in the Data Window, depending on the configuration.

If a configuration contains symbols that are not yet supported in the new default view, the Tabular page Branched page are displayed.
The following symbols are only supported in the tabular view: MOST signals, BAP signals, communication objects (CO), CAT parameters, J1587 signals and environment variables.
