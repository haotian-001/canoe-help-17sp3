Functional Overview

 

Configure the Display of the Data Window
  • Symbols
    You You can add/remove symbols in the Data Window using drag and drop or the context menu.
  • Columns
    There are different columns available to display measurement data.
  • Sorting Symbols
    You can move, sort, insert blank rows and change the color of the text manually to display the symbols like you want in the Data Window.
  • Display Symbolic Values
    Defined discrete values in the database value tables can be displayed in the Data Window if you select the Symbolic option in the Value or Raw value column.
  • Display Scientific Values
    The symbol value is displayed as a scientific number, that is mantissa and exponent.
  • Display Global and Local Minimums and Maximums
    On the Tabular page you can display minimums and maximums when showing columns Current Min/Current Max using context menu of the table head. Set the time interval using the context menu command Times Settings... to display local minimum and maximum values.
    On the Branched page, the value range is determined by the current min/max values during the measurement, you cannot configure any values.
  • Display Symbol Values as Progress Bars
    Symbol values can be displayed as progress bars in the Bar column. Define the symbol operating value range using the context menu command Settings in the Bar column.
  • Display 1-Bit Signal Inverted
    A 1-bit signal can be displayed inverted using the context menu command Display a 1-bit signal inverted in the Bar column.
  • Minimum and maximum values can be displayed additionally to the current value using the context menu command Display minimum and maximum value in the Bar column.

Other Functions
  • Logging of Data Window signals during measurement

To log signals from the Data Window manually or automatically during measurement, you can use the context menu command Logging Configuration. The signals are extracted from messages and binary stored in signal-based MDF files (MDF version 2.0–4.1).
