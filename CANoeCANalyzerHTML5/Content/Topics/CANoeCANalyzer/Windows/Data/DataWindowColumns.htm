Columns

 

You can use the context menu of the table head to configure how columns are displayed.

You can configure the following columns:

Column

Description

Page

State/Update Indicator

The activity display of the symbol is displayed in this column.

Tabular/Branched

Bar

The physical value is displayed in this column as a progress bar.

Tabular/Branched

Comment

If you have added a comment for the symbol in the database, it is displayed here.

Tabular/Branched

Current Max

The current maximum value of the symbol is displayed in this column.

Tabular/Branched

Current Min

The current minimum value of the symbol is displayed.

Tabular/Branched

DA

J1939 destination address

Tabular

Dest

Receiver (consumer) side of the communication with communication objects.

Tabular

Favorites

In this column you can assign defined favorites to symbols and open the favorites management.

Branched

FS

Only with Option AFDX

In addition to the default columns you can display the Functional State (FS) of a signal. To do this move the mouse cursor over the column header and open the context menu with a right click. Then select AFDX FS to display the column FS.

The FS attribute is not applicable for signals that represent a Functional State itself. For these signals the value -- is displayed. For signals in the same Functional Data Set the last transmitted Functional State is displayed.

Tabular

Last Value Time

The absolute time of the last value change since measurement start.

Branched

Name

The name of the symbol is displayed. You can configure the name view.
As default all information is displayed that is necessary for the unique identification. You can configure the display in the Options dialog.

For array elements you can configure filters.

Tabular/Branched

Object Detail

  • Details of the type of the distributed object member, e.g. function call or function return of method member type.

  • Details of the type of the communication object, e.g. function call or function return of Method type.

Tabular/Branched

Object Type

  • Type of the distributed object member.

  • Type of the communication object.

Tabular/Branched

Raw Value

You can configure the display type of the raw value.

Tabular/Branched

Role

Role of the distributed object (consumer/provider).

Branched

SA

J1939 source address

Tabular

Side

Specifies to which side the value update of the communication object belongs.

Tabular

Src

Sender (provider) side of the communication with communication objects.

Tabular

Unit

If a unit has been defined for the symbol, it is displayed here.

Tabular/Branched

Value

The physical value is extracted from the message raw data and scaled with the conversion formula. The display type of the value can be configured.

Tabular/Branched

The Adapt Column Width/Best Fit (All Columns) context menu command allows you to adjust all column widths to the width of the displayed content.

On the Branched page you can use the context menu command Best Fit to adjust the width of the column selected via the table header to its content.
