Toolbar (Data Window)

Context Menu: Show Toolbar…

The toolbar provides the following buttons for controlling the Data Window:

Symbol

Function

Pages

Tabular/Branched

Removes no longer updated entries of a dynamic list from the view of the Data Window.

Branched

With the Pause function, you can switch off the updating of the Data Window during longer measuring processes to view and analyze the measured values.

If you activate the Pause function again, the display of the Data Window is continued at the current measuring point.

Branched
