Filter for Arrays and Dynamic Lists

 

For arrays you can select which elements should be displayed in the view.

You can open the configuration of the filter by clicking on the filter icon (Select visible elements...).

Symbol

Description

Array without configured filter

Array with active configured filter

Array with inactive configured filter

You can activate, deactivate and reset (delete filter) the state of a configured filter using the context menu of the filter.

Arrays With Fixed Length

For fixed-length arrays, you can select individual elements from a list or type directly into the text box which elements you want to see.

Elements and ranges are separated by ;. A range from element a to b (both included) is expressed by a..b.

Dynamic Lists

For dynamic lists with and without maximum length you can define an open area with a..*. This will display all elements starting from a to the end of the list. This is especially useful for lists that may lengthen or shorten dynamically.

Activate Select from available, then you can choose which element you want to display from a list of currently available elements
