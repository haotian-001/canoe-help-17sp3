Display of Values as Bars

Table Head Context Menu: Bar
Columns Context Menu (Bar): Settings…

In the Bar column, the physical value is displayed in the form of a progress bar.

Minimum/Maximum

Tabular page

With the minimum and maximum value you can limit the value range of the symbol to the range that the symbol actually passes through.

Here, you have to configure the minimum and maximum value for the progress bar. To do this, open the configuration dialog via the Settings... context menu in the corresponding cell.

You can additionally display the minimum and maximum of received measured values in the Bar column. Select the context menu Display Minimum and Maximum Value in the Bar column. The selected cell is divided into 3 parts, where the upper part shows the minimum value, the middle part shows the current value and the lower part shows the maximum value.

Branched page

Here, you cannot configure any values here. Min/max are determined either via the symbol definition or during the measurement by the current min/max values.

Display a 1-bit Signal Inverted

Note

Only available on Tabular page.

You can display the value of a 1-bit signal inverted in the progress bar via the context menu Display a 1-bit signal inverted.

  • If the 1-bit signal has the value 0, a full bar is drawn in the Bar column.
  • If the signal has the value 1, no bar is displayed.
