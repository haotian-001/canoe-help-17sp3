drag and drop of Symbols between Windows

 

You can select the symbols shown in the Graphics Window and Data Window either from the Symbol Selection dialog / MOST Symbol Explorer or by drag and drop from

  • a Trace Window
  • the legend of another Graphics Window
  • another Data Window
  • the Symbol Explorer / MOST Symbol Explorer

Apart from signals and variables you also can drag parameters of an unfolded diagnostic primitive from a Trace Window to a Graphics or Data Window.

When you drag symbols from one Graphics or Data Window to another Graphics or Data Window they retain their preconfigured properties. Therefore you may wish to select a new color.

Note

drag and drop without pressing <Shift> copies a symbol from one window to the other window.
drag and drop with pressing <Shift> moves a symbol to the other window.

You can drag symbols and messages from the Trace Window in the fixed position mode, in Pause mode or after the end of measurement. When you drag a message to the Graphics or Data Window all of the message’s signals are pasted there too. In the Graphics Window the already passed values are displayed as signal curve.

Note

You can use the Clipboard instead of drag and drop.

You can also use either method to move symbols between two opened CANoe applications.

Note on drag and drop in the Graphics Window

  • System variables of type String are also disabled automatically.
