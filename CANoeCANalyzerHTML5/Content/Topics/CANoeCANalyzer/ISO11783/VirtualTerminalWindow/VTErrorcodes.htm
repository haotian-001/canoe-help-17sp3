Virtual Terminal Error Codes

 

The following errors can be output by the Virtual Terminal.

Error Code

Error Text

Description

500

Delete working set of node ... after timeout.

The maintenance message of the Working Set Master was not received within three seconds. The Working Set is deleted.

501

Initiating flag of the working set maintenance message of node ... is still 0.

After deleting the Working Set cause by time-out of the maintenance message, all further maintenance messages are responded with the NACK message until the initializing bit of it is set.

502

'Graphics context' (object ...): Value of 'number of polygon points' (...) is bigger than the number of following points.

The command for drawing a polygon contains less points than specified.

503

'Graphics context' (object) ...: Value of 'number of bytes' (...) is bigger than the number of following bytes.

The command for drawing text contains less bytes than specified.

504

Cannot assign auxiliary input (...) to auxiliary function (...) because single-assignment flag is set.

An auxiliary function cannot assigned multiple because the single-assignment of the auxiliary input is activated.

505

Virtual Terminal changes function instance of the device name from ... to ... because original function instance is already used by another Virtual Terminal.

Virtual Terminal is stopped because function instance ... of the device name is already used by another Virtual Terminal and there can be found no other unused function instance.

The Virtual Terminal could not successfully execute the Address Claiming because no clear function instance for the device name could be found.

1000

Cannot create timer to check function instances.

Internal error: A timer could not be set.

1001

'Change active mask' command is sent to unknown object ID (ID=...).

A Virtual Terminal command references an unknown object ID. The command and the ID are listed in the error text.

1002

'Change font attribute' (object ...) uses invalid color (...).

A Virtual Terminal command contains an invalid value. The command and the ID are listed in the error text.

1003

Input boolean object (...) does not reference to an object of type number variable (...).

A boolean input object references to an object of an object, which is not of type "Number".

1004

'Change active mask' command is sent to a non Working Set object (ID=...).

A Virtual Terminal command references to an object which does not have the correct object type for this command.

1005

Object ...: Cannot handle unknown macro function ... .

The macro contains an unknown function.

1006

'Change attribute' (object ...) contains invalid attribute id (...).

'Get attribute value' (object ...) contains invalid attribute id (...).

The "Get/Change Attribute" command contains an invalid attribute ID.

1007

No valid response received for assignment of input object ... (node ...) to function object ... .

The response of an auxiliary assignment is missing or contains an error.

1008

Read/write over end of message data block.

A Virtual Terminal command message contains invalid data.

1009

Received message VT21 (PGN=0xE700, SA=..., FCT=...) but no Working Set is assigned to this address.

A Virtual Terminal command was received from an unknown Working Set Master.

1010

'Object Pool transfer': Error(s) in Object Pool of Working Set of node ... while extracting object ... .

The message "Object Pool transfer" contains invalid objects.

1011

'StoreVersion': error(s) while storing version '...' of Working Set (node ...).

The command "StoreVersion" could not be executed, probably because an abortive file access.

1012

'LoadVersion': version label '...' of Working Set (node ...) is not correct or unknown.

'LoadVersion': error(s) in version '...' of Working Set (node ...).

The command "Load Version" could not be executed. No object pool with the set label is saved or the saved object pool contains errors.

1013

'Delete Version': version label '...' of Working Set (node ...) is not correct or unknown.

A saved object pool could not be deleted with the "Delete Version" command. No object pool with the set label is existing or the file access failed.

1014

'Change active mask' command is sent to an unselectable Working Set object (ID=...).

The command "Change active mask" could not be executed. The Working Set object is not selectable.

1015

Command (...) cannot be handled by object (ID=...).

A Virtual Terminal command could not be executed with the referenced object because the object does not support this command.

1016

'Object Pool transfer': Cannot build bitmap of object ... .

The bitmap object of an object pool contains invalid data.

1017

'Object Pool transfer': Unknown object type ... is used (object pool will be deleted).

An object pool contains an object with unknown object type.

1018

'Object Pool transfer': ID ... of macro object is bigger than 255.

The object ID of a macro objects is out of range. The valid range is 0..255.

1020

The received message VT21 (PGN=0xE700, SA=...d, FCT=...d) is sent to global destination address.

A Virtual Terminal command was sent to target address 255 (global). It must be sent to the address of the Virtual Terminal instead.
