VT Diagnostic Requests

 

The following information of the Virtual Terminal can be queried using a diagnostic request:

Message

PGN

Content of the Response

Product identification

0xFC8D

Character string 001*CANoe Virtual Terminal*<current CANoe version>*.

Control function functionalities

0xFC8E

Functionality Universal Terminal Server with configured VT version.

ECU identification information

0xFDC5

Character string 001*<current CANoe version>*Unknown*Virtual Terminal*Vector Informatik GmbH*01*.

Diagnostic protocol

0xFD32

Note, that no further diagnostic protocols are supported.

ISOBUS certification

0xFD42

Note, that no certification exists.

Language command

0xFE0F

Language settings, date and time format and units.

Active diagnostic trouble codes (DM1)

0xFECA

Note, that there are no errors.

Previously active diagnostic trouble codes (DM2)

0xFECB

Note, that there are no errors.

Diagnostic data clear/reset previously active DTCs (DM3)

0xFECC

As response to this request a positive acknowledgment message is sent.

ECU software identification

0xFEDA

A software identification field with following content Virtual Terminal <current CANoe version>.

Time/Date

0xFEE6

Current date and current time.
