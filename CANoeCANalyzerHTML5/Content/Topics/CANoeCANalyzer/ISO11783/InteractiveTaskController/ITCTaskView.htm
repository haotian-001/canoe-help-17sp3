Task view of the Interactive Task Controller

 

The task view shows the available tasks. You can modify these tasks (Treatment zones, peer control assignments and process data), create new tasks and delete existing tasks. During measurement you can send the values of a treatment zone or a single process value to the implement with the [Set] buttons.

To show the properties of a task select an entry below the entry Tasks in the tree view. On the right side of the dialog the properties of the selected task are shown.

To create a new task open the context menu of the entry Tasks in the tree view and select the menu command Add Task. A new task is created for which you can set the designator, the default treatment zone, further treatment zones and peer control assignments.

To delete a task, select the task and open the context menu. Select the entry Remove, the task will be deleted.

Information

In this section the designator is shown and you can select the default treatment zone (Default Zone). This is applied automatically when a task is started.

Treatment Zones

In this section you can modify and delete existing treatment zones and its process data, as well as create new treatment zones and process data.

To create and delete treatment zones and process data the following buttons are available:

Button

Description

creates a new treatment zone

creates a new entry for the selected treatment zone

deletes the selected entry

With [] you can insert new process data. Afterwards you can insert the values manually. If the entered process data can be assigned to adequate device process data, the designator and the values with unit are taken from these device process data. In this case raw data and physical values are automatically converted and displayed.

If you don't want to insert the process data manually, you can select the available process data from an overview list. Use the button [...] to open the overview. Select the process data and close the dialog with [OK]. The values are automatically inserted in the appropriate columns.

To change the values of process data later, double-click in the column Value and insert the new value. Alternatively you can use the key <F2>.

Peer Control

If the version of the Interactive Task Controller is at least 4 (see the Configuration of the Interactive Task Controller page), the process data which can be used as the Setpoint Value User is displayed in this section. Use the […] button to assign a Setpoint Value Source. If the task is started, the Interactive Task Controller attempts to set up the peer control assignments by sending the corresponding commands to the Setpoint Value Source and Setpoint Value User. A successful setup is indicated by the icon.

To remove the Setpoint Value User reopen the Assignment dialog with the […] button and click the [Reset] button. If the task is still active, the Interactive Task Controller sends commands to the Setpoint Value Source and the Setpoint Value User, to terminate the connection.
