Device name

 

The 64-bit device name of an ECU consists of the following individual segments:

Segment

Description

Arbitrary Address Capable (AAC)

represents the arbitrary address capable flag

Industry Group

represents the industry group the device belongs to

Vehicle System Instance

represents the instance of the vehicle system

Vehicle System

represents the vehicle system

Function

represents the function

Function Instance

represents the function instance

If the check box Dynamic is activated, the Interactive Task Controller checks before registering to the network, if the selected Function Instance is already used by another Interactive Task Controller or Data Logger. In that case the Interactive Task Controller changes its Function Instance to an unused value.

ECU Instance

represents the ECU instance

Manufacturer Code

represents the manufacturer code of the device

Identity Number

represents the identity number
