Object Pool API

 

The Object Pool API of the ISO11783 Node Layer manages the object pool which is needed for the user interface on a Virtual Terminal. It is part of the ISO11783 Node Layer and controlled with CAPL functions. The Object Pool API requires an object pool file (IOP).

 

Features

  • Connection Management with the Virtual Terminal (Initialization procedure, Maintenance message)
  • Load an object pool file (IOP)
  • Transmit the object pool to the Virtual Terminal
  • Access on the local object pool
  • Sending commands to the Virtual Terminal
  • Handle messages from the Virtual Terminal
  • Bind object values from the object pool to environment variables
  • Management of auxiliary inputs and functions
