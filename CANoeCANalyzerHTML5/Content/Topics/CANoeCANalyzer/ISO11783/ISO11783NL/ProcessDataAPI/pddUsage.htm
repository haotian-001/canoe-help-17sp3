Process Data API - Usage

 

To simulate a process data dictionary, the following additional steps must also be executed:

  1. After an ECU has been created with Iso11783CreateECU the process data dictionary can be created. To do this, the CAPL function Iso11783PDDLoadDeviceDescription is used. The handle of the ECU is needed as a parameter. Furthermore, the filename of the machine configuration file must also be entered. The process data dictionary is created after this configuration file and then contains all process variables with the initial value 0.
  2. Next, the process variables must be initialized. To do this, the function Iso11783PDDSetValue can be used.
  3. Process variables that are to be changed during the measurement or displayed in a panel can be combined with an environment variable. Every change to a process variable then has an impact on the environment variable and vice versa. The function Iso11783PDDConnectEnvVar can be used to combine a process variable with an environment variable. The environment variable must be defined in a database.
  4. During the measurement, the process variables can be set with Iso11783PDDSetValue or requested with Iso11783PDDGetValue.
  5. So that the CAPL program can react to changes to the process variables, the callback function Iso11783PPDOnDataChanged must be implemented. This function is called up by the node layer when a value command has been received and the process variable changed. The CAPL program is responsible for the process variables being applied at sub-element.

The following example demonstrates the steps to load a device description file and connect to the Task Controller.

Example

ecuHandle = Iso11783CreateECU( gECUName );
if (Iso11783PDDLoadDeviceDescription( ecuHandle, "Sprayer.XML" ) == 0) {
  Iso11783PDDSetValue( ecuHandle, 0x0048, 0, 10 );
  Iso11783PDDSetValue( ecuHandle, 0x0048, 1, 20 );
  Iso11783PDDSetValue( ecuHandle, 0x0048, 2, 30 );
  
  Iso11783PDDConnectEnvVar( ecuHandle, 0x0048, 1, "EvSprayer_TankVolume" );
  Iso11783PDDConnectEnvVar( ecuHandle, 0x0052, 2, "EvSprayer_AppRatePerArea" );
}
else {
  write( "Error while load task file" );
}
Iso11783ECUGoOnline(handle, gECUAddress );

The CAPL functions of the process data API are listed in the CAPL functions overview.
