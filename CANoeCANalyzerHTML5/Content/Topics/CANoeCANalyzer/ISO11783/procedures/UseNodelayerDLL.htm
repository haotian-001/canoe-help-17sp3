Interaction Layer Integration

 

In the CANoe option. ISO11783 different interaction layer can be used to implement the functionality of a simulated node. Each interaction layer provides you with many other context-specific CAPL functions that can be used to influence the behavior of the simulated node. Assign the interaction layer to the node in the CANoeSimulation Setup so that it can be used there. To do this, use one of the methods described on the Assigning Modeling Libraries page.

Using an Interaction Layer in vTESTstudio

The CAPL functions of an interaction layer are not available in vTESTstudio by default. To use it in vTESTstudio, you have to insert the DLL from the CANoe folder Exec32 as modeling library in the Test Unit configuration.

Copying the Interaction Layer DLL from the CANoe folder Exec32 and storing it in a subfolder of the vTESTstudio project has proven successful. If the Test Unit is executed in CANoe, the Interaction Layer DLL from CANoe is used.
