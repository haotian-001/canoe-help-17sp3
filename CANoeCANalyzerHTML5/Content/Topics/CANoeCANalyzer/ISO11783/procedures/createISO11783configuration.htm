ISO11783 » How to create a configuration for ISO11783

Create ISO11783 Configuration

 

When you create a new configuration from a ISO11783 template, it is already prepared to work in a ISO11783 network:

  • A ISO11783-specific baudrate is set.
  • For ISO11783 helpful windows are configured.
  • Possibly the ISO11783 standard database is already assigned.

To build an enhanced ISO11783 configuration that includes a standard ISO11783 database, follow the steps below:

  1. In File Ribbon Tab select the New command.

    The available templates are displayed.
  2. Select one of the a ISO11783 templates, whose icon has a yellow folder symbol, and click on Create Configuration From Template.
  3. Select a folder where the new configuration will be stored. Confirm with [OK].
