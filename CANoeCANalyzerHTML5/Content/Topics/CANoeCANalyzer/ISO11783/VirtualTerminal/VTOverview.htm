Virtual Terminal

 

General
  • In an ISO11783 network one of the important components is the Virtual Terminal (VT). CANoe Option .ISO11783 offers the whole functionality of such a terminal.
  • Besides, big value was put on the flexibility with the description of the qualities of the terminal. So it can be configured, e.g. whether the simulated terminal supports color or black and white representation. The number of the Softkeys as well as the resolution are configurable freely. With it all possible terminals are available to the user with one tool. Manufacturers of Virtual Terminals can check therefore without big expenditure different operating drafts.
  • With only a few steps you can activate a Virtual Terminal and work with it. Additional options are available to analyze an object pool, so all objects of the object pool can be displayed and analyzed in an additional view.
  • Within the passive mode the bus traffic of a real Virtual Terminal can be monitored and the object pool can be analyzed in detail in the CANoe Virtual Terminal if any errors occur.
  • Starting with version 6, the VT adds a transaction number (TAN) to the Activation Messages and controls the corresponding Response Messages.
