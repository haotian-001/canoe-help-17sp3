Configuration

 

After setting up the ISO11783 IL, perform the following additional steps in the database:

  • Add the message PD (PGN CB00h) to the Tx list of the node.
  • Set the node attributes ISO11783PDDFilename and ISO11783PDDVersion.

Version of the ISO11783 specification (part 10) supported by the interaction layer.

Alternatively, the process data directory file and the version can also be specified in the Component Configuration dialog of the node in the Simulation Setup or at runtime via the CAPL commands Iso11783IL_PDDLoadDeviceDescription and Iso11783IL_PDDSetParameter.
