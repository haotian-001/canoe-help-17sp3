Bus Statistics Window

 

The Bus Statistics Window displays ARINC-429-specific data and values.

Note

All bus statistics values are available as system variables: SysVars::_Statistics::A429_x::<variable>

  • x: ARINC-429 channel number (1, 2, 3 â¦)
  • <variable>: Value from the Variable column

The following information is available:

Statistics Value

Variable

Description

Direction

Dir

Communication direction

  • Rx for receive channels
  • Tx for transmit channels
  • Tx (Sim.) in Simulated bus mode

Max. Bitrate [bit/s]

MaxBitRate

Maximum communication speed of the connected channel

  • Tx: n/a (not applicable) is always displayed.
  • Rx: Maximum value of Bitrate [bit/s]

Bitrate [bit/s]

BitRate

Communication speed of the connected channel

  • Tx: The setting of the driver is applied as initial value.
  • Rx: Value for the last received message

Min. Bitrate [bit/s]

MinBitRate

Minimum communication speed of the connected channel

  • Tx: n/a (not applicable) is always displayed.
  • Rx: Minimum value of Bitrate [bit/s]

Busload [%]

BusLoad

Current busload

Peak load [%]

PeakLoad

Maximum value of Busload [%]

Words [fr]

TotalWords

Number of ARINC 429 words that were processed error-free

Word rate [fr/s]

WordRate

Number of ARINC 429 words per second that were processed error-free

This error detection is also dependent on the set (channel) parameters.

Word errors [fr]

WordErrors

Number of ARINC 429 words that were processed with set error flag

Line errors [fr]

LineErrors

Number of detected channel errors

These are errors that prevented transmission of an ARINC word. This error detection is independent on the set (channel) parameters.
