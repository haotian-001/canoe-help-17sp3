Transmit List

 

Via the Send List view you can configure single words that you want to send over the ARINC 429 bus. You can add ARINC 429 words from an assigned database via the symbolic selection dialog and adjust their properties before sending. Alternatively you can add raw value defined ARINC 429 words (without using the database) and modify the label or further properties.

To customize the columns use the toolbar button . The view ARINC 429 word Properties displays all properties of the ARINC 429 word.

The Send List view displays the following columns.

Column

Options

Displayed by default

Description

Num.

—

Yes

Displays the row number of the ARINC 429 word.

Send

Yes

If the trigger condition type is manual, this button sends the ARINC 429 word.

If the trigger condition type is periodic, this starts the periodic transmission of the ARINC 429 word on the set channel.
 

—

If the trigger condition is periodic and currently active, this button appears. Clicking it stops the periodic transmission of the ARINC 429 word.

Trigger

Manual / On Key

Yes

The ARINC 429 word is send on bus when the button  is pressed. Additionally a trigger key can be defined for the frame. If the key is pressed the ARINC 429 word is send on bus. Manual and OnKey trigger conditions can be used together.
 

Periodic

No

The ARINC 429 word is send on bus, with the defined period, after the button  is pressed. If the (too small) cycle time is set leading to a busload of over 100 % with the current set transmission interval or bitrate of the ARINC 429 channel, an error notification is output on the left of the cyle time entry field.

Name

—

Yes

The name of the ARINC 429 word from an assigned database is used (not editable).

Label

—

No

The identifier (label byte) of the ARINC 429 word.

Channel

—

Yes

The ARINC 429 channel the ARINC 429 word should be send on.

Chn. Presets

Acive

Yes

If channel presettings are activated for a specific ARINC 429 word, the properties parity and min. gap from the currently valid settings of the ARINC 429 channel are used. The columns parity calculation and gap are not editable in this mode.

Not Active

—

ARINC 429 channel settings do not have any influence on properties of the ARINC 429 word. Values from the columns parity calculation and gap are used for the transmission. These columns are not editable.

DB Comment

—

No

Displays the comment from the database assigned to the ARINC 429 word.

Comment

—

No

Editable user defined comment for the ARINC 429 word.

Word Gap

 

No

Gap to be respected between ARINC 429 words following one another. Entry in bit time in 0,5 increments (1..128 bit times).

Parity Calculation

 

No

Parity calculation for the ARINC 429 word: even, uneven, deactivated

Tx. Settings

 

Yes

Displays the overview of the properties of the ARINC 429 word that can be set both locally as well as in the channel presettings.

To show the context menu set a right click within the Send List view. The context menu shows following commands:

Command

Description

Add ARINC-429-Word from database

The toolbar dialog is opened to add an ARINC 429 word from an assigned database.

Add ARINC-429-Word

Add an ARINC 429 word (without database connection) parameterizable via raw values.

Cut

cuts the selected ARINC 429 word from the Send List view and stores it in the clipboard.

Copy

Copies the selected ARINC 429 words to the clipboard.

Paste

Adds the previously copied ARINC 429 words under the currently selected word. Adding ARINC 429 words copied to the CANOE/ CANalyzer Trace Window is supported.

Delete

Deletes the selected ARINC 429 words from the view Send List.

Select all

Selects all ARINC 429 words in the view Send List.

To change the position of one or more ARINC 429 words within the Send List view, a drag and drop functionality is available.
