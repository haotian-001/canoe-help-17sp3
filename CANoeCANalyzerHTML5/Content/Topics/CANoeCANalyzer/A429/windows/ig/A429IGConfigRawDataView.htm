Raw Data

 

The view raw data allows editing single bytes of the ARINC 429 words selected in the view Send List. It is possible to switch between the view Raw Data and the view Signals if the ARINC 429 word is defined in the database.

The label value of the ARINC word (first byte of the word) can only be modified in the view Send List. The remaining 3 bytes of the ARINC 429 word contain the optional bit fields SDI and SSM, signal data bits and the parity bit.

In the view raw data, two modes for data input are supported.

Signal Data Bits

Integer, hexadecimal value

0..variable

A part of the ARINC 429 word limited by SDI, SSM and Parity bit that can be used for the transmission of the signal values.

The start and end bit of the area signal data bits is not constant and depends on the usage of SDI and/or SSM of the ARINC 429 word.

There is no coding of the entered data (e.g. in BCD-Format) but only a conversion to an integer value without sign that is directly entered into the ARINC 429 word.

Bytewise Parameterization

In the upper area of the view raw data, data of the ARINC 429 word can be edited bytewise. It is possible to add values from the clipboard and to navigate between the bytes by means of the keyboard. The editing mode of a byte must be activated before the value can be edited. To activate the editing mode, double click the byte value or press <F2>.

To display the context menu, right click the entry-field of a byte. The context menu contains the following commands.

Command

Description

Copy

Copies the selected value of a byte to the clipboard.

Paste

Pastes the copied bytes from the clipboard.

Note: It is possible to paste values from the clipboard. The values must have the format 01 02 03.

Reset selection

The currently selected and focused byte is reset to 0.

Reset all

All editable byte values are reset to 0.

Byte 0 (Label) of the ARINC 429 words can only be modified in the view Send List.
