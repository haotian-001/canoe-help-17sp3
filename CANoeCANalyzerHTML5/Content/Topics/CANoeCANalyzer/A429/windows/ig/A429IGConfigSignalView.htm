Signals

 

The Signals view shows all signals of a selected ARINC 429 Word from an assigned database. It is also possible to toggle between the Signals view and the Raw Data view. The customization of the columns can be configured with the toolbar button .

Note

If the frame is no ARINC 429 word from an assigned database or not connected to the database, the Signals view is disabled. Only the Raw Data view is available in this case.

Column

Description

Name

The database name of the signal.

Raw value

The raw value of the signal can be increased or decreased by the defined step size via the arrow buttons.

Raw step

The raw step size. Default is 1/20 of the raw value range.

Phys. value

Physical value of the signal, user-configurable

Converting the raw value into the physical value and vice versa is completed using the conversion formula: [physical value] = ( [raw value] * [factor] ) + [offset]

Phys. step

The physical value step size. Default is 1/20 of the physical value range.

Unit

Physical unit of the signal value, taken from the database.

StartBit

Start bit of the signal inside the ARINC 429 word payload, taken from the database.

Length

The length of the signal.

DB comment

(Not displayed by default)

The comment for the signal in the database.

Generator settings

The Signal Generator settings of the signal.

  • Starts the Signal Generator (appears only when the generator is not started).
  • Stops the Signal Generator.
  • Pauses the Signal Generator (appears only when the generator is started).
  • Opens settings of the Signal Generator.

Generator Type

The generator type of the signal. See Signal Generator.

General

You only need to enter the raw value or physical value, since the corresponding value is always also calculated by the signal's conversion formula.

If intermediate values cannot be represented as raw values when entering physical signal values, they are automatically rounded up or down to the next closest value.

Depending on the signal type, the representation forms of raw and physical values may deviate between signals.

Signal Types

The properties of the signal of an ARINC 429 word are set by means of signal description in the database. The following signal properties are specially handled in the user interface:

Value tables

Predefined couples (signal value and value description) of a signal in the database are displayed in form of a table in the column phys value in the view Signals and can be selected there.

Multiplexor Signal

When signal-multiplexing, various signals are transferred in one ARINC 429 word depending on a multiplex value.

  • The signal that contains the multiplex value is designated as the multiplexor signal.
  • The signals that are transferred depending on the multiplex value are designated as multiplexed signals.

The multiplex value indicates the current valid signals from all possible word signals. The following appears in the configuration dialog of the Interactive Generator:

  • multiplexor signals
  • multiplexed signals

If the multiplex value is changed, the multiplexed signals are automatically updated.

Note

If defined values in the Interactive Generator in the database are changed, the corresponding field is shaded orange.

Example

The signal value entered in the Raw Value or Phys Value columns are outside the value range that was defined in the database using a minimum and maximum value.
