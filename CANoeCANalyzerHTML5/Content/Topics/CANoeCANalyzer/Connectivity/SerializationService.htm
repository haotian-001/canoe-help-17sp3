Serialization Service

 

The serialization service provides a simple solution to serialize and deserialize data. For this purpose, an object with the attribute [Binding="SerializationService"] is built in the data model, in which the methods required for serialization or deserialization are defined. These methods can be called in application models or in the application panel. All members of an object in the serialization service must be of type consumed method. Methods for serialization must have exactly one parameter of the data type to be serialized and the return type bytes, while the reverse is true for methods for deserialization.

Attributes To Use With the Serialization Service

The following attributes can be used to configure the serialization service:

Attribute

Values

Description

Binding

SerializationService

Defines the object as part of the serialization service.

Serialization

Default, JSON, GoogleProtobuf, Plain

The serialization format to be used. The serialization service does not support a default value or Plain serialization. Therefore, these values are invalid for the attribute. For more details on the particular serialization, see the help pages for JSON and GoogleProtobuf.

SerializationService::Method

Deserialize, Serialize

Indicates whether the defined method should serialize or deserialize.

Example

In the following example, a serialization service object is created in the data model and used in application models for serialization and deserialization using JSON. The data and the serialized byte string are output in the Write Window. The two application models in C# and Python have the same functionality and can be used alternatively to each other.

vCDL

version 1.4;

import module "SerializationService";

namespace SerializationExample
{
  struct ExampleType
{
    int32 number;
    string text;
  }

  [Binding="SerializationService"]
  [Serialization="JSON"]
  interface ISerializer
  {
    [SerializationService::Method=Serialize]
    consumed method bytes ToJSON(ExampleType input);

    [SerializationService::Method=Deserialize]
    consumed method ExampleType FromJSON(bytes input);
  }

  ISerializer serializer;
}

C#

using Vector.Tools;
using Vector.CANoe.Runtime;

public class Application : MeasurementScript
{
  public override void Start()
  {
    //Serialization

    var dataValue = new SerializationExample.ExampleType();

    dataValue.number.ImplValue = 25;
    dataValue.text = "This is an example text for JSON Serialization";

    SerializationExample.serializer.ToJSON.CallAsync(dataValue, (serializedData) => {
      Output.WriteLine(
        "______________________________________________________\n" +
        "JSON Serialization finished.\n" +
        "Input data:\n" +
        " number = " + dataValue.number.ImplValue + "\n" +
        " text = '" + dataValue.text + "'\n" +
        "Serialized data:\n" +
        System.Text.Encoding.ASCII.GetString(serializedData) + "\n" +
        "______________________________________________________"
      );
    });

    //Deserialization

    var serializedValue = System.Text.Encoding.ASCII.GetBytes("{\"number\":42,\"text\":\"This is an example text for JSON Deserialization\"}");

    SerializationExample.serializer.FromJSON.CallAsync(serializedValue, (deserializedData) => {
      Output.WriteLine(
        "______________________________________________________\n" +
        "JSON Deserialization finished.\n" +
        "Input data:\n" +
        System.Text.Encoding.ASCII.GetString(serializedValue) + "\n" +
        "Deserialized data:\n" +
        " number = " + deserializedData.number.ImplValue + "\n" +
        " text = '" + deserializedData.text + "'\n" +

        "______________________________________________________"
      );
    });
  }
}

Python

import vector.canoe
import application_layer

@vector.canoe.measurement_script
class Application:
  def start(self):

    # Serialization

    dataValue = application_layer.SerializationExample.ExampleType()

    dataValue.number = 25
    dataValue.text = "This is an example text for JSON Serialization"

    application_layer.SerializationExample.serializer.ToJSON.call_async(
      dataValue,
      lambda serializedData: vector.canoe.write(
        """______________________________________________________
JSON Serialization finished.
Input data:
number = %d
text = '%s'
Serialized data:
%s
______________________________________________________"""
        % (dataValue.number, dataValue.text, serializedData.decode("utf-8"))
      ),
    )

    # Deserialization

    serializedValue = '{"number":42,"text":"This is an example text for JSON Deserialization"}'.encode(
      "utf-8"
    )

    application_layer.SerializationExample.serializer.FromJSON.call_async(
      serializedValue,
      lambda deserializedData: vector.canoe.write(
        """______________________________________________________
JSON Deserialization finished.
Input data:
%s
Deserialized data:
number = %d
text = '%s'
______________________________________________________"""
        % (
          serializedValue.decode("utf-8"),
          deserializedData.number,
          deserializedData.text,
        )
      ),
    )

on start
{
  // Serialization
  struct SerializationExample::ExampleType dataValue;
  callContext SerializationExample::serializer.ToJSON callContextSerialize;
  char output[100];

  // Deserialization
  char serializedValue[100] = "{\"number\":42,\"text\":\"This is an example text for JSON Deserialization\"}";
  callContext SerializationExample::serializer.FromJSON callContextDeserialize;
  byte input[100];


  // Serialization
  dataValue.text = "This is an example text for JSON Serialization";
  dataValue.number = 25;

  callContextSerialize = SerializationExample::serializer.ToJSON.CallAsync(dataValue);
  memcpy(output, callContextSerialize.Result);

  write("______________________________________________________");
  write("JSON Serialization finished.");
  write("Input data:");
  write(" number = %d", dataValue.number);
  write(" text = '%s'", dataValue.text);
  write("Serialized data:");
  write("%s", output);
  write("______________________________________________________");

  // Deserialization
  memcpy(input, serializedValue, 100);
  callContextDeserialize = SerializationExample::serializer.FromJSON.CallAsync(input);

  write("______________________________________________________");
  write("JSON Deserialization finished.");
  write("Input data:");
  write("%s", serializedValue);
  write("Deserialized data:");
  write(" number = %d", callContextDeserialize.Result.number);
  write(" text = '%s'", callContextDeserialize.Result.text);
  write("______________________________________________________");
}
