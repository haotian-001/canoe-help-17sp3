MQTT Data Types

 

The MQTT binding loads certain predefined data types that are used in the MQTT Client API and in the MQTT fault injection:

1. _MQTT::DataTypes::MQTTPublishMessage

Name

_MQTT::DataTypes:: MQTTPublishMessage

Data Type

Struct

Member

Name

Data Type

Description

Payload

Bytes

Describes the payload of the MQTT message.

PublishTopic

String

The topic to to which the message should be sent.

QoS

Uint8

The used Quality of Service level for the message.

Retain

Bool

This parameter decides whether the message should be stored on the broker by the retain mechanism or not.

DUP

Bool

Marks whether the message was sent for the first time or whether it is a resend attempt for the message. When sending a message for the first time, DUP must always be false.
