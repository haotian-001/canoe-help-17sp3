TCP/UDP Ports used by CANoe

 

Note

  • Note that the tables below show the default ports used for communication. If you change port usage settings, you have to enable the corresponding ports.

  • In addition to the specified ports, on both the RT server and user computer sides, the ports used by Vector Tool Platform must be taken into account, see TCP/UDP ports used by Vector Tool Platform.
