PortLink

 

General

PortLink is used to link CANoe environment variables to analog and digital I/O ports on I/O cards.

The value of an environment variable is either read to a port or output by a port.

Glossary

Additional PortLink Topics
  • Device DLL
    A device DLL supports one or more I/O cards.
    In the Options dialog you can choose the device DLLs that should be used for the configuration.
  • Device
    A device is a I/O card.
  • Device Number
    Each device gets a device number.
    Device numbers are unique within a device DLL.
  • DLL Identifier
    The internal name of the device DLL.
  • Port Name
    The port name as used in the configuration of the I/O card.
  • I/O Cards
  • Configuration
  • Porting a PortLink configuration to another computer

Notes on Performance

  • Performance is very dependent on the type of I/O card and the number of ports used.
  • It may take up to 300µs to read in an analog port.
  • If several analog ports are being used, the user must make sure that the polling interval is not too short.
