GL3000/GL4000/GL5000

Ribbon: File Tab|Options|Extensions|GL Logger

General

You can use the GL3000, GL4000 and GL5000 loggers as network interface devices in CANoe and CANalyzer. The logger is connected to the CANoe computer via Ethernet connection. Once the measurement begins, the logger sends the bus data to CANoe. The data can then be analyzed in the Measurement Setup. During measurement the logger does not write data to the ring buffers. Classifications are continued even during measurement.

Configuring CANoe

Configuring the Logger
  1. Create a new configuration from a template.
  2. Activate GL Logger mode (as shown below).
  3. Specify the number of CAN, LIN, FlexRay and MOST150 channels.
  4. If you want to use databases, you need to create networks in the Simulation Setup and assign the databases to them.
  5. Open the network hardware configuration dialog and specify the IP address and port for the logger.
  6. Click Connect.
    If the logger is recognized, its configuration data are listed. The list remains empty if the logger is not recognized. In this case, please check the network connection, the IP address and the port.
  1. Open the Vector Logger Suite.
  2. Create a new project or open an existing project via File ribbon tab (Create Configuration/Load Configuration) and add interface functionality to it.
  3. Enable the interface mode on page Hardware|Interface Mode.
  4. Set the IP address of the logger and adjust the subnet mask and the port if necessary. Make sure that the settings match to your computer or network.
  5. Optionally you can change the channel and the CAN ID for the marker message that marks the beginning and the end of the interface mode in the logging file.
  6. In the Transmit behavior section you can set which messages, sent by the logger, should be displayed in CANoe if required. In the same way you can specify whether Error Frames are displayed.
  7. Load the configuration to the logger (see Vector Logger Suite manual).

Activating GL-Logger Mode

Setting the Network Parameters
  • If you want to use a GL Logger as an interface device, you need to turn GL Logger mode on in CANoe Options dialog (Extensions|GL Logger).
  • In GL Logger mode, you can only use GL3000, GL4000 or GL5000 loggers as interface devices. You cannot operate these alongside other interfaces.
  1. Set the IP address and the port for the logger.
  2. Click Connect.
    If the logger is recognized, its configuration data are listed.

Further Information
  • CAN, LIN, FlexRay and MOST150 bus systems are supported. You can insert an additional interface card or license dongle to provide the licenses required for CANoe.
  • It is not possible to transmit messages. The Simulation Setup is therefore deactivated. Messages (real and virtual) sent by the logger configuration are still sent after measurement start. They are displayed as Tx messages if configured properly in the logger.
  • You can use databases. This lets you display data received by the logger symbolically and analyze it in CAPL blocks.
  • You cannot operate a mix of loggers and other network interfaces at the same time.
