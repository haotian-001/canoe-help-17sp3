Installation of the MATLAB Integration Package

 

System Requirements

The CANoe MATLAB Integration Package requires the following minimum product versions:

  • CANoe 7.5
  • MATLAB® R2014a 64 bit
  • Visual C++® 12.0 (Visual Studio 2013)

Installation

You can start the setup for the MATLAB Integration Package with help of the following hyperlink: Open Setup Folder

During the installation, you associate the MATLAB Integration Package with a single MATLAB instance on the system. You can install multiple instances of the MATLAB Integration Package on the same system.

It is recommended that you use CANoe with the appropriate version of MATLAB Integration Package that comes with each CANoe version. However, there is some degree of upward and downward compatibility. See the Version History for more details.

The installer of the MATLAB Integration Package comes with additional components like the Linux toolchain. Select the components you need during installation.

File Locations

After installation you should find the following path in your MATLAB environment:
$(INSTALLDIR)\rtw\c\canoe\devices

CANoe MATLAB Integration Package files (canoe\*.*) are located in the following directory:
$(INSTALLDIR)\rtw\c

The with CANoe delivered sample files (CANoe configurations and Simulink models) are located in the following directory: Open Directory

The additional sample configurations provided with the installation of the MATLAB Integration Package are located in the following folder, then choose the respective installation:

%PUBLIC%\Documents\Vector\MATLAB Integration

Here you find descriptions for the MATLAB sample configurations.
