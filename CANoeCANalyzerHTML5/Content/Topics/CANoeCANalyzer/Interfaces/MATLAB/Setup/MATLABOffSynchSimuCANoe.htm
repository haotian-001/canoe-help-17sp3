Offline Mode / Synchronous Mode - Simulink and CANoe

 

Offline Mode

To use the offline mode as described in the Execution Modes, the following steps are required. For a configuration example, see the Offline Mode sample configuration.

Simulink Model Configuration

To use the offline mode, the MATLAB Integration Package Simulation Step block hast to be added to the model. Configure the Simulation Mode in the block dialog of the Simulation Step block to Offline.

The are no requirements for the solver or step time settings in Simulink.

Execute the Simulation

Since Simulink is the master in this execution mode, the simulation must be started from within Simulink with the Run button. This will automatically start the measurement in CANoe as well.

Synchronous Mode

To use the synchronous mode as described in the Execution Modes. Use the same steps, as with the offline mode, only change the setting of the Simulation Step block to Synchronized.

Note

Both the offline mode and synchronous mode rely on the CANoe COM Interface. If no CANoe is open, it will open the COM registered CANoe version. Be aware, if multiple CANoe instances are open it is not defined which CANoe Instance will act as the slave.

Note

The Simulation Step block does not influence the code generation for HIL mode in any way. Therefore, a Simulink model can be used for offline mode and synchronous mode as well HIL mode with the same configuration. Be aware that the HIL mode does require more specific configurations in the Simulink model. For further details see HIL Mode - Simulink Configuration (MATLAB Integration) and HIL Mode - CANoe Configuration (MATLAB Integration).

Analysis Models

To use the variant of analysis models which is similar to synchronous mode, as described in the Execution Modes, the following steps are required. For a configuration example, see the Analysis sample configuration.

Simulink Model Configuration

To use this variant of the analysis models, the MATLAB Integration Package Analysis Step block has to be added to the model. There are no further configurations in the block dialog of the Analysis Step block required. The MATLAB Integration Package Simulation Step block is not required.

Execute the Simulation

The simulation can be started identically to offline mode and synchronous mode.
