Typically the command mex – setup (for MATLAB Version up to and including R2013b) or mex – setup C++ (starting with MATLAB Version R2014a) must be run from the command window. Microsoft Visual C++ Compiler

See your MATLAB Simulink Coder documentation for a correct setup of the compiler environment. Typically the command mex – setup (for MATLAB Version up to and including R2013b) or mex – setup C++ (starting with MATLAB Version R2014a) must be run from the command window.

Mingw-w64 Compiler

Limitations

The MinGW compiler does not yet support all the features of the Microsoft Visual C++ compiler. Currently not available is the following feature:

  • Extended Real-Time DLL

Required Compiler Configuration

For the MATLAB Integration Package both architectures (32Bit/ 64Bit) of the MinGW compiler are necessary. With MinGW each architecture comes as a separate compiler, so two compilers have to be installed.

Obtaining the Compiler

Note

Vector is not affiliated with the compiler distributor in any way. Please be aware that this is FOSS (Free and open-source software) with all the advantages and risks.

The 64-bit MinGW compiler must be installed through the free official MATLAB add-on. The add-on, named MATLAB Support for MinGW-w64 C/C++ Compiler, can be obtained either through the MATLAB Add-on Explorer or directly from the Mathworks website.

The 32-bit version of the MinGW compiler can be obtained free-of-charge from the Vector Download Center. There, the two supported versions MinGW 5.3 and MinGW 6.3 are available in the form of the CANoe MATLAB Integration Package Compiler Bundle 1.0. Alternatively, the compilers can be downloaded from the SourceForge website. Direct links to the recommended versions are MinGW 5. 3 and MinGW 6.3.

The compressed folder only has to be unzipped and placed in a suitable directory. Installing the compiler it is not necessary.

Configuring the Compiler

The 64 bit MinGW compiler only needs to be registered in MATLAB after setup, all you need to do is execute the mex -setup command in the MATLAB command window.

The 32-bit MinGW compiler for the MATLAB Integration Package can’t be set from the MATLAB command window. The paths to its/bin directory has to be configured as Windows system variable.

Name of the System Variable

Configuration

Example Content

CANoe_MatlabIntegrationPackage_MinGW_32_bin

Path to the /bin directory of the 32 Bit MinGW installation

C:\mingw32\bin

You can create the system environment variables with the following steps:

  1. Use the Windows (Start Menu) search input box, type in env, and choose Edit the system environment variables.
  2. In the System Properties dialog on page Advanced, click the Environment Variables button.
  3. Use the New button to define the variable as required.
  4. Restart MATLAB, so these changes will take effect.

Linux Compiler

The MATLAB Integration Package allows you to compile Simulink models for use in Linux RT/ERT. To install this optional feature, select the Custom setup type and activate Support for Linux RT/ERT during the MATLAB Integration Package setup. Afterwards, the target cn_linux is additionally available to you.

Currently, the range of functions is still limited. The following features are not yet supported:

  • Multitasking mode
  • CAPL blocks and CAPL functions
  • XCP
  • Signal analysis

To use all the features of the MATLAB Integration Package, you still need to register a C compiler in MATLAB. This can be either a Microsoft Visual C compiler or the 64-bit version of the MinGW compiler.

System requirements:

  • MATLAB Version: MATLAB R2018b
  • Linux RT available starting with CANoe 13
  • Linux ERT available starting with CANoe 17

Verification

To verify proper installation open the Simulink library browser. You should see an additional library: Vector CANoe. When you open a model (or create a new one) open the Tools menu and click the Code Generation Options. Under Code generation select the Browse button to open the System Target File Browser. In this list you should see the following line:
cn.tlc Vector CANoe Realtime Target DLL (Visual C++ Project Makefile)
