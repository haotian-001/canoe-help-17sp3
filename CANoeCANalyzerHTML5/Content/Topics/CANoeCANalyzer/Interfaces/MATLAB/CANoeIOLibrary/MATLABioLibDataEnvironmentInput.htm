Data Environment Input

 

Description

The Data Environment Input block is used for an environment variable of type data. The bytes of the environment variable are output as a vector from the block.

  

Parameters
  • Sample Time
    The settings are in agreement with the Simulink semantics:
    0 = continuous sample time
    -1 = inherited sample time
    Continuous sample time is set by default. If the Environment Input block e.g. in a triggered subsystem, the sample time must be set to inherited because only constant blocks or blocks with inherited Sample Time are allowed in triggered subsystems.
  • Data type
    Data type of the second Output port of the block. For the inherited data type, the data type is derived from the blocks connected to the block. For the DB type data type, the data type from the database is used and the Simulink data type that fits best is used.
  • Select variable
    Specify environment variable from database.
