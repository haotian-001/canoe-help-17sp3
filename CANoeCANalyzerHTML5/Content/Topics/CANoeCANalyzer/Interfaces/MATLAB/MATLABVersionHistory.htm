New Features in Version 8.3:

  • Support of MATLAB Release R2023a.
  • Support of Microsoft Visual Studio 2022.
  • Support of fields for distributed objects.
  • Support of communication direction internal for distributed objects.

New Features in Version 8.2:

  • Support of MATLAB Releases R2022a and R2022b.

New Features in Version 8.1:

  • Support of ERT (Linux Target).
  • Improved vModule support.

New Features in Version 8.0:

  • Support of MATLAB Release R2021b.
  • Support of Linux RT.
  • Support of vModules for analysis models.

New Features in Version 7.2:

  • Support of MATLAB Release R2020b and R2021a.
  • Support of distributed object functions and structs in offline mode and synchronous mode.
  • Mingw-64 compiler now supports XCP and multitasking models.

New Features in Version 7.1:

  • Support of distributed objects (data and events) in offline mode and synchronous mode.
  • Support of SOME/IP signals in HIL mode.
  • Support of Mingw-w64 compiler.
  • Multidimensional parameters in ERT.

New Features in Version 7.0:

  • Support of distributed object functions and structs in HIL mode.
  • vModules can now be used in Communication Setup.

New Features in Version 6.5:

  • Support of XCP in MATLAB Release R2020a.

New Features in Version 6.4:

  • Support of distributed objects (data and events) in HIL mode.
  • Model generation can now create a file of the type *.vmodule.
  • Models can now be built as a 64-bit DLL.
  • Arrays of type int and double supported in ERT.
  • Support of Visual Studio 2019 in MT mode.
  • Support of MATLAB Release R2020a.
  • Support of system variables of the type Double Array and Integer Array in ERT DLLs.

New Features in Version 6.3:

  • Filter option to exclude blocks from parameterization.
  • Export CAPL Functions option support for ERT.
  • Support of Microsoft Visual Studio 2019.
  • Support of MATLAB Release R2019b.

New Features in Version 6.2:

  • Support of parameterization in Extended Real Time DLLs.
  • CAPL function that returns the name of the model.
  • Support of MATLAB Release R2019a.

New Features in Version 6.1:

  • Support of communication objects in offline mode and synchronous mode.
  • Support of system variables of the type Integer and Double in Extended Real Time DLLs.
  • Support of MATLAB Release R2018b.

New Features in Version 6.0:

  • DLLs can be built for ERT target. See Extended Real Time (ERT). Only signal blocks are supported.

New Features in Version 5.1:

  • Support of Communication Object Function Calls in HIL mode

New Features in Version 5.0:

  • Support of Communication Object Signals in HIL mode
  • Support of multitasking in HIL mode
  • Support of MATLAB release R2018a

New Features in Version 4.2:

  • Parameterization and signal analysis in referenced models
  • Support of MATLAB release R2016b, R2017a and R2017b
  • Support of Microsoft Visual Studio 2017

New Features in Version 4.1:

  • Blocks for communication with CANoe via system variable structs

New Features in Version 4.0:

  • Support of any data type in the Simulink model
  • Distinction between raw values and physical values for signals and system variables
  • Support of MATLAB release R2016a
  • Support of Microsoft Visual Studio 2015

New Features in Version 3.2:

  • You can build models as Debug DLLs
  • Support of parameters in CAPL Call block and CAPL Handler block
  • Support of protected model references
  • Support of MATLAB release R2015b

New Features in Version 3.1:

  • Support of PDUs in the Message Input and the Message Output Block
  • Support of MATLAB release R2014b, R2015a
  • Support of Microsoft Windows SDK 7.1 as compiler

New Features in Version 3.0:

  • Automatic system variables for model performance (min, max, avg)
  • Support of MATLAB 64bit versions
  • Support of release 2014a

New Features in Version 2.5:

  • Generically integrate Simulink models into CANoe configuration
  • Support of R2011b, R2012a, R2012b

New Features in Version 2.4:

  • You can calibrate MATLAB Models with XCP on Ethernet or XCP on CAN
  • Support of R2011a

New Features in Version 2.3:

  • The Model Viewer displays values of parameters, signals and system variables
  • You can edit parameters in the Model Viewer
  • From the Model Viewer you can also open the new Symbol Panel to change values of signals and system variables

New Features in Version 2.2:

  • Support of Model Referencing
  • Support of MATLAB release R2008b and R2009a

New Features in Version 2.1:

  • Simulink models can be used for analysis in the Measurement Setup
  • Improved Model Viewer; e.g. display of Stateflow diagrams
  • Support of MATLAB release 2008a

New Features in Version 2.0:

  • New synchronous mode introduced. Simulation can be started in MATLAB and CANoe can be operated in real mode.
  • Model Viewer Integration

New Features in Version 1.7:

  • Improved configuration of signal blocks regarding database changes
  • When generating a model or by using the Simulink Model Advisor the correctness of the configuration of the CANoe blocks can be checked

Start the setup via the following link: Open Setup Folder

New Features in Version 1.6:

  • Blocks for communication with CANoe via system variables
  • When using parameterization generated models automatically create system variables for the parameters
  • Generated models can also automatically create system variables for Simulink signals
  • Support of MATLAB release 2007b

New Features in Version 1.5:

  • Support of CANoe 7.0

New Features in Version 1.4:

  • Parameterization of generated models before and during a measurement (since MATLAB release 14)
  • Message blocks that receive/output all signals of a message (since MATLAB release 14)
  • Support of Visual C++ 2005 (since MATLAB release 2006a)
  • Support of MATLAB release 2006a, 2006b, 2007a

New Features in Version 1.3:

  • Direct signal access with CANoe Interaction Layer
  • Support of Visual C++ 7.1 (since MATLAB release 14)
