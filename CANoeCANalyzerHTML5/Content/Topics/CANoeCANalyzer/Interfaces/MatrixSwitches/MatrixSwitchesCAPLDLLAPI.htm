Switches: CAPL DLL API

 

This function documentation explains the API of the DLL pl_nise.DLL.

long NISEClose (long handle)

Description

This function closes an open device. After calling this function, the handle must no longer be used.

Parameters

handle
The handle references a NI Switch Executive virtual device.

Return Value

0 on success

long NISEConnect (long handle, char connectSpec[], long multiconnectMode, long waitForDebounce)

Description

This function performs the specified routes. For more details on the parameters please refer to NI Switch Executive help.

Parameters

handle
The handle references a NI Switch Executive virtual device.

connectSpec
This string describes the connection.

multiconnectMode
This parameter specifies whether a switch can be closed (logically) multiple times.

waitForDebounce
This parameter is used as boolean. Must be set to false.

Return Value

0 on success

long NISEConnectAndDisconnect (long handle, char connectSpec[], char disconnectSpec[], long multiconnectMode, long operationOrder, long waitForDebounce)

Description

This function performs the specified routes. For more details on the parameters please refer to NI Switch Executive help.

Parameters

handle
The handle references a NI Switch Executive virtual device.

connectSpec
This string describes the connection.

disconnectSpec
This string describes the routes to be disconnected.

multiconnectMode
This parameter specifies whether a switch can be closed (logically) multiple times.

operationOrder
1 = first disconnect, then connect;
2 = first connect

waitForDebounce
This parameter is used as boolean. Must be set to false.

Return Value

0 on success

long NISEDisconnect (long handle, char disconnectSpec[])

Description

This function disconnects the specified routes. For more details on the parameters please refer to NI Switch Executive help.

Parameters

handle
The handle references a NI Switch Executive virtual device.

disconnectSpec
This string describes the routes to be disconnected.

Return Value

0 on success

long NISEDisconnectAll (long handle)

Description

This function disconnects all connections on the virtual device.

Parameters

handle
The handle references a NI Switch Executive virtual device.

Return Value

0 on success

long NISEFindRoute (long handle, char channel1[], char channel2[], char routeSpec[], long routeSpecSize[], long routeCapability[1])

Description

This function finds an existing or potential route between the specified channels. For details how to specify channels and routes and route capabilities please refer to NI documentation.

Parameters

handle
The handle references a NI Switch Executive virtual device.

channel1
This parameter is the channel name of one of the endpoints of the route to find.

channel2
This parameter is the other endpoint of the route.

routeSpecSize
This parameter is the size of the caller supplied routeSpecArray.

routeSpec
This parameter returns the fully specified route path.

routeCapability
This parameter returns route capabilities.

Return Value

0 on success

long NISEIsDebounced (long handle, long isDebounced[1])

Description

This function returns whether the system is debounced.

Parameters

handle
The handle references a NI Switch Executive virtual device.

isDebounced
Set to true if system is debounced, otherwise set to false.

Return Value

0 on success

long NISEOpen (char virtualDeviceName[], char options[], long handle[1])

Description

This function opens the specified virtual device.

Parameters

virtualDeviceName
This parameter is the name of the NI-SEVD as specified in MAX.

options
The option string can be used to pass information to each of the IVI devices on start-up. It can be used to set things such as simulation, range checking, etc. Consult your driver documentation for more information about valid entries for the option string.

handle
The handle references a NI Switch Executive virtual device.

Return Value

0 on success
