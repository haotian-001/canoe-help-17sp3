IOpiggy: Use Cases

 

For the following examples, the Standard mode has been set for IOpiggy1.

Example 1: Digital output on DOUT0

// Set High Level
@sysvar::IO::IOPiggy1::DOUT0 = 1;

...

// Set Low Level
@sysvar::IO::IOPiggy1::DOUT0 = 0;

Example 2: Analog output on AOUT0

// Set Voltage to 2.5
@sysvar::IO::IOPiggy1::AOUT0 = 2.5;

Example 3: PWM output

// Set Output Frequency to 2000Hz
@sysvar::IO::IOPiggy1::PWMOutFreq = 2000;

// Set Duty Cycle to 75%
@sysvar::IO::IOPiggy1::PWM = 75;
