Protocol Information

UDP (User Data Protocol) is a network protocol with minimal protocol header. It is a connectionless protocol and doesn't use data segmentation, that means the packet size in CANoe is restricted to the maximum Ethernet size. With a checksum transmission errors can be identified. With UDP a service on the target system can be addressed with port numbers.
