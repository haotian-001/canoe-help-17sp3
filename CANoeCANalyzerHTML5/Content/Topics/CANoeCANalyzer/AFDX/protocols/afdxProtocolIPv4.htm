Protocol Information

Besides IPv6 the IPv4 protocol (Internet Protocol Version 4) is the lowermost protocol of the internet protocol family respectively the TCP/IP model that is independent of the transmitting medium. With Ethernet IPv4 uses the Ethertype 0800.

Option X

option<X>_<Y>

m

8..n
(optional)

 - - X

option with option type X

An option consists of type, length and data of an option IPv4 (except options that have a size of 1 byte and the option type only).

Note

Options can occur several times in a packet. In that case the token designator gets a postfix _<Y>. Via this postfix you can access the appropriate token.

E.g. the token designator "option3_2.code" describes the type of option 3 that occurs in the packet a second time.
