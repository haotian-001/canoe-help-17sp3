Functional Data Set (FDS)

 

General

A Functional Data Set in the option AFDX conceptionally corresponds to a PDU in CANoe.

Composition

A Functional Data Set (FDS) is organized as a block of up to four data sets (DS) with an associated Functional Status Set (FSS). This way, each data set (DS) is transmitted with the associated Functional Status (FS).

Functional Status

A Functional Status is part of a 32 bit value and occupies 8 bit.
For the Functional Status (FS) the following values are valid:
