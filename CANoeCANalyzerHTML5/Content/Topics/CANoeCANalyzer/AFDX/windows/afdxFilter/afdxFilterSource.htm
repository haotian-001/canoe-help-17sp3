You insert a source filter:

  • Using the context menu (right mouse button)
  • Using the menu bar (Edit menu)
  • Add ES from database...: Opens the selection dialog via which you can insert the known end systems from the database.
  • Add Source Filter...: Opens the parameter dialog for configuration of a source filter using the sender IP address or sender MAC address and the AFDX application channel.
  • Insert from Trace Window...: Takes the channel and sender IP information from the AFDX packets copied in the Trace Window beforehand and adds it to the list.

The parameters for source filters are divided into two views. The left side displays all source filters (main view). The right side displays the properties of the currently selected filter (properties view).

Columns in the main view

Column

Description

Active

activates or deactivates a filter

ES

name of the sender end system, if known from the database; otherwise, the representation of the sender IP address

Identifier

sender IP address or sender MAC address that is filtered by (depending on the IP/MAC setting)

Application

application name from database, if known; otherwise unknown

Partition

4th byte of the sender IP address

The partition can be set explicitly to *. The last byte is then ignored when filtering.

IP/MAC

Specifies whether the sender IP address or the sender MAC address is to be filtered by.

Channel

number of the application channel

The possible selections depend on the number of configured channels.
