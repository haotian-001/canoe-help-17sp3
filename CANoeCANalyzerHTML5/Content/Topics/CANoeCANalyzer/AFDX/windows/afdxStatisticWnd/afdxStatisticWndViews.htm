AFDX VL Statistics Window » Views

Views

 

The following views are available, all of them show the same statistic data.

View

Description

Display Hierarchy

SV
(Statistic quantity)

This view focuses on the statistic quantities. The displayed value is a hierarchical summation of each virtual link and end system.

This view is suitable to create statistic system variables too.

Statistic quantity | End system | Virtual Link

ES
(End system)

The hierarchical summation is done for each Virtual Link. The topmost hierarchy level is built by the end systems.

This simplifies the VL related analysis of the statistic quantities.

Note

The name of the end system is taken from the DBC. If no DBC information is available, or if the symbolic display is deactivated, the name of the end system is taken from the User-ID (per ARINC-664 specification). Here, the prefix ES_ is extended by the two middle bytes of the Source-IP, and the bytes are delimited by a dot (.) to make the character of the address clearly discernible.

If a system variable is added, the dot between the bytes is replaced by an underscore (_) in CAPL.

End system | Statistic quantity | Virtual Link

VL
(Virtual Link)

This view focuses on the detailed display of the statistic quantities of a single Virtual Link.

End system | Virtual Link | Statistic quantity
