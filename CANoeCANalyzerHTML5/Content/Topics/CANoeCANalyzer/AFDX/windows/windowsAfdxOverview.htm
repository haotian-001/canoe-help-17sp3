Short Description

AFDX VL Statistics Window

The AFDX VL Statistics Window displays the number of packets and bytes for the Tx and Rx directions for each AFDX channel and Virtual Link.

AFDX Filter

With the AFDX Filter you can remove AFDX messages from the data stream.

Trace Window

With the Trace Window for AFDX you can specifically analyze AFDX packets.

Data Window

You can display AFDX signals in the Data Window.

Bus Statistics Window

The Bus Statistics Window shows AFDX-specific or Ethernet-specific data and values.
