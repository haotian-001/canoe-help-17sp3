FS View

 

In the FS View, the signal states (Functional Status) are shown for all messages that are selected in the Message List. An entry is created in the FS View for each message. Expand the message in the FS View to show all signal states of the message.

The different signal states are highlighted in color in the FS View. In addition, the signals assigned to the status are shown in color in the Signal List.

The following colors are used to display signal states.

Signal State

Description

Color

ND

No Data

pink

NO

Normal Operation

white

FT

Functional Test

yellow

NCD

No Computed Data

blue

Erroneous signal states are marked with an error icon. Messages with such signal states cannot be sent.

Change signal states

Signal states can be modified in the FS View as follows:

  • Select the desired value from the list in the Status column.
  • Select one or more states or one or more messages. Set the value from the context menu by the commands Status | ND, Status | NO, Status | FT or Status | NCD. You can modify multiple signal states simultaneously using this method.

Filter signal states

Under the FS View you will find the FS Filter. You can use this filter to reduce the number of signal states shown in the list. To do this, enter the desired character string in the input box. The list of displayed signal states is then automatically adapted. If only messages are being displayed in the FS View, no signal state satisfies the filter criteria.

Notes

  • Wildcards (*) are not supported in the FS filter.
  • Message names are not filtered.
