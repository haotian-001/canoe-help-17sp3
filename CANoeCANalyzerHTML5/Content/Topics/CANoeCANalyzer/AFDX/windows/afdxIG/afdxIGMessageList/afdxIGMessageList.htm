Message List

 

In the upper area of the dialog, you will find the message list. Insert all of the messages that you wish to send with the AFDX IG in this list.

The messages are parameterized via the signal list and the FS view in the lower area of the dialog. There, you can set all signal states (Functional Status) and signal values of the messages that are selected in the Message List.

Erroneous messages are marked with an error icon. Such messages cannot be sent.

To select multiple messages, hold down the <Ctrl> key and select the desired messages with the left mouse button.

Insert messages

Messages that you add to the Message List are always inserted at the end of the list.

You have the following ways to add a message to the list:

  • Choose the Add | Messageâ¦ command. In the Selection of Messages and Bus Events dialog, select the desired messages.
  • Choose the Add | Message via Signal Selectionâ¦ command. In the Selection of Signals dialog, select the desired signals. All messages to which the selected signals are assigned are entered into the Message List.
  • Select a message in the Symbol Explorer and move it to the Message List by drag and drop.
  • Select a message in the Trace Window and move it to the Message List by drag and drop. With this method you not only copy the message, but also the signal values of the selected message. The message can be sent immediately with the same signal layout.
  • Copy a message in the Trace Window. Then open the context menu of the Message List by right clicking and choose the Paste from Trace command. Together with the copied message its signal values are inserted too.
  • Select a message in the Message List and open the context menu by right clicking. Then choose the Duplicate command.
