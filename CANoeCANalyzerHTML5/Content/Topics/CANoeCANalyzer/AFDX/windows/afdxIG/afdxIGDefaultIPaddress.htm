Information About Using the Default Source IP Address

 

Global setting of the default source IP address

You can set the default source IP address as global so that it is used for all new AFDX configurations and AFDX Interactive Generators. You can do this in the Options dialog of CANoe.

Setting of the default source IP address in the AFDX IG

You can set the default source IP address independently for each AFDX IG. The address is relevant only for the managed messages.

Thus if you want to send messages with the IP addresses of different end systems, you can do that with several AFDX IGs and the appropriate messages.

Note

Consider that setting the default source IP address in the AFDX IG may overwrite the global setting.
