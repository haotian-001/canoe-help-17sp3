Bytes Data Type

 

The bytes data type is used to store or transfer an opaque sequence of bytes. The number of bytes can vary with each value. In contrast to a list of bytes (which could be declared with list<uint8>), it is not possible to access an individual element (a single byte) in the programming APIs directly or to react on updates or changes of individual elements with a handler. On the other hand, it is possible to copy all bytes directly into and out of a byte value (which does not work with a list because it also contains internal control data).

Example: Byte sequence as a data member of a distributed object.

object SomeObject
{
  consumed data bytes SomeData;
}

Supported Attributes

The bytes data type does not support any attributes.
