vCDL 1.4

 

vCDL allows the CANoecommunication concept to be extended by a tool for the simple and text-based configuration ofcommunication objects anddistributed objects.

Example

version1.0;
namespace Audio
{
  service CDPlayer
  {
    consumer TouchScreen, Dashboard;
    provider Console;

    field uint16 NumberOfTracks { get; }
    field uint16 CurrentTrack; // get, set and notify implied
    void NextTrack();
    void PrevTrack();

    bindingAbstract;
  }
}
