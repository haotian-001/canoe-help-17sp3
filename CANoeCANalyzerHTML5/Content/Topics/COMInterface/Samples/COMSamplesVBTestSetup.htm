CentralLockingSystem – CANoe_TestSetup.vbs

 

This sample shows how an external application can start a test configuration. Also the usage of test configuration events is shown.
The script can be found under Demo_CAN_CN\TestFeatureSet\CentralLockingSystem\Script and belongs to the CentralLockingSystem configuration.

The following operations can be done using the COM Interface:

Access to the Test Configuration

Create a TestConfigurations object

The TestConfigurations object is a property of the Configuration object, it represents all test configurations (collection) within the CANoe configuration.

Set TestConfigurations = Config.TestConfigurations

Get access to a specific TestConfiguration object

The TestConfiguration object represents a specific test configuration of the test configurations collection.
Use the Item property of the TestConfigurations object to have access to the specific TestConfiguration object.

Set TestConfiguration = TestConfigurations.Item(1)

Get access to the first test unit (TestUnit) of the TestConfiguration object

The TestUnit object represents all test units in a test configuration.
The access to the TestUnits object is done using the TestUnits property of the TestConfiguration object.
Use the Item property of the TestUnits object to access a specific test unit (TestUnit object).

Set FirstTestUnit = TestConfiguration.TestUnits.Item(1)

Start the test configuration

Use the Start method of the TestConfiguration object to start the test configuration.

TestConfiguration.Start()

Get the verdict of the test configuration

Use the Verdict property of the TestConfiguration object to get the verdict of a test configuration.

TMVerdict = TestConfiguration.Verdict

Samples of TestConfiguration object events

OnStart event

The OnStart event occurs, when the test configuration is started.
The following procedure is started then:

Sub TestConfiguration_OnStart()
  IsTesterExecuting = True
End Sub

OnStop event

The OnStop event occurs, when the test configuration is stopped.
The following procedure is started then:

Sub TestConfiguration_OnStop(reason)
  IsTesterExecuting = False
  TesterStopReason = reason
End Sub
