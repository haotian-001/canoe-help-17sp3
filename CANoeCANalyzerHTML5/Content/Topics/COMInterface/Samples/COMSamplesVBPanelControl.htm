Easy – Panel Control.exe

 

This sample shows how an external application can control the CANoe panels.
It demonstrates the saving of panel positions, create a panel, add & remove a panel from a panel group as well as setting a panel visible/invisible.

The script has to be used with the CANoe CANSystemDemo configuration.
The Visual Basic application provides buttons for closing and opening panels which are assigned to various panel groups.

The following operations can be done using the COM Interface:

Access the COM Interface

See COM Samples Introduction for general information on accessing and using the COM Interface.

In the global declaration the Application object has to be defined

'Globals
Dim WithEvents app As CANoe.Application

Note the declaration of global variables app with the modifier WithEvents. This is necessary for receiving events.

A new Application object has to be created in the form load (file: PanelControl.frm)

Set app = New CANoe.Application

Load easy sample configuration

The Easy sample configuration is loaded using method Open method:

App.Open (full)

Access to CAPL functions — Compile all nodes

See CANSystemDemo – CANoe Environment.vbs; the same procedure is used here.

Further Steps in CANoe

Use the Quit method to close CANoe:

App.Quit

Get the path of the CANoe application

With the Path property of the Application object you can get the path of the CANoe application.

HelpPath = app.Path

Get the full name of the loaded CANoe configuration

With the FullName property of the Configuration object you can get the full name of the loaded CANoe configuration file.

app.Configuration.FullName

Working with panels

Save the current panel positions

To save the current panel positions use the SavePositions method of the PanelSetup object.

App.Configuration.GeneralSetup.PanelSetup.SavePositions

Create a panel group

A panel group is created using the Panels property of the PanelSetup object.
The parameter of the property defines the panel type.
The Value 4 means "temporary panels".

The following example assigns a panel object containing all temporary loaded panels to a variable:

Set PanelsComfortBus = App.Configuration.GeneralSetup.PanelSetup.Panels(4)

Add a panel to a panel group

A panel will be added to a panel group using the Add method of the Panels object.

PanelsComfortBus.Add(fullName)

Remove a panel from a panel group

A panel will be removed from a panel group using the Remove method of the Panels object.

PanelsComfortBus.Remove(fullname)

Get the number of panels assigned to a panel group

Use the Count property of the Panels object to get the number of panels in a panel group.

count = PanelsComfortBus.Count

Set a panel visible/invisible

A panel can be set visible or invisible with the Visible property of the Panel object.
The Item property of the PanelSetup object returns an object from the PanelSetup container.

Panels.Item(i).Visible = true
