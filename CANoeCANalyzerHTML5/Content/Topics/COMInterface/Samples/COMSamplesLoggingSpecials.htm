CANSystemDemo – Logging Specials.vbs

 

This sample shows some special features of the Exporter object of the COM Interface. It can be used with the CANoe CANSystemDemo configuration.

The Exporter object has two special features used here:

  1. The batch export/conversion of more than one source file to the same number of target files ("n to n")
  2. The batch export/conversion of more than one source file to one target file ("n to 1")

The source files are the logging files specified in the Export dialog of the 2nd and 3rd Logging Block. By default the 2nd and 3rd Logging Block in the CANSystemDemo configuration are not configured to start automatically the logging at the start of measurement. You first have to make sure that the logging files are created by:

  1. Starting the Logging Block manually in the Measurement Setup.
  2. Changing the trigger configuration of the Logging Block.

Access the COM Interface

See COM Samples Introduction for general information on accessing and using the COM Interface.

Export conversion of multiple logging files

The online mode is represented in the OnlineSetup property of the Configuration object. It returns the MeasurementSetup object. The LoggingCollection object comprises all Logging Blocks belonging to the MeasurementSetup object.

The LoggingCollection object is used to assign the second and third Logging Block to a logging variable

Set logging = app.Configuration.OnlineSetup.LoggingCollection(2)
Set logging2 = app.Configuration.OnlineSetup.LoggingCollection(3)

The Exporter property of the Logging object returns the Exporter object, which represents the Export dialog. The dialog is used to save a logging file under a new format.

The Export dialog can be opened via a Logging Block context menu in the Measurement Setup

Set exporter = logging.Exporter

In the next step two logging files (source files) are defined for the export/conversion ("n to n" conversion) using the Sources property of the Exporter object.

Example

With exporter.Sources
  Clear
  Add(logging.FullName)
  Add(logging2.FullName)
End With

Note

The Sources property collects all source files sequentially assigned by the Add method. The Clear method shall guarantee, that the collection is empty at the beginning.

The target files are defined using Destinations property of the Exporter object. The target file format is *.ASC.
The return value of the Destinations property is the Files object.

Example

With exporter.Destinations
  Clear
  Add(path + "\N_TO_N_1.asc")
  Add(path + "\N_TO_N_2.asc")
End With

Note

The Destinations property collects all target files sequentially assigned by the Add method of the Files object. The Clear method of the Files object shall guarantee, that the collection is empty at the beginning.

Before the export/conversion the logging files must be loaded at first using method Load method of the Exporter object.

Example

exporter.Load

The source files are loaded and the content can now be used.

The export/conversion is done using the Save method of the Exporter object.

Example

exporter.Save True

In the second part both source files are exported/converted into one target file ("n to 1"); source files already specified.
Now the target file has to be defined using the Destinations property of the Exporter object.

Example

With exporter.Destinations
  Clear
  Add(path + "\N_TO_1.asc")
End With

The export/conversion is done with the following programming code

exporter.Save True
