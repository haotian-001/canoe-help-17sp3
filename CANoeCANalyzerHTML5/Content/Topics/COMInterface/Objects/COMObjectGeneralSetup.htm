Example

Example — Channels property

The following example shows how to increase the number of available CAN, LIN and KLine channels.

Background

If you add an additional network via COM Interface and there is no corresponding channel number in the Vector hardware configuration dialog available, an error message appears.

Solution

Before adding an additional network the number of available channels must be increased.

Const cCANChannelType   = 1
Const cLINChannelType   = 5
Const cKLineChannelType = 15

'set CAN channel number to 4
app.Configuration.GeneralSetup.Channels(cCANChannelType ) = 4

'set LIN channel number to 2
app.Configuration.GeneralSetup.Channels(cLINChannelType ) = 2

'set KLine channel number to 2
app.Configuration.GeneralSetup.Channels(cKLineChannelType ) = 2

Example — SnippetSetup property

This snippet adds the file Snippet1.cs in sub-folder Snippets of the current configuration.

The first snippet in the first file is executed. The running state is checked every 10 seconds.

Configuration.GeneralSetup.SnippetSetup.SnippetFiles.Add("Snippet1", "Snippets\Snippet1.cs")

Set snippet = Configuration.GeneralSetup.SnippetSetup.SnippetFiles.Item(1).Snippets.Item(1)

snippet.Run
While snippet.IsRunning
  WScript.Sleep 10
WEnd
