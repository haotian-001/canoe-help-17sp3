Description

The IUTSettings object represents the IUT specific configuration settings of a LinSlaveConformanceTester object.

DeviceClass

This property sets or returns the setting determining the LIN device class of the IUT.

Syntax: object.DeviceClass [=value]

(see example below)

8.0 SP3

The setting determining the LIN device class of the IUT; possible values:

1|cClassB

The IUT is a device of class B (microcontroller-based with RxD and TxD connectors).

2|cClassC

The IUT is a device of class C (with transceiver and analog LIN bus connector).

FunctionID

This property sets or returns the setting determining the Function ID of the IUT.

If the Function ID is set using this property, the property value overrides any existing database value; valid value range: 0…65535.

The database value for the Function ID will be returned if it has not been overridden previously.

In that case, the override value will be returned.

Syntax: object.FunctionID [=value]

8.0 SP3

Type: INT

The setting determining the Function ID of the IUT.

—

InitialNAD

This property sets or returns the setting determining the overridden initial NAD of the IUT.

If the NAD is set using this property, the property value overrides any existing database value; valid value range: 1…125.

Manually assigned initial NAD of the IUT. A value of -1 signifies that no NAD has been assigned. The database value will be used in this case.

Syntax: object.InitialNAD [=value]

(see example below)

8.0 SP3

Type: INT

The setting determining the overridden initial NAD of the IUT.

SupportsAssignNAD

This property determines if (and how) the IUT supports the AssignNAD service.

When setting this property, the LIN protocol version of the IUT has to be taken into account.

For a LIN 2.1 node, the value 2 (cBasicConfigurationServiceSupport) is not valid. Use the SlaveNodeProtocolVersion property of the LinSlaveConformanceTester object to determine the protocol version used by the slave node.

Syntax: object.SupportsAssignNAD (=value)

(see example below)

8.0 SP3

The support type for the AssignNAD service; possible values:

1|cNoConfigurationServiceSupport

The IUT does not support the service.

2|cBasicConfigurationServiceSupport

The IUT supports the service, but does not support a positive response.

3|cPositiveResponseConfigurationServiceSupport

The IUT supports the service and a positive response for it.

—

Example

Example — DeviceClass/InitialNAD/SupportsInterbyteSpace/VarianID property

In this example, the ability of the slave node to send wake-up signals is specified by setting the SendsWakeupSignal property to True.

Dim App, environments, testEnv, linCTs, slaveCT1, iutSettings

Set App          = CreateObject("Canoe.Application")
Set environments = App.Configuration.TestSetup.TestEnvironments
Set testEnv      = environments.Add("testenvironment1")
Set linCTs       = testEnv.TestModules.LinSlaveConformanceTesters
Set slaveCT1     = linCTs.Add()
Set iutSettings  = slaveCT1.IUTSettings

iutSettings.VariantID              = 2
iutSettings.InitialNAD             = 100
iutSettings.DeviceClass            = 2
iutSettings.SendsWakeupSignal      = True
iutSettings.SupportsInterbyteSpace = True

Example — ReconfigureBeforeEachTestCase/SupportsAssignNAD/SupportsConditionalChangeNAD property

In this example, some IUT settings are configured for a LIN Slave conformance test:

Dim App
Dim TSCollection, TestFolder
Dim TMs, CTs
Dim CT1
Dim IUTSettings

Set App          = CreateObject("CANoe.Application")
Set TSCollection = App.Configuration.TestSetup.TestEnvironments
Set TestFolder   = TSCollection.Add("Environment1")
Set TMs          = testFolder.TestModules

' Get the collection of LIN Slave Conformance Testers
Set CTs = TMs.LinSlaveConformanceTesters

'Add a new Conformance Tester
CTs.Add()

'Retrieve the Conformance Tester
Set CT1 = CTs.Item(1)

'Get the IUT Settings of the tester
Set IUTSettings = CT1.IUTSettings

'Set the AssignNAD support of the IUT to cNoConfigurationServiceSupport
IUTSettings.SupportsAssignNAD = 1

'Set the ConditionalChangeNAD support of the IUT to cNoConfigurationServiceSupport
IUTSettings.SupportsConditionalChangeNAD = 1

'Disable reconfiguration of the IUT before each test case
IUTSettings.ReconfigureBeforeEachTestCase = false
