Description

The TesterSettings object represents the tester configuration settings of a LinSlaveConformanceTester object.

Properties

Name

Availability

(since version)

Property Value/Type

Property Value Element

AddTraceSnapshotOnFailedTestCase

This property sets or returns the value determining whether a Trace Window snapshot will be taken when a LIN Slave Conformance Tester test case fails.

Syntax: object.AddTraceSnapshotOnFailedTestCase [=value]

(see example below)

8.0 SP3

Parameters

object

Type: TesterSettings

value

(optional)

Type: BOOL

Decides whether a Trace Window snapshot will be taken; possible values:

TRUE

If snapshots are taken for failed test cases.

FALSE

If snapshots are not taken for failed test cases.

—

TraceWindowName

This property sets or returns the name of the Trace Window of which a snapshot may be taken if a LIN Slave Conformance Tester test case fails.

Syntax: object.TraceWindowName [=value]

(see example below)

8.0 SP3

object

Type: TesterSettings

value

Example

Example — General

In the following example, the TesterSettings object is used to configure Trace Window snapshots of failed test cases.

Dim App, environments, testEnv, linCTs, tester, testSettings

Set App = CreateObject("Canoe.Application")
Set environments = App.Configuration.TestSetup.TestEnvironments
Set testEnv = environments.Add("environment1")

'Get the collection of LIN Slave Conformance Testers
Set linCTs = testEnv.TestModules.LinSlaveConformanceTesters

'Create new LIN Slave Conformance Tester
linCTs.Add()

'Get the first LIN Slave Conformance Tester
Set tester = linCTs(1)

'Select the Node that will be tested and its channel
tester.SlaveNodeName = "DWF_Left"
channelNumber = tester.ChannelNumber

'Modify IUT Settings of the Tester
tester.IUTSettings.SendsWakeupSignal = False

'Modify Tester Settings. Trace Window "Trace LIN" should be captured when a test case fails
Set testSettings = tester.TesterSettings
testSettings.TraceWindowName = "Trace LIN"
testSettings.AddTraceSnapshotOnFailedTestCase = True

'Create new LIN Slave Conformance Tester
tester.GenerateTestCases()                'Test Sequence is updated
tester.Sequence.Item(6).Enabled = False   'Disable a specific Test Case
