IUTInitializationSettings

Description

The IUTInitializationSettings object represents the IUT initialization configuration settings of a LinSlaveConformanceTester object.

ResetType

This property sets or returns the value determining how the IUT is reset before each test case.

Syntax: object.ResetType [=value]

(see example below)

8.0 SP3

Sets or returns how the IUT is reset before each test case; possible values:

1|cNoReset

The IUT is not reset before each test case.

2|cVH11xx

Vector VH11xx (VH1100, VH1150,…) hardware is used to reset the IUT.

3|cVT7001

Vector VT7001 hardware is used to reset the IUT.

4|cManual

The IUT is reset manually.

5|cSoftware

Only the IUT software is reset by waiting for a LIN bus timeout before each test case.

6|cRelayAndIOCab

(deprecated)

A third party relay circuit and a Vector IOcab is used to reset the IUT; this reset type is obsolete can no longer be set.

—

SupplyVoltage

This property sets or returns the voltage that should be supplied to the IUT when using a Vector VH1100 or VT7001 to reset the IUT.

The setting has no effect if the ResetType property of the IUTInitializationSettings object is not set to cVT7001 or cVH1100.

Syntax: object.SupplyVoltage [=value]

(see example below)

8.0 SP3

Type: INT

Sets or returns the voltage that is supplied to the IUT.

—

UseAutomaticSleepDetection

This property sets or returns the value determining if automatic sleep detection is used.

The property has no effect if the ResetType property is not set to cVT7001 or cVH1100. It is used in conjunction with the MinCurrentConsumptionAwake property and MaxCurrentConsumptionSleep property.

Syntax: object.UseAutomaticSleepDetection [=value]

(see example below)

8.0 SP3

Type: BOOL

Sets or returns whether automatic sleep detection is used; possible values:

TRUE

Automatic detection of wake-up and sleep is used.

FALSE

Automatic detection of wake-up and sleep is not used.

Example

Example

In the following example, the initialization options for the slave node are configured via the IUTInitializationSettings object.

Dim App, environments, testEnv, linCTs, tester, initSettings

Set App          = CreateObject("Canoe.Application")
Set environments = App.Configuration.TestSetup.TestEnvironments
Set testEnv      = environments.Add("environment1")

'Get the collection of LIN Slave Conformance Testers
Set linCTs = testEnv.TestModules.LinSlaveConformanceTesters

'Create new LIN Slave Conformance Tester
linCTs.Add()

'Get the first LIN Slave Conformance Tester
Set tester = linCTs(1)

'Configure the initialization for the slave node using the IUTInitializationSettings object
Set initSettings                        = tester.IUTInitializationSettings
initSettings.ResetType                  = 2     'Vector VH1100 hardware is used for reset
initSettings.BusIdleWaitTime            = 4000
initSettings.PowerOffTime               = 500
initSettings.IUTInitTime                = 100
initSettings.SupplyVoltage              = 12
initSettings.MaxCurrentConsumptionSleep = 14
initSettings.MinCurrentConsumptionAwake = 22
initSettings.UseAutomaticSleepDetection = True
tester.GenerateTestCases()                      'Test Sequence is updated
