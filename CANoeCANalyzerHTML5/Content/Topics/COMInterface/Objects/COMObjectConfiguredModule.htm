Description

Example

The following example written in PowerShell demonstrates how to work with configured modules:

# Get the VT System configuration instance
$canoe = New-Object -ComObject CANoe.Application
$vts   = $canoe.Configuration.VTSystem

# Print the current configuration
Foreach ($module in $vts.ConfiguredModules)
{
  Write-Host("Configured module " + $module.Name)
}

# Change the name of the first module
$vts.ConfiguredModules[1].Name = "NewName"

# Add a basic module to the configuration
$newModule1 = $vts.ConfiguredModules.AddBasicModule(2516) # Add a VT2516

# Add an application specific module to the configuration
$id = $vts.AvailableModules.ApplicationSpecificModules[1].ID
$newModule2 = $vts.ConfiguredModules.AddApplicationSpecificModule($id)

# Remove all modules from the configuration
$vts.ConfiguredModules.RemoveAll()
