DataSourceFiles

Application » Configuration » CommunicationSetup » DataSourceSetup » DataSources » FileGroupDataSource » DataSourceFiles

Description

The DataSourceFiles object is a snapshot of the collection of the DataSourceFile objects of a FileGroupDataSource. The snapshot is taken at the time of the creation of the collection object. It is updated with the modifications done using the same instance, but note that it does not update with modifications done using another instance of the collection.

The object can be used to modify (add, remove) the files of the corresponding data source.

Example
Copy

The following VBScript illustrates how to use group data sources:

set Application     = CreateObject("CANoe.Application")
set DataSourceSetup = Application.Configuration.CommunicationSetup.DataSourceSetup

' Note: All error handling is omitted in the following.
set Source = DataSourceSetup.DataSources.AddGroupDataSource("Complete System Description")
call Source.SourceFiles.Add("SystemDescription_Part_A.arxml")
call Source.SourceFiles.Add("SystemDescription_Part_B.arxml")
call Source.Import()

' Illustrate the base properties: It is a group, it is of default type
if not Source.IsGroupSource then
  call MsgBox("This message box should never be displayed or something went completely wrong.", vbCritical or vbOkOnly)
end if
if Source.SourceType <> cSourceType_Default then
  call MsgBox("This message box should never be displayed or something went completely wrong.", vbCritical or vbOkOnly)
end if

Msg = "The file set of the source (name: '" & Source.Name & ") " & " contains the following " & CStr(Source.SourceFiles.Count) & " files:"
for each File in Source.SourceFiles
  Msg = Msg & vbCrLf & "* File name: '" & File.Name & "'"
next
call MsgBox(Msg, vbInformation or vbOkOnly)

' Removing is possible also by path:
call Source.SourceFiles.Remove("SystemDescription_Part_A.arxml")
Msg = "The file set of the source (name: '" & Source.Name & ") " & " contains the following " & CStr(Source.SourceFiles.Count) & " files:"
for each File in Source.SourceFiles
  Msg = Msg & vbCrLf & "* File name: '" & File.Name & "'"
next
call MsgBox(Msg, vbInformation or vbOkOnly)

' Enable usage of symbolic names.
const cSourceType_Default = 0
const cSourceType_VTT     = 1
