Description

The Channels object represents the channel configuration in the Simulation Setup of the CANoe application.

Add

This method assigns a channel to a Bus object.

Only one channel can be assigned to a bus. Therefore you have to remove an already assigned channel before you can assign a new channel.

Syntax: object.Add(type, number)

Example

Example — Add method

The following example assigns the CAN channel 2 to the first bus in the Simulation Setup:

Dim Channels, Channel
Set Channels = Application.Configuration.SimulationSetup.Buses(1).Channels
Set Channel  = Channels.Add(1, 2)

Example — Remove method

The following example removes the first channel that is assigned to the first bus in the Simulation Setup:

Set app = CreateObject("CANoe.Application")
Set channels = app.Configuration.SimulationSetup.Buses(1).Channels
channels.Remove(1)
