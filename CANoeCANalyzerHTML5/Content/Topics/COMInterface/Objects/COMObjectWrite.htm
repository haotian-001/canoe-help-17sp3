DisableOutputFile

This method disables logging of all outputs of the Write Window for the certain page.

Syntax: object.DisableOutputFile([tabIndex])

(see example below)

✔

tabIndex

tabIndex

(optional)

The index of the tab, for which logging of the output is to be deactivated.

If no tabIndex is passed (or tabIndex=0), logging of the output is deactivated for the first page (All page).

You can activate/deactivate the logging of the Write Window in its configuration dialog (option Activated). You can also set/unset this option with the COM methods EnableOutputFile and DisableOutputFile.

After activating the logging (EnableOutputFile) all future Write Window notifications are written to the logging file. Existing Write Window notifications are not incorporated in the logging file. When you stop and restart the logging to the same logging file, the new information will be appended the existing content.

EnableOutputFile

This method enables logging of all outputs of the Write Window in the output file for the certain page.

Syntax: object.EnableOutputFile([[outputFile], tabIndex])

(see example below)

✔

If no outputFile is passed, logging of the output in predefined output file is activated.

Example

Example — DisableOutputFile method

  • Sample 1

    The following examples deactivate logging of the output of the Write Window for the first tab (All tab):

    Application.UI.Write.DisableOutputFile
    Application.UI.Write.DisableOutputFile 0

  • Sample 2

    The following example deactivates logging of the output of the Write Window for the second tab (System tab):

    Application.UI.Write.DisableOutputFile 1

Example — EnableOutputFile method

  • Sample 1

    The following examples activate logging of the output of the Write Window to the file All.TXT for the first page (page All):

    Application.UI.Write.EnableOutputFile "C:\All.TXT"
    Application.UI.Write.EnableOutputFile "C:\All.TXT", 0

  • Sample 2

    The following example activates logging of the output of the Write Window to the file System.TXT for the second page (page System):

    Application.UI.Write.EnableOutputFile "C:\System.TXT", 1

Example — Output method

The following example outputs a notification in the Write Window:

Application.UI.Write.Output "Hello world!"
