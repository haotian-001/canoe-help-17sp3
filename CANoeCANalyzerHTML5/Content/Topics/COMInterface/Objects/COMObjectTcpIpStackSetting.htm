Description

The TcpIpStackSetting object represents the TCP/IP stack settings of an object in CANoe.

The TCP/IP stack settings are only available if the object is attached to an Ethernet bus.

Example

Print settings of global stack:

Set App                 = CreateObject("CANoe.Application")
Set GlobalStackSettings = App.Configuration.GlobalTcpIpStackSetting

Select Case GlobalStackSettings.StackSelection
  Case 0:
    Wscript.Echo "Stack Selection: Windows Stack"
  Case 1:
    Wscript.Echo "Stack Selection: Global CANoe Stack"
    PrintEnabled GlobalStackSettings.DelayedAckEnabled, "Delayed ACK"
    PrintEnabled GlobalStackSettings.RoutingEnabled, "Routing"
    Wscript.Echo "Default IPv4 Gateway: " & GlobalStackSettings.IPv4DefaultGateway
    Wscript.Echo "Default IPv6 Gateway: " & GlobalStackSettings.IPv6DefaultGateway
  Case 2:
    Wscript.Echo "Stack Selection: Node Stack"
    PrintEnabled GlobalStackSettings.DelayedAckEnabled, "Delayed ACK"
    PrintEnabled GlobalStackSettings.RoutingEnabled, "Routing"
    Wscript.Echo "Default IPv4 Gateway: " & GlobalStackSettings.IPv4DefaultGateway
    Wscript.Echo "Default IPv6 Gateway: " & GlobalStackSettings.IPv6DefaultGateway
End Select

Sub PrintEnabled(value, name)
  if(value) Then
    Wscript.Echo name & " Enabled"
  Else
    Wscript.Echo name &" Disabled"
  End If
End Sub
