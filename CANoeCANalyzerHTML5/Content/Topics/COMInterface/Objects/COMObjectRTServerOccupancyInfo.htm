RTServerOccupancyInfo

Application » Configuration » DistributedMode » RTServerOccupancyInfo

Description

The RTServerOccupancyInfo object contains the occupancy information of the RT server specified by its remote address.

The occupancy information reflects the state of the RT server at the time this object is queried, i. e. there is no automatic update of the data contained in the object. If an updated occupancy information is required at a later point in time, a new RTServerOccupancyInfo object must be requested.

This object is interesting, e. g., if you want to check whether the device is already used by another client.

Client

Get the name of the client which currently occupies the device for usage as RT server in distributed mode.

The returned client name string can be one of the following:

  • username

  • computername

  • username@computername

An empty string is returned if device is not used by another client in distributed mode.

Example

Example — Client property

Set RTServerOccupancyInfo = App.Configuration.DistributedMode.GetRTServerOccupancyInfo(RemoteAddress)
ClientName = RTServerOccupancyInfo.Client

If len(ClientName) = 0 Then
  App.UI.Write.Output "No client name available!"
Else
  App.UI.Write.Output "The device is occupied by: " & ClientName

End If

Example — UsedInDistributedMode property

Check if device is currently used in distributed mode by some client:

Set RTServerOccupancyInfo = App.Configuration.DistributedMode.GetRTServerOccupancyInfo(RemoteAddress)
UsedInDistributedMode = RTServerOccupancyInfo.UsedInDistributedMode

App.UI.Write.Output "Is " & remoteAddress & " currently used in distributed mode? " & UsedInDistributedMode

Example — UsedInStandaloneMode property

Check if device is currently used in standalone mode by some client:

Set RTServerOccupancyInfo = App.Configuration.DistributedMode.GetRTServerOccupancyInfo(RemoteAddress)
UsedInStandaloneMode = RTServerOccupancyInfo.UsedInStandaloneMode

App.UI.Write.Output "Is " & remoteAddress & " currently used in standalone mode? " & UsedInStandaloneMode
