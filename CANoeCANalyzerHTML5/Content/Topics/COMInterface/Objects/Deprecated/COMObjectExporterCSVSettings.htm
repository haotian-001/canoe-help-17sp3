Layout

This property sets or determines the increment of the time markers for the consistent time channel if the destination files shall be stored in CSV format.

Syntax: object.Layout[=layout]

deprecated

The layout to be used; possible values:

  • 0|cOneTableEmptyCells
    One table will be written in the file.
    The signals in this table only contain values at this points in time at which (these) values have been measured.
  • 1|cOneTableKeepLastValues
    One table will be written in the file.
    The signals in this table contain values for each point in time.
    Each point in time that doesn't have a (real measured) value gets the last measured value. This makes it easier to create diagrams e.g. in Microsoft Excel.
  • 2|cSeveralTables
    Several tables will be written in the file.
    For each signal a separate table with time column and value column will be created.
    All tables are arranged one below the other.
  • 3|cSeveralTablesSideBySide
    Several tables will be written in the file.
    For each signal a separate table with time column and value column will be created.
    All tables are arranged side by side.

—
