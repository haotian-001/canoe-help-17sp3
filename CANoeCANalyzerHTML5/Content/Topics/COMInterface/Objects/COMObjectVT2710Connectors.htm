Description

The VT2710Connectors object represents the collection of connector settings on a configured VT2710 module.

Example

The following example written in PowerShell demonstrates how to use the custom settings available on VT2710 modules:

# Get the VT System configuration instance
$canoe = New-Object -ComObject CANoe.Application
$vts   = $canoe.Configuration.VTSystem

# Get the VT2710 module
$vt2710 = $vts.GetModuleByName("M1_VT2710")

# Use internal voltage generation for both external pins
$vt2710.InternalVoltage = 3.3
$vt2710.Connectors[1].VoltageSource = 1 # Internal
$vt2710.Connectors[1].OutputIntVoltOnExtPinEnabled = $true
$vt2710.Connectors[2].VoltageSource = 1 # Internal
$vt2710.Connectors[2].OutputIntVoltOnExtPinEnabled = $true

# Enable LVDS voltage output
$vt2710.LVDSVoltage = 5.0
$vt2710.LVDSVoltageOutputEnabled = $true

# Configure connector 1 for UART, RS485 and I2C
$vt2710.Connectors[1].Pins[1].Mode = 3 # Pin 2 -> I2C
$vt2710.Connectors[1].Pins[2].Mode = 3 # Pin 3 -> I2C
$vt2710.Connectors[1].Pins[3].Mode = 5 # Pin 4 -> RS485
$vt2710.Connectors[1].Pins[4].Mode = 5 # Pin 5 -> RS485
$vt2710.Connectors[1].Pins[5].Mode = 5 # Pin 6 -> RS485
$vt2710.Connectors[1].Pins[6].Mode = 5 # Pin 7 -> RS485
$vt2710.Connectors[1].Pins[7].Mode = 4 # Pin 8 -> UART
$vt2710.Connectors[1].Pins[8].Mode = 4 # Pin 9 -> UART

# Configure connector 2 for DIO
$vt2710.Connectors[2].Pins[1].Mode = 0 # Pin 2 -> DI
$vt2710.Connectors[2].Pins[2].Mode = 0 # Pin 3 -> DI
$vt2710.Connectors[2].Pins[3].Mode = 0 # Pin 4 -> DI
$vt2710.Connectors[2].Pins[4].Mode = 0 # Pin 5 -> DI
$vt2710.Connectors[2].Pins[5].Mode = 1 # Pin 6 -> DO
$vt2710.Connectors[2].Pins[6].Mode = 1 # Pin 7 -> DO
$vt2710.Connectors[2].Pins[7].Mode = 1 # Pin 8 -> DO
$vt2710.Connectors[2].Pins[8].Mode = 1 # Pin 9 -> DO
