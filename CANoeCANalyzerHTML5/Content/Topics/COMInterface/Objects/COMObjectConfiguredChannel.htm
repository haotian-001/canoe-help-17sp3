Example

Example — ConfiguredChannel Object

The following example written in PowerShell shows how to configure a VT System channel:

# Get the VT System configuration instance
$canoe = New-Object -ComObject CANoe.Application
$vts   = $canoe.Configuration.VTSystem

# Get the channel
$channel = $vts.GetChannelByName("M3_Ch4")

# Change name and comment of the channel
$channel.Name = "MyChannel"
$channel.Comment = "My Channel Comment"

# Enable two measurement values
$channel.Measurementvalues["Avg"].IsEnabled = $true
$channel.Measurementvalues["Cur"].IsEnabled = $true
