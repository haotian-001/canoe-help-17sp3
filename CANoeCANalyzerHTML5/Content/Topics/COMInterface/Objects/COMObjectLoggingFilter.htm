LoggingFilter

Application » Configuration » MeasurementSetup » LoggingCollection » Logging » LoggingFilter
Application » Configuration » StandaloneMode » StandaloneLogging » LoggingFilter

Description

The LoggingFilter object enables logging of the specified events.

Properties

—

Disable

This method disables logging of the specified events.

Syntax: object.Disable(events)

(see example below)

9.0 SP2

events

0|cLogAllEvents

Disable logging of all events.

1|cLogBusEvents

Disable logging of bus events (e.g. frames, messages, PDUs).

2|cLogTriggerEvents

Disable logging of Start/Stop trigger events.

3|cLogStatisticEvents

Disable logging of system variables for statistics.

4|cLogSysVarEvents

Disable logging of environment and system variables.

5|cLogInternalEvents

Disable logging of internal program events (e.g. CANoe status reports).

6|cLogCaplEvents

Disable logging of outputs of the writeToLog CAPL function and program internal comments.

7|cLogTestEvents

Disable logging of activity of test modules and test units.

8|cLogDiagEvent

Disable logging of diagnostic events.

9|cLogCommunicationObjects

Disable logging of communication object and attribute events.

Example

The following example assigns the Exporter object to a variable:

Const cLogAllEvents            = 0
Const cLogBusEvents            = 1
Const cLogTriggerEvents        = 2
Const cLogStatisticEvents      = 3
Const cLogSysVarEvents         = 4
Const cLogInternalEvents       = 5
Const cLogCaplEvents           = 6
Const cLogTestEvents           = 7
Const cLogDiagEvents           = 8
Const cLogCommunicationObjects = 9

Set app = CreateObject("CANoe.Application")
Set config = app.Configuration

Set logging = config.OnlineSetup.LoggingCollection(1)
Set logFilter = logging.Filter

logFilter.Enable(cLogAllEvents)

logFilter.Disable(cLogTestEvents)
logFilter.Disable(cLogDiagEvents)

Set logFilter = Nothing
Set logging   = Nothing
Set config    = Nothing
Set app       = Nothing
