Example

Example — General

CANSystemDemo – Logging Filter.vbs (COM Sample)

Example — AnimationFactor property / WorkingMode property

This example sets the working mode to simulated bus with animation factor 5:

Set CanoeApp    = CreateObject("CANoe.Application")
Set OnlineSetup = CanoeApp.Configuration.OnlineSetup

OnlineSetup.WorkingMode     = 2
OnlineSetup.AnimationFactor = 5

Example — EthernetStatisticsSysVarAsStruct property / ExtendedEthernetLogging property

This example sets the extended Ethernet logging and bus statistics as struct as recommended in network based mode:

Set CanoeApp    = CreateObject("CANoe.Application")
Set OnlineSetup = CanoeApp.Configuration.OnlineSetup

OnlineSetup.ExtendedEthernetLogging          = true
OnlineSetup.EthernetStatisticsSysVarAsStruct = true

Example — ActivateEndBlock method

This example activates all the end blocks named Trace Powertrain:

Set App   = CreateObject("CANoe.Application")
Set Setup = App.Configuration.OnlineSetup
Setup.ActivateEndBlock "Trace Powertrain", True

Example — CreateMappingSet method

Create a new default channel mapping that swaps CAN 1 and CAN:

' Create a new default channel mapping that swaps CAN 1 and CAN 2
Set App = CreateObject( "CANoe.Application")
Set offlineSetup = App.Configuration.OfflineSetup
Set defaultMapping = offlineSetup.CreateMappingSet("DefaultMapping")
defaultMapping.PutByBusName "CAN", 1, 2
defaultMapping.PutByBusName "CAN", 2, 1
offlineSetup.OfflineSourceRoot.SetMappingSet(defaultMapping)

Example — ExportMappingSets method

Export all mappings:

' export all mappings
Set App   = CreateObject( "CANoe.Application")
Set Setup = App.Configuration.OfflineSetup
Setup.ExportMappingSets "C:\mappings.chmap"

Example — GetAllMappingSets method

Get all mapping sets

' Get all mapping sets
Set App   = CreateObject( "CANoe.Application")
Set Setup = App.Configuration.OfflineSetup

MsgBox "Printing all mappings..."
Set mappings = Setup.GetAllMappingSets
for i = 1 to mappings.Count
  MsgBox mappings(i).Name + " " + mappings(i).Id
next

Example — GetMappingSetById method

Set App   = CreateObject( "CANoe.Application")
Set Setup = App.Configuration.OfflineSetup
Set ms    = Setup.GetMappingSetById "276665B6-AC1F-42AA-BAA2-8BC4C0FE1AC7"

Example — ImportMeasurementSetup method

The following example imports the measurement setup of a configuration file:

Application.Configuration.OnlineSetup.ImportMeasurementSetup("C:\Configuration.cfg")

Example — RemoveMappingSet method

Removes the default mapping:

' Removes the default mapping
Set App = CreateObject( "CANoe.Application")
Set offlineSetup = App.Configuration.OfflineSetup
Set defaultMapping = offlineSetup.OfflineSourceRoot.GetMappingSet
if not defaultMapping is Nothing then
  offlineSetup.RemoveMappingSet(defaultMapping)
end if
