Node

Application » Configuration » SimulationSetup » Nodes » Node

Definition

"Node" describes a Program Node, e.g. CAPL/C# nodes of the Simulation Setup, or Application Models of the Communication Setup.

Description

The Node object represents a »node of the Simulation Setup / Communication Setup of the CANoe application.

DetachBus

Removes the node from a bus.

In case the node is attached to one single bus, detaching this bus would be equal to removing the node at all.

Therefore, detaching the last bus of the node is not allowed.

This behavior of the COM interface is equal to the behavior of the CANoe UI.

Syntax: object.DetachBus

(see example below)

11.0 SP3

Type: Bus

Example

Example — AttachedBuses property

The following example displays the number of attached buses.

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
msgbox Node.AttachedBuses.Count

Example — DriftJitter property

The following example sets the min (5%) and max (10%) value of the drift/jitter of a CAPL node within the Simulation Setup:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
// object.DriftJitter[min, max]
Node.DriftJitter 5, 10

Example — DriftJitterActive property

The following example deactivates the drift/jitter of a CAPL node within the Simulation Setup:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
Node.DriftJitterActive = False

Example — DriftJitterFixedDeviation property

The following example sets the value of the drift/jitter (fixed deviation) of a CAPL node within the Simulation Setup:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")

Node.DriftJitterFixedDeviation = 25

Example — DriftJitterFromDb property

The following example sets the value of drift/jitter of a CAPL node within the Simulation Setup to the database value:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
Node.DriftJitterFromDb = True

Example — DriftJitterMax property

The following example displays the max value of the drift/jitter of a CAPL node within the Simulation Setup:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
msgbox "DriftJitterMax=" & Node.DriftJitterMax

Example — DriftJitterMin property

The following example displays the min value of the drift/jitter of a CAPL node within the Simulation Setup:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
msgbox "DriftJitterMin=" & Node.DriftJitterMin

Example — DriftJitterMode property

The following example sets the fixed deviation mode for a CAPL node within the Simulation Setup:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
Node.DriftJitterMode = 0

Example — ERTNetworkNode property

The following example verifies whether a node is an ERT network node:

If Node.ERTNetworkNode Then
  Wscript.Echo "Node is ERT network node"
End If

Example — ERTNodeConfiguration property

The following example gets the node configuration of an ERT network node:

Dim ErtNodeConfiguration

If Node.ERTNetworkNode Then
  Set ErtNodeConfiguration = Node.ERTNodeConfiguration
End If

Example — IsGateway property

The following example displays whether the node is a gateway:

Dim Nodes
Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
msgbox Node.IsGateway

Example — PortCreation property

To disable automatic port creation of the node ECU1:

Set Ports = Bus.Ports
Set Node  = App.Configuration.SimulationSetup.Nodes("ECU1")

if not Ports.IsPortBasedConfig Then
  Wscript.Echo "Configuration is not network based"
Else
  Node.PortCreation = 0
End If

Example — StartDelay property

The following example sets a start delay of a CAPL node within the Simulation Setup to 200 ms:

Dim Nodes
Set Node        = Application.Configuration.SimulationSetup.Nodes("comauto")
Node.StartDelay = 200

Example — StartDelayActive property

The following example activates the start delay of a CAPL node within the Simulation Setup:

Dim Nodes
Set Node              = Application.Configuration.SimulationSetup.Nodes("comauto")
Node.StartDelayActive = True

Example — StartDelayFromDb property

The following example sets the value of start delay of a CAPL node within the Simulation Setup to the CANdb value:

Dim Nodes
Set Node              = Application.Configuration.SimulationSetup.Nodes("comauto")
Node.StartDelayFromDb = True

Example — TestModule property

The following example verifies whether a node is a test module:

If Node.TestModule Then
  ‘ do something
End If

See also COM Samples: CentralLockingSystem — CANoe_TestSetup.vbs

Example — AttachBus method

The following example adds a new bus and attaches the new bus to a node.

Dim Nodes

Set Node   = Application.Configuration.SimulationSetup.Nodes("comauto")
Set NewBus = Application.Configuration.SimulationSetup.Buses.Add("NewBus")

Node.AttachBus(NewBus)

Example — DetachBus method

The following example detaches a bus from the node.

Dim Nodes

Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
Set Bus  = Application.Configuration.SimulationSetup.Buses.Item(1)

Node.DetachBus(Bus)

Example — IsBusAttached method

The following example displays whether the bus is attached to the node.

Dim Nodes

Set Node = Application.Configuration.SimulationSetup.Nodes("comauto")
Set Bus  = Application.Configuration.SimulationSetup.Buses.Item(1)

msgbox Node.IsBusAttached(Bus)
