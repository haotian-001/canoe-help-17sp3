Description

The StartState object represents the start state of a VT System module or channel.

This flag indicates if the start state is enabled or not.

Example

The following example written in PowerShell demonstrates how to modify the start state of a module or channel:

# Get the VT System configuration instance
$canoe = New-Object -ComObject CANoe.Application
$vts   = $canoe.Configuration.VTSystem

# Get the start state of a VT1104 channel
$startState = $vts.GetChannelByName("M3_Ch4").StartState

# Change the start state
$startState.IsEnabled = $true
$startState.StartValues["MeasurementMode"].Value = 1 # Filtered
$startState.StartValues["Threshold"].Value = 3.0

# Get the names of all availabe start values
Foreach ($startValue in $startState.StartValues)
{
  Write-Host("Start value " + $startValue.Name)
}

# Reset the start state to defaults
$startState.ResetToDefaults()
