Description

The Signal object represents a signal on the bus.

Properties

Name

Availability

(since version)

Property Value/Type

Property Value Element

FullName

This property returns the fully qualified name of a signal.

In connection with an Exporter object a Signal object can also represent an environment variable.

Syntax: object.FullName[=fullname]

(read-only)

(see example below)

✔

Parameters

object

Type: Signal

fullname

(optional)

Type: STRING

The fully qualified name in the following format: <DatabaseName>::<EnvironmentVariable>.

—

IsOnline

This property checks whether the measurement is running and the signal has been received.

Syntax: object.IsOnline

(read-only)

(see example below)

✔

Type: BOOL

TRUE

The measurement is running and the signal has been received.

FALSE

The measurement is not running and/or the signal has not been received.

—

RawValue

This property sets or returns the current value of the signal as it will be / has been transmitted on the bus.

Syntax: object.RawValue

(see example below)

✔

The raw value of the signal.

—

State

This property returns the state of the signal; see return values.

Syntax: object.State

(read-only)

(see example below)

✔

Type: INT

The state of the signal; possible values:

0

The default value of the signal is returned.

1

The measurement is not running; the value set by the application is returned.

2

The measurement is not running; the value of the last measurement is returned.

3

The signal has been received in the current measurement; the current value is returned.

—

Value

This property sets or returns the active value of the signal.

Syntax: object.Value[=value]

✔

object

value

(optional)

The value of the signal.

Example

Example — FullName/IsOnline/State property

Set Application = CreateObject( "CANoe.Application")
Set Signal = Application.Bus.GetSignal( 2, "Ignition_Info", "StarterKey")

Text = "State of signal " & Signal.FullName
State = Signal.State

If Signal.IsOnline Then
  Text = Text & " : Online, State = " & State
Else
  Text = Text & " : Offline, State = " & State
End If

MsgBox text

Set Signal      = Nothing
Set Application = Nothing

Example — RawValue property

The following example indicates the raw value of a signal:

MsgBox "Last transmitted value for CarSpeed is " & CarSpeed.RawValue
