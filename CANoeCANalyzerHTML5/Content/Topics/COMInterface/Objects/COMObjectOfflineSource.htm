OfflineSource

Application » Configuration » MeasurementSetup » OfflineSource

Description

The OfflineSource object sets or returns the name of the offline source file including the path, and it configures its channel mapping.

Sources

This property returns a Files object for each source file of the exporter or offline source.

On the first access to the source files of the exporter the Files object contains the source file as defined in the Export dialog.

Typically this is the logging file as set in the Logging Block, as long as no other source file has been defined manually.

Example

Example — Sources property

The following example deletes all assigned logging files and adds a new logging file to the offline sources:

Set Application = CreateObject("CANoe.Application")
Set Sources     = Application.Configuration.OfflineSetup.Source.Sources

'delete all assigned source files
Sources.Clear

'add a source file to the offline source object
Sources.Add( "F:\Demo_CAN_CN\CANSystemDemo\Logging\EngineDiagData.asc")

Example — TimeSection property

The following example configures a time section for the offline source and defines a start and end time stamp to be considered.

Set offlineSource               = Application.Configuration.OfflineSetup.Source
offlineSource.TimeSection.Type  = 2
offlineSource.TimeSection.Start = "01:12:15.123"
offlineSource.TimeSection.End   = "02:13:16.321"
Application.Measurement.Start

Example — ImportMappingTable method

This example imports the channel mapping configuration from the XML file D:\MyMapping.chmap:

App.Configuration.OfflineSetup.Source.ImportMappingTable "D:\MyMapping.chmap"

Example of a valid XML file:

<?xml version="1.0" encoding="utf-8"?>

<ChannelMapExport xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xmlns:xsd="http://www.w3.org/2001/XMLSchema">
  <mMappingList>
    <Data>
      <Source>1</Source>
      <Bus>CAN</Bus>
      <Destination>ignore</Destination>
    </Data>
    <Data><Source>2</Source><Bus>CAN</Bus><Destination>ignore</Destination></Data>
    <Data><Source>101</Source><Bus>CAN</Bus><Destination>1</Destination></Data>
    <Data><Source>102</Source><Bus>CAN</Bus><Destination>2</Destination></Data>
  </mMappingList>
</ChannelMapExport>
