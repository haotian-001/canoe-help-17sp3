MostApplicationFilter

Application » Networks » Network » Devices » Device » MostNetworkInterface » MostApplicationFilter

Description

The MostApplicationFilter object represents a filter, that filters the messages and packets in the corresponding MostNetworkInterface before they are fired as events.

The purpose of this object is to optimize the performance of the client-server communication.

The MostApplicationFilter object is based on the MOST Application Filter, that can be placed in the Measurement Setup, but it only affects the corresponding MostNetworkInterface object.

Enabled

This property Enables/Disables the MOST Application Filter.

By default the MOST Application Filter is a pass filter with no message/packet configured. So just activating it will result in no message/packet events been fired anymore by the MOST network interface until some messages/packets have been added to the filter or the filter type has been switched to block filter.

AddApplicationEvent

Adds an application event to the MOST Application Filter (CANoe).

The application event added to the MOST Application Filter using this method is applied to all messages as well as all packets.

In addition to the valid values described below, all parameters (except object) have 'wildcard' (long -1 / 0xFFFFFFFF) as valid value.

The filter type and the enable state of the MOST Application Filter are not affected by this method.

Example

Example — FilterType property

The following example sets the filter type for the MOST Application Filter to Pass Filter:

MostApplicationFilter.FilterType = 0

Example — AddApplicationEvent method

The following example adds an application event with fblockID = 1 (NetBlock) to the MOST Application Filter:

Dim wildcard
wildcard = &HFFFFFFFF
MostApplicationFilter.AddApplicationEvent wildcard, wildcard, 1

Example — AddRawPacket method

The following example adds a raw packet with destinationAddress = 0x101 to the MOST Application Filter:

Dim wildcard
wildcard = &HFFFFFFFF
MostApplicationFilter.AddRawPacket wildcard, &H101

Example — AddSystemMessage method

The following example adds a system message with rType = 3 (Allocate) to the MOST Application Filter:

Dim wildcard
wildcard = &HFFFFFFFF
MostApplicationFilter.AddSystemMessage wildcard, wildcard, 3
