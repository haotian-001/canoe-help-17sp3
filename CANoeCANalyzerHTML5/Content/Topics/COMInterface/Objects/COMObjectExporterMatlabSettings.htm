Description

The ExporterMatlabSettings object represents the exporter settings for the MATLAB format.

ExportMode

This property sets or determines the export mode for the CSV or MATLAB format..

Syntax: object.ExportMode[=ExportMode]

(see example below)

✔

Parameters:

object

Type: ExporterMatlabSettings

ExportMode

(optional)

Type: ENUM

The export mode; possible values:

0|cOneTimeChannelPerSignal

Every signal has its one time channel.

1|cOneTimeChannelForAllSignals_CurrentValue

All signals share one time channel. Cells without value are empty.

2|cOneTimeChannelForAllSignals_LastValue

All signals share one time channel.

Cells without value are filled with the last valid value.

3|cOneTimeChannelForAllSignals_NextValue

All signals share one time channel.

Cells without value are filled with the next valid value.

4|cOneTimeChannelForAllSignals_InterpolatedValue

All signals share one time channel.

If possible, cells without value are filled with an interpolated value else the last value will be used.

5|cNewTimeChannelForAllSignals_LastValue

All signals share one new created time channel.

Cells without value are filled with the last valid value.

6|cNewTimeChannelForAllSignals_NextValue

All signals share one new created time channel.

Cells without value are filled with the next valid value.

7|cNewTimeChannelForAllSignals_InterpolatedValue

All signals share one new created time channel.

If possible, cells without value are filled with an interpolated value else the last value will be used.

NameLength

This property sets the length of the signal name.

The signal names are shortened during export to the appropriate length.

The exported file can then be used in older MATLAB versions.

Syntax: object.NameLength[=length]

(see example below)

✔

Parameters:

object

Type: ExporterMatlabSettings

length

(optional)

The length of the signal name; possible values:

0

No limitation.

19

MATLAB up to and including version 4.x (19 characters).

31

MATLAB version 5.x and higher (31 characters).

63

MATLAB version 6.5 and higher (63 characters).
