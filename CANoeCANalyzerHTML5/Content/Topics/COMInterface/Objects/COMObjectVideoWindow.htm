VideoWindow

Application » Configuration » MeasurementSetup » VideoWindows » VideoWindow

Description

The VideoWindow object represents a single Video Window of the current CANoe configuration.

Example

Example — General

Dim App, Measurement, VideoWindows, VideoWindow
Set App          = CreateObject("CANoe.Application")
Set Measurement  = App.Measurement

Set VideoWindows = App.Configuration.OnlineSetup.VideoWindows
Set VideoWindow  = VideoWindows(1)

‘Configure recorder
VideoWindow.RecordFile ="RecorderFile.avi"
VideoWindow.SetTriggerTimes 5000, 10000

'Start measurement and wait for 10 sec
Measurement.Start
Wscript.Sleep 10000

'Start video recording and stop it after 30 sec
VideoWindow.StartRecording
Wscript.Sleep 30000
VideoWindow.StopRecording

'Stop measurement after 20 sec
Wscript.Sleep 20000
Measurement.Stop
