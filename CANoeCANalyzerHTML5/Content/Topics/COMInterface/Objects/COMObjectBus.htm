Description

The Bus object represents a bus of the CANoe application.

The instantiation of the Bus object is done by the Bus property of the Application object.

If no bus system is handed over a CAN bus object will be instantiated: Set Bus_CAN = App.Bus

GetSignal

Returns a Signal object.

Other bus systems (LIN, FlexRay, MOST, AFDX, Ethernet) can be accessed in the same manner as CAN.

Syntax: object.GetSignal(channel, message, signal)

(see example below)

✔

channel, message, signal

channel

Type: INT

The channel on which the signal is sent; possible values:

0…65535

the maximum valid value range for the channel number

0

0 is the wildcard for the channel selection.

If this is set, however, the channel number is internally set to -1 and the signal is searched for in the database for all existing channels.

The possibility to pass the 0 is especially relevant in the VB context, because there is no signed integer data type.

-1

Passing the -1 directly in the interface leads to the same result as passing 0.

message

Type: STRING

The name of the message to which the signal belongs.

signal

Type: STRING

The name of the signal.

Type: Signal

Example

Example — FlexRay bus

Accessing a signal:

Set App     = CreateObject( "CANoe.Application")
Set FlexRay = App.Bus( "FlexRay")
Set Signal  = FlexRay.GetSignal( 1, "ABSData", "CarSpeed")

Example — AutogenerateNodes method

The example gets the bus named LIN1 from the Simulation Setup, assigns a database to it, and automatically generates the network nodes afterwards.

Set App    = CreateObject( "CANoe.Application")
Set LinBus = App.Configuration.SimulationSetup.Buses("LIN1")

LinBus.Databases.Add("C:\Path\to\ExampleDb.ldf")
LinBus.AutogenerateNodes()

Example — GetSignal method

Set App      = CreateObject( "CANoe.Application")

// Access to the signal value
Set Signal = App.Bus( "CAN").GetSignal( 1, "ABSData", "CarSpeed")
value      = Signal.Value
Signal.Value = 12

Example — GetJ1939Signal method

The following example assigns a Signal object to a variable:

Dim EngSpeed
Set EngSpeed = Application.Bus( "CAN" ).GetJ1939Signal( 1, "EEC1_EMS", "EngSpeed", 0, 255 )
