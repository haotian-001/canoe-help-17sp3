TestConfigurations

Application » Configuration » TestConfigurations

Description

The TestConfigurations object represents the collection of test configurations within the CANoe configuration.

Example

Example — Add method

In this example a new test configuration is added:

Dim CanoeApp, Configuration, TestConfigurations, TestConfiguration

Set CanoeApp           = CreateObject("CANoe.Application")
Set Configuration      = CanoeApp.Configuration
Set TestConfigurations = Configuration.TestConfigurations

Set TestConfiguration  = TestConfigurations.Add
WScript.Echo TestConfiguration.Name

Example — Remove method

In this example two test configurations are added and removed respectively by index and by name:

Dim CanoeApp, Configuration, TestConfigurations
Dim TestConfiguration, index, name

Set CanoeApp           = CreateObject("CANoe.Application")
Set Configuration      = CanoeApp.Configuration
Set TestConfigurations = Configuration.TestConfigurations

' remove a test configuration by index
Set TestConfiguration = TestConfigurations.Add
index                 = TestConfigurations.Count
WScript.Echo "Removing test configuration number" & CStr(index) & "."
TestConfigurations.Remove index

' remove a test configuration by name
Set TestConfiguration = TestConfigurations.Add
name                  = TestConfiguration.Name
WScript.Echo "Removing test configuration """ & name & """"
TestConfigurations.Remove CStr(name)
