OfflineGroup

Application » Configuration » MeasurementSetup » OfflineGroup

Description

The OfflineGroup object represents a group of offline sources.

This object inherits from OfflineSourceEx.

Example

Example — General

Set Application = CreateObject( "CANoe.Application")
' get the root group of all offline sources
Set offlineSourceRoot = Application.Configuration.OfflineSetup.OfflineSourceRoot

Example — MergeMode property

The following example sets the MergeMode:

OfflineGroup.MergeMode = 1  //eOfflineSourceMergeMode.cByWallClockTime

Example — GetChildren method

on error resume next
Set Application = CreateObject( "CANoe.Application")
' get the path of all offline sources
Set offlineSources = Application.Configuration.OfflineSetup.OfflineSourceRoot.GetChildren
for i = 1 to offlineSources.Count
  ' log files and video sources provide a path property
  MsgBox "Source " + CStr(i) + ": " + offlineSources(i).Path
  if Err.Number <> 0 Then
    Err.Clear
    ' image sequences provide a directory and a pattern
    MsgBox "Source " + CStr(i) + ": " + offlineSources(i).Directory + offlineSources(i).Pattern + "*"
  end if
  if Err.Number <> 0 Then
    Err.Clear
    MsgBox "Source " + CStr(i) + " is not a file-based offline source"
  end if
next

Example — GetImageSequences method

Set Application = CreateObject( "CANoe.Application")
' get directory and file pattern of all offline image sequences
Set imageSequences = Application.Configuration.OfflineSetup.OfflineSourceRoot.GetImageSequences
for i = 1 to imageSequences.Count
  MsgBox "Image sequence " + CStr(i) + ": " + imageSequences(i).Directory + imageSequences(i).Pattern + "*"
next

Example — GetLogFiles method

Set Application = CreateObject( "CANoe.Application")
' get the path of all offline log files
Set offlineSources = Application.Configuration.OfflineSetup.OfflineSourceRoot.GetLogFiles
for i = 1 to offlineSources.Count
  MsgBox "Log file " + CStr(i) + ": " + offlineSources(i).Path
next

Example — GetMappingSet method

' Get the default channel mapping of CAN 1 to 4
Set App = CreateObject( "CANoe.Application")
Set defaultMapping = App.Configuration.OfflineSetup.OfflineSourceRoot.GetMappingSet
if defaultMapping is Nothing then
  MsgBox "Default mapping set is not set"
else
  for src = 1 to 4
    dst = defaultMapping.GetByBusName("CAN", src)
    MsgBox "CAN " + CStr(src) + " -> " + CStr(dst)
  next
end if

Example — GetMappingTableByBusName method

This example maps CAN channel 101 from a logging file to channel 1 in CANoe.

Messages on channel 1 in the logging file are ignored.
Set Application = CreateObject( "CANoe.Application")
Set Mapping = Application.Configuration.OfflineSetup.OfflineSourceRoot.GetMappingTableByBusName("CAN")
Mapping.Put 1, 0
Mapping.Put 101,1

Example — GetVideoSources method

Set Application = CreateObject( "CANoe.Application")
' get the path of all offline videos
Set offlineVideos = Application.Configuration.OfflineSetup.OfflineSourceRoot.GetVideoSources
for i = 1 to offlineVideos.Count
  MsgBox "Video " + CStr(i) + ": " + offlineVideos(i).Path
next

Example — SetMappingSet method

This example imports channel mapping sets from a file, and assigns each mapping set to the corresponding logging file:

' get offline files
Set App = CreateObject( "CANoe.Application")
Set Setup = App.Configuration.OfflineSetup
Set logFiles = Setup.OfflineSourceRoot.GetLogFiles
' import some mappings
Set mappingSets = Setup.ImportMappingSets("D:\myMappingSets.chmap")
' Assign mappings to logfiles
for i = 1 to logFiles.Count
  for j = 1 to mappingSets.Count
    fileName = Mid(logFiles(i).Name, InStrRev(logFiles(i).Name, "\") + 1)
    if fileName = mappingSets(j).Name then
      logFiles(i).SetMappingSet mappingSets(j)
    end if
  next
next
