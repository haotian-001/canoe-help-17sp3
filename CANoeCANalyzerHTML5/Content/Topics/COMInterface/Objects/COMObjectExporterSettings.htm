NameDelimiter

This property sets or determines the delimiter used to separate the specific parts of a qualified signal/variable name.

The maximum length of the delimiter is limited to 3 characters. All further characters will be ignored.

Syntax: object.NameDelimiter[=nameDelimiter]

✔

Parameters:

object

Type: ExporterSettings

nameDelimiter

(optional)

The delimiter used to separate the specific parts of a qualified signal/variable name.

—

SaveInvalidJ1939Values

This property places or defines whether invalid values of J1939 signals should be considered with the export.

On changing this setting the source files of the exporter must be loaded again, so that a correct export can be executed.

Syntax: object.SaveInvalidJ1939Values[=saveInvalidJ1939Values]

✔

Parameters:

object

Type: ExporterSettings

saveInvalidJ1939Values

(optional)

Type: BOOL

Indicates whether invalid values of J1939 signals are considered with the export; possible values:

TRUE

Invalid values are considered.

FALSE

Invalid values are not considered.

WriteSignalValuesRaw

This property sets or determines if signal values are written raw or physical.

If WriteSignalValuesSymbolic is true and if a symbolic value exists it has priority over raw/physical values.

Syntax: object.WriteSignalValuesRaw[=writeSignalValuesRaw]

✔

Parameters:

object

Type: ExporterSettings

writeSignalValuesRaw

(optional)

Type: BOOL

Determines whether signal values are written raw or physical; possible values:

1|TRUE

Signal values are written raw.

0|FALSE

Signal values are written physical.
