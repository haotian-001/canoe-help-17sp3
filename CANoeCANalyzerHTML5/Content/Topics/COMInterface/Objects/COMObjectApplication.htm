Application

Application

Description

The Application object represents the CANoe application.

The Application object is the foundation for the object hierarchy. You can reach all other objects from the Application object.

If none of the parameters is set, a Networks object is returned that contains all networks of all bus systems of the CANoe configuration.

Only one of the optional parameters can be set.

New

This method creates a new configuration.

Syntax: object.New([autoSave[, promptUser]])

(see example below)

3.0

autoSave, promptUser

autoSave

(optional)

Type: BOOL

Indicates whether the active configuration should be saved if it has been changed; possible values:

TRUE

Automatic saving is executed.

FALSE

Automatic saving is not executed.

If FALSE is entered, the function will fail if the active configuration was changed.

promptUser

(optional)

Type: BOOL

Indicates whether the user should intervene in error situations; possible values:

TRUE

The user is prompted for input in error situations.

FALSE

The user will not be notified.

If FALSE is entered for the promptUser parameter, the function will fail if an error situation occurs; possible error situations include:

  • the active configuration was created or changed and not saved yet
  • automatic saving fails for some reason (write protection, full disk,…)

Example

Example — Bus property

The following example instantiates a Bus object:

Set App          = CreateObject( "CANoe.Application")
Set Bus_CAN      = App.Bus( "CAN")
Set Bus_LIN      = App.Bus( "LIN")
Set Bus_FlexRay  = App.Bus( "FlexRay")
Set Bus_AFDX     = App.Bus( "AFDX")
Set Bus_Ethernet = App.Bus( "Ethernet")

Example — ChannelMappingName property

Simple VBScript example:

Set Spp = CreateObject("CANoe.Application")

'set the name:
App.ChannelMappingName = "CANoe_Instance1"

'read the name
appNameRead = App.ChannelMappingName

Example — Networks property

  • The following example assigns Networks objects with different content to several variables:

    Set App           = CreateObject( "CANoe.Application")
    Set Networks_All  = App.Networks      'all networks of all bus systems of the CANoe configuration
    Set Networks_CAN  = App.Networks(1)   'all CAN networks of the CANoe configuration
    Set Networks_MOST = App.Networks(6)   'all MOST networks of the CANoe configuration

  • The following example assigns a Network object to a variable:

    Set App          = CreateObject( "CANoe.Application")
    Set MOST_Network = App.Networks("MOST Ring")
    'The (one) MOST network.
    '(There is always exactly one MOST network. It's name is "MOST Ring".)

Example — Performance property

This example sets the maximum number of additional threads for logging branches in the Measurement Setup to 16:

Application.Performance.MaxNumMeasurementSetupThreads = 16

Example — Visible property

The following example hides the CANoe program window and shows a tray icon.

Set App = CreateObject( "CANoe.Application")
App.Visible = false

If App.Visible Then
  MsgBox "CANoe is visible"
Else
  MsgBox "CANoe is not visible"
End If

Example — New method

The following example creates a new configuration:

Application.New

Example — Open method

The following example opens a configuration:

Application.Open "C:\CANWIN\DEMO_CN\CANSYSTEMDEMO\CANSYSTEMDEMO.CFG"

Example — Quit method

The following example quits CANoe:

Application.Quit

A configuration was changed during the measurement (the dirty flag * was set). The following example quits CANoe without saving changes in the configuration. No message for saving the configuration is displayed.

Set App           = CreateObject( "CANoe.Application")
Set Measurement   = App.Measurement
Set Configuration = App.Configuration

If Not Measurement.Running Then
  Measurement.Start

  WScript.Sleep 7000
  App.Measurement.Stop
  WScript.Sleep 7000
  App.Configuration.Modified = False
  App.Quit

Example — OnOpen event

The following example shows a procedure which is called when a configuration is opened:

Sub Application_OnOpen(fullname)
  Set Configuration = App.Configuration
End Sub

Example — OnQuit event

The following example shows a procedure that is called when CANoe is quit:

Sub Application_OnQuit()
  Set App = Nothing
End Sub
