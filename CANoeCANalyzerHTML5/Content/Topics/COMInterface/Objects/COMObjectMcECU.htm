Description

DAQTimeout

This property sets the maximum timeout between DAQ messages received from the ECU.

If the ECU does not send a DAQ command within the timeout, the communication will be considered as broken and a new connection will be established depending on the ReconnectAllowedAfterError and NumberOfConnectionAttempts settings.

Syntax: object.DAQTimeout

✔

Type: INT

The maximum time between DAQ messages.

    Example

    Example — General

    Dim gApp, gConfig, gGeneralSetup, gXCPSetup, gECUs, gECU, gSignals

    gApp          = New CANoe.Application
    gConfig       = gApp.Configuration
    gGeneralSetup = gConfig.GeneralSetup
    gXCPSetup     = gGeneralSetup.XCPSetup

    'Find existing ECU
    gECUs = gXCPSetup.ECUs
    gECU  = gECUs("XCPsim")

    'Change settings
    gECU.ConnectOnMeasurementStart        = 1
    gECU.DisconnectOnMeasurementStop      = 1
    gECU.ObserverActive                   = 1
    gECU.PageSwitchingActive              = 1
    gECU.RAMpage                          = 1
    gECU.ResetVariablesAfterDisconnect    = 1
    gECU.UseDAQTimestampsOfECU            = 1
    gECU.UseDAQTimestampsOfECUDivOperator = 0
