StartValueList

Application » Configuration » StartValueList

Description

The StartValueList object provides access to the start values which are defined in the Start Values window.

You must have read and write access on the directory where the start values file is located.

Append

This method adds the start values defined in the file to the Start Values Window.

You must have read and write access on the directory where the start values file is located.

The contents of the Start Values Window will not be overwritten by the start values contained in the file.

Syntax: object.Append (startValuesFile, validationMessage)

Example

Example — Append method

Set mCANoeApp = CreateObject("CANoe.Application")

' Open CANoe configuration
mCANoeApp.Open "MyConfiguration.cfg", True, True

Set ocResult = mCANoeApp.Configuration.OpenConfigurationResult

If (ocResult.Result > 0) Then
  MsgBox "Configuration could not be opened."
  Set ocResult = Nothing
  mCANoeApp.Quit
  Set mCANoeApp = Nothing
  MsgBox "Script done..."
End If

Set mStartValueList = mCANoeApp.Configuration.StartValueList
Dim mValidationMessage

' Append
If (mStartValueList.Append("File2.svc", mValidationMessage)) Then
  If (Not (Len(mValidationMessage & "") = 0)) Then
    MsgBox "Start values file successfully appended. Message: " & mValidationMessage
  Else
    MsgBox "Start values file successfully appended."
  End If
Else
  MsgBox "Start values file append failed. Message: " & mValidationMessage
End If

' Close CANoe
Set mStartValueList = Nothing
mCANoeApp.Quit
Set mCANoeApp = Nothing

MsgBox "Script done..."
