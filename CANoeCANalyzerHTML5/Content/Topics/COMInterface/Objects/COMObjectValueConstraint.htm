Description

The ValueConstraint object represents a single value constraint on a VT System module or channel.

Operator that is used to check if a new setting value is allowed or not; possible values:

1|cGtEq

Setting must be greater than or equal to the parameter.

2|cGt

Example

The following example written in PowerShell demonstrates how to create and configure different types of constraints:

# Get the VT System configuration instance
$canoe = New-Object -ComObject CANoe.Application
$vts   = $canoe.Configuration.VTSystem

# Get a channel
$channel = $vts.GetChannelByName("M1_Ch1")

# Create a value constraint
$valueConstraint = $channel.ValueConstraints.Add()
$valueConstraint.Setting   = "Voltage"
$valueConstraint.Operator  = 3 # Less than
$valueConstraint.Parameter = 3.5 # 3.5V

# Create a relay contraint
$relayConstraint = $channel.RelayConstraints.Add()
$relayConstraint.Setting            = "RelayShortCircuit"
$relayConstraint.Operation          = 1 # Closing
$relayConstraint.Parameter.Sysvar   = "MyVariables::MyVoltage"
$relayConstraint.Parameter.Operator = 0 # Equals
$relayConstraint.Parameter.Value    = 0.0 # 0V
