UI

Application » UI

Description

The UI object represents the user interface in CANoe.

Properties

Name

Availability

(since version)

Property Value/Type

Property Value Element

CommandEnabled

Sets or defines the availability of a command on the user interface.

Activation/deactivation of all commands is not supported yet.

Syntax: object.CommandEnabled([command])[=boolean]

(see example below)

✔

Parameters:

object

Type: UI

command

(optional)

The command.

Currently only the values start and stop can be input for the command parameter.

If no command is entered the function acts on all commands that can currently be influenced via this interface.

boolean

(optional)

Type: BOOL

A boolean value that indicates whether the command should be available; possible values:

TRUE

the command is available

FALSE

the command is not available

Example

Example — CommandEnabled property

The following example disables the Stop command:

App.UI.CommandEnabled( "stop") = False

Example — CreateDesktop method

The following example creates a new desktop:

Application.UI.CreateDesktop("MyDesktop")

Example — ActivateDesktop method

The following example switches to the desktop with the name "Simulation Setup":

Application.UI.ActivateDesktop "Simulation Setup"

Example — OpenBaudrateDialog method

The following example opens the dialog for configuring the bus parameters:

Application.UI.OpenBaudrateDialog

Example — OpenDbDialog method

The following example shows a list of all assigned databases:

Application.UI.OpenDbDialog
