Description

The DiagDescription object represents one diagnostic "master" description configured in CANoe.

Use the AdditionalDescriptions property to access additional descriptions.

ReplaceDescriptionFile

This method replaces the diagnostic description loaded for the description with a different database, keeping all settings.

Replacing is only possible when measurement is not running.

The new database file may have any of the supported database formats, e.g. CDD, PDX, MDX…

If more than 1 ECU is defined in the database, the ECU with the identifier originally configured is selected. If no such ECU is found, replacing the database fails.

The new ECU must define a communication interface with the same qualifier as currently configured, i.e. when no such interface is found, replacing the database fails.

Syntax: object.ReplaceDescriptionFile (filePath)

(see example below)

✔

filePath

The location of the diagnostic description file to load.

Example

Example — AdditionalDescriptions property

This example adds two additional descriptions to the first "master" description and defines a certain interpretation order for them.

All diagnostic windows for the first additional description and the fault memory window for the second additional description are opened.

set app               = CreateObject("CANoe.Application")
set masterDescription = app.Configuration.GeneralSetup.DiagnosticsSetup.DiagDescriptions.Item(1)
set addDesc1          = desc.AdditionalDescriptions.Add("CDD\UDS-ExampleEcu-4.0.2.cdd")
set addDesc2          = desc.AdditionalDescriptions.Add("ODX\UDS-ExampleEcu-8.0.1.pdx", "Door")

masterDescription.InterpretationOrder.Move 1, 2

addDesc1.OpenWindows
addDesc2.OpenWindows 4

Print some information about a diagnostic description

Set description = Application.Configuration.GeneralSetup.DiagnosticsSetup.DiagDescriptions( "Any_ECU_Example")

Example — Mode property

If description.Mode = 0 Then
  Wscript.echo "InterpretationOnly"
ElseIf description.Mode = 1 Then
  Wscript.echo "SendOnly"
ElseIf description.Mode = 2 Then
  Wscript.echo "EcuSimulation: " & description.Node
ElseIf description.Mode = 3 Then
  Wscript.echo "PNR"
ElseIf description.Mode = 4 Then
  Wscript.echo "FNR"
End If

Example — ReplaceDescriptionFile method

This examples replaces the diagnostic description file Airbag_V1.1.pdx with the new file Airbag_V1.2.pdx:

Dim App, diagSetup, diagDescription, descr

set App = createobject( "CANoe.Application")

If App.Measurement.Running Then
  WScript.echo "Measurement running! Expect that change will fail"
End If

Set diagSetup = App.Configuration.GeneralSetup.DiagnosticsSetup
Set descr = diagSetup.DiagDescriptions( "ECU1")

Err.clear
' replace old version of the airbag description with the new one
descr.ReplaceDescriptionFile "Airbag_V1.2.pdx"
If Err.number = 0 Then
  Wscript.echo "Replace reported no problem"
Else
  Wscript.echo "Replace reported error " & Hex( Err.number) & " '" & Err.description & "'"
End If
