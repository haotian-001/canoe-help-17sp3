ViewSynchronization

Application » Configuration » MeasurementSetup » ViewSynchronization

Description

The ViewSynchronization object represents the Synchronize Windows feature of CANoe, which is available e.g. in the Trace Window's context menu.

During a measurement synchronization only works if the appropriate CANoe window (e.g. Trace, Graphics) is paused.

Properties

Name

Availability

(since version)

Property Value/Type

Property Value Element

Enabled

This property activates/deactivates the synchronization of windows.

The activation state of the synchronization of windows can be queried.

The activation state is set globally for all CANoe windows and all connected COM clients.

Syntax: object.Enabled [=enabled]

Example

Example: SetTime method

The following example sets the time stamp 2.5 seconds:

Dim viewSynchronization
Set viewSynchronization = Application.Configuration.OnlineSetup.ViewSynchronization
viewSynchronization.SetTime 0, 250000

Example: SetTimeNS method

The following example sets the time stamp 1.7 seconds:

Dim viewSynchronization
Set viewSynchronization = Application.Configuration.OnlineSetup.ViewSynchronization
viewSynchronization.SetTimeNS 0, 1700000000
