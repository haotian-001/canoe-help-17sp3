ERTNodeConfiguration

Application » Configuration » SimulationSetup » Nodes » Node » ERTNodeConfiguration

Description

The ERTNodeConfiguration object represents the ERT specific configuration of an ERT network node.

CAPLTiming

This property sets or returns whether CAPL timing variables will be / are enabled.

This property is only valid if a CAPL source file is attached.

Syntax: object.CAPLTiming[=status]

(see example below)

16 SP2

Parameters

object

Type: ERTNodeConfiguration

status

(optional)

Type: BOOL

The synchronization status of ERT variables for time measurements of CAPL timers; possible values:

TRUE

CAPL timing variables will be / are enabled.

FALSE

CAPL timing variables will be / are disabled.

Example

Example — CAPLTiming property

The following example checks if CAPL timing variables are enabled and enables it if necessary:

Dim ErtNodeConfiguration
Set ErtNodeConfiguration = ErtNode.ERTNodeConfiguration

If ErtNodeConfiguration.CAPLTiming = False Then
  'Only valid if a CAPL source file is attached
  ErtNodeConfiguration.CAPLTiming = True

End If

Example — DataConsistency property

The following example checks the DataConsistency and swaps it:

Dim ErtNodeConfiguration
Set ErtNodeConfiguration = ErtNode.ERTNodeConfiguration

If ErtNodeConfiguration.DataConsistency = 0 Then
  ErtNodeConfiguration.DataConsistency = 1   'cSnapshot
ElseIf ErtNodeConfiguration.DataConsistency = 1 Then
  ErtNodeConfiguration.DataConsistency = 0   'cWriteThrough
End If

Example — TaskPriority property

The following example checks the task priority and sets it to "normal"

Dim ErtNodeConfiguration
Set ErtNodeConfiguration = ErtNode.ERTNodeConfiguration

If Not ErtNodeConfiguration.TaskPriority = 3 Then
  ErtNodeConfiguration.TaskPriority = 3   'cTaskPrioNormal
End If
