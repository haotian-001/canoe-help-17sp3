Description

The McParameter object represents a single parameter from the database.

Update!

This object replaces the deprecated objects McConfiguredSignal and McDatabaseSignal!

Example

Example — General

This sample adds an XCP connection to the ECU an enables one parameter for DAQ measurement:

Dim app, config, generalSetup, XCPSetup, ECUs, ECU, groups, group, parameters, parameter

app          = New CANoe.Application
config       = app.Configuration
generalSetup = config.GeneralSetup
XCPSetup     = generalSetup.XCPSetup
ECUs         = XCPSetup.ECUs

'Add a new ECU with XCP on CAN
ECU = ECUs.Add("C:\XCPsim.a2l", CANoe.eMcTransportLayer.cTlCan)

'Access the parameters contained in the database
groups     = ECU.MeasurementGroups
group      = groups.Item(1)
group.Name = "My Group"
group.BeginUpdate
parameters = group.Parameters

'Enable a parameter for DAQ measurement
parameter            = parameters("channel2")
parameter.Configured = 1
parameter.ReadMode   = CANoe.eMcReadMode.cDAQ
parameter.EventCycle = 1
parameter.AutoRead   = 1
group.EndUpdate
