Devices

Returns the Devices object or the Device object, respectively.

In a MOST environment you can use the channel (e.g. MOST 1 for channel 1) instead of the real name of the Device object (e.g. HeadUnit).

Syntax: object.Devices [deviceName]

(see example below)

✔

Parameters:

object

Type: Network

deviceName

Type: Devices (collection)

The name of a device.

This is a shortcut that skips the Devices object in the object hierarchy and directly returns the Device object with the given name.

Type: Device

Name

This property returns the name of the network.

Syntax: object.Name

(read-only)

✔

Example

Example — Devices property

  • Sample 1

    The following example assigns the Devices object to a variable:

    Dim Devices
    Set Devices = Application.Networks.Item(1).Devices

  • Sample 2

    The following example assigns the Device object to a variable: 

    Set App        = CreateObject( "CANoe.Application")
    Set Device_CAN = App.Networks("PowerTrain").Devices("Engine")
      'Precondition: "PowerTrain" network and "Engine" device must exist.
    Set Device_MOST = App.Networks("MOST Ring").Devices("MOST 1")
      'Precondition: at least one MOST channel is configured.
