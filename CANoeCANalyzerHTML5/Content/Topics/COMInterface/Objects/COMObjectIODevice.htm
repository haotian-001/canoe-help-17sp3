IODevice

Application » Configuration » IOHardware » VectorIO » IODevices » IODevice

Description

The IODevice object represents the configuration of a single Vector I/O device in CANoe.

Example

Example — General

Copy

The following example written in PowerShell shows how to remove, create and iterate Vector I/O device configurations:

# Get the Vector I/O configuration instance
$CANoe    = New-Object -ComObject CANoe.Application
$vectorIO = $CANoe.Configuration.IOHardware.VectorIO

# Remove all devices from the configuration
$vectorIO.Devices.RemoveAll()

# Add an IOpiggy and a VN1630 I/O to the configuration
$iopiggy  = $vectorIO.Devices.Add(0)
$vn1630io = $vectorIO.Devices.Add(3)

# Configure the IOpiggy
$iopiggy.Mode  = 2 # Custom
$vn1630io.Mode = 1 # Standard

# Enumerate all configured devices
Foreach ($device in $vectorIO.Devices)
{
  Write-Host($device.Name + " on DAIO channel " + $device.DAIOChannel)
}
