Description

The CANController object represents a CAN controller for a specific channel.

The "Philips 82C200-compatible" view is assumed for the NEC 72005.

CANSetConfig

This method can be used to set the configuration of a CAN channel.

The methods switches the mode of the channel to CAN, if it was in CAN FD mode.

Syntax:
object.CANSetConfig (double baudrate,
// in bit/s
unsigned char tseg1,
// length of the time segment 1 in time quanta
unsigned char tseg2,
// length of the time segment 2 in time quanta
unsigned char sjw,
// sync jump width in time quanta
unsigned char sam,
// number of sampling points (1 or 3)
unsigned int flags);
// see description below

(see example below)

8.2 SP3

baudrate, tseg1, tseg2, sjw, sam, flags

object

Type: CANController

baudrate

The baudrate in bits/s; e.g. 500 kBaud/s = 500000 bits/s.

tseg1

Length of time segment 1 in time quanta.

tseg2

Length of time segment 2 in time quanta.

sjw

Sync jump width in time quanta; cannot be larger than tseg2.

sam

Number of sampling points in a bit around the sample point; possible values:

1

Bit is sampled only one time at the sample point.

3

Bit is sampled two times before the sample point and one time at the sample point.

flags

Valid values (all other values are reserved):

0

Normal mode

1

Silent mode (acknowledge not created)

Example

Example — Acknowledge property

The following example determines the acknowledge of the CAN controller:

Dim Acknowledge
Acknowledge = CAN1.Acknowledge

Example — BTR0 property

The following example determines the BTR0 register of the CAN controller:

Dim BTR0
BTR0 = CAN1.BTR0

Example — BTR1 property

The following example determines the BTR1 register of the CAN controller:

Dim BTR1
BTR1 = CAN1.BTR1

Example — OutputControl property

The following example determines the Output Control Register of the CAN controller:

CAN1.OutputControl = &HFA

Example — Samples property

The following example sets the number of times the CAN controller samples.

CAN1.Samples = 1

Example — SelfAckEnabled property

The following example determines the acknowledge of the CAN controller:

Dim SelfAckEnabled
SelfAckEnabled = CAN1.SelfAckEnabled

Example — Synchronisation property

The following example sets the synchronization type of the CAN controller to recessive-dominant:

CAN1.Synchronisation = 0

Example — CANSetConfig method

The following example shows the usage of the function for the CAN controller:

// Set the baudrate to 500 kBit/s, TSEG1 = 5, TSEG2 = 2 (-> SamplePoint = 75%)
// Jump width is 2 time quanta, bit is only once sampled, controller is in normal mode
CAN1.CANSetConfig (500000,5,2,2,1,0);

Example — CANSetFDArbPhaseConfig method

The following example shows the usage of the function for the CAN controller:

// Set the baudrate to 500 kBit/s, TSEG1 = 5, TSEG2 = 2 (-> SamplePoint = 75%)
// Jump width is 2 time quanta,
// controller is in normal mode

// 1 ISO CAN FD call:
CAN1.CANSetConfig (500000,5,2,2,1,0);
// 2. Non-ISO CAN FD call:
CAN1.CANSetFDArbPhaseConfig (500000,5,2,2,1);

Example — CANSetFDDataPhaseConfig method

The following example shows the usage of the function for the CAN controller:

// Set the baudrate to 500 kBit/s, TSEG1 = 5, TSEG2 = 2 (-> SamplePoint = 75%)
// Jump width is 2 time quanta,
// controller is in normal mode

// 1 ISO CAN FD call:
CAN1.CANSetFDDataPhaseConfig (500000,5,2,2,0);
// 2. Non-ISO CAN FD call:
CAN1.CANSetFDDataPhaseConfig (500000,5,2,2,1);

Example — SetBTR method

The following example determines the BTR register of the CAN controller:

CAN1.SetBTR( &H01, &H23)
