Description

The CANController object represents a CAN controller for a specific channel.

The "Philips 82C200-compatible" view is assumed for the NEC 72005.

CANSetConfig

This method can be used to set the configuration of a CAN channel.

The methods switches the mode of the channel to CAN, if it was in CAN FD mode.

Syntax:
object.CANSetConfig (double baudrate,
// in bit/s
unsigned char tseg1,
// length of the time segment 1 in time quanta
unsigned char tseg2,
// length of the time segment 2 in time quanta
unsigned char sjw,
// sync jump width in time quanta
unsigned char sam,
// number of sampling points (1 or 3)
unsigned int flags);
// see description below

(see example below)

8.2 SP3

baudrate, tseg1, tseg2, sjw, sam, flags

object

Type: CANController

baudrate

The baudrate in bits/s; e.g. 500 kBaud/s = 500000 bits/s.

tseg1

Length of time segment 1 in time quanta.

tseg2

Length of time segment 2 in time quanta.

sjw

Sync jump width in time quanta; cannot be larger than tseg2.

sam

Number of sampling points in a bit around the sample point; possible values:

1

Bit is sampled only one time at the sample point.

3

Bit is sampled two times before the sample point and one time at the sample point.

flags

Valid values (all other values are reserved):

0

Normal mode

1

Silent mode (acknowledge not created)
