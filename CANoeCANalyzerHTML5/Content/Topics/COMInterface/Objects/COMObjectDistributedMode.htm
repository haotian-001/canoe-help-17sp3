DistributedMode

Application » Configuration » DistributedMode

Description

The DistributedMode object offers the possibility to access application settings related to distributed mode (e.g. usage of an RT server).

The functionality provided here corresponds to the Distributed Mode page of the CANoe plugin in the Vector Tools Environment dialog.

AutoDisconnectOnMeasurementStop

Set AutoDisconnectOnMeasurementStop to automatically disconnect from the RT server at the end of the measurement and automatically reestablish the connection before the next measurement start.

This option is mainly used to share the RT server with other users as long as you do not need it for a measurement.

Example

Example — AutoConnectOnProgramStart property

Set and get auto connect on program start feature:

'Activate auto connect on program start feature.
App.Configuration.DistributedMode.AutoConnectOnProgramStart = True

'Get activation status of auto connect on program start feature.
AutoConnectOnProgramStart = App.Configuration.DistributedMode.AutoConnectOnProgramStart
App.UI.Write.Output "Is auto connect on program start feature activated? " & AutoConnectOnProgramStart

Example — AutoDisconnectOnMeasurementStop property

Set and get auto disconnect on measurement stop feature:

'Activate auto disconnect on measurement stop feature.
App.Configuration.DistributedMode.AutoDisconnectOnMeasurementStop = True

'Get activation status of auto disconnect on measurement stop feature.
AutoDisconnectOnMeasurementStop = App.Configuration.DistributedMode.AutoDisconnectOnMeasurementStop
App.UI.Write.Output "Is auto disconnect on measurement stop feature activated? " & AutoDisconnectOnMeasurementStop

Example — Connected property

Check if we are connected to the configured RT server:

Connected = App.Configuration.DistributedMode.Connected
App.UI.Write.Output "Is connection etablished to configured RT server? " & Connected

Example — RemoteTimeAutoAdjust property

Set and get remote time auto adjustment:

'Activate remote time auto adjustment feature.
App.Configuration.DistributedMode.RemoteTimeAutoAdjust = True

'Get activation status of remote time auto adjustment feature.
RemoteTimeAutoAdjust = App.Configuration.DistributedMode.RemoteTimeAutoAdjust
App.UI.Write.Output "Is remote time auto adjustment feature activated? " & RemoteTimeAutoAdjust

Example — RTServer property

Set and get configured RT server:

Set App = CreateObject( "CANoe.Application")

'Set a device as configured RT server
App.Configuration.DistributedMode.RTServer = RemoteAddress   'remote address string

'Get configured RT server address
RTServer = App.Configuration.DistributedMode.RTServer

If len(RTServer) = 0 Then
  App.UI.Write.Output "No RT server configured!"
Else
  App.UI.Write.Output "configured RT server: " & RTServer
End If

Set App = Nothing SourceCode

Example — Connect method

Establish connection to configured RT server:

App.Configuration.DistributedMode.Connect()

Example — Disconnect method

Terminate connection to configured RT server:

App.Configuration.DistributedMode.Disconnect()

Example — GetRTServerOccupancyInfo method

Query the occupation information object of a desired remote address:

Set RTServerOccupancyInfo = App.Configuration.DistributedMode.GetRTServerOccupancyInfo(RemoteAddress)
