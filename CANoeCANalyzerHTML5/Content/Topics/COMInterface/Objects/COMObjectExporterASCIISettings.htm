Description

The ExporterASCIISettings object represents the exporter and logging settings for the ASCII format.

Example

Example

The following example sets all ASCIISettings options:

Set Exporter      = app.Configuration.OnlineSetup.LoggingCollection(1).Exporter
Set ASCIISettings = Exporter.Settings.ASCII

ASCIISettings.LogTimeStampAccuracy = 4
ASCIISettings.NumberBase           = 0
ASCIISettings.NumericDecimalPlaces = 14
ASCIISettings.Representation       = 1
ASCIISettings.TimeFormat           = 0
