Variable

Application » System » Namespaces » Namespace » Variables » Variable

Description

The Variable object represents a system variable or one of its struct members.

Note

  • A variable must be unique in its namespace and therefore it can only be defined once.

  • Before a given variable is defined again, the first definition must be taken back.

  • Only the creator of a variable is permitted to delete it.

BeginStructUpdate

Starts the update of several elements of a system variable of type Struct or Generic Array. Use this function and the corresponding EndStructUpdate to change several elements at the same time without the variable having an intermediate value where only some elements are changed.

Each call must be followed by a call to EndStructUpdate, else the variable value will not change.

Syntax: object.BeginStructUpdate()

✔

—

—
