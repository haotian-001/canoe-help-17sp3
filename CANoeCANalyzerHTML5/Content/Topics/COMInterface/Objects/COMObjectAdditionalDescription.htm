AdditionalDescription

Application » Configuration » GeneralSetup » DiagnosticsSetup » DiagDescriptions » DiagDescription » AdditionalDescriptions » AdditionalDescription

Description

The AdditionalDescription object represents one additional diagnostic description configured in CANoe.

Use the DiagDescriptions property of the DiagnosticsSetup object to access the diagnostic "master" description.

Example

Example — Language property

This example toggles between two language settings:

Set description = Application.Configuration.GeneralSetup.DiagnosticsSetup.DiagDescriptions.DiagDescription.AdditionalDescriptions.AdditionalDescription( "Any_ECU_Example")
If description.Language = "de-DE" Then
  description.Language = "en-US"
Else
  description.Language = "de-DE"
End If

Example — Variant property

This example toggles between two variant settings:

Set description = Application.Configuration.GeneralSetup.DiagnosticsSetup.DiagDescriptions.DiagDescription.AdditionalDescriptions.AdditionalDescription( "Any_ECU_Example")
If description.Variant = "Common" Then
  description.Variant = "2012_June"
Else
  description.Variant = "Common"
End If

Example — OpenWindows method

This example adds two additional descriptions to the first "master" description and defines a certain interpretation order for them.

All diagnostic windows for the first additional description and the fault memory window for the second additional description are opened.

set app               = CreateObject("CANoe.Application")
set masterDescription = app.Configuration.GeneralSetup.DiagnosticsSetup.DiagDescriptions.Item(1)
set addDesc1          = desc.AdditionalDescriptions.Add("CDD\UDS-ExampleEcu-4.0.2.cdd")
set addDesc2          = desc.AdditionalDescriptions.Add("ODX\UDS-ExampleEcu-8.0.1.pdx", "Door")

masterDescription.InterpretationOrder.Move 1, 2

addDesc1.OpenWindows
addDesc2.OpenWindows 4
