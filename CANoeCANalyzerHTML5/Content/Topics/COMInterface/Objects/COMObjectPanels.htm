Panels

Application » Configuration » GeneralSetup » PanelSetup » Panels

Description

The Panels object represents panels of a specific type of the CANoe application.

Remove

Removes a panel from a specific panel group.

A panel can either be removed by using its name or by using its index in the panels collection.

  • The index starts with 1.
  • Basically the index may change when a new panel is added.
  • It is recommended to use the name instead of the index.

Syntax 1: object..Remove(name)

Syntax 2: object.Remove(index)

(see example below)

Example — Add property

The following example adds a new panel to the panels that are to be permanently loaded:

Dim Panels, Panel
Set Panels = Application.Configuration.GeneralSetup.PanelSetup.Panels(1)
Set Panel  = Panels.Add("c:\canoe\demo_cn\cansystemdemo\panels\motor.xvp")

Example — Remove property

The following example removes a panel from the (list of) permanently loaded panels by its name:

Dim Panels
Set Panels = Application.Configuration.GeneralSetup.PanelSetup.Panels(0)
Panels.Remove("motor")

The following example removes a panel from the (list of) permanently loaded panels by its index number:

Dim Panels
Set Panels = Application.Configuration.GeneralSetup.PanelSetup.Panels(0)
Panels.Remove(3)
