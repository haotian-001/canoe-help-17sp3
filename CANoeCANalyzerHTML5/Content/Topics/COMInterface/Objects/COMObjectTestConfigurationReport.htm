TestConfigurationReport

Application » Configuration » TestConfigurations » TestConfiguration » TestConfigurationReport

Description

The TestConfigurationReport object represents the reporting settings of a test configuration.

LastWrittenFullPath

Returns the path of the last written test report file.

If the CANoe Test Report Viewer output format is used, and the test configuration is still running at the time this property is evaluated, the temporary path of the report currently being written is returned.

This path can be used to open the report even before it is completed.

Call the property again after the test run is finished to retrieve the final file path.

Example

   

Example — FullPath/LastWrittenFullPath property

This example changes the report path of an existing test configuration, starts the test and displays the path to the generated test report file:

Dim CanoeApp, Measurement, TestConfiguration, TestUnit, TestConfigurationReport
Dim testing, reported

Set CanoeApp                = CreateObject("CANoe.Application")
Set Measurement             = CanoeApp.Measurement
Set TestConfiguration       = CanoeApp.Configuration.TestConfigurations(1)
Set TestConfigurationReport = TestConfiguration.Report

WScript.ConnectObject TestConfiguration, "TestConfiguration_"
WScript.ConnectObject TestConfigurationReport, "TestConfigurationReport_"

TestConfigurationReport.UseJointReport = true
TestConfigurationReport.FullPath       = "D:\report.vtestreport"

testing  = false
reported = false

' start the measurement
CanoeApp.Measurement.Start

' wait for measurement to start
While not CanoeApp.Measurement.Running
  WScript.Sleep 500
Wend

TestConfiguration.Start

While not reported
  WScript.Sleep 100
Wend

WScript.DisconnectObject TestConfiguration
WScript.DisconnectObject TestConfigurationReport

WScript.Echo "HTML Report: " & TestConfigurationReport.LastWrittenFullPath

Sub TestConfiguration_OnStart()
  testing = True
End Sub

Sub TestConfiguration_OnStop(reason)
  testing = False
End Sub

Sub TestConfigurationReport_OnGenerated(success, sourceFullPath, generatedFullPath)
  reported = true
End Sub

Example — StyleSheet property

The following example shows how to set report stylesheet to use the multipage.xslt file:

Dim CanoeApp, TestConfiguration

Set CanoeApp          = CreateObject("CANoe.Application")
Set TestConfiguration = CanoeApp.Configuration.TestConfigurations(1)

TestConfiguration.Report.StyleSheetEnabled = true
TestConfiguration.Report.Stylesheet        = CanoeApp.Path & "testunit_report\multipage.xslt"

Example — StyleSheetEnabled property

The following example shows how to set report stylesheet to use the multipage.xslt file:

Dim CanoeApp, TestConfiguration

Set CanoeApp          = CreateObject("CANoe.Application")
Set TestConfiguration = CanoeApp.Configuration.TestConfigurations(1)

TestConfiguration.Report.StylesheetEnabled = true
TestConfiguration.Report.StyleSheet        = CanoeApp.Path & "testunit_report\multipage.xslt"

Example — TestRun property

This example shows how to use TestRun in combination with a field code.

The generated report file will be: "D:\CANoe_Reports\tester_0042.xml".

Dim CanoeApp, TestConfiguration

Set CanoeApp          = CreateObject("CANoe.Application")
Set TestConfiguration = CanoeApp.Configuration.TestConfigurations(1)

TestConfiguration.Report.TestRun  = 42
TestConfiguration.Report.FullPath = "D:\CANoe_Reports\tester_{TestRun|4}.xml"

Example — UseJointReport

This example deactivates the reporting for the test configuration and for all contained test units:

Dim CanoeApp, TestConfig, TestUnit

Set CanoeApp   = CreateObject("CANoe.Application")
Set TestConfig = CanoeApp.Configuration.TestConfigurations(1)

' deactivate the reporting
TestConfig.Report.UseJointReport = false
For Each TestUnit in TestConfig.TestUnits
  TestUnit.Report.Enabled = false
Next

Example — OnGenerated event

This example starts a CANoe test configuration, waits for the report generation and opens the HTML report with the default browser when the report generation is finished:

Option Explicit

Dim WshShell, CanoeApp, Configuration, Measurement
Dim TestConfiguration, TestConfigurationReport
Dim TestUnit, TestUnitReport
Dim measuring, testing, reported

WScript.Quit(Main)

Function Main
  Set WshShell                = WScript.CreateObject("WScript.Shell")
  Set CanoeApp                = CreateObject("CANoe.Application")
  Set Configuration           = CanoeApp.Configuration
  Set Measurement             = CanoeApp.Measurement
  Set TestConfiguration       = Configuration.TestConfigurations.Item(1)
  Set TestConfigurationReport = TestConfiguration.Report
  Set TestUnit                = TestConfiguration.TestUnits(1)
  Set TestUnitReport          = TestUnit.Report

  WScript.ConnectObject Measurement, "Measurement_"
  WScript.ConnectObject TestConfiguration, "TestConfiguration_"
  If TestConfigurationReport.UseJointReport Then
    WScript.ConnectObject TestConfigurationReport, "TestConfigurationReport_"
  Else
    If TestUnitReport.Enabled Then
      WScript.ConnectObject TestUnitReport, "TestUnitReport_"
    Else
      ' reporting is disabled
      WScript.Echo "HTML reporting is disabled."
      Main = -1
      Exit Function
    End If
  End If

  ' start measurement
  measuring = Measurement.Running
  If Not measuring Then
    Measurement.Start
  End If

  ' wait for measurement start
  While Not measuring
    Wscript.Sleep 100
  Wend

  testing = False
  ' start testing
  TestConfiguration.Start

  ' wait for testing start or measurement end
  While Not testing and measuring
    Wscript.Sleep 100
  Wend

  ' wait for reporting event or measurement end
  reported = False
  While Not reported and measuring
    Wscript.Sleep 100
  Wend
    Main = 0
End Function

Sub Measurement_OnStart()
  measuring = True
End Sub

Sub Measurement_OnStop()
  measuring = False
End Sub

Sub TestConfiguration_OnStart()
  testing = True
End Sub

Sub TestConfiguration_OnStop(reason)
  testing = False
End Sub

Sub TestUnitReport_OnGenerated(success, sourceFullPath, generatedFullPath)
  reported = True
  If success Then
    OpenReport(generatedFullPath)
  Else
    WScript.Echo "Test unit HTML report failed."
  End If
End Sub

Sub TestConfigurationReport_OnGenerated(success, sourceFullPath, generatedFullPath)
  reported = True
  If success Then
    OpenReport(generatedFullPath)
  Else
    WScript.Echo "Test configuration HTML report failed."
  End If
End Sub

Sub OpenReport(path)
  WshShell.Run """" & path & """"
End Sub
