Technical Reference: COM Interface

 

With the help of the COM Interface you can control CANoe from external programs.

Besides applications, scripts also come into consideration as external programs.

Certainly the most well-known script and programming languages available to you are: VBScript, JScript, Perl, VBA, Visual Basic, Delphi and C/C++.

  • When starting CANoe by use of a script (or similar) the disclaimer might not be displayed.

    To activate the display of the disclaimer in self-written scripts you can use the following sample code.

VBScript sample to demonstrate the usage of the disclaimers in scripts and VisualBasic programs

Copy 
' get the ToolInfo object
set tool  = CreateObject("CANutil.ToolInfo")

' show disclaimer (language, strict, beta, education)
dim language, strict, beta, education
language  = 1
strict    = 1
beta      = 0
education = 0

if tool.ShowDisclaimer(language, strict, beta, education) <> 1 Then
  wscript.quit
end if

' release the ToolInfo object
set tool  = nothing
