Report

 

Description

The Report command can be used to enter comments in text form as part of test cases or preparation and finalization steps.

  • In addition to the title attributes of the test/control commands, comments are used to provide information concerning the test sequence as part of the test report (test unit/test module). They are inserted in the report at the relevant location.
  • In addition to the text, object values can be inserted into the report. Use the Object Value property to add the current value of a signal, system variable or environment variable to the report.
  • Additionally, the comment can be supplemented with an image in extended syntax. Supported image types are: BMP, GIF, JPG, PNG. The existence of the file is not checked.
  • A file path or URL can be provided for the image resource. Relative file paths are based on the folder that will contain the test report.
  • Optionally, the size to which the image should be scaled in the report can be specified in the width and height properties.
    If width or height is specified, the HTML report will automatically show the image with a link which opens the image in full size in a new window. The width and height parameters are transferred to the HTML report as is, their values can be given as pixels or percentage. See an HTML reference for details.
