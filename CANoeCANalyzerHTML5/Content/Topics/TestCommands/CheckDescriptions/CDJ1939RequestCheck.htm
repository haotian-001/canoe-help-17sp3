J1939 Request Check

 

Description

This check observes the behavior of a node when it receives a J1939 Request message (PGN EA00h).

After a J1939 Request is detected, the check validates if the specified response message or Acknowledgmentmessage (PGN E800h) is received or if a timeout is detected.

The check can be used for a specific requested PGN, a specific node or all requests in the network.

Following validation are done:

  • The response must be sent before timeout

  • The DLC of the J1939 Request message must be 3

  • Same request is not allowed in request is pending (configurable)

  • Within a request inhibit time (default 50 ms) the same request is not allowed

  • Only three retries of the same request are allowed if no response is received

  • The Acknowledgmentmessage (PGN E800h) shall be sent to global address (configurable because for ISO11783 the Acknowledgmentmessage shall be sent to a specific address)

  • A negative Acknowledgmentmessage (PGN E800h, NACK) shall not be sent after a global request

  • If the check is used in vTESTstudio it can validate signal values of the response

The request check differs between global request, which are addressed to all nodes and specific request which are addressed to a single node.

Depending on this the response must be sent global or specific.

Note

In a test configuration you can activate Report only first background check violation to reduce the number of error messages in the report.

Error messages are then reported once for every combination of the values marked bold in the table above.

If, for example, there is the following sequence of error messages...

Timeout, no response received for requested PGN 0xEE00 (60928)

Timeout, no response received for requested PGN 0xEE00 (60928)

Timeout, no response received for requested PGN 0xF101 (61697)

Timeout, no response received for requested PGN 0xEE00 (60928)

Timeout, no response received for requested PGN 0xF101 (61697)

...only these messages are listed in the test report:

Timeout, no response received for requested PGN 0xEE00 (60928)

Timeout, no response received for requested PGN 0xF101 (61697)
