Messages Known

 

Description

This check observes the current bus and reports messages that are not defined in any of the databases that are associated to the bus.

This check additionally checks some consistence of the databases. Multiply defined messages (of the same id) are reported by a warning notification during creation of the check.

For FlexRay only valid data frames are recognized as communication, Null frames and Erroneous frames are ignored.

Since CANoe 8.1 SP2 for a FlexRay PDU database (FIBEX ≥ 3.0 or AUTOSAR System Description) the mapped frames of all PDUs will be observed to be known. The check will fail on receiving any other frame. Before that CANoe version the check will always succeed for a FlexRay PDU database, because only the PDUs will be observed to be valid. (Unknown PDUs does not exist.)
